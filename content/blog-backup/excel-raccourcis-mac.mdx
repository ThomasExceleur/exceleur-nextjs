# Excel Raccourcis Mac : Les Essentiels

Sur Mac, les raccourcis Excel utilisent principalement **Cmd** au lieu de **Ctrl**. Voici tous les raccourcis indispensables.

## Raccourcis de base

| Raccourci | Action |
|-----------|--------|
| **Cmd + C** | Copier |
| **Cmd + X** | Couper |
| **Cmd + V** | Coller |
| **Cmd + Z** | Annuler |
| **Cmd + Y** | Retablir |
| **Cmd + S** | Enregistrer |
| **Cmd + O** | Ouvrir |
| **Cmd + N** | Nouveau classeur |
| **Cmd + P** | Imprimer |
| **Cmd + F** | Rechercher |
| **Cmd + H** | Remplacer |

## Navigation

| Raccourci | Action |
|-----------|--------|
| **Cmd + Fleche** | Aller au bord de la zone de données |
| **Fn + Cmd + Gauche** | Aller à la cellule A1 |
| **Fn + Cmd + Droite** | Dernière cellule utilisée |
| **Ctrl + G** | Atteindre (equivalent F5) |
| **Fn + Fleche Bas** | Page suivante |
| **Fn + Fleche Haut** | Page précédente |
| **Ctrl + Tab** | Feuille suivante |
| **Ctrl + Maj + Tab** | Feuille précédente |

## Sélection

| Raccourci | Action |
|-----------|--------|
| **Cmd + A** | Sélectionner tout |
| **Cmd + Maj + Fleche** | Etendre la sélection |
| **Ctrl + Espace** | Sélectionner la colonne |
| **Maj + Espace** | Sélectionner la ligne |
| **Cmd + Maj + L** | Activer/desactiver les filtres |

## Astuces Mac spécifiques

### Utiliser Fn

De nombreuses touches de fonction (F1-F12) necessitent la touche **Fn** sûr Mac :

- **Fn + F2** = Éditer la cellule
- **Fn + F4** = Figer la référence
- **Fn + F7** = Orthographe
- **Fn + F9** = Calculer

### Préférences clavier

Pour utiliser F1-F12 sans Fn :

1. Préférences Système > Clavier
2. Cochez "Utiliser les touches F1, F2, etc. comme touches de fonction standard"

## Raccourcis personnalises

### Créer un raccourci

Excel Mac ne permet pas de personnaliser les raccourcis directement, mais vous pouvez :

1. Utiliser les macros avec des raccourcis assignes
2. Utiliser Automator où Keyboard Maestro

## Comparaison Windows/Mac

| Windows | Mac | Action |
|---------|-----|--------|
| Ctrl + C | Cmd + C | Copier |
| F2 | Ctrl + U | Éditer |
| F4 | Cmd + T | Figer |
| Alt + = | Cmd + Maj + T | Somme auto |
| Ctrl + Home | Fn + Cmd + Gauche | Aller en A1 |
