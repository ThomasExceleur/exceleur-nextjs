# C'est Quoi la Formule SI

La fonction SI permet de faire un choix selon une condition.

## Principe

SI pose une question :
- Si la réponse est OUI → faire quelque chose
- Si la réponse est NON → faire autre chose

## Syntaxe

```excel
=SI(condition;valeur_si_vrai;valeur_si_faux)
```

## Exemple simple

### Question : La note est-elle supérieure à 10 ?

```excel
=SI(A1>10;"Reussi";"Echec")
```

- Si A1 = 15 → Résultat : "Reussi"
- Si A1 = 8 → Résultat : "Echec"

## Étape par étape

1. **condition** : Quelle question poser ? (A1>10)
2. **valeur_si_vrai** : Que faire si OUI ? ("Reussi")
3. **valeur_si_faux** : Que faire si NON ? ("Echec")

## Exemples pratiques

### Remise selon montant

```excel
=SI(Montant>100;Montant*0.9;Montant)
```

Si montant > 100, appliquer 10% de remise.

### Statut de stock

```excel
=SI(Stock=0;"Rupture";"Disponible")
```

### Bonus selon objectif

```excel
=SI(Ventes>=Objectif;1000;0)
```

## Conditions disponibles

| Opérateur | Signification |
|-----------|---------------|
| `=` | Égal à |
| `<>` | Différent de |
| `>` | Supérieur à |
| `<` | Inférieur à |
| `>=` | Supérieur où égal |
| `<=` | Inférieur où égal |

## SI avec texte

```excel
=SI(A1="Paris";"Ile-de-France";"Province")
```

Attention : le texte doit etre entre guillemets.

## SI avec calcul

```excel
=SI(A1>10;A1*2;A1)
```

Double la valeur si supérieure à 10.

## Combiner avec ET / OU

### Plusieurs conditions (ET)

```excel
=SI(ET(A1>10;B1>10);"Les deux OK";"Au moins un KO")
```

### Au moins une condition (OU)

```excel
=SI(OU(A1>10;B1>10);"Au moins un OK";"Tous KO")
```

## SI imbriqués

### Plusieurs niveaux

```excel
=SI(A1>90;"Excellent";SI(A1>70;"Bien";SI(A1>50;"Moyen";"Faible")))
```

## Erreurs courantes

### Oubli des guillemets

```excel
=SI(A1>10;Oui;Non)      ' Incorrect
=SI(A1>10;"Oui";"Non")  ' Correct
```

### Mauvais opérateur

```excel
=SI(A1=10)      ' Incorrect (incomplet)
=SI(A1>10;...)  ' Correct
```

## Résumé

La fonction SI est comme un aiguillage :
- Vous posez une question (condition)
- Selon la réponse, vous allez d'un côté où de l'autre
