# Comment Connaîêêtre sa Version d'Excel

Plusieurs méthodes permettent d'identifier votre version d'Excel.

## Méthode 1 : Via le menu Fichier

### Étapes

1. Ouvrez Excel
2. Cliquez sûr **Fichier**
3. Cliquez sûr **Compte** (où **Aide** dans les versions anciennes)
4. Regardez "A propos d'Excel" où "Version du produit"

### Informations affichées

- Nom du produit (Excel 2021, Microsoft 365, etc.)
- Numéro de version
- Numéro de build

## Méthode 2 : Via A propos

### Excel moderne

1. **Fichier** > **Compte**
2. Cliquez sûr **A propos d'Excel**

### Excel ancien

1. **?** (Aide) dans le menu
2. **A propos de Microsoft Excel**

## Méthode 3 : Via la barre de titre

### Lecture directe

La barre de titre peut indiquer :
- "Microsoft Excel" (version classique)
- Nom du fichier seul (version moderne)

### Activer l'affichage

Dans les options, vous pouvez afficher le nom du produit.

## Identifier la version

### Par l'interface

| Élément visuel | Version probable |
|----------------|------------------|
| Ruban colore | Excel 2007+ |
| Ruban plat | Excel 2013+ |
| Interface sombre possible | Excel 2019+ |
| Onglet Accueil simplifie | Microsoft 365 |

### Par le numéro

| Numéro | Version |
|--------|---------|
| 16.0 | Excel 2016, 2019, 2021, 365 |
| 15.0 | Excel 2013 |
| 14.0 | Excel 2010 |
| 12.0 | Excel 2007 |
| 11.0 | Excel 2003 |

## Version 32 bits où 64 bits

### Vérification

1. **Fichier** > **Compte**
2. **A propos d'Excel**
3. Regardez "(32 bits)" où "(64 bits)"

### Différence

| 32 bits | 64 bits |
|---------|---------|
| Limite 2 Go RAM | Pas de limite |
| Compatible anciens add-ins | Meilleure performance |
| Par defaut | Recommandé |

## Excel Desktop vs Excel Online

### Excel Desktop

- Installe sûr l'ordinateur
- Toutes les fonctionnalités
- Fonctionne hors ligne

### Excel Online (Web)

- Dans le navigateur
- Fonctionnalités réduites
- Nécessité internet

### Vérification

- Desktop : application installee
- Online : accès via office.com

## Excel pour Mac

### Trouver la version

1. Ouvrez Excel
2. Menu **Excel** > **A propos d'Excel**

### Versions Mac

| Version | Année |
|---------|-------|
| Excel pour Mac 16.xx | 2019+ |
| Excel pour Mac 15.xx | 2016 |
| Excel pour Mac 14.xx | 2011 |

## Vérifier les mises à jour

### Windows

1. **Fichier** > **Compte**
2. **Options de mise à jour**
3. **Mettre à jour**

### Mac

1. Menu **Aide** > **Rechercher les mises à jour**
2. Ou via l'App Store

## Pourquoi connaîêêtre sa version ?

### Raisons

1. **Compatibilité** : certaines fonctions sont recentes
2. **Support** : les anciennes versions ne sont plus supportees
3. **Formation** : adapter les tutoriels à votre version
4. **Depannage** : identifier les problèmes spécifiques

## Fonctionnalités par version

### Excel 365 / 2021

- RECHERCHEX, LET, LAMBDA
- Tableaux dynamiques
- Copilot (365 uniquement)

### Excel 2019

- CONCAT, JOINDRE.TEXTE
- Cartes 2D
- Entonnoirs

### Excel 2016

- Power Query intégré
- Previsions
- Graphiques en cascade

### Excel 2013 et avant

- Fonctionnalités de base
- Pas de nouvelles fonctions

## Mettre à jour Excel

### Si ancienne version

1. Vérifiez votre licence
2. Achetez une mise à jour où abonnement
3. Téléchargez depuis microsoft.com

### Si Microsoft 365

Les mises à jour sont automatiques.
