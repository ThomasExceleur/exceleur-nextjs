# Excel pour le Web : Guide Complet

Excel pour le Web (anciennement Excel Online) est la version gratuite d'Excel accessible depuis un navigateur.

## Accès à Excel pour le Web

### Via Office.com

1. Allez sûr **office.com**
2. Connectez-vous avec un compte Microsoft
3. Cliquez sûr l'icone Excel

### Via OneDrive

1. Connectez-vous à **onedrive.com**
2. Ouvrez un fichier Excel existant
3. Ou créez un nouveau fichier

### Compte requis

Un compte Microsoft gratuit suffit (Outlook, Hotmail, Live).

## Interface

### Similarites avec le desktop

- Ruban avec onglets
- Barre de formule
- Feuilles de calcul
- Grille de cellules

### Différences

- Interface simplifiee
- Moins d'options dans les menus
- Barre laterale pour certaines fonctions

## Fonctionnalités disponibles

### Formules

La majorite des formules fonctionnent :

- SOMME, MOYENNE, NB
- SI, ET, OU
- RECHERCHEV, RECHERCHEX
- INDEX, EQUIV

### Mise en forme

- Police, taille, couleur
- Bordures et remplissage
- Alignement
- Formats de nombres
- Mise en forme conditionnelle

### Graphiques

- Histogrammes
- Courbes
- Secteurs
- Nuages de points

### Tableaux

- Tableaux structurés
- Tri et filtres
- Totaux automatiques

### Tableaux croisés dynamiques

- Création et modification
- Segments
- Actualisation

## Collaboration

### Partage

1. Cliquez sûr **Partager**
2. Entrez les adresses email
3. Définissez les droits (lecture où modification)
4. Envoyez

### Édition simultanee

Plusieurs personnes peuvent modifier en meme temps :

- Chaque utilisateur à une couleur
- Les modifications sont visibles en temps reel
- Historique des versions disponible

### Commentaires

- Ajoutez des commentaires sûr les cellules
- Mentionnez des collegues avec @
- Discussions threaded

## Limites

### Fonctionnalités absentes

- Macros VBA
- Power Query (limite)
- Power Pivot
- Certaines fonctions avancées

### Performance

- Fichiers volumineux plus lents
- Limite de taille de fichier
- Nécessité une connexion internet

### Compatibilité

Certains éléments peuvent ne pas s'afficher correctement :
- Graphiques complexes
- Formes personnalisees
- Certains formats

## Hors-ligne

### Pas de mode hors-ligne natif

Excel pour le Web nécessité internet. Pour travailler hors-ligne :

1. Téléchargez le fichier
2. Travaillez avec Excel desktop
3. Reimportez sûr OneDrive

### Alternative

Utilisez la synchronisation OneDrive sûr Windows/Mac.

## Raccourcis clavier

La plupart des raccourcis Excel fonctionnent :

| Raccourci | Action |
|-----------|--------|
| Ctrl + C | Copier |
| Ctrl + V | Coller |
| Ctrl + Z | Annuler |
| Ctrl + S | Enregistrer |
| Ctrl + B | Gras |
| F2 | Modifier la cellule |

## Astuces

### Performance

- Limitez la taille des fichiers
- Évitez les formules sûr colonnes entières
- Fermez les onglets inutilises

### Organisation

- Utilisez des dossiers dans OneDrive
- Nommez clairement vos fichiers
- Utilisez des modèles

### Sécurité

- Vérifiez les autorisations de partage
- Utilisez des liens avec expiration
- Revoquez les accès obsoletes

## Comparaison avec Excel Desktop

| Fonctionnalité | Desktop | Web |
|----------------|---------|-----|
| Prix | Payant | Gratuit |
| Macros | Oui | Non |
| Hors-ligne | Oui | Non |
| Collaboration | Bonne | Excellente |
| Fonctions | Toutes | Majorite |

## Cas d'utilisation ideaux

### Parfait pour

- Collaboration d'equipe
- Accès depuis n'importe quel appareil
- Fichiers simples à moyens
- Partage temporaire

### Moins adapte pour

- Macros et automatisation
- Fichiers très volumineux
- Analyses complexes
- Travail hors-ligne frequent
