# Modifier Plusieurs Cellules d'un Coup

Gagnez du temps avec l'edition multiple.

## Sélection multiple

### Méthodes

| Méthode | Action |
|---------|--------|
| Ctrl+Clic | Cellules non adjacentes |
| Maj+Clic | Plage continue |
| Ctrl+Maj+Fin | Jusqu'à la fin |

## Saisie simultanee

### Étapes

1. Sélectionnez les cellules
2. Tapez la valeur
3. **Ctrl+Entrée** (pas juste Entrée)

### Résultat

Toutes les cellules sélectionnées recoivent la valeur.

## Mise en forme multiple

### Étapes

1. Sélectionnez les cellules
2. Appliquez la mise en forme
3. Toutes sont modifiées

## Formule sûr plusieurs cellules

### Meme formule

1. Sélectionnez la plage
2. Tapez la formule
3. **Ctrl+Entrée**

## Copier vers plusieurs destinations

### Étapes

1. Copiez la source (**Ctrl+C**)
2. Sélectionnez les destinations (Ctrl+Clic)
3. Collez (**Ctrl+V**)

## Rechercher et remplacer

### Multi-cellules automatique

**Ctrl+H** remplace dans toute la sélection où la feuille.

## Cas pratiques

| Situation | Solution |
|-----------|----------|
| Meme texte | Ctrl+Entrée |
| Meme format | Sélection + format |
| Remplir des vides | Atteindre + Ctrl+Entrée |

## Remplir les vides

### Méthode

1. Sélectionnez la plage
2. **Ctrl+G** > Cellules vides
3. Tapez une formule (=cellule_au_dessus)
4. **Ctrl+Entrée**

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Ctrl+Entrée | Ne pas oublier Ctrl |
| Sélection precise | Vérifiez avant de valider |
| Annuler | Ctrl+Z si erreur |
