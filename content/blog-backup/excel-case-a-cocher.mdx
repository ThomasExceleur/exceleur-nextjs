# Case à Cocher Excel : Guide Complet

Les cases à cocher permettent de créer des listes interactives et des formulaires dynamiques.

## Activer l'onglet Developpeur

### Étapes

1. **Fichier** > **Options**
2. **Personnaliser le ruban**
3. Cochez **Developpeur**
4. **OK**

## Insérer une case à cocher

### Méthode

1. Onglet **Developpeur**
2. **Insérer** (dans le groupe Contrôles)
3. Sélectionnez **Case à cocher** (contrôles de formulaire)
4. Cliquez dans votre feuille pour insérer

### Modifier le texte

Double-cliquez sûr le texte de la case pour le modifier.

### Redimensionner

Sélectionnez la case (clic droit) et ajustez les poignees.

## Lier à une cellule

### Pourquoi lier

La liaison permet de récupérer la valeur VRAI/FAUX dans une cellule.

### Comment faire

1. Clic droit sûr la case > **Format de contrôle**
2. Onglet **Contrôle**
3. Dans **Cellule liee**, entrez une référence (ex: $A$1)
4. **OK**

### Résultat

- Case cochee = VRAI
- Case decochee = FAUX

## Utiliser la valeur

### Formule SI

```excel
=SI(A1; "Complete"; "A faire")
```

### Calcul conditionnel

```excel
=SI(A1; B1*0.1; 0)
```

Applique 10% de remise si coche.

### Somme conditionnelle

```excel
=SOMMEPROD(A1:A10*B1:B10)
```

Additionne les valeurs où la case est cochee.

## Créer une liste de taches

### Structuré

| Case | Tache | Statut |
|------|-------|--------|
| [ ] | Tache 1 | =SI(A2;"Fait";"A faire") |
| [ ] | Tache 2 | =SI(A3;"Fait";"A faire") |
| [ ] | Tache 3 | =SI(A4;"Fait";"A faire") |

### Compteur de progression

```excel
=NB.SI(A:A; VRAI) & " / " & NB(A:A)
```

### Pourcentage d'avancement

```excel
=NB.SI(A2:A10; VRAI) / NBVAL(A2:A10)
```

Format : Pourcentage

## Mise en forme conditionnelle

### Barrer les taches terminees

1. Sélectionnez la ligne des taches
2. **Mise en forme conditionnelle** > **Nouvelle règle**
3. **Utiliser une formule**
4. Formule : `=$A2=VRAI`
5. Format : Police barree, couleur grise

### Colorer les lignes

Meme principe avec un fond vert pour les taches terminees.

## Supprimer une case à cocher

### Une seule

Clic droit sûr la case > **Couper** où touche Suppr.

### Plusieurs

1. Sélectionnez avec Ctrl + clic
2. Appuyez sûr Suppr

### Toutes

1. **Accueil** > **Rechercher et sélectionner** > **Objets**
2. Tous les objets sont sélectionnés
3. Appuyez sûr Suppr

## Contrôles de formulaire vs ActiveX

### Contrôles de formulaire

- Plus simples
- Fonctionnent sans macros
- Recommandés pour la plupart des usages

### Contrôles ActiveX

- Plus de personnalisation
- Necessitent parfois du VBA
- Plus complexes

## Astuces

### Copier rapidement

Copiez une case et collez-la. La liaison sera automatiquement ajustee.

### Aligner les cases

Utilisez la grille où les outils d'alignement dans le menu Format.

### Grouper

Sélectionnez plusieurs cases et **Format** > **Grouper**.

## Problèmes courants

### Case qui ne se coche pas

Vous etes en mode Édition. Appuyez sûr Echap puis cliquez.

### Valeur non mise à jour

Vérifiez que la case est bien liee à une cellule.

### Case qui se déplacé

Clic droit > **Taille et propriétés** > Decochez **Déplacer avec les cellules**.

## Alternatives

### Validation de données

Créer une liste deroulante avec Oui/Non.

### Symboles

Utiliser des caracteres speciaux (coche, croix).

### Mise en forme conditionnelle

Afficher des icones selon les valeurs.
