# Mettre un 0 Quand la Cellule est Vide

Plusieurs méthodes permettent de remplacer les cellules vides par 0 dans Excel.

## Méthode 1 : Formule SI

### Syntaxe

```excel
=SI(A1=""; 0; A1)
```

### Explication

- Si A1 est vide, affiché 0
- Sinon, affiché la valeur de A1

### Variante avec ESTVIDE

```excel
=SI(ESTVIDE(A1); 0; A1)
```

## Méthode 2 : Rechercher-Remplacer

### Étapes

1. Sélectionnez la plage
2. **Ctrl + H** (Rechercher et remplacer)
3. Laissez "Rechercher" vide
4. Dans "Remplacer par" : 0
5. **Options** > Cochez **Totalite du contenu de la cellule**
6. **Remplacer tout**

### Attention

Cette méthode modifié definitivement les cellules.

## Méthode 3 : Collage special

### Étapes

1. Tapez 0 dans une cellule
2. Copiez cette cellule
3. Sélectionnez les cellules vides
4. **Accueil** > **Rechercher** > **Atteindre** > **Cellules** > **Cellules vides**
5. **Collage special** > **Valeurs**
6. **OK**

## Méthode 4 : Format personnalise

### Cette méthode n'affiché que le 0

Les cellules restent vides mais affichent 0.

1. Sélectionnez les cellules
2. **Ctrl + 1** (Format de cellule)
3. **Personnalisee**
4. Tapez : `0;-0;0;@`
5. **OK**

### Avantage

Les cellules restent vides pour les calculs.

## Méthode 5 : Formule matricielle

### Remplacer toute une colonne

```excel
=SI(A1:A100=""; 0; A1:A100)
```

Validez avec **Ctrl + Maj + Entrée** (versions anciennes).

### Excel 365

Les formules sont automatiquement dynamiques.

## Méthode 6 : Power Query

### Étapes

1. **Données** > **Obtenir des données** > **A partir d'une table**
2. **Transformer** > **Remplacer les valeurs**
3. "Valeur à rechercher" : laisser vide
4. "Remplacer par" : 0

### Avantage

Transformation automatique à chaque actualisation.

## Formule avec calcul

### Si le résultat d'un calcul est vide

```excel
=SI(B1-C1=""; 0; B1-C1)
```

### Version simplifiee

```excel
=B1-C1+0
```

L'ajout de 0 force un résultat numérique.

### Avec SIERREUR

```excel
=SIERREUR(B1/C1; 0)
```

Retourne 0 en cas d'erreur.

## Afficher 0 au lieu de vide dans les formules

### Formule retournant vide

```excel
=SI(A1>10; A1; "")
```

### Formule retournant 0

```excel
=SI(A1>10; A1; 0)
```

## Ne pas afficher les zeros

### Si vous voulez l'inverse

1. **Fichier** > **Options**
2. **Options avancées**
3. Decochez **Afficher un zero dans les cellules ayant une valeur nulle**

### Par mise en forme

Format personnalise : `#;-#;;@`

Les zeros ne s'affichent pas.

## Cas pratiques

### Tableau de données

Avant :
| A | B | C |
|---|---|---|
| 1 |   | 3 |
|   | 2 |   |

Après :
| A | B | C |
|---|---|---|
| 1 | 0 | 3 |
| 0 | 2 | 0 |

### Somme avec cellules vides

Les cellules vides sont ignorees par SOMME. Si vous voulez les compter comme 0, c'est déjà le cas.

## Choisir la bonne méthode

| Méthode | Modification | Dynamique | Complexité |
|---------|--------------|-----------|------------|
| Formule SI | Non | Oui | Simple |
| Rechercher-Remplacer | Oui | Non | Très simple |
| Collage special | Oui | Non | Simple |
| Format personnalise | Non | Visuel seulement | Moyen |
| Power Query | Non | Oui | Moyen |

## Recommandation

- **Pour un affichage** : Format personnalise
- **Pour les calculs** : Formule SI
- **Pour une modification définitive** : Rechercher-Remplacer
