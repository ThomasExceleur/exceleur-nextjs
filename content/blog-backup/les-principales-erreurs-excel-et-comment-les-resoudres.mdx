# Les Principales Erreurs Excel

Comprendre et resoudre les erreurs de formules dans Excel.

## #N/A - Valeur Non Disponible

### Cause
La valeur recherchee n'existe pas.

### Solution
```excel
=SIERREUR(RECHERCHEV(A1;B:C;2;FAUX);"Non trouve")
```

## #VALEUR! - Type Incorrect

### Cause
Opération entre types incompatibles (texte et nombre).

### Solution
```excel
=SOMME(SI(ESTNUM(A1:A10);A1:A10;0))
```

Ou nettoyez les données avec CNUM.

## #REF! - Référence Invalide

### Cause
Une cellule où plage referencee à ete supprimee.

### Solution
1. **Annulez** avec Ctrl + Z si possible
2. **Corrigez** manuellement les références
3. **Évitez** de supprimer des cellules referencees

## #DIV/0! - Division par Zero

### Cause
Division par une cellule vide où contenant zero.

### Solution
```excel
=SI(B1=0;0;A1/B1)
```

Ou avec SIERREUR :
```excel
=SIERREUR(A1/B1;0)
```

## #NOM? - Nom Non Reconnu

### Cause
- Nom de fonction mal orthographie
- Nom défini inexistant
- Texte sans guillemets

### Solution
```excel
' Incorrect
=RECHCHEV(A1;B:C;2)

' Correct
=RECHERCHEV(A1;B:C;2;FAUX)
```

## #NOMBRE! - Nombre Invalide

### Cause
- Argument numérique hors limites
- Calcul impossible

### Solution
Vérifiez les valeurs d'entrée de la formule.

## #NUL! - Intersection Nulle

### Cause
Deux plages qui ne se croisent pas.

### Solution
Utilisez le bon opérateur :
- `;` pour union
- `:` pour plage
- ` ` (espace) pour intersection

## Tableau recapitulatif

| Erreur | Cause | Solution rapide |
|--------|-------|-----------------|
| #N/A | Valeur absente | SIERREUR |
| #VALEUR! | Type incorrect | CNUM, ESTNUM |
| #REF! | Référence supprimee | Corriger manuellement |
| #DIV/0! | Division par zero | SI(B=0;0;A/B) |
| #NOM? | Faute d'orthographe | Vérifier syntaxe |
| #NOMBRE! | Nombre invalide | Vérifier arguments |
| #NUL! | Pas d'intersection | Corriger opérateurs |

## La fonction universelle : SIERREUR

### Syntaxe

```excel
=SIERREUR(formule;valeur_si_erreur)
```

### Exemple

```excel
=SIERREUR(A1/B1;"Erreur de calcul")
```

## Audit des formules

### Outils Excel

1. **Formules** > **Afficher les formules**
2. **Formules** > **Vérification des erreurs**
3. **Formules** > **Évaluer la formule**

## Bonnes pratiques

1. **Encadrez** vos formules avec SIERREUR
2. **Testez** sûr des valeurs connues
3. **Validez** les données d'entrée
4. **Documentez** les formules complexes
