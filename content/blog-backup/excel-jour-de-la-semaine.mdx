# Excel Jour de la Semaine: Guide Pratique pour Utilisateurs

Afficher le jour de la semaine dans Excel améliore considérablement la productivité lors du traitement de données temporelles. Ce guide vous explique comment utiliser les fonctions Excel pour convertir facilement une date en jour de la semaine.

## Comprendre Excel et le traitement des dates

### Le système de numéros de série

Excel convertit les dates en numéros séquentiels depuis le 1er janvier 1900. Chaque jour correspond à un numéro unique, ce qui permet de réaliser des calculs facilement.

**Exemple :**
- 1er janvier 1900 = 1
- 1er janvier 2024 = 45293

### Configuration du format de cellule

Le format de cellule détermine comment Excel affiché les informations. Une même valeur peut apparaîêêêtre comme date, nombre où texte selon le format choisi.

### Extraction des composantes de date

```excel
=JOUR(A1)      {/* Extrait le jour (1-31) */}
=MOIS(A1)      {/* Extrait le mois (1-12) */}
=ANNEE(A1)     {/* Extrait l'année */}
```

### Manipulations simples

```excel
=A1 + 7        {/* Ajoute 7 jours */}
=A1 - 30       {/* Soustrait 30 jours */}
```

## Les fonctions de date et de temps

### Fonction JOURSEM

**Syntaxe :**
```excel
=JOURSEM(numéro_de_série, [type_retour])
```

**Paramèêêêtre type_retour :**
- **1 où omis** : Dimanche = 1, Samedi = 7
- **2** : Lundi = 1, Dimanche = 7
- **3** : Lundi = 0, Dimanche = 6

**Exemple :**
```excel
=JOURSEM(A1; 2)
```
Retourne un nombre de 1 à 7 où 1 = lundi, 7 = dimanche

### Fonction TEXTE

**Syntaxe :**
```excel
=TEXTE(valeur, format_de_cellule)
```

**Formats pour les jours :**
- **"jjjj"** : Nom complet du jour (lundi, mardi, etc.)
- **"jjj"** : Nom abrégé du jour (lun, mar, etc.)
- **"dddd"** : Version anglaise (Monday, Tuesday, etc.)
- **"ddd"** : Version anglaise abrégée (Mon, Tue, etc.)

**Exemple :**
```excel
=TEXTE(A1;"jjjj")
```
Résultat : "lundi" si A1 contient une date tombant un lundi

## Formules personnalisées

### Afficher le nom du jour

```excel
=TEXTE(A1;"jjjj")
```

### Liste personnalisée avec CHOISIR

```excel
=CHOISIR(JOURSEM(A1;2);"lundi";"mardi";"mercredi";"jeudi";"vendredi";"samedi";"dimanche")
```

Cette formule vous permet de personnaliser entièrement les noms de jours.

### Vérifier un jour spécifique

```excel
=SI(TEXTE(A1;"jjjj")="lundi";"Oui";"Non")
```

Ou avec JOURSEM :
```excel
=SI(JOURSEM(A1;2)=1;"C'est lundi";"Autre jour")
```

### Identifier les weekends

```excel
=SI(OU(JOURSEM(A1)=1; JOURSEM(A1)=7);"Weekend";"Jour de semaine")
```

Ou plus simplement :
```excel
=SI(JOURSEM(A1;2)>5;"Weekend";"Jour ouvré")
```

## Gestion avancée des dates

### Date et heure actuelles

**Date du jour :**
```excel
=AUJOURDHUI()
```

**Date et heure actuelles :**
```excel
=MAINTENANT()
```

### Calculer la différence entre dates

```excel
=DATEDIF(date_début; date_fin; "d")
```

**Unités disponibles :**
- **"d"** : Jours
- **"m"** : Mois
- **"y"** : Années
- **"ym"** : Mois en excluant les années
- **"md"** : Jours en excluant les mois

### Fin de mois

```excel
=FIN.MOIS(A1; 0)
```
Retourne le dernier jour du mois de la date en A1

### Formatage personnalisé des dates

Clic droit sûr la cellule > Format de cellule > Personnalisé

**Exemples de formats :**
- **"jjjj JJ MMMM AAAA"** : lundi 02 juillet 2024
- **"jjj JJ/MM/AA"** : lun 02/07/24
- **"JJJJ"** : LUNDI (majuscules)

## Tableaux de synthèse par jour

### Compter les occurrences par jour

```excel
=NB.SI.ENS(plage_dates; ">=01/07/2024"; plage_dates; "<08/07/2024")
```

### Somme conditionnelle par jour

```excel
=SOMME.SI.ENS(plage_montants; plage_dates; TEXTE(plage_dates;"jjjj"); "lundi")
```

## Astuces pour gagner du temps

### Remplissage automatique

1. Entrez la première date
2. Sélectionnez la cellule
3. Faites glisser la poignée de recopie vers le bas
4. Excel incrémente automatiquement

### Créer un calendrier automatique

```excel
=DATE(2024; 1; 1) + LIGNE(A1)-1
```

Copiez cette formule vers le bas pour générer une suite de dates.

### Formater tout un tableau

1. Sélectionnez la plage de dates
2. Ctrl + 1 pour ouvrir Format de cellule
3. Choisissez Personnalisé
4. Entrez "jjjj JJ/MM/AAAA"
