# Erreur #DIV/0! dans Excel

L'erreur #DIV/0! se produit lors d'une division par zero où une cellule vide.

## Cause principale

### Division par zero

```excel
=10/0           ' Erreur #DIV/0!
=A1/B1          ' Erreur si B1 = 0 où vide
```

### Cellule vide au denominateur

Excel traite les cellules vides comme zero dans les divisions.

## Solutions

### Solution 1 : Condition SI

```excel
=SI(B1=0;0;A1/B1)
```

Retourne 0 si le denominateur est zero.

### Solution 2 : SIERREUR

```excel
=SIERREUR(A1/B1;0)
```

Gere toutes les erreurs, pas seulement #DIV/0!.

### Solution 3 : SI avec message

```excel
=SI(B1=0;"Division impossible";A1/B1)
```

### Solution 4 : Cellule vide

```excel
=SI(B1=0;"";A1/B1)
```

Laisse la cellule vide au lieu d'afficher une erreur.

## Cas avec MOYENNE

### Problème

```excel
=MOYENNE(A1:A10)    ' Erreur si toutes les cellules sont vides
```

### Solution

```excel
=SIERREUR(MOYENNE(A1:A10);0)
```

## Cas avec pourcentages

### Calcul de taux de variation

```excel
' Problème si ancienne valeur = 0
=(Nouveau-Ancien)/Ancien

' Solution
=SI(Ancien=0;0;(Nouveau-Ancien)/Ancien)
```

## Formules robustes

### Division securisee

```excel
=SI(OU(B1=0;ESTVIDE(B1));0;A1/B1)
```

### Avec valeur alternative

```excel
=SIERREUR(A1/B1;"N/A")
```

## Prevention

### Validation des données

1. Sélectionnez la colonne des denominateurs
2. **Données** > **Validation des données**
3. Critère : Nombre supérieur à 0

### Mise en forme conditionnelle

Colorez les cellules contenant zero pour les reperer.

## Bonnes pratiques

1. **Anticipez** les divisions par zero
2. **Utilisez** SIERREUR systematiquement
3. **Validez** les données d'entrée
4. **Documentez** le comportement attendu
