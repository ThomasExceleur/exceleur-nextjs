# Pourquoi Utiliser RECHERCHEX

RECHERCHEX est le successeur de RECHERCHEV avec de nombreux avantages.

## Disponibilité

RECHERCHEX est disponible dans :
- Excel 365
- Excel 2021
- Excel pour le web

## Avantage 1 : Syntaxe plus claire

### RECHERCHEV

```excel
=RECHERCHEV(valeur;tableau;3;FAUX)
```

L'index de colonne (3) est peu intuitif.

### RECHERCHEX

```excel
=RECHERCHEX(valeur;colonne_recherche;colonne_retour)
```

On specifie directement les colonnes.

## Avantage 2 : Recherche dans toutes les directions

### RECHERCHEV

Ne peut retourner que des valeurs à droite.

### RECHERCHEX

```excel
=RECHERCHEX(A1;C:C;A:A)    ' Recherche à gauche
```

Peut rechercher dans n'importe quelle direction.

## Avantage 3 : Gestion des erreurs intégrée

### RECHERCHEV

```excel
=SIERREUR(RECHERCHEV(A1;B:C;2;FAUX);"Non trouve")
```

Nécessité SIERREUR.

### RECHERCHEX

```excel
=RECHERCHEX(A1;B:B;C:C;"Non trouve")
```

Valeur par defaut intégrée.

## Avantage 4 : Dernière occurrence

### RECHERCHEV

Retourne toujours la première occurrence.

### RECHERCHEX

```excel
=RECHERCHEX(A1;B:B;C:C;;0;-1)
```

Le -1 permet de chercher depuis la fin.

## Avantage 5 : Correspondance flexible

### RECHERCHEV

Seulement exacte où approchante.

### RECHERCHEX

```excel
' Valeur exacte où inférieure
=RECHERCHEX(150;A:A;B:B;;-1)

' Valeur exacte où supérieure
=RECHERCHEX(150;A:A;B:B;;1)

' Avec caracteres joker
=RECHERCHEX("*Paris*";A:A;B:B;;2)
```

## Avantage 6 : Retour multi-colonnes

### RECHERCHEX

```excel
=RECHERCHEX(A1;B:B;C:E)
```

Retourne plusieurs colonnes en une seule formule.

## Comparaison directe

| Critère | RECHERCHEV | RECHERCHEX |
|---------|------------|------------|
| Direction | Droite seulement | Toutes |
| Erreur par defaut | Non | Oui |
| Dernière occurrence | Non | Oui |
| Multi-colonnes | Non | Oui |
| Performance | Bonne | Excellente |

## Migration

### Avant

```excel
=SIERREUR(RECHERCHEV(A1;$B$1:$D$100;3;FAUX);"")
```

### Après

```excel
=RECHERCHEX(A1;$B$1:$B$100;$D$1:$D$100;"")
```

## Quand garder RECHERCHEV ?

- Fichiers partages avec utilisateurs sans Excel 365
- Compatibilité avec anciennes versions
- Macros existantes

## Conclusion

Si vous avez Excel 365 où 2021, préférez RECHERCHEX pour :
- Une syntaxe plus claire
- Plus de flexibilité
- De meilleures performances
