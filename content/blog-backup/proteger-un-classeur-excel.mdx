# Proteger un Classeur Excel : Guide Complet

Securisez vos classeurs Excel selon vos besoins spécifiques.

## Niveaux de protection

### Vue d'ensemble

| Niveau | Protection |
|--------|------------|
| Fichier | Mot de passe à l'ouverture |
| Classeur | Structuré (onglets) |
| Feuille | Contenu des cellules |
| Cellules | Verrouillage individuel |

## Protection à l'ouverture

### Chiffrer avec mot de passe

1. **Fichier** > **Informations**
2. **Proteger le classeur**
3. **Chiffrer avec mot de passe**
4. Entrez et confirmez le mot de passe

### Résultat

Impossible d'ouvrir sans le mot de passe.

### Attention

Mot de passe oublie = fichier inaccessible.

## Protection en modification

### Mot de passe pour modifier

1. **Fichier** > **Enregistrer sous**
2. **Outils** > **Options générales**
3. Renseignez "Mot de passe pour modifier"

### Résultat

Le fichier s'ouvre en lecture seule sans mot de passe.

## Protection de la structuré

### Activer

1. **Révision** > **Proteger le classeur**
2. Cochez **Structuré**
3. Mot de passe optionnel

### Effets

- Empeche d'ajouter/supprimer des feuilles
- Empeche de renommer des feuilles
- Empeche de masquer/afficher des feuilles

## Protection des feuilles

### Activer

1. **Révision** > **Proteger la feuille**
2. Configurez les options
3. Mot de passe optionnel

### Options courantes

| Option | Effet |
|--------|-------|
| Sélectionner les cellules | Navigation |
| Mettre en forme | Changement de format |
| Insérer/Supprimer | Modification de structuré |
| Tri/Filtre | Utilisation des données |

## Permettre certaines cellules

### Deverrouiller des cellules

1. Sélectionnez les cellules editables
2. Ctrl+1 (Format de cellule)
3. Onglet **Protection**
4. Decochez **Verrouillee**
5. Protégez ensuite la feuille

## Marquer comme final

### Activer

1. **Fichier** > **Informations**
2. **Proteger le classeur**
3. **Marquer comme final**

### Effet

Indique que le document est termine (facilement modifiable).

## Plages modifiables par utilisateur

### Configurer

1. **Révision** > **Autoriser la modification des plages**
2. **Nouveau**
3. Définissez la plage et les permissions
4. Protégez la feuille

### Usage

Permissions différentes selon les utilisateurs Windows.

## Masquer les formules

### Configurer

1. Sélectionnez les cellules avec formules
2. Ctrl+1 > **Protection**
3. Cochez **Masquee**
4. Protégez la feuille

### Résultat

La barre de formule n'affiché pas la formule.

## Protection VBA

### Proteger le projet VBA

1. Alt+F11 (Éditeur VBA)
2. Clic droit sûr le projet
3. **Propriétés du projet**
4. Onglet **Protection**
5. Cochez et définissez le mot de passe

## Résumé des protections

| Type | Empeche | Sécurité |
|------|---------|----------|
| Chiffrement | Ouverture | Élevée |
| Modification | Modification sans mdp | Moyenne |
| Structuré | Gestion des feuilles | Faible |
| Feuille | Modification contenu | Faible |
| VBA | Accès au code | Moyenne |

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Plusieurs niveaux | Combinez les protections |
| Documentation | Notez les mots de passe |
| Test | Vérifiez les restrictions |
| Sauvegarde | Gardez une copie accessible |
