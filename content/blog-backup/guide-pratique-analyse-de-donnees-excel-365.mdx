# Analyse de Données avec Excel 365

Exploitez les nouvelles fonctionnalités d'analyse.

## Outils d'analyse

### Fonctions dynamiques

| Fonction | Usage |
|----------|-------|
| FILTRE | Extraction |
| TRIER | Ordonnancement |
| UNIQUE | Deduplication |
| SEQUENCE | Génération |

### Power Query

Import et transformation des données.

### Power Pivot

Modelisation et DAX.

## Analyse rapide

### Accès

Sélectionnez des données > Ctrl+Q

### Options

- Mise en forme conditionnelle
- Graphiques
- Totaux
- Tableaux
- Sparklines

## Idées (Insights)

### Usage

1. Sélectionnez vos données
2. **Accueil** > **Idées**
3. Excel suggere des analyses
