# C'est Quoi Excel : Guide Complet

Tout ce que vous devez savoir.

## Définition

Microsoft Excel est un tableur pour :

- Calculs
- Analyse de données
- Graphiques
- Rapports

## Historique

| Année | Version |
|-------|---------|
| 1985 | Excel 1.0 (Mac) |
| 1987 | Excel 2.0 (Windows) |
| 2024 | Excel 365 |

## Interface

### Éléments clés

- Cellules (intersection ligne/colonne)
- Feuilles (onglets)
- Ruban (menus)
- Barre de formule

## Fonctionnalités

### Basiques

- Calculs simples
- Tableaux
- Tri et filtres

### Avancées

- Formules complexes
- Tableaux croisés
- Power Query
- VBA

## Versions actuelles

| Version | Type |
|---------|------|
| Excel 365 | Abonnement |
| Excel 2021 | Achat unique |
| Excel Online | Gratuit |

## Alternatives

- Google Sheets
- LibreOffice Calc
- Numbers

## Pour qui ?

Tout le monde : etudiants, professionnels, particuliers.
