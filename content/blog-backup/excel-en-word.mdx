# Excel en Word : Guide d'Intégration

Insérer des tableaux et graphiques Excel dans Word est essentiel pour créer des rapports professionnels.

## Méthode 1 : Copier-Coller Simple

### Étapes

1. Dans Excel, sélectionnez les cellules
2. **Ctrl + C** pour copier
3. Dans Word, **Ctrl + V** pour coller

### Résultat

Le tableau est converti en tableau Word. Les données sont statiques (non liees à Excel).

### Avantages

- Simple et rapide
- Tableau modifiable dans Word
- Pas besoin du fichier Excel

### Inconvenients

- Pas de mise à jour automatique
- Perte de certains formats

## Méthode 2 : Coller avec Liaison

### Étapes

1. Copiez dans Excel (**Ctrl + C**)
2. Dans Word : **Accueil** > **Coller** > **Collage special**
3. Sélectionnez **Coller avec liaison**
4. Choisissez le format

### Options de format

- **Objet Feuille de calcul Microsoft Excel** : Conserve l'apparence Excel
- **Texte mis en forme (RTF)** : Tableau Word avec mise en forme
- **Image** : Image statique du tableau

### Mise à jour

- La liaison garde le lien avec le fichier Excel
- Modifications dans Excel = mise à jour dans Word
- Clic droit sûr le tableau > **Mettre à jour la liaison**

## Méthode 3 : Incorporation

### Étapes

1. Copiez dans Excel
2. Dans Word : **Collage special** > **Coller** (pas "avec liaison")
3. Sélectionnez **Objet Feuille de calcul Microsoft Excel**

### Résultat

- Le tableau Excel est incorpore dans Word
- Double-clic pour modifier avec l'interface Excel
- Les données sont dans le document Word

### Avantages

- Pas besoin du fichier Excel original
- Modification complete possible
- Toutes les fonctionnalités Excel disponibles

## Méthode 4 : Insérer un Objet

### Étapes

1. Dans Word : **Insertion** > **Objet**
2. Onglet **Créer à partir d'un fichier**
3. Parcourir et sélectionner le fichier Excel
4. Cochez **Liaison** si vous voulez un lien

### Options

- **Liaison** : Mise à jour automatique
- **Afficher sous forme d'icone** : Affiche une icone cliquable

## Insérer un Graphique Excel

### Copier-Coller

1. Cliquez sûr le graphique dans Excel
2. **Ctrl + C**
3. Dans Word, **Ctrl + V**

### Options de collage

Après le collage, cliquez sûr l'icone d'options :

- **Graphique (lie aux données Excel)** : Mise à jour auto
- **Graphique Excel (classeur entier)** : Incorpore
- **Coller en tant qu'image** : Image statique

## Mise en forme dans Word

### Redimensionner

1. Cliquez sûr le tableau/graphique
2. Utilisez les poignees de coin
3. Maintenez **Maj** pour conserver les proportions

### Position

1. Clic droit > **Options de disposition**
2. Choisissez l'habillage du texte

### Bordures

Pour les tableaux colles en format Word :
- Sélectionnez le tableau
- **Conception de tableau** > **Bordures**

## Problèmes courants

### Tableau trop large

1. Sélectionnez le tableau
2. **Disposition** > **Ajustement automatique** > **Contenu**

Ou reduisez le nombre de colonnes dans Excel avant de copier.

### Mise à jour ne fonctionne pas

1. Vérifiez que le fichier Excel existe encore
2. Vérifiez le chemin d'accès
3. Clic droit > **Mettre à jour la liaison**

### Perte de mise en forme

Utilisez l'option "Objet Feuille de calcul" au lieu du collage simple.

## Bonnes pratiques

### Pour les rapports réguliers

- Utilisez la liaison pour la mise à jour automatique
- Gardez Excel et Word dans le meme dossier
- Nommez clairement vos fichiers

### Pour les documents à partager

- Utilisez l'incorporation (pas besoin d'Excel)
- Ou convertissez en image pour un rendu fixe

### Pour les grandes données

- Sélectionnez uniquement les données nécessaires
- Évitez de copier des plages trop larges

## Comparatif des méthodes

| Méthode | Mise à jour | Modifiable | Fichier Excel requis |
|---------|-------------|------------|---------------------|
| Copier-coller simple | Non | Oui (Word) | Non |
| Avec liaison | Oui | Non | Oui |
| Incorporation | Non | Oui (Excel) | Non |
| Image | Non | Non | Non |
