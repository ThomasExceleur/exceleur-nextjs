# Optimisez vos Previsions avec l'Analyse de Sensibilite Excel

L'analyse de sensibilite permet d'évaluer l'impact des variations sûr vos résultats.

## Qu'est-ce que l'analyse de sensibilite ?

### Définition

L'analyse de sensibilite etudie comment les variations des variables d'entrée affectent le résultat final.

### Applications

- Previsions financieres
- Budgetisation
- Gestion de projets
- Decisions d'investissement

## Outils Excel pour l'analyse de sensibilite

### Tables de données

| Type | Usage |
|------|-------|
| Une variable | Impact d'un facteur |
| Deux variables | Croisement de facteurs |

### Gestionnaire de scenarios

Comparez plusieurs hypotheses simultanement.

### Valeur cible

Trouvez la valeur d'entrée pour un résultat cible.

## Créer une table de données à une variable

### Étape 1 : Préparer le modèle

```excel
Cellule B1 : Variable (ex: taux d'intérêt)
Cellule B5 : Résultat (formule utilisant B1)
```

### Étape 2 : Configurer la table

1. Listez les valeurs à tester en colonne
2. Sélectionnez la plage
3. **Données** > **Analyse de scenarios** > **Table de données**

### Étape 3 : Définir la cellule d'entrée

Specifiez la cellule de la variable (B1).

## Table de données à deux variables

### Configuration

| | Valeur 1 | Valeur 2 | Valeur 3 |
|---------|----------|----------|----------|
| Facteur A | Résultat | Résultat | Résultat |
| Facteur B | Résultat | Résultat | Résultat |

### Utilisation

Analysez l'interaction entre deux variables.

## Gestionnaire de scenarios

### Créer un scenario

1. **Données** > **Analyse de scenarios** > **Gestionnaire de scenarios**
2. **Ajouter** pour créer un nouveau scenario
3. Nommez et définissez les cellules variables
4. Entrez les valeurs du scenario

### Synthèse des scenarios

Générez un rapport comparatif de tous les scenarios.

## Exemple pratique : Prevision de ventes

### Modèle de base

```excel
Prix unitaire : B1 = 100
Quantité : B2 = 1000
Cout variable : B3 = 60
Couts fixes : B4 = 20000
Profit : B5 = (B1-B3)*B2-B4
```

### Scenarios à tester

| Scenario | Prix | Quantité | Résultat |
|----------|------|----------|----------|
| Optimiste | 110 | 1200 | 40000 |
| Base | 100 | 1000 | 20000 |
| Pessimiste | 90 | 800 | 4000 |

## Interpreter les résultats

### Points clés

- Identifiez les variables les plus sensibles
- Évaluez les risques de chaque scenario
- Préparez des plans d'action

### Visualisation

Créez des graphiques pour illustrer l'impact des variations.

| Validation | Vérifiez la cohérence |
| Présentation | Résumez les conclusions |
