# Créer un Rapport d'Activité dans Excel

Excel est l'outil ideal pour créer des rapports d'activité clairs et professionnels.

## Structuré d'un rapport d'activité

### Éléments essentiels

1. **En-tête** : période, service, responsable
2. **Résumé executif** : indicateurs clés
3. **Données détaillées** : tableaux et graphiques
4. **Commentaires** : analyse et recommandations

## Créer le modèle de base

### En-tête du rapport

| Champ | Exemple |
|-------|---------|
| Période | Janvier 2024 |
| Service | Commercial |
| Responsable | Jean Dupont |
| Date edition | 15/02/2024 |

### Indicateurs clés (KPI)

```excel
=SOMME(Ventes[Montant])
=MOYENNE(Ventes[Montant])
=NB(Ventes[Client])
=SOMME.SI(Ventes[Statut];"Conclu";Ventes[Montant])
```

## Tableau recapitulatif

### Structuré recommandée

| Objectif | Réalisé | Ecart | % |
|----------|---------|-------|---|
| 100 000 | 95 000 | -5 000 | 95% |

### Formules

```excel
=B2-A2                    ' Ecart
=B2/A2                    ' Pourcentage
=SI(C2<0;"A améliorer";"OK")  ' Statut
```

## Graphiques pour le rapport

### Graphique d'evolution

1. Sélectionnez les données temporelles
2. **Insertion** > **Graphique** > **Courbe**
3. Ajoutez un titre et une legende

### Graphique de comparaison

1. Utilisez un **histogramme groupe**
2. Comparez objectifs vs réalisé
3. Ajoutez des etiquettes de données

### Graphique de répartition

```excel
' Camembert pour la répartition par catégorie
Sélectionnez les catégories et valeurs
Insertion > Graphique > Secteurs
```

## Mise en forme conditionnelle

### Colorer selon la performance

1. Sélectionnez la colonne "% objectif"
2. **Mise en forme conditionnelle** > **Échelles de couleurs**
3. Rouge-Jaune-Vert

### Icones de tendance

1. **Mise en forme conditionnelle** > **Jeux d'icones**
2. Fleches où feux tricolores

## Automatisation du rapport

### Avec les tableaux croisés dynamiques

1. Créez un TCD à partir de vos données
2. Regroupez par période
3. Ajoutez des filtres segments

### Actualisation automatique

```excel
=AUJOURDHUI()             ' Date du jour
=TEXTE(AUJOURDHUI();"mmmm aaaa")  ' Mois en cours
```

### Liaison avec d'autres feuilles

```excel
='Données brutes'!A1
=SOMME('Jan:Dec'!B2)      ' Somme sûr plusieurs feuilles
```

## Modèle de rapport mensuel

### Feuille 1 : Résumé

- KPI principaux
- Graphique d'evolution
- Points clés

### Feuille 2 : Données détaillées

- Tableau complet des données
- Filtres et tris

### Feuille 3 : Analyse

- Commentaires
- Recommandations
- Plan d'action

## Exportation du rapport

### En PDF

1. **Fichier** > **Exporter** > **PDF**
2. Sélectionnez les feuilles à inclure
3. Vérifiez la mise en page

### En PowerPoint

1. Copiez les graphiques
2. Collage special > Image
3. Ou liaison pour mise à jour automatique

## Bonnes pratiques

### Présentation

1. **Cohérence** : memes couleurs et polices
2. **Clarté** : titres explicites
3. **Simplicité** : evitez la surcharge

### Données

1. **Fiabilite** : vérifiez vos sources
2. **Actualite** : dates de mise à jour
3. **Tracabilite** : documentez vos calculs

## Conseils pour un rapport efficace

1. Commencez par les conclusions
2. Utilisez des visuels parlants
3. Limitez le nombre de pages
4. Adaptez au public cible
5. Incluez des recommandations
