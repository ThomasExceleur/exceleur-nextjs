# Graphiques 3D Excel

Ajoutez de la profondeur à vos visualisations.

## Types de graphiques 3D

### Disponibles

| Type | Usage |
|------|-------|
| Colonnes 3D | Comparaisons |
| Barres 3D | Comparaisons horizontales |
| Secteurs 3D | Répartitions |
| Lignes 3D | Tendances |
| Aires 3D | Evolution |

## Créer un graphique 3D

### Étapes

1. Sélectionnez vos données
2. **Insertion** > **Graphiques**
3. Choisissez une variante 3D

## Personnaliser la 3D

### Rotation

1. Clic droit sûr le graphique
2. **Rotation 3D**
3. Ajustez les angles

### Options

| Option | Effet |
|--------|-------|
| X | Rotation horizontale |
| Y | Rotation verticale |
| Perspective | Profondeur |

## 3D Maps

### Pour données geographiques

**Insertion** > **3D Map**

## Avantages et inconvenients

### Avantages

- Visuellement attractif
- Impression de profondeur

### Inconvenients

- Peut deformer la perception
- Plus difficile à lire

## Recommandations

| Situation | Recommandation |
|-----------|----------------|
| Precision | Évitez 3D |
| Présentation | 3D acceptable |
| Analyse | 2D preferable |

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Sobriete | N'abusez pas |
| Lisibilité | Priorisez la clarté |
| Contexte | Adaptez au public |
