# C'est Quoi Excel

Introduction au tableur de Microsoft.

## Définition

### Microsoft Excel

Logiciel de tableur permettant de :

- Organiser des données
- Effectuer des calculs
- Créer des graphiques
- Analyser des informations

## Interface

### Éléments principaux

| Élément | Description |
|---------|-------------|
| Cellules | Cases de la grille |
| Lignes | Numerotees 1, 2, 3... |
| Colonnes | Lettrees A, B, C... |
| Feuilles | Onglets en bas |

## A quoi çà sert ?

### Calculs

- Additions, moyennes
- Formules complexes
- Statistiques

### Organisation

- Listes et tableaux
- Bases de données simples
- Inventaires

### Visualisation

- Graphiques
- Tableaux de bord
- Rapports

## Qui utilisé Excel ?

| Metier | Usage |
|--------|-------|
| Comptables | Calculs financiers |
| RH | Suivi des employes |
| Commerciaux | Suivi des ventes |
| Etudiants | Projets, statistiques |
