# Optimisez vos Données avec FILTRE

Extrayez et analysez vos données efficacement.

## Syntaxe rappel

```excel
=FILTRE(tableau;inclure;[si_vide])
```

## Strategies d'optimisation

### Pre-filtrage

Filtrez d'abord les données volumineuses.

### Combinaison avec calculs

```excel
=SOMME(FILTRE(Montants;Region=\"Nord\"))
```

### Tableaux structurés

Utilisez des tableaux pour des références dynamiques.

## Filtres cascades

```excel
=LET(
  etape1;FILTRE(Données;Année=2024);
  etape2;FILTRE(etape1;Region=\"Nord\");
  etape2
)
```

## Avec agregations

### Somme filtree

```excel
=SOMME(FILTRE(Ventes[Montant];Ventes[Region]=E1))
```

### Moyenne filtree

```excel
=MOYENNE(FILTRE(Notes;Classe=\"A\"))
```

### Compte filtre

```excel
=LIGNES(FILTRE(Données;Statut=\"Actif\"))
```
