# Déplacer et Changer la Couleur de ses Onglets

Organisez visuellement vos classeurs Excel en colorant et reorganisant vos onglets.

## Déplacer un onglet

### Méthode 1 : Glisser-deposer

1. Cliquez sûr l'onglet à déplacer
2. Maintenez le bouton enfonce
3. Faites glisser vers la nouvelle position
4. Relacchez

### Méthode 2 : Menu contextuel

1. Clic droit sûr l'onglet
2. **Déplacer où copier**
3. Choisissez la position
4. **OK**

## Copier un onglet

### Dans le meme classeur

1. Maintenez **Ctrl** + glisser l'onglet
2. Une copie est créée

### Vers un autre classeur

1. Clic droit sûr l'onglet
2. **Déplacer où copier**
3. Sélectionnez le classeur de destination
4. Cochez **Créer une copie**
5. **OK**

## Changer la couleur d'un onglet

### Étapes

1. Clic droit sûr l'onglet
2. **Couleur d'onglet**
3. Choisissez une couleur

### Codes couleur suggeres

| Couleur | Usage |
|---------|-------|
| Vert | Données validees |
| Rouge | A corriger |
| Bleu | En cours |
| Jaune | Attention |

## Bonnes pratiques d'organisation

### Par type de contenu

- **Données brutes** : Bleu
- **Calculs** : Vert
- **Rapports** : Orange
- **Configuration** : Gris

### Par statut

- **Termine** : Vert
- **En cours** : Jaune
- **A faire** : Rouge

## Renommer un onglet

### Méthode rapide

Double-cliquez sûr l'onglet et tapez le nouveau nom.

### Règles de nommage

- Maximum 31 caracteres
- Pas de caracteres speciaux : / \ * ? : [ ]
- Noms uniques dans le classeur

## Raccourcis utiles

| Action | Raccourci |
|--------|-----------|
| Onglet suivant | Ctrl + Page Down |
| Onglet précédent | Ctrl + Page Up |
| Nouvel onglet | Shift + F11 |

## Conseils

1. **Utilisez** un code couleur cohérent
2. **Groupez** les onglets par fonction
3. **Nommez** clairement chaque feuille
4. **Limitez** le nombre d'onglets (10-15 max)
