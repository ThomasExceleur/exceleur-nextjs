# Combiner Plusieurs Fichiers avec Power Query

Fusionnez automatiquement vos fichiers.

## Depuis un dossier

### Étapes

1. **Données** > **Obtenir des données** > **A partir d'un fichier** > **A partir d'un dossier**
2. Sélectionnez le dossier
3. **Combiner** > **Combiner et transformer**

## Configuration

### Fichier exemple

Power Query vous demande de choisir un fichier modèle.

### Options

| Option | Description |
|--------|-------------|
| Première ligne en-tête | Utiliser comme noms |
| Ignorer les erreurs | Continuer si fichier corrompu |

## Transformations

### Ajouter la source

La colonne "Source.Name" indique le fichier d'origine.

### Filtrer les fichiers

Excluez certains fichiers si nécessaire.

## Types de fichiers

| Type | Support |
|------|---------|
| Excel | Oui |
| CSV | Oui |
| Texte | Oui |
| JSON | Oui |

## Actualisation

### Automatique

Ajoutez des fichiers au dossier, actualisez la requête.

### Manuelle

**Données** > **Actualiser tout**

## Structuré identique

### Prerequis

Tous les fichiers doivent avoir la meme structuré.

### Si différent

Utilisez plusieurs requêtes et ajoutez-les.

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Structuré | Meme format partout |
| Nommage | Noms de fichiers cohérents |
| Dossier dedie | Organisez vos sources |
