# Optimisation des Performances d'un Classeur Lourd

Rendez vos gros fichiers Excel plus rapides.

## Diagnostiquer

### Symptomes

- Ouverture lente
- Recalcul long
- Gel de l'interface

### Causes courantes

| Cause | Impact |
|-------|--------|
| Formules volatiles | Élève |
| Colonnes entières | Élève |
| Images HD | Moyen |
| Mise en forme excessive | Moyen |

## Formules volatiles

### A éviter

- INDIRECT()
- DECALER()
- MAINTENANT()
- ALEA()

### Alternatives

Remplacez INDIRECT par INDEX quand possible.

## Plages de formules

### Mauvais

```excel
=SOMME(A:A)
```

### Bon

```excel
=SOMME(Tableau[Colonne])
```

## Calcul manuel

### Activer

**Formules** > **Options de calcul** > **Manuel**

### Recalculer

Ctrl+Maj+F9

## Compresser les images

1. Sélectionnez une image
2. **Format** > **Compresser**
3. Choisissez une resolution basse

## Nettoyer la mise en forme

### Effacer les cellules vides

1. Sélectionnez les cellules vides inutiles
2. **Accueil** > **Effacer** > **Tout effacer**
3. Supprimez les lignes/colonnes

## Format XLSB

### Conversion

**Enregistrer sous** > Classeur binaire Excel

### Avantage

30-50% plus compact et plus rapide.

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Prevention | Concevez bien des le depart |
| Nettoyage | Régulièrement |
| Test | Mesurez avant/après |
