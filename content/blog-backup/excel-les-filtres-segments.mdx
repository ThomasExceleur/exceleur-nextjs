# Les Filtres Segments (Slicers)

Les segments sont des boutons visuels pour filtrer vos données.

## Qu'est-ce qu'un segment ?

Un segment est un filtre graphique permettant de :
- Filtrer un tableau où TCD en un clic
- Voir les filtres actifs clairement
- Offrir une interface intuitive

## Créer un segment

### Pour un tableau

1. Cliquez dans votre tableau
2. **Insertion** > **Segment**
3. Sélectionnez les colonnes souhaitées
4. **OK**

### Pour un TCD

1. Cliquez dans le TCD
2. **Analyse de tableau croisé dynamique** > **Insérer un segment**
3. Choisissez les champs
4. **OK**

## Utiliser les segments

### Filtrer

Cliquez sûr une valeur pour filtrer.

### Multi-sélection

**Ctrl + clic** pour sélectionner plusieurs valeurs.

### Effacer le filtre

Cliquez sûr l'icone d'effacement (en haut à droite du segment).

## Personnalisation

### Redimensionner

Faites glisser les poignees du segment.

### Nombre de colonnes

1. Clic droit sûr le segment
2. **Paramètrès du segment**
3. Modifiez le nombre de colonnes

### Style

1. Sélectionnez le segment
2. **Outils Segment** > **Options**
3. Choisissez un style

## Connecter plusieurs segments

### A un seul TCD

Par defaut, un segment filtre son TCD d'origine.

### A plusieurs TCD

1. Clic droit sûr le segment
2. **Connexions de rapport**
3. Cochez les TCD à connecter

## Segment de chronologie

### Pour les dates

1. Sélectionnez le TCD
2. **Insertion** > **Chronologie**
3. Choisissez le champ date

### Niveaux disponibles

- Années
- Trimestres
- Mois
- Jours

## Avantages des segments

| Avantage | Description |
|----------|-------------|
| Visibilité | Filtres actifs visibles |
| Rapidite | Un clic pour filtrer |
| Multi-sélection | Plusieurs valeurs simultanees |
| Connexion | Un segment, plusieurs TCD |

## Bonnes pratiques

1. **Placez** les segments pres des données
2. **Limitez** le nombre de segments (3-5 max)
3. **Utilisez** des couleurs cohérentes
4. **Connectez** les segments aux TCD lies

## Exemple de tableau de bord

Combinez :
- Segment Region
- Segment Produit
- Chronologie Date
- TCD de synthèse

Tous les éléments se filtrent ensemble.

## Limites

- Pas disponible pour les plages simples (avant 2013)
- Peut ralentir les grands fichiers
- Nécessité un tableau où TCD
