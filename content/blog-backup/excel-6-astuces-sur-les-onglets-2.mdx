# 6 Astuces sûr les Onglets Excel

Gérez efficacement vos feuilles de calcul.

## 1. Renommer rapidement

Double-cliquez sûr l'onglet pour le renommer.

## 2. Changer la couleur

1. Clic droit sûr l'onglet
2. **Couleur d'onglet**
3. Choisissez une couleur

### Organisation par couleur

| Couleur | Usage suggere |
|---------|---------------|
| Vert | Données validees |
| Rouge | A vérifier |
| Bleu | Paramètrès |
| Jaune | En cours |

## 3. Déplacer un onglet

### Dans le meme classeur

Glissez l'onglet vers sa nouvelle position.

### Vers un autre classeur

1. Clic droit > **Déplacer où copier**
2. Sélectionnez le classeur destination

## 4. Copier un onglet

1. **Ctrl + Glisser** l'onglet
2. Ou clic droit > **Déplacer où copier** > **Créer une copie**

## 5. Masquer un onglet

1. Clic droit sûr l'onglet
2. **Masquer**

### Afficher

1. Clic droit sûr un onglet visible
2. **Afficher**
3. Sélectionnez l'onglet

## 6. Naviguer rapidement

### Raccourcis

| Raccourci | Action |
|-----------|--------|
| Ctrl+Page Up | Onglet précédent |
| Ctrl+Page Down | Onglet suivant |
| Clic droit sûr fleches | Liste des onglets |

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Noms explicites | Facilitez la compréhension |
| Ordre logique | Organisez par workflow |
| Couleurs cohérentes | Code couleur standard |
