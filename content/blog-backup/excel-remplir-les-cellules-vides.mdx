# Remplir les Cellules Vides

Plusieurs méthodes pour combler les cellules vides dans vos données.

## Méthode 1 : Atteindre + Formule

### Étapes

1. Sélectionnez la plage contenant les vides
2. **Ctrl + G** (Atteindre)
3. **Special...**
4. Cochez **Cellules vides**
5. **OK** (toutes les cellules vides sont sélectionnées)
6. Tapez `=` puis la cellule au-dessus (ex: `=A1`)
7. **Ctrl + Entrée** (valide pour toutes les cellules)

### Résultat

Chaque cellule vide contient maintenant une formule copiant la valeur au-dessus.

### Convertir en valeurs

1. Copiez les cellules
2. Collez en valeurs (**Ctrl + Alt + V** > **Valeurs**)

## Méthode 2 : Rechercher et remplacer

### Pour remplir avec une valeur fixe

1. Sélectionnez la plage
2. **Ctrl + H**
3. **Rechercher** : laissez vide
4. **Remplacer par** : votre valeur (ex: "N/A")
5. **Options** > Cochez **Totalite du contenu de la cellule**
6. **Remplacer tout**

## Méthode 3 : Formule SI

### Éviter les vides dans les calculs

```excel
=SI(A1="";"Valeur par defaut";A1)
```

### Copier la valeur précédente

```excel
=SI(A2="";A1;A2)
```

## Méthode 4 : Power Query

### Étapes

1. Sélectionnez vos données
2. **Données** > **A partir d'un tableau**
3. Dans Power Query, sélectionnez la colonne
4. **Transformer** > **Remplir** > **Vers le bas**
5. **Fermer et charger**

### Avantage

Le remplissage se fait automatiquement lors des actualisations.

## Méthode 5 : VBA

### Macro de remplissage

```vba
Sub RemplirVides()
    Dim rng As Range
    For Each rng In Sélection
        If rng.Value = "" Then
            rng.Value = rng.Offset(-1, 0).Value
        End If
    Next rng
End Sub
```

## Cas pratiques

### Données importees

Les exports de bases de données ont souvent des cellules vides à remplir.

### Tableaux croisés depivotes

Après depivotement, les catégories sont souvent vides.

### Données hiérarchiques

Les sous-catégories peuvent avoir des cellules de catégorie vides.

## Comparaison des méthodes

| Méthode | Rapidite | Automatique | Sans formule |
|---------|----------|-------------|--------------|
| Atteindre | Rapide | Non | Non |
| Remplacer | Rapide | Non | Oui |
| Formule | Moyen | Oui | Non |
| Power Query | Moyen | Oui | Oui |
| VBA | Rapide | Non | Oui |

## Bonnes pratiques

1. **Sauvegardez** avant de modifier
2. **Testez** sûr une copie
3. **Préférez** Power Query pour les données recurrentes
4. **Documentez** les modifications effectuees
