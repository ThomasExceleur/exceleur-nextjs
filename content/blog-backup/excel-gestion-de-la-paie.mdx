# Gestion de la Paie dans Excel

Excel permet de gérer la paie pour les petites structurés où de comprendre les mecanismes de calcul.

## Avertissement

Ce guide est à but pedagogique. Pour une gestion officielle de la paie, utilisez un logiciel certifie.

## Structuré du tableau de paie

### Informations salarie

| Champ | Description |
|-------|-------------|
| Matricule | Identifiant unique |
| Nom Prenom | Identite |
| Poste | Fonction |
| Date entrée | Anciennete |
| Salaire brut | Remuneration mensuelle |

### Éléments de paie

| Élément | Type |
|---------|------|
| Salaire de base | Fixe |
| Heures sup | Variable |
| Primes | Variable |
| Avantages | Fixe/Variable |

## Calcul du salaire brut

### Salaire de base

```excel
=Salaire_annuel/12
```

### Heures supplémentaires

```excel
=NB_heures_sup * Taux_horaire * 1.25    ' Majoration 25%
=NB_heures_sup * Taux_horaire * 1.50    ' Majoration 50%
```

### Taux horaire

```excel
=Salaire_mensuel / 151.67    ' Base 35h
```

## Calcul des cotisations

### Cotisations salariales (approximatif)

| Cotisation | Taux |
|------------|------|
| Sécurité sociale | 6.9% |
| Retraite | 4% |
| Chomage | 0.95% |
| CSG/CRDS | 9.7% |

### Formule de calcul

```excel
=Salaire_brut * Taux_cotisation
```

### Total cotisations

```excel
=SOMME(Cotisation1:CotisationN)
```

## Calcul du salaire net

### Formule de base

```excel
=Salaire_brut - Total_cotisations_salariales
```

### Net à payer

```excel
=Salaire_net - Acomptes - Retenues + Remboursements
```

## Tableau recapitulatif mensuel

### Structuré

| Salarie | Brut | Cotisations | Net | Net à payer |
|---------|------|-------------|-----|-------------|
| Dupont | 3000 | =FORMULE | =FORMULE | =FORMULE |

### Formules

```excel
=SOMME.SI(Paie[Salarie];A2;Paie[Brut])
```

## Gestion des heures

### Feuille de temps

| Date | Entrée | Sortie | Heures | Heures sup |
|------|--------|--------|--------|------------|
| 01/01 | 9:00 | 18:00 | 8:00 | 0:00 |

### Calcul des heures

```excel
=C2-B2-"1:00"              ' Moins pause dejeuner
=SI(D2>"7:00";D2-"7:00";0) ' Heures supplémentaires
```

## Conges et absences

### Suivi des conges

| Salarie | Acquis | Pris | Solde |
|---------|--------|------|-------|
| Dupont | 25 | 10 | =B2-C2 |

### Impact sûr la paie

```excel
=Salaire_journalier * Jours_absence
=Salaire_mensuel / 21.67   ' Salaire journalier
```

## Primes et avantages

### Types de primes

- Prime d'anciennete
- Prime de performance
- Prime exceptionnelle
- 13eme mois

### Calcul prime anciennete

```excel
=SI(Anciennete>=3;Salaire_brut*0.03;0)    ' 3% après 3 ans
```

## Charges patronales

### Taux approximatifs

| Charge | Taux |
|--------|------|
| Sécurité sociale | 13% |
| Retraite | 8% |
| Chomage | 4% |
| Formation | 1% |

### Cout total employeur

```excel
=Salaire_brut * (1 + Total_charges_patronales)
```

## Tableau de bord RH

### Indicateurs clés

```excel
=SOMME(Paie[Brut])                    ' Masse salariale
=MOYENNE(Paie[Brut])                   ' Salaire moyen
=NB(Paie[Salarie])                     ' Effectif
=SOMME(Paie[Brut])/NB(Paie[Salarie])  ' Cout moyen
```

## Export et reporting

### Pour le comptable

1. Créez un recapitulatif mensuel
2. Detaillez par compte comptable
3. Exportez en PDF

### Pour les declarations

Préparez les totaux par type de cotisation.

## Bonnes pratiques

1. **Securisez** le fichier avec un mot de passe
2. **Archivez** chaque mois
3. **Vérifiez** les calculs régulièrement
4. **Mettez à jour** les taux de cotisation

## Limites d'Excel pour la paie

- Pas de génération de fiches de paie officielles
- Pas de declarations automatiques
- Pas de mise à jour des taux reglementaires
- Risque d'erreurs de calcul
