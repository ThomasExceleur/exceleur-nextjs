# Créer une Heatmap dans Excel

Une heatmap (carte de chaleur) utilisé les couleurs pour visualiser les données.

## Qu'est-ce qu'une heatmap ?

Une représentation visuelle où :
- Les valeurs élevées sont en couleur chaude (rouge)
- Les valeurs faibles sont en couleur froide (vert/bleu)
- Les couleurs intermediaires indiquent les valeurs moyennes

## Méthode : Mise en forme conditionnelle

### Étapes

1. Sélectionnez vos données
2. **Accueil** > **Mise en forme conditionnelle**
3. **Nuances de couleurs**
4. Choisissez un schéma (ex: Rouge-Jaune-Vert)

### Schémas disponibles

| Schéma | Usage |
|--------|-------|
| Rouge-Jaune-Vert | Performance (vert = bon) |
| Vert-Jaune-Rouge | Risque (rouge = mauvais) |
| Bleu-Blanc-Rouge | Ecarts (neutre au centre) |

## Personnaliser les couleurs

### Échelle personnalisee

1. **Mise en forme conditionnelle** > **Gérer les règles**
2. Sélectionnez la règle
3. **Modifier la règle**
4. Choisissez **Échelle à 3 couleurs**
5. Définissez les seuils et couleurs

### Exemple de seuils

| Position | Valeur | Couleur |
|----------|--------|---------|
| Minimum | Automatique | Vert |
| Milieu | Percentile 50 | Jaune |
| Maximum | Automatique | Rouge |

## Heatmap avec valeurs spécifiques

### Seuils fixes

1. **Mise en forme conditionnelle** > **Nouvelle règle**
2. **Mettre en forme les cellules qui contiennent**
3. Définissez vos conditions

### Exemple

- Inférieur à 50 : Rouge
- Entre 50 et 80 : Jaune
- Supérieur à 80 : Vert

## Masquer les valeurs

### Afficher uniquement les couleurs

1. Sélectionnez les cellules
2. **Ctrl + 1** (Format de cellule)
3. **Personnalise**
4. Tapez `;;;` (trois points-virgules)

Les nombres sont masques, les couleurs restent.

## Cas pratiques

### Matrice de correlation

Affiche les relations entre variables.

### Calendrier de chaleur

Visualisez l'activité par jour.

### Performance par region

Comparez les résultats geographiques.

### Tableau de bord

Identifiez rapidement les anomalies.

## Astuces avancées

### Combiner avec des icones

1. Ajoutez une mise en forme conditionnelle
2. Choisissez **Jeux d'icones**
3. Les icones completent les couleurs

### Heatmap negative/positive

Utilisez une échelle centree sûr zero :
- Negatif : Rouge
- Zero : Blanc
- Positif : Vert

## Bonnes pratiques

1. **Choisissez** des couleurs intuitives
2. **Ajoutez** une legende si nécessaire
3. **Évitez** trop de catégories de couleurs
4. **Testez** avec des daltoniens en tete

## Limites

- Pas de vraie carte geographique
- Nécessité des données numériques
- Peut etre difficile à imprimer
