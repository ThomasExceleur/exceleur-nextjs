# Nettoyer et Transformer les Données

Les bases du nettoyage de données dans Excel.

## Problèmes courants

| Problème | Solution |
|----------|----------|
| Espaces | SUPPRESPACE |
| Casse | MAJUSCULE/MINUSCULE |
| Doublons | Supprimer les doublons |
| Formats | Convertir |

## Fonctions de nettoyage

### SUPPRESPACE

```excel
=SUPPRESPACE(A1)
```

Supprime les espaces superflus.

### EPURAGE

```excel
=EPURAGE(A1)
```

Supprime les caracteres non imprimables.

### NOMPROPRE

```excel
=NOMPROPRE(A1)
```

Première lettre en majuscule.

## Convertir le texte

### En majuscules

```excel
=MAJUSCULE(A1)
```

### En minuscules

```excel
=MINUSCULE(A1)
```

## Supprimer les doublons

### Étapes

1. Sélectionnez la plage
2. **Données** > **Supprimer les doublons**
3. Choisissez les colonnes

## Remplir les cellules vides

### Avec la valeur précédente

1. Sélectionnez la plage avec vides
2. F5 > Special > Cellules vides
3. Tapez = puis fleche haut
4. Ctrl+Entrée
