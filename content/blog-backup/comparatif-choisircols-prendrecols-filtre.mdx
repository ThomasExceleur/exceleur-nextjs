# Comparatif CHOISIRCOLS, PRENDRECOLS et FILTRE

Trois fonctions pour manipuler les colonnes et filtrer les données.

## CHOISIRCOLS : Sélectionner des colonnes spécifiques

### Syntaxe

```excel
=CHOISIRCOLS(tableau;num_col1;[num_col2];...)
```

### Exemples

```excel
=CHOISIRCOLS(A1:D10;1;3)      ' Colonnes 1 et 3
=CHOISIRCOLS(A1:D10;4;2;1)    ' Ordre inverse
=CHOISIRCOLS(A1:D10;-1)       ' Dernière colonne
```

### Cas d'usage

- Réorganiser les colonnes
- Exclure des colonnes intermediaires
- Inverser l'ordre des colonnes

## PRENDRECOLS : Extraire un nombre de colonnes

### Syntaxe

```excel
=PRENDRECOLS(tableau;[nb_colonnes])
```

### Exemples

```excel
=PRENDRECOLS(A1:D10;2)    ' 2 premières colonnes
=PRENDRECOLS(A1:D10;-2)   ' 2 dernières colonnes
```

### Cas d'usage

- Obtenir les N premières colonnes
- Obtenir les N dernières colonnes

## FILTRE : Extraire selon des critères

### Syntaxe

```excel
=FILTRE(tableau;inclure;[si_vide])
```

### Exemples

```excel
=FILTRE(A1:D10;B1:B10>100)              ' Ou B>100
=FILTRE(A1:D10;(B1:B10>100)*(C1:C10="Oui"))  ' Multicriteres
=FILTRE(A1:D10;B1:B10>100;"Aucun")     ' Avec message si vide
```

### Cas d'usage

- Filtrer les lignes selon conditions
- Créer des sous-ensembles dynamiques

## Tableau comparatif

| Aspect | CHOISIRCOLS | PRENDRECOLS | FILTRE |
|--------|-------------|-------------|--------|
| Colonnes spécifiques | Oui | Non | Non |
| N colonnes | Non | Oui | Non |
| Filtrer lignes | Non | Non | Oui |
| Critères | Non | Non | Oui |
| Ordre flexible | Oui | Non | Non |

## Combinaisons utiles

### CHOISIRCOLS + FILTRE

Filtrer puis sélectionner des colonnes :

```excel
=CHOISIRCOLS(
  FILTRE(A1:D10;B1:B10>100);
  1;3
)
```

### PRENDRECOLS + TRIER

Prendre les colonnes après tri :

```excel
=PRENDRECOLS(
  TRIER(A1:D10;2;-1);
  2
)
```

### FILTRE + CHOISIRCOLS + TRIER

Chaine complete :

```excel
=TRIER(
  CHOISIRCOLS(
    FILTRE(A1:D10;B1:B10>100);
    1;3;4
  );
  1
)
```

## Equivalents pour les lignes

### CHOISIRLIGS

```excel
=CHOISIRLIGS(tableau;num_lig1;num_lig2;...)
```

### PRENDRELIGS

```excel
=PRENDRELIGS(tableau;nb_lignes)
```

## Exemples pratiques

### 1. Rapport avec colonnes spécifiques

```excel
=CHOISIRCOLS(Ventes;1;3;5)  ' Nom, Montant, Statut
```

### 2. Top 5 des ventes

```excel
=PRENDRELIGS(
  TRIER(Ventes;2;-1);
  5
)
```

### 3. Clients avec commandes recentes

```excel
=FILTRE(
  Clients;
  MOIS(Clients[DernierAchat])=MOIS(AUJOURDHUI())
)
```

### 4. Export personnalise

```excel
=CHOISIRCOLS(
  FILTRE(Données;Données[Region]="Paris");
  1;4;5;2
)
```

## Gestion des erreurs

### FILTRE sans résultat

```excel
=FILTRE(A:A;B:B="Inexistant";"Aucun résultat")
```

### CHOISIRCOLS avec colonne hors limites

Vérifiez que les numéros de colonnes existent.

### PRENDRECOLS depassant le tableau

Retourne toutes les colonnes disponibles.

## Performance

| Fonction | Performance |
|----------|-------------|
| CHOISIRCOLS | Très rapide |
| PRENDRECOLS | Très rapide |
| FILTRE | Depend des critères |

### Optimisation

1. Limitez les plages sources
2. Évitez les critères complexes dans FILTRE
3. Utilisez LET pour les calculs intermediaires

## Quand utiliser quoi ?

### CHOISIRCOLS

- Sélectionner colonnes spécifiques
- Reordonner les colonnes
- Exclure des colonnes

### PRENDRECOLS

- Premières où dernières colonnes
- Simplifier les extractions

### FILTRE

- Extraire selon conditions
- Créer des vues dynamiques
- Tableaux de bord interactifs
