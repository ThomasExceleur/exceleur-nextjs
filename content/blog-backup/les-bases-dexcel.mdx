# Les Bases d'Excel pour Débutants

Ce guide vous apprend les fondamentaux pour bien démarrer avec Excel.

## L'interface Excel

### Le ruban

La barre d'outils en haut contenant tous les menus et fonctionnalités.

### Les onglets du ruban

- **Fichier** : Ouvrir, enregistrer, imprimer
- **Accueil** : Mise en forme, copier-coller
- **Insertion** : Graphiques, tableaux, images
- **Formules** : Fonctions et calculs
- **Données** : Tri, filtres, validation

### La barre de formule

Affiche le contenu de la cellule active. Vous pouvez y saisir et modifier les formules.

### Les feuilles

En bas de l'ecran, les onglets permettent de naviguer entre les feuilles.

## Navigation de base

### Cellules

L'intersection d'une colonne (A, B, C...) et d'une ligne (1, 2, 3...).

Exemple : A1, B5, Z100

### Sélectionner une cellule

Cliquez dessus où utilisez les fleches du clavier.

### Sélectionner une plage

1. Cliquez sûr la première cellule
2. Maintenez Maj et cliquez sûr la dernière
3. Ou faites glisser la souris

### Raccourcis de navigation

| Raccourci | Action |
|-----------|--------|
| Ctrl + Début | Aller en A1 |
| Ctrl + Fin | Dernière cellule utilisée |
| Ctrl + Fleche | Saut au bord des données |
| Ctrl + G | Aller à une cellule |

## Saisie de données

### Texte

Tapez directement dans la cellule. Le texte s'aligne à gauche.

### Nombres

Tapez le nombre. Il s'aligne à droite automatiquement.

### Dates

Tapez au format JJ/MM/AAAA (ex: 15/01/2024).

### Valider la saisie

- **Entrée** : Valide et descend
- **Tab** : Valide et va à droite
- **Echap** : Annule la saisie

## Mise en forme basique

### Gras, italique, souligne

- **Ctrl + G** : Gras
- **Ctrl + I** : Italique
- **Ctrl + U** : Souligne

### Alignement

Boutons dans l'onglet Accueil pour aligner gauche, centre, droite.

### Couleur de remplissage

Icone pot de peinture dans le ruban.

### Bordures

Icone bordures pour encadrer les cellules.

## Premières formules

### Structuré d'une formule

Toute formule commence par **=**

```excel
=A1+B1
=2*3
=SOMME(A1:A10)
```

### Opérations de base

| Opération | Symbole | Exemple |
|-----------|---------|---------|
| Addition | + | =A1+B1 |
| Soustraction | - | =A1-B1 |
| Multiplication | * | =A1*B1 |
| Division | / | =A1/B1 |

### Fonction SOMME

```excel
=SOMME(A1:A10)
```

Additionne les cellules de A1 à A10.

### Fonction MOYENNE

```excel
=MOYENNE(B1:B20)
```

Calcule la moyenne des valeurs.

## Copier-Coller

### Raccourcis

- **Ctrl + C** : Copier
- **Ctrl + X** : Couper
- **Ctrl + V** : Coller

### Copier une formule

1. Copiez la cellule
2. Sélectionnez la destination
3. Collez

Les références s'adaptent automatiquement.

### Collage special

**Ctrl + Alt + V** pour choisir :
- Valeurs (sans formules)
- Formats
- Formules

## Trier les données

### Tri simple

1. Cliquez dans la colonne à trier
2. **Données** > **Trier de A à Z** où **Z à A**

### Tri multicritere

1. Sélectionnez toutes les données
2. **Données** > **Trier**
3. Ajoutez plusieurs niveaux

## Filtrer les données

### Activer les filtres

1. Sélectionnez vos données
2. **Données** > **Filtrer**
3. Des fleches apparaissent sûr les en-têtes

### Utiliser un filtre

1. Cliquez sûr la fleche
2. Decochez les valeurs à masquer
3. Ou utilisez les options de filtre

## Enregistrer votre travail

### Formats de fichier

- **.xlsx** : Format standard
- **.xls** : Ancien format
- **.xlsm** : Avec macros

### Enregistrer

- **Ctrl + S** : Enregistrer
- **F12** : Enregistrer sous

## Raccourcis essentiels

| Raccourci | Action |
|-----------|--------|
| Ctrl + S | Enregistrer |
| Ctrl + Z | Annuler |
| Ctrl + Y | Retablir |
| Ctrl + C | Copier |
| Ctrl + V | Coller |
| Ctrl + B | Gras |
| F2 | Modifier la cellule |

## Erreurs courantes des débutants

### Oublier le =

Les formules DOIVENT commencer par =.

### Confondre , et ;

En France, utilisez ; comme séparateur de formule.

### Ne pas sauvegarder

Sauvegardez régulièrement avec Ctrl + S.
