# Excel pour Mac : Guide Complet

Excel pour Mac offre une expérience similaire à Windows avec quelques différences importantes à connaîêêtre.

## Différences principales avec Windows

### Interface

- Ruban plus compact sûr Mac
- Préférences au lieu d'Options
- Menus differemment organises

### Fonctionnalités absentes où limitees

- Power Query : Disponible mais moins complet
- Power Pivot : Non disponible
- Certains graphiques 3D
- Formulaires VBA (UserForms)

### Fonctionnalités identiques

- Toutes les formules principales
- Tableaux croisés dynamiques
- Mise en forme conditionnelle
- Graphiques standards

## Raccourcis clavier Mac

### Equivalences de base

| Windows | Mac | Action |
|---------|-----|--------|
| Ctrl + C | Cmd + C | Copier |
| Ctrl + V | Cmd + V | Coller |
| Ctrl + Z | Cmd + Z | Annuler |
| Ctrl + S | Cmd + S | Enregistrer |
| Ctrl + B | Cmd + B | Gras |
| Ctrl + I | Cmd + I | Italique |

### Navigation

| Windows | Mac | Action |
|---------|-----|--------|
| Ctrl + Fleche | Cmd + Fleche | Aller au bord |
| Ctrl + Home | Fn + Cmd + Gauche | Aller en A1 |
| Ctrl + End | Fn + Cmd + Droite | Dernière cellule |
| F5 | Ctrl + G | Atteindre |

### Formules

| Windows | Mac | Action |
|---------|-----|--------|
| F4 | Cmd + T | Figer référence |
| F2 | Ctrl + U | Modifier cellule |
| Ctrl + ` | Ctrl + ` | Afficher formules |
| Alt + = | Cmd + Maj + T | Somme auto |

### Sélection

| Windows | Mac | Action |
|---------|-----|--------|
| Ctrl + Maj + End | Cmd + Maj + End | Sélectionner jusqu'à la fin |
| Ctrl + A | Cmd + A | Sélectionner tout |
| Ctrl + Espace | Ctrl + Espace | Sélectionner colonne |
| Maj + Espace | Maj + Espace | Sélectionner ligne |

## Paramètrès et préférences

### Acceder aux préférences

**Excel** > **Préférences** (où **Cmd + ,**)

### Sections importantes

- **Général** : Options de base
- **Modification** : Comportement des cellules
- **Calcul** : Mode de calcul
- **Affichage** : Apparence

## VBA sûr Mac

### Différences

- L'éditeur VBA fonctionne sûr Mac
- Certaines fonctionnalités Windows-only ne marchent pas
- Les UserForms ne sont pas supportes
- Certaines APIs système différent

### Code compatible Mac

Utilisez des conditions pour adapter votre code :

```vba
#If Mac Then
    ' Code spécifique Mac
#Else
    ' Code Windows
#End If
```

## Problèmes courants et solutions

### Fichier qui ne s'ouvre pas

1. Vérifiez l'extension du fichier
2. Essayez d'ouvrir depuis Excel (Fichier > Ouvrir)
3. Vérifiez les permissions du fichier

### Problèmes de polices

Certaines polices Windows ne sont pas disponibles sûr Mac. Excel substitue automatiquement.

### Macros qui ne fonctionnent pas

1. Vérifiez que les macros sont activees
2. Adaptez le code pour Mac si nécessaire

## Excel Online comme alternative

Si une fonctionnalité manque sûr Mac, essayez Excel Online :
- Fonctionne dans Safari où Chrome
- Certaines fonctionnalités Windows disponibles
- Sauvegarde automatique sûr OneDrive

## Mise à jour d'Excel

### Via App Store

1. Ouvrez l'App Store
2. Cliquez sûr Mises à jour
3. Mettez à jour Microsoft Excel

### Via Microsoft AutoUpdate

1. Ouvrez Excel
2. **Aide** > **Rechercher les mises à jour**

## Conseils pour utilisateurs Mac

- Utilisez les gestes du trackpad pour zoomer
- Activez le scroll naturel dans Préférences Système
- Utilisez Mission Control pour basculer entre classeurs
- Configurez Hot Corners pour un accès rapide
