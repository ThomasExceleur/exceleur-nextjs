# Planning de Production dans Excel

Un planning de production bien concu optimise vos ressources et respecte les délais.

## Éléments du planning

### Informations essentielles

| Élément | Description |
|---------|-------------|
| Ordre de fabrication | Référence OF |
| Produit | Article à fabriquer |
| Quantité | Nombre à produire |
| Date début | Lancement production |
| Date fin | Livraison prévue |
| Statut | État de l'OF |

## Structuré du tableau

### Tableau des OF

| OF | Produit | Qte | Début | Fin | Poste | Statut |
|----|---------|-----|-------|-----|-------|--------|
| OF001 | Produit A | 100 | 01/01 | 05/01 | Ligne 1 | En cours |
| OF002 | Produit B | 50 | 02/01 | 04/01 | Ligne 2 | Planifie |

## Calcul des délais

### Duree de production

```excel
=Quantité / Cadence_horaire / Heures_jour
```

### Date de fin

```excel
=Date_debut + Duree_jours - 1
=JOURS.OUVRES(Date_debut;Duree_jours)    ' Jours ouvres
```

## Diagramme de Gantt

### Créer la grille

1. Créez des colonnes pour chaque jour
2. Les lignes representent les OF

### Formule de mise en forme

```excel
=ET(F$1>=[@Début];F$1<=[@Fin])
```

### Mise en forme conditionnelle

1. Sélectionnez la grille
2. **Mise en forme conditionnelle** > **Nouvelle règle**
3. Utilisez la formule ci-dessus
4. Appliquez un remplissage colore

## Capacité de production

### Tableau de capacité

| Poste | Capacité/jour | Charge | Dispo |
|-------|---------------|--------|-------|
| Ligne 1 | 100 | 80 | 20 |
| Ligne 2 | 150 | 150 | 0 |

### Calcul de charge

```excel
=SOMME.SI(Planning[Poste];A2;Planning[Qte])
```

### Taux d'utilisation

```excel
=Charge / Capacité
```

## Ordonnancement

### Règles de priorité

| Règle | Description |
|-------|-------------|
| FIFO | Premier arrive, premier servi |
| Date livraison | Plus proche d'abord |
| Priorité client | Clients importants d'abord |

### Tri automatique

```excel
Trier par : Date livraison (croissant)
Puis par : Priorité (decroissant)
```

## Suivi de production

### États de production

- Planifie
- Lance
- En cours
- Termine
- En retard

### Mise à jour du statut

```excel
=SI([@Fin]<AUJOURDHUI();"En retard";
  SI([@Début]<=AUJOURDHUI();"En cours";"Planifie"))
```

## Indicateurs de production

### KPI essentiels

| KPI | Formule |
|-----|---------|
| OTD (On Time Delivery) | OF à l'heure / Total OF |
| TRS | Temps prod / Temps dispo |
| Taux de rebut | Rebuts / Production totale |

### Calcul OTD

```excel
=NB.SI.ENS(Planning[Statut];"Termine";Planning[Fin];"<="&Planning[Date livraison]) / NB.SI(Planning[Statut];"Termine")
```

## Gestion des aleas

### Types d'aleas

| Alea | Impact | Action |
|------|--------|--------|
| Panne machine | Retard | Replanifier |
| Absence | Capacité réduite | Remplacement |
