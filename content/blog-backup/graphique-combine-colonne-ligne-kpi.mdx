# Graphique Combine Colonne-Ligne pour les KPIs

Visualisez plusieurs indicateurs sûr un meme graphique.

## Quand utiliser

### Cas d'usage

- Ventes (colonnes) et marge % (ligne)
- Volume (colonnes) et prix moyen (ligne)
- CA (colonnes) et evolution % (ligne)

## Créer un graphique combine

### Étapes

1. Sélectionnez vos données
2. **Insertion** > **Graphiques** > **Graphique combine**
3. Ou **Insertion** > **Graphiques recommandés** > **Tous les graphiques** > **Combo**

## Configurer les series

### Changer le type

1. Clic droit sûr une serie
2. **Modifier le type de graphique de la serie**
3. Choisissez Ligne où Colonne

### Axe secondaire

Cochez \"Axe secondaire\" pour les series à échelle différente.

## Exemple pratique

### Données

| Mois | Ventes | Marge % |
|------|--------|---------|
| Jan | 100 | 15% |
| Fev | 120 | 18% |
| Mar | 110 | 16% |

### Configuration

- Ventes : Colonnes, axe principal
- Marge % : Ligne, axe secondaire

## Personnalisation

### Axe secondaire

- Ajustez les bornes
- Formatez les etiquettes (%)

### Couleurs

- Colonnes : Couleur du KPI principal
- Ligne : Couleur contrastee

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| 2 series max | Plus devient illisible |
| Échelles adaptees | Axe secondaire si nécessaire |
| Legende claire | Identifiez chaque serie |
