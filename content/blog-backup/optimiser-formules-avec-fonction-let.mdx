# Optimiser vos Formules avec LET

La fonction LET permet de créer des variables dans vos formules pour plus de performance et de lisibilité.

## Disponibilité

- Excel 365
- Excel 2021

## Pourquoi utiliser LET ?

### Problème classique

Sans LET, vous répétez les memes calculs :

```excel
=SI(SOMME(A1:A100)>1000;SOMME(A1:A100)*0.1;SOMME(A1:A100)*0.05)
```

SOMME(A1:A100) est calcule 3 fois.

### Solution avec LET

```excel
=LET(
  total;SOMME(A1:A100);
  SI(total>1000;total*0.1;total*0.05)
)
```

Le calcul n'est fait qu'une seule fois.

## Syntaxe

```excel
=LET(nom1;valeur1;[nom2;valeur2;...];calcul)
```

## Avantages

### Performance

Les calculs complexes ne sont executes qu'une fois.

### Lisibilité

Les formules deviennent auto-documentees.

### Maintenance

Plus facile de modifier une variable que plusieurs occurrences.

## Exemples pratiques

### Calcul de remise

```excel
=LET(
  montant;B2;
  taux;SI(montant>1000;0.15;SI(montant>500;0.10;0.05));
  remise;montant*taux;
  montant-remise
)
```

### Statistiques

```excel
=LET(
  données;A1:A100;
  moy;MOYENNE(données);
  ecart;ECARTYPE.STANDARD(données);
  "Moyenne: "&moy&" | Ecart-type: "&ecart
)
```

### Recherche avec traitement

```excel
=LET(
  résultat;RECHERCHEX(E1;A:A;B:B);
  SI(ESTERREUR(résultat);"Non trouve";résultat*1.2)
)
```

## Variables en cascade

### Reutiliser les variables précédentes

```excel
=LET(
  x;A1;
  y;x*2;
  z;y+10;
  résultat;z/5;
  résultat
)
```

Chaque variable peut utiliser les précédentes.

## Combinaison avec autres fonctions

### LET + FILTRE

```excel
=LET(
  filtre;FILTRE(A:D;B:B="Actif");
  TRIER(filtre;3;-1)
)
```

### LET + RECHERCHEX

```excel
=LET(
  id;E1;
  nom;RECHERCHEX(id;A:A;B:B);
  prix;RECHERCHEX(id;A:A;C:C);
  nom&" - "&prix&" euros"
)
```

## Règles de nommage

### Noms valides

- Commencer par une lettre
- Lettres, chiffres, underscore
- Pas de noms reserves (VRAI, FAUX)

### Exemples

```excel
=LET(total;...;...)           ' OK
=LET(mon_total;...;...)       ' OK
=LET(1total;...;...)          ' ERREUR
=LET(VRAI;...;...)            ' ERREUR
```

## Bonnes pratiques

1. **Nommez** clairement vos variables
2. **Limitez** le nombre de variables (5-7 max)
3. **Groupez** les calculs logiquement
4. **Testez** chaque variable separement
