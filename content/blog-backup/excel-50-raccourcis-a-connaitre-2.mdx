# 50 Raccourcis Excel Essentiels

Gagnez du temps avec ces raccourcis clavier.

## Navigation

| Raccourci | Action |
|-----------|--------|
| Ctrl+Début | Aller en A1 |
| Ctrl+Fin | Dernière cellule utilisée |
| Ctrl+Fleche | Sauter au bord des données |
| Ctrl+Page Up/Down | Changer d'onglet |

## Sélection

| Raccourci | Action |
|-----------|--------|
| Ctrl+Maj+Fin | Sélectionner jusqu'à la fin |
| Ctrl+Espace | Sélectionner la colonne |
| Maj+Espace | Sélectionner la ligne |
| Ctrl+A | Tout sélectionner |

## Édition

| Raccourci | Action |
|-----------|--------|
| F2 | Éditer la cellule |
| Ctrl+C / Ctrl+V | Copier / Coller |
| Ctrl+X | Couper |
| Ctrl+Z | Annuler |
| Ctrl+Y | Retablir |
| Suppr | Effacer le contenu |

## Mise en forme

| Raccourci | Action |
|-----------|--------|
| Ctrl+G | Gras |
| Ctrl+I | Italique |
| Ctrl+U | Souligne |
| Ctrl+1 | Format de cellule |
| Ctrl+Maj+$ | Format monnaie |
| Ctrl+Maj+% | Format pourcentage |

## Formules

| Raccourci | Action |
|-----------|--------|
| F4 | Référence absolue |
| Ctrl+` | Afficher les formules |
| Tab | Valider suggestion |
| Ctrl+Maj+Entrée | Formule matricielle |

## Fichier

| Raccourci | Action |
|-----------|--------|
| Ctrl+N | Nouveau classeur |
| Ctrl+O | Ouvrir |
| Ctrl+S | Enregistrer |
| F12 | Enregistrer sous |
| Ctrl+P | Imprimer |
| Ctrl+W | Fermer |

## Divers

| Raccourci | Action |
|-----------|--------|
| Ctrl+E | Remplissage instantane |
| Ctrl+T | Créer un tableau |
| Ctrl+Maj+L | Activer les filtres |
| Alt+= | Somme automatique |
| F11 | Créer un graphique |
| Ctrl+; | Date du jour |
