# Excel en PDF

Convertissez vos feuilles Excel en documents PDF professionnels.

## Méthode 1 : Enregistrer sous

### Étapes

1. **Fichier** > **Enregistrer sous**
2. Choisissez l'emplacement
3. **Type** : PDF
4. **Enregistrer**

### Raccourci

**F12** ouvre directement "Enregistrer sous".

## Méthode 2 : Exporter

### Étapes

1. **Fichier** > **Exporter**
2. **Créer un document PDF/XPS**
3. **Créer PDF/XPS**
4. Choisissez l'emplacement
5. **Publier**

## Options d'export

### Avant de publier

Cliquez sûr **Options** pour configurer :

| Option | Description |
|--------|-------------|
| Feuilles sélectionnées | Seules les feuilles actives |
| Classeur entier | Toutes les feuilles |
| Sélection | Seulement les cellules sélectionnées |
| Tableau | Un tableau spécifique |

## Mise en page avant export

### Vérifier l'apercu

**Fichier** > **Imprimer** pour voir le rendu.

### Ajuster la zone

1. **Mise en page** > **Zone d'impression** > **Définir**
2. Sélectionnez les cellules à exporter

### Orientation

1. **Mise en page** > **Orientation**
2. Portrait où Paysage selon le contenu

### Ajuster à la page

1. **Mise en page** > **Largeur** : 1 page
2. **Mise en page** > **Hauteur** : Automatique

## En-têtes et pieds de page

### Configuration

1. **Insertion** > **En-tête et pied de page**
2. Ajoutez titre, date, numéro de page

### Codes utiles

| Code | Résultat |
|------|----------|
| &[Page] | Numéro de page |
| &[Pages] | Nombre total de pages |
| &[Date] | Date du jour |
| &[Fichier] | Nom du fichier |

## Qualité du PDF

### Standard

Qualité equilibree, taille réduite.

### Minimale

Qualité réduite, fichier léger. Pour partage web.

## Exporter plusieurs feuilles

### Classeur entier

Dans les options, sélectionnez "Classeur entier".

### Feuilles spécifiques

1. Maintenez **Ctrl** et cliquez sûr les onglets
2. Exportez les "Feuilles sélectionnées"

## Problèmes courants

### Colonnes coupees

1. Reduisez les marges
2. Passez en paysage
3. Ajustez la mise à l'échelle

### Lignes sûr plusieurs pages

Définissez les lignes à répéter :
**Mise en page** > **Imprimer les titres**

### PDF trop lourd

Reduisez la qualité des images.

2. **Définissez** la zone d'impression
3. **Ajustez** les marges et l'orientation
4. **Testez** sûr plusieurs pages
