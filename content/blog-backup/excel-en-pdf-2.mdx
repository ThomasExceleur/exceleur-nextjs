# Convertir Excel en PDF

Exportez vos classeurs au format PDF.

## Méthode Enregistrer sous

### Étapes

1. **Fichier** > **Enregistrer sous**
2. Choisissez l'emplacement
3. **Type** : PDF
4. **Enregistrer**

## Méthode Exporter

### Étapes

1. **Fichier** > **Exporter**
2. **Créer un document PDF/XPS**
3. **Publier**

## Options d'export

### Ce qui est exporte

| Option | Description |
|--------|-------------|
| Classeur entier | Toutes les feuilles |
| Sélection | Zone sélectionnée |
| Feuilles actives | Feuille courante |

### Qualité

| Option | Usage |
|--------|-------|
| Standard | Publication en ligne |
| Minimum | Fichier plus léger |

## Mise en page avant export

### Vérifier l'apercu

**Fichier** > **Imprimer** (Apercu)

### Ajuster

- Marges
- Orientation
- Échelle
- Zone d'impression

## Plusieurs feuilles

### Sélectionner plusieurs onglets

Ctrl+Clic sûr les onglets souhaités.

## PDF protege

### Dans les options

Ajoutez un mot de passe si nécessaire.

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Apercu | Vérifiez avant export |
| Zone | Définissez la zone d'impression |
| Qualité | Adaptez au besoin |
