# Premiers Pas avec Python dans Excel

Microsoft à intégré Python directement dans Excel 365 pour l'analyse de données avancée.

## Disponibilité

### Actuellement

- Excel 365 (Insiders Program)
- Deploiement progressif

### Prerequis

- Abonnement Microsoft 365
- Version recente d'Excel

## Activer Python

### Dans une cellule

1. Tapez `=PY(` pour commencer
2. Ou utilisez **Formules** > **Insérer Python**

### Syntaxe

```excel
=PY("votre code Python")
```

## Exemple simple

### Dans Excel

```excel
=PY("import pandas as pd; df = xl('A1:C10'); df.describe()")
```

### Ce que çà fait

1. Importe pandas
2. Lit les données Excel (A1:C10)
3. Retourne les statistiques descriptives

## Fonction xl()

### Lire des données Excel

```python
xl("A1:D100")           # Plage
xl("Tableau1")          # Tableau nommé
xl("Feuil1!A:A")        # Colonne entière
```

### Retourne un DataFrame pandas

## Bibliotheques disponibles

### Incluses

- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

### Installation

Pas d'installation nécessaire - tout est dans le cloud Microsoft.

## Cas d'usage

### Analyse statistique

```python
import pandas as pd
df = xl("Ventes")
df.groupby("Region").sum()
```

### Visualisation

```python
import matplotlib.pyplot as plt
df = xl("A1:B100")
df.plot()
plt.show()
```

### Machine Learning

```python
from sklearn.linear_model import LinearRegression
# ... code de prediction
```

## Avantages

| Avantage | Description |
|----------|-------------|
| Intégration | Python directement dans Excel |
| Cloud | Calculs dans le cloud Microsoft |
| Bibliotheques | Pandas, NumPy, etc. inclus |
| Collaboration | Partage facile des classeurs |

## Limites

### Actuelles

- Calcul dans le cloud (nécessité internet)
- Temps de réponse variable
- Pas de pip install personnalise

## Comparaison avec VBA

| Critère | Python | VBA |
|---------|--------|-----|
| Analyse de données | Excellent | Limite |
| Automatisation | Bon | Excellent |
| Courbe d'apprentissage | Moyenne | Raide |
| Communaute | Très large | Excel-spécifique |
