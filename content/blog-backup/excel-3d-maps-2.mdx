# Excel 3D Maps

Visualisez vos données sûr des cartes 3D.

## Qu'est-ce que 3D Maps ?

### Définition

Outil de visualisation geographique intégré à Excel.

### Avantages

- Cartes interactives
- Visualisation temporelle
- Tours guides

## Acceder à 3D Maps

### Étapes

1. Sélectionnez vos données geographiques
2. **Insertion** > **3D Map** > **Ouvrir 3D Maps**

## Types de données geographiques

| Type | Exemple |
|------|---------|
| Pays | France, Allemagne |
| Villes | Paris, Lyon |
| Codes postaux | 75001, 69000 |
| Coordonnees | Latitude, Longitude |

## Types de visualisations

### Cartes empilees

Colonnes verticales sûr la carte.

### Carte thermique

Zones colorees selon l'intensite.

### Regions

Coloration par zone geographique.

### Points

Marqueurs individuels.

## Créer un tour guide

### Étapes

1. Ajoutez des scenes
2. Configurez les transitions
3. Ajoutez des annotations
4. Exportez en video

## Personnalisation

| Option | Description |
|--------|-------------|
| Hauteur | Dimension des colonnes |
| Couleur | Échelle de couleurs |
| Etiquettes | Texte sûr les éléments |
| Fond | Style de carte |

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Données propres | Nettoyez les localisations |
| Cohérence | Un seul type par carte |
| Simplicité | Évitez la surcharge |
