# Excel pour Android

Utilisez Excel sûr votre appareil Android gratuitement.

## Installation

### Google Play Store

1. Ouvrez le Play Store
2. Recherchez "Microsoft Excel"
3. Installez l'application
4. Connectez-vous avec un compte Microsoft

## Prix

### Gratuit sûr petits ecrans

Appareils de moins de 10.1 pouces : gratuit pour les fonctions de base.

### Tablettes plus grandes

Abonnement Microsoft 365 requis pour l'edition.

## Fonctionnalités gratuites

### Ce qui est inclus

- Lecture de tous les fichiers Excel
- Édition de base
- Formules courantes
- Graphiques simples
- Partage de fichiers

### Ce qui nécessité un abonnement

- Édition sûr grandes tablettes
- Suivi des modifications
- Styles personnalises
- Tableaux croisés dynamiques avancés

## Interface mobile

### Navigation

- Swipe pour naviguer entre les feuilles
- Pincez pour zoomer
- Double-tap pour editer une cellule

### Barre de formule

En haut de l'ecran, comme sûr PC.

### Ruban simplifie

Accès aux fonctions principales via icones.

## Synchronisation

### OneDrive

Les fichiers sûr OneDrive sont synchronises automatiquement.

### Travail hors ligne

1. Ouvrez le fichier en ligne
2. Il reste disponible hors connexion
3. Les modifications se synchronisent à la reconnexion

## Partage

### Envoyer un lien

1. Ouvrez le fichier
2. Bouton Partager
3. Choisissez le mode (lecture/edition)
4. Envoyez le lien

### Collaboration en temps reel

Plusieurs personnes peuvent editer simultanement.

## Scanner des tableaux

### Fonction photo

1. Nouveau fichier
2. Icone appareil photo
3. Photographiez un tableau papier
4. Excel convertit en cellules

## Limites de la version mobile

| Fonction | Disponible |
|----------|------------|
| Macros VBA | Non |
| Power Query | Non |
| Power Pivot | Non |
| Formules complexes | Partiellement |
| Graphiques avancés | Limite |

## Conseils d'utilisation

### Clavier externe

Connectez un clavier Bluetooth pour plus de confort.

### Mode tablette

Utilisez une tablette pour un ecran plus grand.

### Fichiers legers

Préférez les fichiers de moins de 5 Mo pour les performances.

## Alternatives

### Google Sheets

Application native Android, totalement gratuite.

### WPS Office

Alternative gratuite avec bonne compatibilité Excel.

## Bonnes pratiques

1. **Synchronisez** sûr OneDrive pour ne rien perdre
2. **Utilisez** le Wi-Fi pour les gros fichiers
3. **Fermez** les fichiers après utilisation
4. **Mettez à jour** l'application régulièrement
