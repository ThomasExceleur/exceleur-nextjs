# Additionner les Heures et les Minutes

Le calcul des heures dans Excel nécessité quelques connaissances spécifiques.

## Format d'heure

### Saisie correcte

Utilisez le format HH:MM :
- 8:30 (pas 8h30)
- 17:45 (pas 17.45)
- 0:45 (quarante-cinq minutes)

## Addition simple

### Formule SOMME

```excel
=SOMME(A1:A10)
```

### Exemple

| Duree |
|-------|
| 2:30 |
| 3:45 |
| 1:15 |
| **7:30** |

## Problème des 24 heures

### Le problème

Si le total dépasse 24 heures :
- 25:30 s'affiché 01:30 (incorrect)

### La solution

Format personnalise : `[h]:mm`

1. Sélectionnez la cellule du total
2. **Ctrl + 1**
3. **Personnalise**
4. Tapez `[h]:mm`

## Formats disponibles

| Format | Affichage max |
|--------|---------------|
| h:mm | 23:59 |
| [h]:mm | Illimite |
| [m]:ss | Minutes totales |
| [s] | Secondes totales |

## Convertir en decimal

### Heures decimales

```excel
=A1*24
```

8:30 devient 8.5

### Pour la paie

```excel
=HeuresDecimales*TauxHoraire
```

## Additionner heures et minutes séparées

### Si heures et minutes dans colonnes différentes

```excel
=TEMPS(A1;B1;0)
```

Puis additionnez les résultats.

### Exemple

| Heures | Minutes | Résultat |
|--------|---------|----------|
| 2 | 30 | =TEMPS(A2;B2;0) |
| 3 | 45 | =TEMPS(A3;B3;0) |

## Soustraire des heures

### Duree entre deux heures

```excel
=B1-A1
```

Ou B1 est l'heure de fin, A1 l'heure de début.

### Attention aux heures negatives

Si fin < début (travail de nuit) :
```excel
=SI(B1<A1;B1+1-A1;B1-A1)
```

## Fonction TEMPS

### Syntaxe

```excel
=TEMPS(heures;minutes;secondes)
```

### Créer une duree

```excel
=TEMPS(8;30;0)    ' 8 heures 30 minutes
```

## Extraire heures et minutes

### Extraire les heures

```excel
=HEURE(A1)
```

### Extraire les minutes

```excel
=MINUTE(A1)
```

### Total en minutes

```excel
=HEURE(A1)*60+MINUTE(A1)
```

## Cas pratique : Feuille de temps

| Jour | Début | Fin | Pause | Travail |
|------|-------|-----|-------|---------|
| Lun | 8:00 | 17:00 | 1:00 | =C2-B2-D2 |
| Mar | 9:00 | 18:00 | 1:00 | =C3-B3-D3 |

Format de la colonne Travail : `[h]:mm`

## Bonnes pratiques

1. **Formatez** en [h]:mm pour les totaux
2. **Saisissez** avec deux-points (8:30)
3. **Utilisez** TEMPS() pour les calculs
4. **Convertissez** en decimal pour la paie
