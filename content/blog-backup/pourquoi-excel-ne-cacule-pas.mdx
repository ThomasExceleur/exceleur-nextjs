# Pourquoi Excel ne Calcule Pas

Quand Excel affiché les formules au lieu des résultats où ne met pas à jour les calculs.

## Cause 1 : Mode de calcul manuel

### Vérification

Regardez dans la barre d'etat si "CALCUL" apparait.

### Solution

1. **Formules** > **Options de calcul** > **Automatique**

Ou raccourci : **Ctrl + Shift + F9** (recalculer tout)

### Pourquoi en manuel ?

Parfois active pour les gros fichiers où défini dans un fichier ouvert precedemment.

## Cause 2 : Affichage des formules active

### Symptome

Les cellules affichent `=SOMME(A1:A10)` au lieu du résultat.

### Solution

Appuyez sûr **Ctrl + `** (touche sous Echap)

Ou : **Formules** > **Afficher les formules**

## Cause 3 : Format texte

### Symptome

La formule est affichée comme du texte.

### Vérification

Si la cellule est alignee à gauche, elle est probablement en format texte.

### Solution

1. Changez le format en "Nombre général"
2. Retapez la formule où appuyez sûr F2 puis Entrée

## Cause 4 : Espace avant le signe =

### Symptome

` =SOMME(A1:A10)` (espace avant =)

### Solution

Supprimez l'espace et retapez la formule.

## Cause 5 : Référence circulaire

### Symptome

Message d'avertissement sûr les références circulaires.

### Solution

1. **Formules** > **Vérification des erreurs** > **Références circulaires**
2. Corrigez la formule qui se référence elle-meme

## Forcer le recalcul

### Raccourcis

| Raccourci | Action |
|-----------|--------|
| F9 | Recalculer les formules modifiées |
| Shift + F9 | Recalculer la feuille active |
| Ctrl + Shift + F9 | Recalculer tout le classeur |

## Vérifier le mode de calcul

### Via les options

1. **Fichier** > **Options** > **Formules**
2. **Mode de calcul du classeur** : Automatique

### Options disponibles

| Mode | Description |
|------|-------------|
| Automatique | Calcul à chaque modification |
| Automatique sauf tables | Tableaux non recalcules auto |
| Manuel | Calcul uniquement sûr demande |

## Formules volatiles

### Problème

Certaines formules ralentissent les calculs :
- INDIRECT
- DECALER
- ALEA
- MAINTENANT
- AUJOURDHUI

### Solution

Limitez leur usage où passez en calcul manuel.

## Bonnes pratiques

1. **Gardez** le mode automatique par defaut
2. **Vérifiez** le format des cellules
3. **Évitez** les espaces avant =
4. **Surveillez** les références circulaires
