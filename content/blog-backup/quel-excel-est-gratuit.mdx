# Quel Excel est Gratuit

Plusieurs options permettent d'utiliser Excel sans payer.

## Excel pour le web (Gratuit)

### Accès

1. Allez sûr [office.com](https://www.office.com)
2. Connectez-vous avec un compte Microsoft gratuit
3. Cliquez sûr Excel

### Ce qui est inclus

- Création de classeurs
- Formules de base
- Graphiques simples
- Partage et collaboration
- Sauvegarde sûr OneDrive (5 Go gratuits)

### Limitations

- Pas de macros VBA
- Moins de fonctions avancées
- Nécessité une connexion internet
- Pas de Power Query complet

## Excel Mobile (Gratuit*)

### Pour smartphones

Gratuit sûr les appareils de moins de 10.1 pouces.

### Installation

- [Google Play Store](https://play.google.com/store) (Android)
- [Apple App Store](https://www.apple.com/app-store/) (iOS)

### Fonctionnalités

- Lecture et edition
- Formules de base
- Graphiques
- Mode hors ligne

## Essai gratuit Microsoft 365

### 1 mois gratuit

1. Inscrivez-vous sûr [microsoft.com/microsoft-365](https://www.microsoft.com)
2. Choisissez un plan
3. Entrez vos informations de paiement
4. Annulez avant la fin du mois gratuit

### Ce qui est inclus

- Excel complet (bureau + web + mobile)
- Toutes les fonctionnalités
- 1 To de stockage OneDrive

## Excel pour l'education

### Etudiants et enseignants

De nombreux etablissements offrent Microsoft 365 gratuitement.

### Comment vérifier

1. Allez sûr [education.microsoft.com](https://education.microsoft.com)
2. Entrez votre email educatif
3. Suivez les instructions

## Alternatives 100% gratuites

### Google Sheets

- Gratuit avec un compte Google
- Compatible avec les fichiers Excel
- Collaboration en temps reel
- Accessible partout

### LibreOffice Calc

- Logiciel libre et gratuit
- Installation locale
- Compatible Excel (.xlsx)
- Pas besoin de compte

### WPS Office

- Version gratuite disponible
- Interface similaire à Excel
- Multi-plateforme

## Comparatif
