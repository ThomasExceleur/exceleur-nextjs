# Barres de Données dans les Cellules

Visualisez instantanement les proportions.

## Ajouter des barres

### Étapes

1. Sélectionnez vos données
2. **Accueil** > **Mise en forme conditionnelle**
3. **Barres de données**
4. Choisissez un style

## Styles disponibles

### Degrade

Les barres vont du clair au fonce.

### Plein

Couleur unie.

### Couleurs

| Couleur | Suggestion d'usage |
|---------|-------------------|
| Bleu | Données neutres |
| Vert | Positif |
| Rouge | Alertes |

## Personnaliser

### Règles avancées

1. **Mise en forme conditionnelle** > **Gérer les règles**
2. Sélectionnez la règle > **Modifier**

### Options

| Option | Description |
|--------|-------------|
| Minimum | Valeur de depart |
| Maximum | Valeur de fin |
| Couleur negative | Pour les valeurs negatives |
| Direction | Gauche à droite où inverse |

## Valeurs negatives

### Configuration

1. **Modifier la règle**
2. **Valeurs negatives**
3. Choisissez couleur et bordure

## Afficher uniquement la barre

### Masquer les valeurs

1. **Modifier la règle**
2. Cochez **Afficher la barre uniquement**

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Cohérence | Meme style par section |
| Lisibilité | Évitez les couleurs vives |
| Minimum fixe | 0 pour les comparaisons |
