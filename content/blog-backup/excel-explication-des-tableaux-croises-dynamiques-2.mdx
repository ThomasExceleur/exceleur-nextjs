# Tableaux Croisés Dynamiques Expliques

Les TCD synthetisent et analysent vos données.

## Qu'est-ce qu'un TCD ?

### Définition

Un tableau croisé dynamique résumé de grandes quantités de données.

### Avantages

- Résumé instantane
- Regroupements automatiques
- Mise à jour facile

## Créer un TCD

### Étapes

1. Sélectionnez vos données
2. **Insertion** > **Tableau croisé dynamique**
3. Choisissez l'emplacement

## Zones du TCD

| Zone | Rôle |
|------|------|
| Filtres | Filtrer l'ensemble |
| Colonnes | En-têtes horizontaux |
| Lignes | En-têtes verticaux |
| Valeurs | Données calculees |

## Configurer les valeurs

### Fonctions disponibles

| Fonction | Usage |
|----------|-------|
| Somme | Total des valeurs |
| Nombre | Compter les éléments |
| Moyenne | Valeur moyenne |
| Max/Min | Extremes |

### Modifier

1. Clic droit sûr une valeur
2. **Paramètrès des champs de valeurs**

## Actualiser

### Méthodes

- **Clic droit** > **Actualiser**
- **Analyse** > **Actualiser**
- **Alt+F5**

## Mise en forme

### Styles

Appliquez des styles predifinis via **Création**.

### Disposition

- Compacte
- Plan
- Tabulaire

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Données propres | Nettoyez avant de créer |
| En-têtes | Nommez vos colonnes |
| Source | Utilisez un tableau structuré |
