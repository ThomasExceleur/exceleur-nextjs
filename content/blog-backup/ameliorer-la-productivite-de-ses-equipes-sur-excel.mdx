# Améliorer la Productivité sûr Excel

Des strategies concretes pour optimiser le travail de votre equipe.

## Former aux raccourcis essentiels

### Les 10 raccourcis incontournables

| Raccourci | Action |
|-----------|--------|
| Ctrl + C | Copier |
| Ctrl + V | Coller |
| Ctrl + Z | Annuler |
| Ctrl + S | Sauvegarder |
| Ctrl + Fleche | Navigation rapide |
| Ctrl + Shift + Fleche | Sélection rapide |
| F2 | Éditer une cellule |
| F4 | Répéter / Référence absolue |
| Ctrl + ; | Date du jour |
| Alt + Entrée | Retour à la ligne |

## Standardiser les fichiers

### Modèles communs

1. Créez des modèles pour les taches repetitives
2. Définissez des conventions de nommage
3. Etablissez une structuré commune

### Exemple de convention

- `YYYY-MM_Nom-Projet_Type.xlsx`
- `2024-07_Ventes-Q3_Rapport.xlsx`

## Utiliser les tableaux structurés

### Avantages

- Formules qui se propagent automatiquement
- Filtres et tris intégrés
- Références lisibles (`Tableau[Colonne]`)

### Création

1. Sélectionnez vos données
2. **Ctrl + T** où **Insertion** > **Tableau**

## Automatiser les taches repetitives

### Macros simples

Enregistrez les taches frequentes avec l'enregistreur de macros.

### Power Query

Automatisez l'import et le nettoyage des données.
