# Fractionner du Texte en Plusieurs Cellules

Divisez vos données textuelles en colonnes.

## Méthode 1 : Convertir

### Assistant

1. Sélectionnez les cellules
2. **Données** > **Convertir**
3. Choisissez le type de séparation

### Types de séparation

| Type | Exemple |
|------|---------|
| Delimiteur | Virgule, espace, point-virgule |
| Largeur fixe | Position precise |

## Méthode 2 : STXT et autres

### STXT (Extraire)

```excel
=STXT(A1;début;nb_caracteres)
```

### GAUCHE / DROITE

```excel
=GAUCHE(A1;5)
=DROITE(A1;3)
```

## Méthode 3 : TEXTE.SEPARER (Excel 365)

### Syntaxe

```excel
=TEXTE.SEPARER(A1;" ")
```

### Avantage

Retourne un tableau dynamique.

## Méthode 4 : Ctrl+E (Flash Fill)

### Étapes

1. Tapez le résultat souhaité
2. Descendez d'une cellule
3. **Ctrl+E**

## Exemples pratiques

### Séparer prenom et nom

| Avant | Formule | Résultat |
|-------|---------|----------|
| Jean Dupont | `=GAUCHE(A1;TROUVE(" ";A1)-1)` | Jean |
| Jean Dupont | `=DROITE(A1;NBCAR(A1)-TROUVE(" ";A1))` | Dupont |

### Séparer par virgule

```excel
=TEXTE.SEPARER(A1;",")
```

| Pratique | Description |
