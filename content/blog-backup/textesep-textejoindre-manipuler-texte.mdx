# FRACTIONNER.TEXTE et JOINDRE.TEXTE

Fonctions modernes pour la manipulation de texte.

## FRACTIONNER.TEXTE (TEXTSPLIT)

### Syntaxe

```excel
=FRACTIONNER.TEXTE(texte;delimiteur_col;[delimiteur_ligne])
```

### Exemple simple

```excel
=FRACTIONNER.TEXTE(\"Paris;Lyon;Marseille\";\";\")"
```

Résultat : Paris | Lyon | Marseille (en colonnes)

### Avec lignes et colonnes

```excel
=FRACTIONNER.TEXTE(\"A1,B1;A2,B2\";\",\";\";\")
```

Créé une matrice 2x2.

## JOINDRE.TEXTE (TEXTJOIN)

### Syntaxe

```excel
=JOINDRE.TEXTE(delimiteur;ignorer_vides;texte1;...)
```

### Exemple

```excel
=JOINDRE.TEXTE(\", \";VRAI;A1:A10)
```

### Ignorer les vides

Le paramèêtre VRAI ignore les cellules vides.

## Cas pratiques

### Séparer nom et prenom

```excel
=FRACTIONNER.TEXTE(A1;\" \")
```

### Créer une liste

```excel
=JOINDRE.TEXTE(\" | \";VRAI;UNIQUE(A:A))
```

### Adresse email depuis éléments

```excel
=JOINDRE.TEXTE(\"\";VRAI;Prenom;\".\";Nom;\"@entreprise.com\")
```

## Combiner les deux

### Nettoyer et reformater

```excel
=JOINDRE.TEXTE(\" \";VRAI;FRACTIONNER.TEXTE(A1;\",\"))
```

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| VRAI pour ignorer vides | Evite les doublons de séparateurs |
| Caracteres speciaux | Echappez si nécessaire |
| Test | Vérifiez avec différentes données |
