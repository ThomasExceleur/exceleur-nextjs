# Les 10 Formules Excel à Connaîêêtre

Maîtrisez ces 10 formules pour devenir efficace sûr Excel.

## 1. SOMME

### Addition de valeurs

```excel
=SOMME(A1:A100)
```

Additionne toutes les valeurs d'une plage.

## 2. MOYENNE

### Calculer une moyenne

```excel
=MOYENNE(B1:B50)
```

Calcule la moyenne arithmetique.

## 3. SI

### Condition logique

```excel
=SI(A1>100;"Élève";"Faible")
```

Retourne une valeur selon une condition.

## 4. RECHERCHEV

### Rechercher une valeur

```excel
=RECHERCHEV(E1;A:D;3;FAUX)
```

Trouve une valeur dans un tableau.

## 5. NB.SI

### Compter avec condition

```excel
=NB.SI(A:A;"Paris")
```

Compte les cellules repondant à un critère.

## 6. SOMME.SI

### Additionner avec condition

```excel
=SOMME.SI(A:A;"Paris";B:B)
```

Additionne les valeurs selon un critère.

## 7. GAUCHE / DROITE

### Extraire du texte

```excel
=GAUCHE(A1;3)       ' 3 premiers caracteres
=DROITE(A1;4)       ' 4 derniers caracteres
```

## 8. CONCATENER / CONCAT

### Assembler du texte

```excel
=CONCAT(A1;" ";B1)
```

Combine plusieurs textes.

## 9. AUJOURDHUI / MAINTENANT

### Date et heure actuelles

```excel
=AUJOURDHUI()       ' Date du jour
=MAINTENANT()       ' Date et heure
```

## 10. ARRONDI

### Arrondir un nombre

```excel
=ARRONDI(A1;2)      ' 2 decimales
=ARRONDI(A1;0)      ' Nombre entier
```

## Formules bonus Excel 365

### RECHERCHEX

```excel
=RECHERCHEX(A1;B:B;C:C;"Non trouve")
```

Remplace RECHERCHEV avec plus de flexibilité.

### FILTRE

```excel
=FILTRE(A:D;B:B="Paris")
```

Filtre dynamique des données.

## Conseils

1. **Pratiquez** chaque formule
2. **Combinez** les formules
3. **Utilisez** les références absolues ($)
4. **Testez** sûr des exemples simples
