# Fractionner du Texte Excel

Separaez vos données en colonnes.

## Convertir (Classique)

### Étapes

1. Sélectionnez la colonne
2. **Données** > **Convertir**
3. Choisissez le delimiteur

## TEXTE.SEPARER (Excel 365)

```excel
=TEXTE.SEPARER(A1;" ")
```

## Ctrl+E (Flash Fill)

1. Tapez le résultat
2. Descendez
3. **Ctrl+E**

## Formules classiques

### GAUCHE

```excel
=GAUCHE(A1;5)
```

### DROITE

```excel
=DROITE(A1;3)
```

### STXT

```excel
=STXT(A1;2;4)
```

## Exemple prenom/nom

### Prenom

```excel
=GAUCHE(A1;TROUVE(" ";A1)-1)
```

### Nom

```excel
=DROITE(A1;NBCAR(A1)-TROUVE(" ";A1))
```

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Ctrl+E | Le plus rapide |
| TEXTE.SEPARER | Pour Excel 365 |
| Convertir | Pour anciennes versions |
