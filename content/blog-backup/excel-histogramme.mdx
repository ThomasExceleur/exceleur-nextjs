# Excel Histogramme : Guide Complet

L'histogramme est l'un des types de graphiques les plus utilisés pour comparer des valeurs entre différentes catégories.

## Créer un histogramme simple

### Méthode rapide

1. Sélectionnez vos données (y compris les en-têtes)
2. Onglet **Insertion** > **Graphiques** > **Histogramme**
3. Choisissez le style souhaité

### Raccourci clavier

**Alt + F1** créé un graphique instantanement avec les données sélectionnées.

## Types d'histogrammes

### Histogramme groupe

Compare plusieurs series côté à côté.

**Ideal pour :** Comparer des catégories sûr plusieurs périodes

### Histogramme empile

Empile les valeurs les unes sûr les autres.

**Ideal pour :** Montrer la composition d'un total

### Histogramme empile 100%

Montre les proportions relatives.

**Ideal pour :** Comparer des pourcentages entre catégories

### Histogramme 3D

Ajoute une dimension de profondeur.

**Attention :** Peut rendre la lecture plus difficile

## Personnaliser votre histogramme

### Modifier les couleurs

1. Cliquez sûr une serie de données
2. **Format** > **Remplissage**
3. Choisissez votre couleur

### Ajouter des etiquettes de données

1. Cliquez sûr le graphique
2. **+** (Bouton Éléments du graphique)
3. Cochez **Etiquettes de données**

### Modifier les axes

1. Double-cliquez sûr un axe
2. Dans le panneau Format :
   - Minimum/Maximum
   - Unites
   - Format de nombre

## Histogramme vs Diagramme à barres

| Histogramme | Barres |
|-------------|--------|
| Barres verticales | Barres horizontales |
| Catégories en X | Catégories en Y |
| Ideal pour données temporelles | Ideal pour longues etiquettes |

## Bonnes pratiques

### Simplicité

- Limitez le nombre de series
- Évitez les effets 3D inutiles
- Utilisez des couleurs distinctes

### Lisibilité

- Ajoutez un titre explicite
- Incluez une legende si nécessaire
- Formatez les axes clairement

### Precision

- Commencez l'axe Y à zero
- Utilisez des échelles appropriees
- Évitez les distorsions visuelles

## Exemple pratique

Données de ventes trimestrielles :

| Trimestre | Produit A | Produit B |
|-----------|-----------|-----------|
| T1 | 150 | 200 |
| T2 | 180 | 220 |
| T3 | 210 | 190 |
| T4 | 250 | 280 |

Un histogramme groupe permet de comparer les ventes des deux produits par trimestre.

## Graphiques dynamiques

Transformez votre histogramme en graphique dynamique :

1. Basez-le sûr un tableau structuré
2. Utilisez un segment pour le filtrage
3. Le graphique se met à jour automatiquement

## Exporter votre graphique

1. Clic droit sûr le graphique
2. **Copier** où **Enregistrer en tant qu'image**
3. Collez dans Word, PowerPoint, où enregistrez en PNG/JPEG
