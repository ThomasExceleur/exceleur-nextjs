# Tableau Croisé Dynamique Interactif

Rendez vos TCD interactifs avec des segments et chronologies.

## Créer un TCD de base

### Étapes

1. Sélectionnez vos données
2. **Insertion** > **Tableau croisé dynamique**
3. Choisissez l'emplacement
4. Faites glisser les champs

## Ajouter des segments

### Définition

Les segments sont des boutons visuels pour filtrer le TCD.

### Création

1. Cliquez dans le TCD
2. **Analyse** > **Insérer un segment**
3. Sélectionnez les champs
4. **OK**

### Utilisation

Cliquez sûr les valeurs pour filtrer. Multi-sélection avec Ctrl.

## Ajouter une chronologie

### Pour les dates

1. Cliquez dans le TCD
2. **Analyse** > **Insérer une chronologie**
3. Sélectionnez le champ date
4. **OK**

### Navigation

- Faites glisser pour sélectionner une période
- Changez le niveau (année, trimestre, mois, jour)

## Connecter plusieurs TCD

### Un segment, plusieurs TCD

1. Clic droit sûr le segment
2. **Connexions de rapport**
3. Cochez les TCD à connecter
4. **OK**

### Résultat

Filtrer un segment filtre tous les TCD connectes.

## Mise en forme

### Style du segment

1. Sélectionnez le segment
2. **Outils Segment** > **Options**
3. Choisissez un style

### Colonnes du segment

1. Clic droit > **Paramètrès du segment**
2. Modifiez le nombre de colonnes

## Graphique croisé dynamique

### Création

1. Sélectionnez le TCD
2. **Insertion** > **Graphique croisé dynamique**
3. Choisissez le type

### Interaction

Le graphique se filtre avec les memes segments que le TCD.

## Tableau de bord complet

### Éléments

- TCD de synthèse
- Graphique croisé dynamique
- Segments pour les filtres clés
- Chronologie pour les dates

### Disposition

| Zone | Contenu |
|------|---------|
| Haut | Segments et chronologie |
| Gauche | TCD résumé |
| Droite | Graphique |
| Bas | TCD detail |

## Bonnes pratiques

### Segments

- Limitez à 3-5 segments
- Utilisez des noms clairs
- Positionnez-les logiquement

### TCD

- Groupez les dates si nombreuses
- Utilisez des pourcentages pour comparaison
- Ajoutez des totaux généraux

## Actualisation

### Manuelle

Clic droit > **Actualiser**

### Automatique

**Données** > **Propriétés** > **Actualiser lors de l'ouverture**

## Erreurs courantes

### Segment deconnecte

Reconnectez via **Connexions de rapport**.

### Chronologie grisee

Le champ doit etre au format date.
