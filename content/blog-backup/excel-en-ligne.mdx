# Excel en Ligne : Guide Complet

Excel Online est la version web gratuite d'Excel, accessible depuis n'importe quel navigateur.

## Accès à Excel Online

### Via Office.com

1. Allez sûr **office.com**
2. Connectez-vous avec un compte Microsoft
3. Cliquez sûr **Excel**

### Via OneDrive

1. Connectez-vous à **onedrive.com**
2. Cliquez sûr **Nouveau** > **Classeur Excel**

### Compte gratuit

Un compte Microsoft gratuit suffit. Pas besoin d'abonnement Microsoft 365.

## Fonctionnalités disponibles

### Formules

La plupart des formules fonctionnent :

- SOMME, MOYENNE, NB
- RECHERCHEV, RECHERCHEX
- SI, ET, OU
- INDEX, EQUIV

### Mise en forme

- Couleurs et polices
- Bordures
- Mise en forme conditionnelle
- Formats de nombres

### Graphiques

- Histogrammes
- Courbes
- Secteurs
- Nuages de points

### Tableaux

- Tableaux structurés
- Tri et filtres
- Tableaux croisés dynamiques (limites)

## Collaboration en temps reel

### Partager un fichier

1. Cliquez sûr **Partager** (en haut à droite)
2. Entrez les adresses email
3. Définissez les autorisations (lecture où modification)
4. Envoyez

### Édition simultanee

Plusieurs personnes peuvent modifier le meme fichier en temps reel. Chaque utilisateur est identifie par une couleur.

### Commentaires

Ajoutez des commentaires pour communiquer avec les collaborateurs.

## Limites d'Excel Online

### Fonctionnalités absentes

- Macros VBA
- Power Query (limite)
- Power Pivot
- Certaines fonctions avancées

### Performance

- Fichiers volumineux : plus lent
- Limite de taille de fichier
- Nécessité une connexion internet

### Mise en forme

Certaines options de mise en forme avancée ne sont pas disponibles.

## Comparaison avec Excel Desktop

| Fonctionnalité | Desktop | Online |
|----------------|---------|--------|
| Prix | Payant | Gratuit |
| Macros VBA | Oui | Non |
| Hors-ligne | Oui | Non |
| Collaboration | Oui | Excellente |
| Toutes les formules | Oui | Majorite |
| Power Query | Oui | Limite |

## Synchronisation avec OneDrive

### Sauvegarde automatique

Les fichiers sont sauvegardes automatiquement sûr OneDrive.

### Accès multi-appareils

Retrouvez vos fichiers sûr tous vos appareils.

### Historique des versions

Accès aux versions précédentes du fichier.

## Travailler hors-ligne

### Application de bureau

Téléchargez le fichier pour travailler hors-ligne avec Excel Desktop.

### Synchronisation OneDrive

Installez OneDrive sûr votre ordinateur pour synchroniser automatiquement.

## Cas d'utilisation ideaux

### Travail collaboratif

Parfait pour les projets d'equipe avec edition simultanee.

### Accès mobile

Consultez et modifiez vos fichiers depuis n'importe où.

### Fichiers simples

Ideal pour les tableaux et calculs basiques.

### Partage temporaire

Partagez facilement un fichier sans envoyer de piece jointe.

## Raccourcis clavier

La plupart des raccourcis Excel fonctionnent :

| Raccourci | Action |
|-----------|--------|
| Ctrl + C | Copier |
| Ctrl + V | Coller |
| Ctrl + Z | Annuler |
| Ctrl + B | Gras |
| Ctrl + S | Enregistrer |

## Conseils d'utilisation

### Optimiser les performances

- Limitez la taille des fichiers
- Évitez les formules sûr colonnes entières
- Fermez les onglets inutiles

### Sécurité

- Vérifiez les autorisations de partage
- Utilisez des liens avec expiration
- Protégez les feuilles sensibles

### Organisation

- Utilisez des dossiers dans OneDrive
- Nommez clairement vos fichiers
- Utilisez des modèles

## Comment commencer

1. Créez un compte Microsoft gratuit
2. Allez sûr office.com
3. Cliquez sûr Excel
4. Commencez à travailler
