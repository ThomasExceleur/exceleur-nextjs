# Remplissage Automatique avec Ctrl+E

Ctrl+E (Flash Fill) est l'un des raccourcis les plus puissants d'Excel.

## Qu'est-ce que Ctrl+E ?

### Principe

Excel detecte un pattern à partir d'un exemple et l'applique à toute la colonne.

### Disponibilité

- Excel 2013 et versions ulterieures
- Excel pour Mac 2016+

## Exemples d'utilisation

### Extraire le prenom

| A (Complet) | B (Prenom) |
|-------------|------------|
| Jean Dupont | Jean |
| Marie Martin | (Ctrl+E) |
| Pierre Durand | |

1. Saisissez "Jean" en B1
2. Placez-vous en B2
3. Appuyez sûr **Ctrl+E**

### Extraire le nom

| A (Complet) | B (Nom) |
|-------------|---------|
| Jean Dupont | Dupont |
| Marie Martin | (Ctrl+E) |

### Combiner des données

| A (Prenom) | B (Nom) | C (Email) |
|------------|---------|-----------|
| Jean | Dupont | jean.dupont@mail.com |
| Marie | Martin | (Ctrl+E) |

### Extraire des numéros

| A (Texte) | B (Numéro) |
|-----------|------------|
| CMD-12345 | 12345 |
| CMD-67890 | (Ctrl+E) |

### Changer le format

| A (Date) | B (Format) |
|----------|------------|
| 15/01/2024 | 15 janvier 2024 |
| 20/03/2024 | (Ctrl+E) |

## Comment çà marche

### Étapes

1. Créez une colonne vide à côté de vos données
2. Saisissez le premier exemple du résultat souhaité
3. Passez à la cellule suivante
4. Appuyez sûr **Ctrl+E**

### Detection du pattern

Excel analyse :
- Les caracteres communs
- Les positions des éléments
- Les séparateurs

## Cas d'usage courants

### Nettoyage de données

- Supprimer des prefixes/suffixes
- Standardiser des formats
- Corriger des erreurs

### Extraction

- Prenom/Nom
- Code postal/Ville
- Numéro de telephone

### Transformation

- Changer la casse
- Reformater des dates
- Créer des codes

## Alternative : Menu

### Via le ruban

1. **Données** > **Remplissage instantane**
2. Ou **Accueil** > **Remplissage** > **Remplissage instantane**

## Limites

### Ne fonctionne pas toujours

- Patterns trop complexes
- Données incoherentes
- Trop peu d'exemples

### Solutions

- Donnez plusieurs exemples
- Vérifiez la cohérence des données
- Utilisez des formules pour les cas complexes

## Comparaison avec les formules

| Aspect | Ctrl+E | Formules |
|--------|--------|----------|
| Rapidite | Instantane | Plus lent |
| Flexibilité | Limitee | Élevée |
| Résultat | Valeurs | Formules |
| Mise à jour | Non | Automatique |

## Bonnes pratiques

1. **Vérifiez** toujours les résultats
2. **Donnez** des exemples clairs
3. **Utilisez** sûr des données cohérentes
4. **Completez** manuellement si nécessaire

## Alternative : Formules

Si Ctrl+E ne fonctionne pas :

```excel
=GAUCHE(A1;TROUVE(" ";A1)-1)          ' Extraire avant l'espace
=DROITE(A1;NBCAR(A1)-TROUVE(" ";A1))  ' Extraire après l'espace
```
