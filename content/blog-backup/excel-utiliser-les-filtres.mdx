# Utiliser les Filtres dans Excel

Les filtres permettent d'afficher uniquement les données qui vous interessent.

## Activer les filtres

### Raccourci

**Ctrl + Maj + L** : active/desactive les filtres

### Via le menu

1. Sélectionnez vos données
2. **Données** > **Filtrer**

### Résultat

Des fleches apparaissent dans les en-têtes de colonnes.

## Filtrer des données

### Filtre simple

1. Cliquez sûr la fleche d'une colonne
2. Decochez **(Sélectionner tout)**
3. Cochez les valeurs souhaitées
4. **OK**

### Filtrer plusieurs colonnes

Répétez l'opération sûr chaque colonne.

## Filtres de texte

### Options disponibles

| Filtre | Description |
|--------|-------------|
| Est égal à | Correspondance exacte |
| N'est pas égal à | Tout sauf |
| Commence par | Début du texte |
| Se termine par | Fin du texte |
| Contient | Presence dans le texte |
| Ne contient pas | Absence dans le texte |

### Utiliser les jokers

| Joker | Signification |
|-------|---------------|
| * | N'importe quels caracteres |
| ? | Un seul caractere |

## Filtres de nombres

### Options

| Filtre | Description |
|--------|-------------|
| Égal à | Exactement |
| Supérieur à | Plus grand |
| Inférieur à | Plus petit |
| Entre | Intervalle |
| 10 premiers | Top N |

### Exemple

Filtrer les ventes supérieures à 1000 :
1. Fleche > **Filtres numériques** > **Supérieur à**
2. Entrez 1000
3. **OK**

## Filtres de dates

### Options

| Filtre | Description |
|--------|-------------|
| Aujourd'hui | Date du jour |
| Cette semaine | Semaine en cours |
| Ce mois-ci | Mois en cours |
| Trimestre | Q1, Q2, Q3, Q4 |
| Année | Année spécifique |
| Avant/Après | Date limite |

## Filtres personnalises

### Créer un filtre complexe

1. Fleche > **Filtres [type]** > **Filtre personnalise**
2. Définissez les conditions
3. Utilisez ET/OU pour combiner

### Exemple

Valeurs entre 100 et 500 :
- Supérieur où égal à 100
- ET Inférieur où égal à 500

## Filtrer par couleur

### Si mise en forme conditionnelle

1. Cliquez sûr la fleche
2. **Filtrer par couleur**
3. Sélectionnez la couleur de cellule où de police

## Segments (Slicers)

### Pour un filtrage visuel

1. Convertissez en tableau (**Ctrl + T**)
2. **Insertion** > **Segment**
3. Sélectionnez les colonnes
4. Cliquez sûr les boutons pour filtrer

### Avantages

- Visuel et intuitif
- Multi-sélection facile
- Peut controler plusieurs tableaux

## Effacer les filtres

### Une colonne

1. Cliquez sûr la fleche
2. **Effacer le filtre de [colonne]**

### Tous les filtres

- **Données** > **Effacer**
- Ou cliquez sûr l'icone d'entonnoir

## Copier les données filtrees

### Méthode

1. Filtrez vos données
2. Sélectionnez les cellules visibles
3. **Ctrl + C**
4. Collez ailleurs

### Astuce

**Alt + ;** : sélectionné uniquement les cellules visibles

## Filtres avancés

### Pour des critères complexes

1. Créez une zone de critères
2. **Données** > **Avancé**
3. Définissez la plage de critères

### Zone de critères

| Colonne1 | Colonne2 |
|----------|----------|
| Critere1 | Critere2 |

## Bonnes pratiques

1. **Vérifiez** que les filtres sont actifs (icone d'entonnoir)
2. **Effacez** les filtres avant de créer des totaux
3. **Utilisez** les segments pour les analyses repetitives
4. **Sauvegardez** avant de supprimer des données filtrees
