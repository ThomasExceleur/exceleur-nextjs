# Tableau Croisé Dynamique Interactif

Rendez vos TCD interactifs avec segments et chronologies.

## Créer un TCD

### Étapes

1. Sélectionnez vos données
2. **Insertion** > **Tableau croisé dynamique**
3. Choisissez l'emplacement

## Ajouter des segments

### Insertion

1. Cliquez dans le TCD
2. **Analyse** > **Insérer un segment**
3. Sélectionnez les champs

### Personnalisation

| Option | Description |
|--------|-------------|
| Colonnes | Nombre d'éléments par ligne |
| Hauteur | Taille des boutons |
| Style | Theme visuel |

## Ajouter une chronologie

### Pour les dates

1. **Analyse** > **Insérer une chronologie**
2. Sélectionnez le champ date

### Niveaux

- Années
- Trimestres
- Mois
- Jours

## Connecter plusieurs TCD

### Connexions de rapport

1. Clic droit sûr segment
2. **Connexions de rapport**
3. Sélectionnez les TCD à connecter

## Mise en forme

### Styles de TCD

| Style | Usage |
|-------|-------|
| Clair | Lecture simple |
| Moyen | Professionnel |
| Fonce | Présentation |

### Segments assortis

Utilisez le meme theme pour cohérence.

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Disposition | Segments en haut où à gauche |
| Nommage | Renommez les segments |
| Taille | Ajustez pour lisibilité |
| Connexion | Liez tous les TCD relies |
