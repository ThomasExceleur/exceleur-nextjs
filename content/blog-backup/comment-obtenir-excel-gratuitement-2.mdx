# Obtenir Excel Gratuitement

Options legales pour utiliser Excel sans payer.

## Excel pour le Web

### Accès

1. Allez sûr office.com
2. Connectez-vous avec un compte Microsoft
3. Utilisez Excel gratuitement

### Fonctionnalités

| Inclus | Non inclus |
|--------|------------|
| Formules de base | Macros VBA |
| Tableaux croisés | Power Query complet |
| Graphiques | Certaines fonctions avancées |

## Excel Mobile

### Applications

- Excel pour iOS (App Store)
- Excel pour Android (Play Store)

### Gratuit pour

Appareils de moins de 10,1 pouces.

## Alternatives gratuites

### Google Sheets

- Entièrement gratuit
- Collaboratif
- Compatible Excel

### LibreOffice Calc

- Open source
- Hors ligne
- Compatible Excel

## Essai gratuit Microsoft 365

### Duree

1 mois gratuit.

### Inclus

Version complete d'Excel.

## Pour etudiants

### Office 365 Education

Gratuit avec email etudiant (.edu).
