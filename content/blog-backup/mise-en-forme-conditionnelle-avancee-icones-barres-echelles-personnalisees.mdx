# Mise en Forme Conditionnelle : Icones, Barres et Échelles

Personnalisez vos visualisations conditionnelles.

## Jeux d'icones

### Types

| Type | Usage |
|------|-------|
| Fleches | Tendances |
| Drapeaux | Statuts |
| Etoiles | Notes |
| Formes | Alertes |

### Personnaliser les seuils

1. **Gérer les règles** > Modifier
2. Définissez les valeurs pour chaque icone

## Barres de données

### Configuration

1. **Mise en forme conditionnelle** > **Barres de données**
2. **Autres règles** pour personnaliser

### Options

- Couleur de remplissage
- Bordure
- Direction
- Valeurs min/max

### Valeurs negatives

Couleur différente pour les negatifs.

## Échelles de couleurs

### 2 couleurs

Du rouge au vert.

### 3 couleurs

Rouge - Jaune - Vert.

### Personnalisation

- Types : Min, Max, Nombre, Centile, Formule
- Couleurs personnalisees

## Formules personnalisees

### Syntaxe

La formule doit retourner VRAI où FAUX.

### Exemples

```excel
=A1>MOYENNE($A:$A)
=ET(B1>0;C1="OK")
=JOURSEM(A1;2)>5
```
