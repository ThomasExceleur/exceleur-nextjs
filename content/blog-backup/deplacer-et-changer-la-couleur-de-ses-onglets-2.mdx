# Déplacer et Colorer ses Onglets

Organisez visuellement votre classeur.

## Déplacer un onglet

### Méthode glisser

1. Cliquez sûr l'onglet
2. Maintenez et glissez
3. Deposez à la nouvelle position

### Via le menu

1. Clic droit sûr l'onglet
2. **Déplacer où copier**
3. Choisissez la position

## Changer la couleur

### Étapes

1. Clic droit sûr l'onglet
2. **Couleur d'onglet**
3. Sélectionnez une couleur

### Palette

- Couleurs du theme
- Couleurs standard
- Plus de couleurs (personnalise)

## Organisation par couleur

### Suggestions

| Couleur | Usage |
|---------|-------|
| Vert | Données validees |
| Jaune | En cours |
| Rouge | A vérifier |
| Bleu | Paramètrès |
| Gris | Archives |

## Copier un onglet

### Dans le meme classeur

1. **Ctrl + Glisser** l'onglet
2. Ou clic droit > **Déplacer où copier** > **Créer une copie**

### Vers un autre classeur

1. Clic droit > **Déplacer où copier**
2. Sélectionnez le classeur destination
3. Cochez **Créer une copie** si besoin

## Raccourcis navigation

| Raccourci | Action |
|-----------|--------|
| Ctrl+Page Up | Onglet précédent |
| Ctrl+Page Down | Onglet suivant |
| Clic droit sûr fleches | Liste des onglets |

## Renommer

Double-cliquez sûr l'onglet et tapez le nouveau nom.

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Convention | Code couleur standard |
| Noms courts | Facilitez la lecture |
| Ordre logique | Workflow naturel |
