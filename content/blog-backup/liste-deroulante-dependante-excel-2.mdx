# Liste Deroulante Dependante : Méthodes Avancées

Allez plus loin avec les listes deroulantes en cascade.

## Méthode FILTRE (Excel 365)

### Avantages

- Pas de plages nommées multiples
- Plus flexible
- Mise à jour dynamique

### Structuré des données

| Niveau1 | Niveau2 | Niveau3 |
|---------|---------|---------|
| Europe | France | Paris |
| Europe | France | Lyon |
| Europe | Allemagne | Berlin |
| Asie | Japon | Tokyo |

### Liste niveau 1

Source : `=UNIQUE(A2:A100)`

### Liste niveau 2

Source : `=UNIQUE(FILTRE(B2:B100;A2:A100=D1))`

D1 contient la sélection niveau 1.

### Liste niveau 3

Source : `=UNIQUE(FILTRE(C2:C100;B2:B100=D2))`

## Méthode INDIRECT amelioree

### Gestion des espaces et caracteres speciaux

```excel
=INDIRECT(SUBSTITUE(SUBSTITUE(A1;" ";"_");"-";"_"))
```

### Nommer les plages

Remplacez espaces et tirets par underscores.

## Listes avec recherche dynamique

### Avec RECHERCHEX

```excel
=UNIQUE(FILTRE(Villes;RECHERCHEX(Region;Table[Region];Table[Pays])=A1))
```

## Validation avec message personnalise

### Configuration

1. **Données** > **Validation**
2. Onglet **Message de saisie**
3. Personnalisez titre et message
4. Onglet **Alerte d'erreur**
5. Personnalisez l'alerte

## Reinitialisation automatique

### Avec VBA

```vba
Private Sub Worksheet_Change(ByVal Target As Range)
    ' Si la cellule niveau 1 change
    If Not Intersect(Target, Range("A1")) Is Nothing Then
        Range("B1").ClearContents
        Range("C1").ClearContents
    ' Si la cellule niveau 2 change
    ElseIf Not Intersect(Target, Range("B1")) Is Nothing Then
        Range("C1").ClearContents
    End If
End Sub
```

## Listes avec filtrage textuel

### Contient un mot

```excel
=UNIQUE(FILTRE(Données;ESTNUM(CHERCHE(CriterePartiel;Données))))
```
