# Trouver les Valeurs Uniques dans 2 Listes

Comparez deux listes pour identifier les éléments presents dans une seule.

## Méthode 1 : FILTRE + ESTNA (Excel 365)

### Valeurs de Liste1 absentes de Liste2

```excel
=FILTRE(A:A;ESTNA(EQUIV(A:A;B:B;0)))
```

### Valeurs de Liste2 absentes de Liste1

```excel
=FILTRE(B:B;ESTNA(EQUIV(B:B;A:A;0)))
```

## Méthode 2 : Mise en forme conditionnelle

### Étapes

1. Sélectionnez la Liste1
2. **Accueil** > **Mise en forme conditionnelle** > **Nouvelle règle**
3. Choisissez "Utiliser une formule"
4. Formule : `=ESTNA(EQUIV(A1;$B:$B;0))`
5. Définissez un format (couleur de fond)

## Méthode 3 : NB.SI pour identifier

### Formule auxiliaire

```excel
=SI(NB.SI($B:$B;A1)=0;"Unique";"Present")
```

Affichez dans une colonne helper, puis filtrez sûr "Unique".

## Méthode 4 : RECHERCHEV

### Identifier les absents

```excel
=SIERREUR(RECHERCHEV(A1;$B:$B;1;FAUX);"Non trouve")
```

## Excel 365 : Solution elegante

### Avec FILTRE et EQUIV

```excel
=LET(
  liste1;A2:A100;
  liste2;B2:B100;
  absents;FILTRE(liste1;ESTNA(EQUIV(liste1;liste2;0)));
  absents
)
```

## Comparaison bidirectionnelle

### Éléments exclusifs à chaque liste

```excel
' Uniquement dans Liste1
=FILTRE(A:A;ESTNA(EQUIV(A:A;B:B;0)))

' Uniquement dans Liste2
=FILTRE(B:B;ESTNA(EQUIV(B:B;A:A;0)))

' Toutes les différences
=ASSEMB.V(
  FILTRE(A:A;ESTNA(EQUIV(A:A;B:B;0)));
  FILTRE(B:B;ESTNA(EQUIV(B:B;A:A;0)))
)
```

## Bonnes pratiques

1. **Nettoyez** les données avant comparaison
2. **Supprimez** les espaces superflus
3. **Uniformisez** la casse (majuscules/minuscules)
4. **Vérifiez** les doublons dans chaque liste
