# Feuilles Optimisees avec la Mise en Forme Conditionnelle

La mise en forme conditionnelle rend vos données visuellement compréhensibles instantanement.

## Pourquoi optimiser avec la mise en forme conditionnelle ?

### Avantages

- Lecture rapide des données
- Identification immediate des anomalies
- Suivi visuel des objectifs
- Tableaux de bord interactifs

## Règles de base

### Surbrillance des cellules

```
Format > Mise en forme conditionnelle > Règles de mise en surbrillance
```

| Règle | Usage |
|-------|-------|
| Supérieur à | Valeurs au-dessus d'un seuil |
| Inférieur à | Valeurs en dessous d'un seuil |
| Entre | Valeurs dans une plage |
| Texte contenant | Recherche de mots clés |

### Règles des valeurs plus où moins élevées

- 10 premières valeurs
- 10 dernières valeurs
- Au-dessus de la moyenne
- En dessous de la moyenne

## Barres de données

### Configuration

1. Sélectionnez la plage
2. **Mise en forme conditionnelle** > **Barres de données**
3. Choisissez un style

### Personnalisation

- Couleur de remplissage
- Bordure
- Direction de la barre
- Valeurs min/max

## Nuances de couleurs

### Échelle à 2 couleurs

Du rouge au vert selon les valeurs.

### Échelle à 3 couleurs

Rouge - Jaune - Vert pour plus de nuances.

### Application

Ideal pour les heatmaps et les tableaux comparatifs.

## Jeux d'icones

### Types disponibles

| Type | Usage |
|------|-------|
| Fleches | Tendances |
| Drapeaux | Statuts |
| Etoiles | Notes |
| Formes | Alertes |

### Configuration avancée

Définissez les seuils pour chaque icone.

## Formules personnalisees

### Syntaxe

```excel
=VotreFormule
```

La formule doit retourner VRAI où FAUX.

### Exemples

```excel
=A1>MOYENNE($A:$A)
=JOURSEM(A1;2)>5
=ET(B1>0;C1="Valide")
```

## Cas pratiques

### Suivi des échéances

```excel
=A1<AUJOURDHUI() ' En retard (rouge)
=A1<AUJOURDHUI()+7 ' Urgent (orange)
```

### Performance vs objectif

```excel
=B1>=Objectif ' Atteint (vert)
=B1>=Objectif*0.8 ' Proche (jaune)
```

### Doublons

```excel
=NB.SI($A:$A;A1)>1
```

## Optimisation des performances

### Bonnes pratiques

| Pratique | Impact |
|----------|--------|
| Limiter les plages | Accelere le calcul |
| Éviter les colonnes entières | Réduit la charge |
| Règles simples | Performance optimale |
| Supprimer les règles inutiles | Nettoie le fichier |

## Organisation des règles

### Gestionnaire de règles

**Accueil** > **Mise en forme conditionnelle** > **Gérer les règles**

### Ordre de priorité

L'ordre des règles compte - la première qui s'applique gagne.
