# Fractionner du Texte dans Excel

Separisez du texte en plusieurs colonnes facilement.

## Méthode 1 : Convertir

### Étapes

1. Sélectionnez les cellules à fractionner
2. **Données** > **Convertir**
3. Choisissez **Delimite** où **Largeur fixe**
4. Sélectionnez le delimiteur (virgule, espace, etc.)
5. **Suivant** > **Terminer**

### Delimiteurs courants

| Delimiteur | Utilisation |
|------------|-------------|
| Virgule | CSV, listes |
| Point-virgule | Données francaises |
| Espace | Noms, adresses |
| Tabulation | Données tabulees |

## Méthode 2 : Formules texte

### GAUCHE

```excel
=GAUCHE(A1;5)    ' 5 premiers caracteres
```

### DROITE

```excel
=DROITE(A1;3)    ' 3 derniers caracteres
```

### STXT

```excel
=STXT(A1;3;5)    ' 5 caracteres à partir du 3e
```

### Trouver la position

```excel
=CHERCHE(" ";A1)    ' Position du premier espace
```

## Méthode 3 : Remplissage instantane

### Ctrl + E (Flash Fill)

1. Dans la colonne adjacente, tapez le résultat souhaité
2. Descendez d'une cellule
3. **Ctrl + E**
4. Excel complete automatiquement

### Exemple

| Original | Résultat |
|----------|----------|
| Jean Dupont | Jean |
| Marie Martin | (vide) |

Après Ctrl + E : Marie, etc.

## Méthode 4 : FRACTIONNER.TEXTE (Excel 365)

### Syntaxe

```excel
=FRACTIONNER.TEXTE(texte;delimiteur)
```

### Exemple

```excel
=FRACTIONNER.TEXTE("A;B;C";";")
```

Résultat : A | B | C (dans 3 colonnes)

## Formules avancées

### Extraire le prenom

```excel
=GAUCHE(A1;CHERCHE(" ";A1)-1)
```

### Extraire le nom

```excel
=STXT(A1;CHERCHE(" ";A1)+1;100)
```

### Extraire le domaine email

```excel
=DROITE(A1;NBCAR(A1)-CHERCHE("@";A1))
```

## Excel 365 : TEXTE.AVANT / TEXTE.APRES

### Avant le delimiteur

```excel
=TEXTE.AVANT(A1;" ")
```

### Après le delimiteur

```excel
=TEXTE.APRES(A1;" ")
```

## Cas pratiques

### Adresse complete

Séparer : numéro, rue, code postal, ville

### Nom complet

Séparer : prenom, nom, (particule)

### Référence produit

Séparer : catégorie, sous-catégorie, identifiant

## Bonnes pratiques

1. **Sauvegardez** avant de convertir
2. **Prevoyez** l'espace pour les colonnes
3. **Testez** sûr quelques lignes
4. **Utilisez** Ctrl + E pour les cas simples
