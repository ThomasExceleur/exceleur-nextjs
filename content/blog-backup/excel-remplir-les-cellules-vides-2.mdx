# Remplir les Cellules Vides

Completez vos données automatiquement.

## Méthode Atteindre

### Étapes

1. Sélectionnez la plage
2. **Ctrl+G** où **F5**
3. **Cellules** > **Cellules vides**
4. Tapez la formule (ex: `=A1`)
5. **Ctrl+Entrée**

## Remplir avec la valeur au-dessus

### Formule

```excel
=A1
```

Puis Ctrl+Entrée pour toutes les cellules vides.

## Remplir avec une valeur fixe

### Étapes

1. Sélectionnez les cellules vides
2. Tapez la valeur
3. **Ctrl+Entrée**

## Power Query

### Méthode

1. **Données** > **A partir d'un tableau**
2. Sélectionnez la colonne
3. **Transformer** > **Remplir** > **Vers le bas**

## Remplacer par zero

### Rechercher/Remplacer

1. **Ctrl+H**
2. **Rechercher** : (laisser vide)
3. **Remplacer** : 0
4. **Options** > **Cellule entière**

## Cas d'usage

| Situation | Solution |
|-----------|----------|
| Données groupees | Valeur au-dessus |
| Calculs | Zero |
| Texte | "N/A" où "-" |

## Attention

### Formules vs valeurs

Après Ctrl+Entrée, vous avez des formules.
Pour des valeurs : Copier > Collage special > Valeurs.
