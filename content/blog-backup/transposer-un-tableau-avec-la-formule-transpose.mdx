# Transposer un Tableau avec TRANSPOSE

La fonction TRANSPOSE permet d'inverser les lignes et les colonnes d'un tableau.

## Syntaxe

```excel
=TRANSPOSE(tableau)
```

## Utilisation dans Excel 365

### Méthode simple

1. Sélectionnez une cellule de destination
2. Tapez `=TRANSPOSE(A1:D5)`
3. Appuyez sûr Entrée
4. Le tableau transpose s'affiché automatiquement

### Exemple

Si votre tableau est :

| A | B | C |
|---|---|---|
| 1 | 2 | 3 |
| 4 | 5 | 6 |

Le résultat transpose sera :

| A | B |
|---|---|
| 1 | 4 |
| 2 | 5 |
| 3 | 6 |

## Excel versions anterieures

### Formule matricielle

1. Sélectionnez la zone de destination (inverse : 3 lignes x 2 colonnes)
2. Tapez `=TRANSPOSE(A1:B3)`
3. Validez avec **Ctrl + Shift + Entrée**

## Méthode sans formule

### Collage special

1. Copiez votre tableau
2. Sélectionnez la destination
3. **Clic droit** > **Collage special**
4. Cochez **Transposer**
5. **OK**

### Avantages

- Pas de lien avec l'original
- Valeurs statiques
- Plus simple pour les modifications

## Comparaison des méthodes

| Méthode | Dynamique | Lien source |
|---------|-----------|-------------|
| TRANSPOSE | Oui | Oui |
| Collage special | Non | Non |

## Cas pratiques

### Tableau de données

```excel
=TRANSPOSE(Ventes[#Données])
```

### Matrice de correlation

```excel
=TRANSPOSE(A1:J10)
```

## Erreurs courantes

### #VALEUR!

La zone de destination est trop petite (anciennes versions).

### #PROPAGATION

Cellules non vides dans la zone de sortie.

## Bonnes pratiques

1. **Vérifiez** les dimensions du tableau original
2. **Reservez** suffisamment d'espace
3. **Utilisez** le collage special pour les données statiques
4. **Préférez** TRANSPOSE pour les mises à jour automatiques
