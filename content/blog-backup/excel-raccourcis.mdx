## Pourquoi utiliser les raccourcis Excel ?

Les raccourcis clavier peuvent vous faire gagner des heures chaque semaine. Au lieu de naviguer dans les menus, un simple appui sûr quelques touches execute l'action souhaitée.

## Raccourcis de navigation

### Se déplacer rapidement

| Raccourci | Action |
|-----------|--------|
| Ctrl + Home | Aller à la cellule A1 |
| Ctrl + End | Aller à la dernière cellule utilisée |
| Ctrl + Fleche | Aller au bord de la zone de données |
| Ctrl + Page Up/Down | Changer d'onglet |

### Sélectionner des plages

| Raccourci | Action |
|-----------|--------|
| Ctrl + Shift + End | Sélectionner jusqu'à la fin |
| Ctrl + Shift + Home | Sélectionner jusqu'au début |
| Ctrl + A | Sélectionner tout le tableau |
| Ctrl + Espace | Sélectionner la colonne |
| Shift + Espace | Sélectionner la ligne |

## Raccourcis d'edition

### Copier/Coller

| Raccourci | Action |
|-----------|--------|
| Ctrl + C | Copier |
| Ctrl + X | Couper |
| Ctrl + V | Coller |
| Ctrl + Alt + V | Collage special |
| Ctrl + D | Remplir vers le bas |
| Ctrl + R | Remplir vers la droite |

### Modifier les cellules

| Raccourci | Action |
|-----------|--------|
| F2 | Modifier la cellule |
| Echap | Annuler la modification |
| Entrée | Valider |
| Tab | Valider et aller à droite |
| Ctrl + Z | Annuler |
| Ctrl + Y | Retablir |

<Callout type="success" title="Astuce pro">
Le raccourci F4 est magique : il répété la dernière action effectuee. Très utile pour appliquer le meme formatage à plusieurs cellules !
</Callout>

## Raccourcis pour les formules

| Raccourci | Action |
|-----------|--------|
| F4 | Figer/defiger une référence |
| Ctrl + ` | Afficher les formules |
| Tab | Valider une suggestion |
| F9 | Calculer la sélection |
| Alt + = | Somme automatique |

### Exemple avec F4

Quand vous tapez une référence comme `A1`, appuyez sûr F4 pour la transformer :

1. Premier appui : `$A$1` (référence absolue)
2. Deuxieme appui : `A$1` (colonne relative, ligne absolue)
3. Troisieme appui : `$A1` (colonne absolue, ligne relative)
4. Quatrieme appui : `A1` (référence relative)
