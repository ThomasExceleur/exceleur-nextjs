# Quelle Version d'Excel Utilisez-Vous ?

Connaîêêtre sa version d'Excel est important pour utiliser les bonnes fonctionnalités.

## Méthode rapide

### Via le menu Fichier

1. Ouvrez Excel
2. Cliquez sûr **Fichier**
3. Cliquez sûr **Compte**
4. Lisez "A propos d'Excel"

## Versions disponibles

### Versions actuelles

| Version | Année | Type |
|---------|-------|------|
| Microsoft 365 | Continue | Abonnement |
| Excel 2021 | 2021 | Achat unique |
| Excel 2019 | 2018 | Achat unique |
| Excel 2016 | 2015 | Achat unique |

### Versions anciennes

| Version | Année |
|---------|-------|
| Excel 2013 | 2013 |
| Excel 2010 | 2010 |
| Excel 2007 | 2007 |

## Différences entre versions

### Excel 365 vs versions perpetuelles

| Fonctionnalité | 365 | 2021 | 2019 |
|----------------|-----|------|------|
| Mises à jour | Continues | Limitees | Limitees |
| RECHERCHEX | Oui | Oui | Non |
| LET, LAMBDA | Oui | Oui | Non |
| Copilot | Oui | Non | Non |

### Nouvelles fonctions par version

**Excel 365 / 2021** :
- RECHERCHEX
- FILTRE
- TRIER
- UNIQUE
- LET
- LAMBDA

**Excel 2019** :
- CONCAT
- JOINDRE.TEXTE
- SI.CONDITIONS
- SI.MULTIPLE

## Comment mettre à jour

### Microsoft 365

Les mises à jour sont automatiques.

### Versions perpetuelles

Seules les mises à jour de sécurité sont disponibles.

## Quelle version choisir ?

### Pour les particuliers

- **Microsoft 365 Personnel** : mises à jour continues
- **Office 2021** : achat unique sans abonnement

### Pour les entreprises

- **Microsoft 365 Business** : collaboration cloud
- **Office LTSC** : environnements deconnectes

## Compatibilité des fichiers

### Formats

| Extension | Compatible avec |
|-----------|-----------------|
| .xlsx | Excel 2007+ |
| .xlsm | Excel 2007+ (macros) |
| .xls | Toutes versions |

### Attention

Les nouvelles fonctions ne fonctionnent pas dans les anciennes versions.
