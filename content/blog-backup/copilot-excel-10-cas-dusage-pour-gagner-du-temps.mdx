# Copilot Excel : 10 Cas d'Usage

Copilot intégré l'IA dans Excel pour automatiser vos taches.

## Disponibilité

- Microsoft 365 avec licence Copilot
- Excel pour Windows et Mac
- Excel pour le web

## Cas 1 : Analyser les données

### Demandez

"Quelles sont les principales tendances dans ces données ?"

### Copilot fait

- Identifie les patterns
- Créé des graphiques
- Résumé les insights

## Cas 2 : Créer des formules

### Demandez

"Ajoute une formule pour calculer le taux de croissance"

### Copilot fait

- Écrit la formule appropriee
- L'applique à la colonne

## Cas 3 : Trier et filtrer

### Demandez

"Trie par ventes decroissantes et montre le top 10"

### Copilot fait

- Applique le tri
- Filtre les résultats

## Cas 4 : Créer des graphiques

### Demandez

"Créé un graphique montrant l'evolution des ventes par mois"

### Copilot fait

- Choisit le type de graphique adapte
- Configure les axes
- Applique une mise en forme

## Cas 5 : Mettre en forme

### Demandez

"Mets en evidence les valeurs supérieures à 1000 en vert"

### Copilot fait

- Applique la mise en forme conditionnelle

## Cas 6 : Nettoyer les données

### Demandez

"Supprime les doublons et les lignes vides"

### Copilot fait

- Identifie et supprime les doublons
- Nettoie les espaces

## Cas 7 : Créer un tableau croisé

### Demandez

"Créé un TCD montrant les ventes par region et produit"

### Copilot fait

- Créé le TCD
- Configure les champs
- Ajoute les calculs

## Cas 8 : Predire des valeurs

### Demandez

"Prevois les ventes pour les 3 prochains mois"

### Copilot fait

- Analyse les tendances
- Créé une prevision
- Affiche les résultats

## Cas 9 : Expliquer les données

### Demandez

"Pourquoi les ventes ont baisse en mars ?"

### Copilot fait

- Analyse les facteurs
- Propose des explications

## Cas 10 : Documenter

### Demandez

"Explique ce que fait cette formule"

### Copilot fait

- Decompose la formule
- Explique chaque partie

## Comment utiliser Copilot

### Accès

1. Cliquez sûr l'icone Copilot dans le ruban
2. Tapez votre demande en langage naturel
3. Copilot propose des actions

### Conseils

- Soyez spécifique dans vos demandes
- Vérifiez les résultats
- Affinez si nécessaire

## Limites

| Limite | Description |
|--------|-------------|
| Licence | Abonnement Copilot requis |
| Données sensibles | Attention à la confidentialite |
| Precision | Vérifiez toujours les résultats |

## Bonnes pratiques

1. **Commencez** par des demandes simples
2. **Vérifiez** les formules générées
3. **Apprenez** des suggestions de Copilot
4. **Combinez** avec vos connaissances Excel
