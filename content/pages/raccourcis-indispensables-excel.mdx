# Les Raccourcis Indispensables d'Excel

Maitriser les raccourcis clavier peut vous faire gagner des heures chaque semaine. Voici les plus essentiels.

## Navigation

| Raccourci | Action |
|-----------|--------|
| **Ctrl + Home** | Aller a la cellule A1 |
| **Ctrl + End** | Aller a la derniere cellule utilisee |
| **Ctrl + Fleche** | Aller au bord de la zone de donnees |
| **Ctrl + Page Up/Down** | Changer d'onglet |
| **F5** | Ouvrir la boite de dialogue "Atteindre" |

## Selection

| Raccourci | Action |
|-----------|--------|
| **Ctrl + Shift + End** | Selectionner jusqu'a la fin |
| **Ctrl + Shift + Home** | Selectionner jusqu'au debut |
| **Ctrl + A** | Selectionner tout le tableau |
| **Ctrl + Espace** | Selectionner la colonne entiere |
| **Shift + Espace** | Selectionner la ligne entiere |
| **Ctrl + Shift + L** | Activer/desactiver les filtres |

## Edition

| Raccourci | Action |
|-----------|--------|
| **Ctrl + C** | Copier |
| **Ctrl + X** | Couper |
| **Ctrl + V** | Coller |
| **Ctrl + Alt + V** | Collage special |
| **Ctrl + D** | Remplir vers le bas |
| **Ctrl + R** | Remplir vers la droite |
| **F2** | Modifier la cellule |
| **Ctrl + Z** | Annuler |
| **Ctrl + Y** | Retablir |

## Mise en forme

| Raccourci | Action |
|-----------|--------|
| **Ctrl + B** | Gras |
| **Ctrl + I** | Italique |
| **Ctrl + U** | Souligne |
| **Ctrl + 1** | Ouvrir Format de cellule |
| **Ctrl + Shift + %** | Format pourcentage |
| **Ctrl + Shift + $** | Format monnaie |

## Formules

| Raccourci | Action |
|-----------|--------|
| **F4** | Figer/defiger une reference |
| **Ctrl + `** | Afficher les formules |
| **Tab** | Valider une suggestion |
| **F9** | Calculer la selection |
| **Alt + =** | Somme automatique |
| **Ctrl + Shift + Entree** | Formule matricielle |

## Le raccourci magique : F4

Appuyez sur F4 apres avoir tape une reference pour la transformer :
1. `A1` → `$A$1` (absolue)
2. `$A$1` → `A$1` (colonne relative)
3. `A$1` → `$A1` (ligne relative)
4. `$A1` → `A1` (relative)

## Productivite

| Raccourci | Action |
|-----------|--------|
| **Ctrl + T** | Creer un tableau |
| **Alt + F1** | Creer un graphique |
| **Ctrl + ;** | Inserer la date du jour |
| **Ctrl + Shift + ;** | Inserer l'heure |
| **Ctrl + K** | Inserer un lien hypertexte |

## Astuces

- **F4** repete aussi la derniere action effectuee
- **Alt** affiche les raccourcis du ruban
- **Ctrl + Tab** bascule entre les classeurs ouverts
