# C'est Quoi Excel

Introduction au tableur de Microsoft.

## Definition

### Microsoft Excel

Logiciel de tableur permettant de :

- Organiser des donnees
- Effectuer des calculs
- Creer des graphiques
- Analyser des informations

## Interface

### Elements principaux

| Element | Description |
|---------|-------------|
| Cellules | Cases de la grille |
| Lignes | Numerotees 1, 2, 3... |
| Colonnes | Lettrees A, B, C... |
| Feuilles | Onglets en bas |

## A quoi ca sert ?

### Calculs

- Additions, moyennes
- Formules complexes
- Statistiques

### Organisation

- Listes et tableaux
- Bases de donnees simples
- Inventaires

### Visualisation

- Graphiques
- Tableaux de bord
- Rapports

## Qui utilise Excel ?

| Metier | Usage |
|--------|-------|
| Comptables | Calculs financiers |
| RH | Suivi des employes |
| Commerciaux | Suivi des ventes |
| Etudiants | Projets, statistiques |
