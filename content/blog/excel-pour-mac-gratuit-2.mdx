# Excel pour Mac Gratuit

Options pour utiliser Excel sur Mac sans payer.

## Excel pour le Web

### Acces

1. Allez sur office.com
2. Connectez-vous
3. Utilisez Excel gratuitement

### Compatible Mac

Fonctionne dans Safari ou Chrome.

## Numbers (Apple)

### Inclus avec macOS

- Gratuit
- Compatible Excel (.xlsx)
- Interface Mac native

### Limites

Certaines fonctions Excel non disponibles.

## Google Sheets

### 100% gratuit

- Acces via navigateur
- Import/Export Excel
- Collaboratif

## LibreOffice Calc

### Open source

- Telechargeable gratuitement
- Compatible Excel
- Hors ligne

## Essai Microsoft 365

### 1 mois gratuit

Excel complet pendant 30 jours.
