# Gestion des Ressources Humaines dans Excel

Excel est un outil precieux pour les fonctions RH, de la gestion des effectifs au reporting.

## Base de données salaries

### Structuré

| Champ | Description |
|-------|-------------|
| Matricule | Identifiant unique |
| Nom | Nom de famille |
| Prenom | Prenom |
| Date naissance | Pour calcul age |
| Date entrée | Pour anciennete |
| Poste | Fonction |
| Service | Departement |
| Contrat | CDI/CDD/Interim |
| Statut | Actif/Sorti |

### Calculs automatiques

```excel
=DATEDIF(Date_naissance;AUJOURDHUI();"Y")    ' Age
=DATEDIF(Date_entree;AUJOURDHUI();"Y")       ' Anciennete
```

## Gestion des conges

### Tableau des droits

| Salarie | Acquis | Pris | Solde |
|---------|--------|------|-------|
| Dupont | 25 | 15 | 10 |
| Martin | 25 | 20 | 5 |

### Formules

```excel
=B2-C2                                        ' Solde
=SOMME.SI(Conges[Salarie];A2;Conges[Jours])  ' Jours pris
```

### Calendrier des absences

| Salarie | Jan | Fev | Mar | ... |
|---------|-----|-----|-----|-----|
| Dupont | CP | | CP CP | |
| Martin | | M | | |

Legende : CP = Conge paye, M = Maladie, F = Formation

## Suivi des formations

### Plan de formation

| Formation | Salarie | Date | Duree | Cout |
|-----------|---------|------|-------|------|
| Excel avancé | Dupont | 15/03 | 2j | 800 |
| Management | Martin | 20/04 | 3j | 1500 |

### Budget formation

```excel
=SOMME(Formations[Cout])                      ' Total
=SOMME.SI(Formations[Salarie];A2;Formations[Cout])  ' Par salarie
```

### Alerte échéance

```excel
=SI(E2-AUJOURDHUI()<30;"A planifier";"OK")
```

## Recrutement

### Suivi des candidatures

| Poste | Candidat | Date | Étape | Statut |
|-------|----------|------|-------|--------|
| Commercial | Durand | 01/02 | Entretien 2 | En cours |

### Pipeline recrutement

```excel
=NB.SI(Recrutement[Étape];"CV reçu")
=NB.SI(Recrutement[Étape];"Entretien 1")
=NB.SI(Recrutement[Statut];"Embauche")
```

## Organigramme

### Tableau hiérarchique

| Salarie | Manager | Service | Niveau |
|---------|---------|---------|--------|
| Dupont | Martin | Commercial | N-1 |

### Calcul des equipes

```excel
=NB.SI(Salaries[Manager];"Martin")    ' Taille equipe
```

## Masse salariale

### Suivi mensuel

| Mois | Brut | Charges | Cout total |
|------|------|---------|------------|
| Jan | 150 000 | 75 000 | 225 000 |
| Fev | 152 000 | 76 000 | 228 000 |

### Analyse

```excel
=SOMME(Salaries[Brut])                ' Masse salariale
=MOYENNE(Salaries[Brut])              ' Salaire moyen
```

## Indicateurs RH

### Tableau de bord

| KPI | Valeur | Objectif |
|-----|--------|----------|
| Effectif | 120 | 125 |
| Turnover | 8% | 10% |
| Absenteisme | 4% | 5% |
| Formation | 2% MS | 2% |

### Formules KPI

```excel
=NB.SI(Salaries[Statut];"Actif")                    ' Effectif
=(Departs/Effectif_moyen)*100                       ' Turnover
=(Jours_absence/Jours_theoriques)*100               ' Absenteisme
```
