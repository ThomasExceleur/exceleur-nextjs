# Modifier Plusieurs Nombres d'un Coup

Gagnez du temps en modifiant plusieurs valeurs numeriques simultanement.

## Methode 1 : Collage special

### Multiplier tous les nombres par 2

1. Tapez **2** dans une cellule vide
2. Copiez cette cellule
3. Selectionnez vos nombres
4. **Edition** > **Collage special** > **Multiplier**

### Operations disponibles

| Operation | Effet |
|-----------|-------|
| Addition | Ajoute la valeur |
| Soustraction | Retire la valeur |
| Multiplication | Multiplie |
| Division | Divise |

## Methode 2 : Ajouter une valeur

### Exemple : Augmenter de 10%

1. Tapez **1.1** dans une cellule vide
2. Copiez
3. Selectionnez vos prix
4. **Collage special** > **Multiplier**

## Methode 3 : Soustraire une valeur

### Exemple : Reduire de 5

1. Tapez **5** dans une cellule
2. Copiez
3. Selectionnez vos donnees
4. **Collage special** > **Soustraction**

## Methode 4 : Conversion d'unites

### Convertir des milliers en unites

1. Tapez **1000** dans une cellule
2. Copiez
3. Selectionnez vos nombres
4. **Collage special** > **Multiplier**

## Methode 5 : Formules

### Appliquer une formule a une colonne

```excel
=A1*1.1
```

Puis faites glisser vers le bas.

### Avec collage valeurs

Pour conserver uniquement les valeurs :
1. Copiez la colonne de formules
2. Collez en valeurs sur l'originale

## Raccourci Collage special

| Raccourci | Action |
|-----------|--------|
| Ctrl + Alt + V | Ouvre Collage special |
| M | Multiplier (dans la boite) |
| A | Addition |
| S | Soustraction |
| D | Division |

## Cas pratiques

### Augmentation salariale de 3%

1. Cellule vide : **1.03**
2. Copier
3. Selectionner les salaires
4. Collage special > Multiplier

### Arrondir tous les prix

```excel
=ARRONDI(A1;0)
```
