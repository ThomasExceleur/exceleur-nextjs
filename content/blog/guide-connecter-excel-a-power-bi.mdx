# Connecter Excel a Power BI

Combinez la puissance d'Excel et de Power BI pour vos analyses.

## Pourquoi connecter Excel a Power BI ?

### Avantages

- Tableaux de bord interactifs
- Partage facilite
- Actualisation automatique
- Visualisations avancees

## Methode 1 : Importer un fichier Excel

### Dans Power BI Desktop

1. **Accueil** > **Obtenir des donnees** > **Excel**
2. Selectionnez votre fichier
3. Choisissez les tables/feuilles
4. **Charger** ou **Transformer**

### Types de donnees importables

- Tableaux Excel
- Plages nommees
- Feuilles entieres
- Modeles Power Pivot

## Methode 2 : Connexion directe

### Depuis OneDrive/SharePoint

1. **Obtenir des donnees** > **Classeur Excel**
2. Connectez-vous a OneDrive/SharePoint
3. Selectionnez le fichier
4. **Connexion** (pas Import)

### Avantage

Les modifications Excel sont automatiquement refletees.

## Methode 3 : Publier vers Power BI

### Depuis Excel

1. **Fichier** > **Publier** > **Publier vers Power BI**
2. Connectez-vous
3. Choisissez l'espace de travail
4. **Publier**

### Ce qui est publie

- Tableaux
- Modele Power Pivot
- Tableaux croises dynamiques

## Actualisation des donnees

### Manuelle

Dans Power BI Service : **Actualiser maintenant**

### Planifiee

1. **Parametres du jeu de donnees**
2. **Actualisation planifiee**
3. Definissez la frequence

### Passerelle

Pour les fichiers locaux, installez la passerelle Power BI.

## Bonnes pratiques Excel

### Avant l'import

1. **Utilisez** des tableaux structures
2. **Nommez** clairement vos plages
3. **Nettoyez** les donnees
4. **Evitez** les cellules fusionnees

### Structure recommandee

| Colonne | Description |
|---------|-------------|
| Noms clairs | Sans espaces si possible |
| Types coherents | Un type par colonne |
| Pas de totaux | Power BI les calculera |

## Cas d'usage

### Reporting automatique

Excel collecte les donnees, Power BI les visualise.

### Dashboard partage

Publiez un dashboard accessible a toute l'equipe.

### Analyse en temps reel

Connexion directe pour des donnees toujours a jour.

## Limites

| Limite | Description |
|--------|-------------|
| Taille | Fichiers Excel volumineux = lenteur |
| Formules | Les formules Excel ne sont pas importees |
| Mise en forme | Seules les donnees sont importees |

## Conseils

1. **Preferez** les tableaux structures
2. **Utilisez** OneDrive pour l'actualisation auto
3. **Nettoyez** les donnees dans Power Query
4. **Testez** l'actualisation avant le partage
