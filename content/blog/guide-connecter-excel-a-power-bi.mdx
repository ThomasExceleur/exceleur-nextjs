# Connecter Excel à Power BI

Combinez la puissance d'Excel et de Power BI pour vos analyses.

## Pourquoi connecter Excel à Power BI ?

### Avantages

- Tableaux de bord interactifs
- Partage facilite
- Actualisation automatique
- Visualisations avancées

## Méthode 1 : Importer un fichier Excel

### Dans Power BI Desktop

1. **Accueil** > **Obtenir des données** > **Excel**
2. Sélectionnez votre fichier
3. Choisissez les tables/feuilles
4. **Charger** où **Transformer**

### Types de données importables

- Tableaux Excel
- Plages nommées
- Feuilles entières
- Modèles Power Pivot

## Méthode 2 : Connexion directe

### Depuis OneDrive/SharePoint

1. **Obtenir des données** > **Classeur Excel**
2. Connectez-vous à OneDrive/SharePoint
3. Sélectionnez le fichier
4. **Connexion** (pas Import)

### Avantage

Les modifications Excel sont automatiquement refletees.

## Méthode 3 : Publier vers Power BI

### Depuis Excel

1. **Fichier** > **Publier** > **Publier vers Power BI**
2. Connectez-vous
3. Choisissez l'espace de travail
4. **Publier**

### Ce qui est publie

- Tableaux
- Modèle Power Pivot
- Tableaux croisés dynamiques

## Actualisation des données

### Manuelle

Dans Power BI Service : **Actualiser maintenant**

### Planifiee

1. **Paramètrès du jeu de données**
2. **Actualisation planifiee**
3. Définissez la frequence

### Passerelle

Pour les fichiers locaux, installez la passerelle Power BI.

## Bonnes pratiques Excel

### Avant l'import

1. **Utilisez** des tableaux structurés
2. **Nommez** clairement vos plages
3. **Nettoyez** les données
4. **Évitez** les cellules fusionnées

### Structuré recommandée

| Colonne | Description |
|---------|-------------|
| Noms clairs | Sans espaces si possible |
| Types cohérents | Un type par colonne |
| Pas de totaux | Power BI les calculera |

## Cas d'usage

### Reporting automatique

Excel collecte les données, Power BI les visualise.

### Dashboard partage

Publiez un dashboard accessible à toute l'equipe.

### Analyse en temps reel

Connexion directe pour des données toujours à jour.

## Limites

| Limite | Description |
|--------|-------------|
| Taille | Fichiers Excel volumineux = lenteur |
| Formules | Les formules Excel ne sont pas importees |
| Mise en forme | Seules les données sont importees |

## Conseils

1. **Préférez** les tableaux structurés
2. **Utilisez** OneDrive pour l'actualisation auto
3. **Nettoyez** les données dans Power Query
4. **Testez** l'actualisation avant le partage
