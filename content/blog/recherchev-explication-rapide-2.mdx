# RECHERCHEV : Explication Rapide

Maitrisez RECHERCHEV en 5 minutes.

## Syntaxe

```excel
=RECHERCHEV(valeur;tableau;colonne;[type])
```

## Les 4 arguments

| # | Argument | Description |
|---|----------|-------------|
| 1 | valeur | Ce que vous cherchez |
| 2 | tableau | Ou chercher |
| 3 | colonne | Quelle colonne retourner |
| 4 | type | FAUX = exact |

## Exemple concret

### Donnees

| ID | Nom | Prix |
|----|-----|------|
| A1 | Pomme | 2 |
| A2 | Poire | 3 |

### Formule

```excel
=RECHERCHEV("A1";A:C;3;FAUX)
```

Retourne : 2

## Regles importantes

### Toujours FAUX

Utilisez FAUX pour une correspondance exacte.

### Premiere colonne

RECHERCHEV cherche dans la premiere colonne du tableau.

### Pas de recherche a gauche

Le resultat doit etre a droite de la recherche.

## Erreur #N/A

### Causes

- Valeur non trouvee
- Espaces invisibles
- Format different

### Solution

```excel
=SIERREUR(RECHERCHEV(...);"Non trouve")
```

## Alternatives

| Fonction | Avantage |
|----------|----------|
| RECHERCHEX | Plus moderne |
| INDEX+EQUIV | Plus flexible |
