# Combiner Plusieurs Fichiers avec Power Query

Fusionnez automatiquement vos fichiers.

## Depuis un dossier

### Etapes

1. **Donnees** > **Obtenir des donnees** > **A partir d'un fichier** > **A partir d'un dossier**
2. Selectionnez le dossier
3. **Combiner** > **Combiner et transformer**

## Configuration

### Fichier exemple

Power Query vous demande de choisir un fichier modele.

### Options

| Option | Description |
|--------|-------------|
| Premiere ligne en-tete | Utiliser comme noms |
| Ignorer les erreurs | Continuer si fichier corrompu |

## Transformations

### Ajouter la source

La colonne "Source.Name" indique le fichier d'origine.

### Filtrer les fichiers

Excluez certains fichiers si necessaire.

## Types de fichiers

| Type | Support |
|------|---------|
| Excel | Oui |
| CSV | Oui |
| Texte | Oui |
| JSON | Oui |

## Actualisation

### Automatique

Ajoutez des fichiers au dossier, actualisez la requete.

### Manuelle

**Donnees** > **Actualiser tout**

## Structure identique

### Prerequis

Tous les fichiers doivent avoir la meme structure.

### Si different

Utilisez plusieurs requetes et ajoutez-les.

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Structure | Meme format partout |
| Nommage | Noms de fichiers coherents |
| Dossier dedie | Organisez vos sources |
