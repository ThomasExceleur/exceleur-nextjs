# Automatiser les Graphiques Excel vers PowerPoint

Gagnez du temps en liant vos graphiques pour des mises a jour automatiques.

## Methodes de transfert

### Copier-coller standard

1. Copiez le graphique Excel
2. Collez dans PowerPoint
3. Le graphique est independant

### Collage avec liaison

1. Copiez le graphique Excel
2. Dans PowerPoint : **Coller** > **Collage special**
3. Choisissez **Coller avec liaison**

## Liaison vs Incorporation

| Aspect | Liaison | Incorporation |
|--------|---------|---------------|
| Mise a jour | Automatique | Manuelle |
| Taille fichier | Petite | Grande |
| Dependance | Fichier Excel requis | Autonome |
| Partage | Complexe | Simple |

## Creer une liaison

### Etape par etape

1. **Excel** : Selectionnez votre graphique
2. **Ctrl+C** pour copier
3. **PowerPoint** : Allez sur la diapositive
4. **Accueil** > **Coller** > **Collage special**
5. Selectionnez **Coller le lien**
6. Format : Objet Graphique Microsoft Excel

### Options de collage

| Option | Description |
|--------|-------------|
| Utiliser theme destination | Style PowerPoint |
| Conserver mise en forme source | Style Excel |
| Image | Non modifiable |

## Mettre a jour les liaisons

### Mise a jour automatique

Par defaut, les liaisons se mettent a jour a l'ouverture.

### Mise a jour manuelle

1. Clic droit sur le graphique
2. **Mettre a jour le lien**

### Gerer les liaisons

1. **Fichier** > **Informations** > **Modifier les liens vers les fichiers**
2. Verifiez les chemins
3. Mettez a jour ou rompez les liens

## Automatisation avec VBA

### Code de mise a jour

```vba
Sub MettreAJourLiaisons()
    Dim lien As Object
    For Each lien In ActivePresentation.ExternalLinks
        lien.Update
    Next lien
End Sub
```

### Exporter graphique automatiquement

```vba
Sub ExporterGraphique()
    Dim chemin As String
    chemin = "C:\Rapports\graphique.png"
    ActiveChart.Export chemin, "PNG"
End Sub
```

## Bonnes pratiques

### Organisation

1. **Nommez** vos graphiques clairement
2. **Placez** Excel et PowerPoint au meme emplacement
3. **Documentez** les liaisons existantes

### Performance

1. **Limitez** le nombre de liaisons
2. **Optimisez** la taille des graphiques
3. **Testez** avant les presentations

## Problemes courants

### Liaison rompue

**Cause** : Fichier Excel deplace ou renomme.

**Solution** : Mettre a jour le chemin via Modifier les liens.

### Graphique ne se met pas a jour

**Cause** : Mise a jour manuelle activee.

**Solution** : Activez la mise a jour automatique.

### Differences d'affichage

**Cause** : Themes differents.

**Solution** : Utilisez le meme theme ou forcez le style source.

## Alternative : Images liees

### Quand utiliser

Pour des mises a jour visuelles sans interactivite.

### Comment faire

1. Copiez le graphique
2. Collez en tant qu'image liee
3. L'image se met a jour avec la source

## Workflow recommande

1. Creez vos graphiques dans un classeur dedie
2. Liez-les a PowerPoint avec le bon format
3. Avant presentation, actualisez les liaisons
4. Testez l'affichage
