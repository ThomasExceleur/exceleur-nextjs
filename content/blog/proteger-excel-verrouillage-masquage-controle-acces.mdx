# Proteger Excel : Verrouillage, Masquage et Controle

Toutes les techniques pour securiser vos classeurs.

## Verrouillage des cellules

### Principe

Par defaut, toutes les cellules sont verrouillees, mais la protection n'est active qu'apres avoir protege la feuille.

### Deverrouiller des cellules

1. Selectionnez les cellules editables
2. Ctrl+1 > Protection
3. Decochez \"Verrouillee\"

## Masquage des formules

### Activer

1. Selectionnez les cellules
2. Ctrl+1 > Protection
3. Cochez \"Masquee\"
4. Protegez la feuille

### Effet

La barre de formule n'affiche pas la formule.

## Controle d'acces

### Plages modifiables par utilisateur

1. **Revision** > **Autoriser la modification des plages**
2. Definissez les plages et permissions
3. Protegez la feuille

## Protection hierarchique

| Niveau | Protection |
|--------|------------|
| 1 | Chiffrement fichier |
| 2 | Structure classeur |
| 3 | Feuilles |
| 4 | Cellules individuelles |

## Combinaisons

### Maximum securite

1. Chiffrez le fichier
2. Protegez la structure
3. Protegez chaque feuille
4. Deverrouillez uniquement les cellules de saisie

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Layered | Plusieurs niveaux |
| Documentation | Notez les protections |
| Test | Verifiez l'acces |
