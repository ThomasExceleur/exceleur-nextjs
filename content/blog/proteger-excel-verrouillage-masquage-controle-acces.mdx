# Proteger Excel : Verrouillage, Masquage et Contrôle

Toutes les techniques pour securiser vos classeurs.

## Verrouillage des cellules

### Principe

Par defaut, toutes les cellules sont verrouillees, mais la protection n'est active qu'après avoir protege la feuille.

### Deverrouiller des cellules

1. Sélectionnez les cellules editables
2. Ctrl+1 > Protection
3. Decochez \"Verrouillee\"

## Masquage des formules

### Activer

1. Sélectionnez les cellules
2. Ctrl+1 > Protection
3. Cochez \"Masquee\"
4. Protégez la feuille

### Effet

La barre de formule n'affiché pas la formule.

## Contrôle d'accès

### Plages modifiables par utilisateur

1. **Révision** > **Autoriser la modification des plages**
2. Définissez les plages et permissions
3. Protégez la feuille

## Protection hiérarchique

| Niveau | Protection |
|--------|------------|
| 1 | Chiffrement fichier |
| 2 | Structuré classeur |
| 3 | Feuilles |
| 4 | Cellules individuelles |

## Combinaisons

### Maximum sécurité

1. Chiffrez le fichier
2. Protégez la structuré
3. Protégez chaque feuille
4. Deverrouillez uniquement les cellules de saisie

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Layered | Plusieurs niveaux |
| Documentation | Notez les protections |
| Test | Vérifiez l'accès |
