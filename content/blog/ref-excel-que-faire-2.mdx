# Erreur #REF! : Que Faire

Comprendre et corriger cette erreur de reference.

## Signification

#REF! indique une reference invalide a une cellule.

## Causes

| Cause | Exemple |
|-------|---------|
| Suppression | Ligne/colonne supprimee |
| Copie invalide | Coller hors plage |
| Index invalide | RECHERCHEV avec index trop grand |

## Identifier le probleme

### Localiser l'erreur

Cliquez sur la cellule et examinez la formule.

### Chercher #REF!

Le #REF! apparait dans la formule elle-meme.

## Solutions

### Apres suppression

1. **Ctrl+Z** pour annuler
2. Ou reconstruire la formule

### Index RECHERCHEV

```excel
=RECHERCHEV(A1;B:E;3;FAUX)
```

Verifiez que l'index (3) n'excede pas le nombre de colonnes.

### Copie problematique

Verifiez les references avant de copier.

## Prevention

### Utiliser des tableaux

Les tableaux structures s'adaptent automatiquement.

### Nommer les plages

Les plages nommees sont plus robustes.

### Verifier avant suppression

Verifiez les dependances avant de supprimer.

## Trouver les dependances

### Audit

**Formules** > **Reperer les dependants**

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Tableaux | Utilisez les tableaux structures |
| Audit | Verifiez les dependances |
| Sauvegarde | Avant modifications majeures |
