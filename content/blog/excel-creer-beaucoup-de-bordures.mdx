# Creer des Bordures Efficacement

Ajoutez des bordures rapidement a vos tableaux Excel.

## Methode rapide : Raccourcis

### Toutes les bordures

1. Selectionnez la plage
2. **Ctrl + Shift + 7** (pave numerique)

Ou :
1. **Ctrl + 1** (Format de cellule)
2. Onglet **Bordure**

## Menu Bordures

### Acces

1. Selectionnez les cellules
2. **Accueil** > **Bordures** (icone grille)
3. Choisissez le type

### Options principales

| Option | Description |
|--------|-------------|
| Bordure inferieure | Ligne sous les cellules |
| Toutes les bordures | Grille complete |
| Bordures exterieures | Contour uniquement |
| Bordures epaisses | Contour epais |

## Dessiner des bordures

### Mode dessin

1. **Accueil** > **Bordures** > **Dessiner une bordure**
2. Faites glisser pour tracer
3. **Echap** pour quitter

### Dessiner une grille

1. **Accueil** > **Bordures** > **Dessiner une grille de bordures**
2. Faites glisser sur la zone
3. Toutes les bordures sont ajoutees

## Personnaliser les bordures

### Boite de dialogue

1. **Ctrl + 1** (Format de cellule)
2. Onglet **Bordure**
3. Choisissez :
   - Le style de ligne
   - La couleur
   - L'emplacement

### Styles de ligne

- Trait fin
- Trait moyen
- Trait epais
- Pointille
- Tirete
- Double

## Bordures conditionnelles

### Avec mise en forme conditionnelle

1. Selectionnez la plage
2. **Mise en forme conditionnelle** > **Nouvelle regle**
3. Definissez la condition
4. **Format** > **Bordure**

## Supprimer les bordures

### Tout supprimer

1. Selectionnez les cellules
2. **Accueil** > **Bordures** > **Aucune bordure**

### Effacer avec outil

1. **Accueil** > **Bordures** > **Effacer la bordure**
2. Cliquez sur les bordures a supprimer

## Astuces

### Bordure double pour totaux

Utilisez une bordure double au-dessus des totaux pour les distinguer.

### Couleur coherente

Utilisez la meme couleur de bordure dans tout le document.

### Eviter le surplus

Trop de bordures rend le tableau difficile a lire.

## Tableaux structures

### Avantage

Les tableaux Excel (Ctrl + T) ont des bordures automatiques.

### Styles de tableau

1. Cliquez dans le tableau
2. **Creation de tableau** > **Styles de tableau**
3. Choisissez un style avec bordures

## Bonnes pratiques

1. **Utilisez** les tableaux structures quand possible
2. **Limitez** les types de bordures (2-3 max)
3. **Privilegiez** les bordures legeres
4. **Reservez** les traits epais aux totaux
