# Calculer l'Anciennete dans Excel

Plusieurs formules permettent de calculer l'anciennete avec precision.

## Formule principale : DATEDIF

### Syntaxe

```excel
=DATEDIF(date_debut;date_fin;unite)
```

### Unites disponibles

| Unite | Resultat |
|-------|----------|
| "Y" | Annees completes |
| "M" | Mois complets |
| "D" | Jours |
| "YM" | Mois (sans les annees) |
| "YD" | Jours (sans les annees) |
| "MD" | Jours (sans mois ni annees) |

## Exemples d'anciennete

### En annees

```excel
=DATEDIF(A1;AUJOURDHUI();"Y")
```

### En mois

```excel
=DATEDIF(A1;AUJOURDHUI();"M")
```

### Format complet

```excel
=DATEDIF(A1;AUJOURDHUI();"Y")&" ans et "&DATEDIF(A1;AUJOURDHUI();"YM")&" mois"
```

## Affichage detaille

### Annees, mois et jours

```excel
=DATEDIF(A1;AUJOURDHUI();"Y")&" an(s), "&DATEDIF(A1;AUJOURDHUI();"YM")&" mois et "&DATEDIF(A1;AUJOURDHUI();"MD")&" jour(s)"
```

### Resultat

```
5 an(s), 3 mois et 15 jour(s)
```

## Alternative sans DATEDIF

### En annees (approximatif)

```excel
=ENT((AUJOURDHUI()-A1)/365.25)
```

### En mois (approximatif)

```excel
=ENT((AUJOURDHUI()-A1)/30.44)
```

## Avec une date de reference

### Pour un calcul a date fixe

```excel
=DATEDIF(A1;DATE(2024;12;31);"Y")
```

## Cas particuliers

### Anciennete inferieure a 1 an

```excel
=SI(DATEDIF(A1;AUJOURDHUI();"Y")<1;DATEDIF(A1;AUJOURDHUI();"M")&" mois";DATEDIF(A1;AUJOURDHUI();"Y")&" an(s)")
```

### Eviter les erreurs

```excel
=SI(A1>AUJOURDHUI();"Date future";DATEDIF(A1;AUJOURDHUI();"Y")&" ans")
```

### Formule

```excel
=DATEDIF(B2;AUJOURDHUI();"Y")&" ans"
```

## Tranches d'anciennete

### Classification

```excel
=SI(DATEDIF(A1;AUJOURDHUI();"Y")<2;"Junior";SI(DATEDIF(A1;AUJOURDHUI();"Y")<5;"Confirme";"Senior"))
```

## Prime d'anciennete

### Calcul selon anciennete

```excel
=SI(DATEDIF(A1;AUJOURDHUI();"Y")>=10;Salaire*0.1;SI(DATEDIF(A1;AUJOURDHUI();"Y")>=5;Salaire*0.05;0))
```

## Nombre de jours exact

### Total jours travailles

```excel
=AUJOURDHUI()-A1
```

### Jours ouvres

```excel
=NB.JOURS.OUVRES(A1;AUJOURDHUI())
```

## Erreurs courantes

### Date au format texte

Convertissez avec DATEVAL :

```excel
=DATEDIF(DATEVAL(A1);AUJOURDHUI();"Y")
```

### Dates inversees

DATEDIF retourne une erreur si la date de fin est anterieure.
