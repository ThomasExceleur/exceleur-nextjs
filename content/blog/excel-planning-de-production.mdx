# Planning de Production dans Excel

Un planning de production bien concu optimise vos ressources et respecte les delais.

## Elements du planning

### Informations essentielles

| Element | Description |
|---------|-------------|
| Ordre de fabrication | Reference OF |
| Produit | Article a fabriquer |
| Quantite | Nombre a produire |
| Date debut | Lancement production |
| Date fin | Livraison prevue |
| Statut | Etat de l'OF |

## Structure du tableau

### Tableau des OF

| OF | Produit | Qte | Debut | Fin | Poste | Statut |
|----|---------|-----|-------|-----|-------|--------|
| OF001 | Produit A | 100 | 01/01 | 05/01 | Ligne 1 | En cours |
| OF002 | Produit B | 50 | 02/01 | 04/01 | Ligne 2 | Planifie |

## Calcul des delais

### Duree de production

```excel
=Quantite / Cadence_horaire / Heures_jour
```

### Date de fin

```excel
=Date_debut + Duree_jours - 1
=JOURS.OUVRES(Date_debut;Duree_jours)    ' Jours ouvres
```

## Diagramme de Gantt

### Creer la grille

1. Creez des colonnes pour chaque jour
2. Les lignes representent les OF

### Formule de mise en forme

```excel
=ET(F$1>=[@Debut];F$1<=[@Fin])
```

### Mise en forme conditionnelle

1. Selectionnez la grille
2. **Mise en forme conditionnelle** > **Nouvelle regle**
3. Utilisez la formule ci-dessus
4. Appliquez un remplissage colore

## Capacite de production

### Tableau de capacite

| Poste | Capacite/jour | Charge | Dispo |
|-------|---------------|--------|-------|
| Ligne 1 | 100 | 80 | 20 |
| Ligne 2 | 150 | 150 | 0 |

### Calcul de charge

```excel
=SOMME.SI(Planning[Poste];A2;Planning[Qte])
```

### Taux d'utilisation

```excel
=Charge / Capacite
```

## Ordonnancement

### Regles de priorite

| Regle | Description |
|-------|-------------|
| FIFO | Premier arrive, premier servi |
| Date livraison | Plus proche d'abord |
| Priorite client | Clients importants d'abord |

### Tri automatique

```excel
Trier par : Date livraison (croissant)
Puis par : Priorite (decroissant)
```

## Suivi de production

### Etats de production

- Planifie
- Lance
- En cours
- Termine
- En retard

### Mise a jour du statut

```excel
=SI([@Fin]<AUJOURDHUI();"En retard";
  SI([@Debut]<=AUJOURDHUI();"En cours";"Planifie"))
```

## Indicateurs de production

### KPI essentiels

| KPI | Formule |
|-----|---------|
| OTD (On Time Delivery) | OF a l'heure / Total OF |
| TRS | Temps prod / Temps dispo |
| Taux de rebut | Rebuts / Production totale |

### Calcul OTD

```excel
=NB.SI.ENS(Planning[Statut];"Termine";Planning[Fin];"<="&Planning[Date livraison]) / NB.SI(Planning[Statut];"Termine")
```

## Gestion des aleas

### Types d'aleas

| Alea | Impact | Action |
|------|--------|--------|
| Panne machine | Retard | Replanifier |
| Absence | Capacite reduite | Remplacement |
| Rupture stock | Arret | Commande urgente |

## Planning glissant

### Structure

Mettez a jour quotidiennement :
- J : Realise
- J+1 a J+5 : Planifie ferme
- J+6 a J+10 : Previsionnel
