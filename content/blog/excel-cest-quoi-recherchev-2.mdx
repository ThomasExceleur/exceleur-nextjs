# C'est Quoi RECHERCHEV

La fonction de recherche la plus connue.

## Definition

RECHERCHEV cherche une valeur et retourne une information associee.

## Syntaxe

```excel
=RECHERCHEV(cherche;tableau;colonne;type)
```

## Arguments

| # | Argument | Description |
|---|----------|-------------|
| 1 | cherche | Valeur a trouver |
| 2 | tableau | Ou chercher |
| 3 | colonne | Quelle colonne retourner |
| 4 | type | FAUX = exact |

## Exemple

### Donnees

| ID | Produit | Prix |
|----|---------|------|
| A1 | Pomme | 2€ |
| A2 | Poire | 3€ |

### Formule

```excel
=RECHERCHEV("A1";A:C;3;FAUX)
```

Resultat : 2€

## Cas d'usage

| Situation | Exemple |
|-----------|---------|
| Prix | Trouver le prix d'un produit |
| RH | Trouver le salaire d'un employe |
| Stock | Trouver la quantite |

## Limites

- Cherche a droite seulement
- Index de colonne fixe

## Alternative moderne

RECHERCHEX (Excel 365)

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| FAUX | Toujours pour exact |
| SIERREUR | Gerez les erreurs |
| Tableaux | Utilisez des tableaux |
