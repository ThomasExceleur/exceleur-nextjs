# Excel Formule SI NON DISP : Guide Rapide et Efficace

La fonction SI.NON.DISP facilite la gestion des erreurs dans Excel, particulièrement les erreurs #N/A. Elle nécessite deux arguments : la formule à vérifier et la valeur à afficher en cas d'erreur. Les fonctions RECHERCHEV et RECHERCHEH sont parmi les plus courantes générant cette erreur.

## Principes de base des formules Excel

### Syntaxe générale

```excel
=FONCTION(argument1;argument2, ...)
```

### Types de formules

- **Formules arithmétiques** : Addition (+), soustraction (-), multiplication (*), division (/)
- **Fonctions logiques** : SI, ET, OU
- **Fonctions statistiques** : MOYENNE, ECARTYPE
- **Fonctions de recherche et référence** : RECHERCHEV, INDEX, EQUIV

## La fonction SI : concepts et applications

### Syntaxe SI

```excel
=SI(condition;valeur_si_vrai;valeur_si_faux)
```

### Exemples d'usage

- Classification de valeurs
- Calcul de remises conditionnelles
- Gestion des erreurs

## Traitement des erreurs avec SI.NON.DISP

### Syntaxe

```excel
=SI.NON.DISP(valeur;valeur_si_na)
```

### Exemple pratique

```excel
=SI.NON.DISP(RECHERCHEV(A1, B1:C10, 2, FAUX), "Non trouvé")
```

Cette formule recherche la valeur de A1 dans la plage B1:C10. Si la valeur n'est pas trouvée (erreur #N/A), elle affiche "Non trouvé" au lieu de l'erreur.

## Alternative : SIERREUR

La fonction SIERREUR gère tous les types d'erreurs, pas seulement #N/A.

```excel
=SIERREUR(RECHERCHEV(A1;B1:C10;2;FAUX);"Erreur")
```

Cette formule capture toutes les erreurs possibles (#N/A, #REF!, #DIV/0!, etc.) et affiche "Erreur" à la place.

## Fonctions avancées (ET, OU, NON)

Vous pouvez combiner SI.NON.DISP avec d'autres fonctions logiques pour créer des formules complexes :

- **ET** : Toutes les conditions doivent être vraies
- **OU** : Au moins une condition doit être vraie
- **NON** : Inverse le résultat d'une condition

## Évolutions selon les versions Excel

- **Excel 2013-2016** : Introduction de SI.NON.DISP
- **Excel 2021 et Microsoft 365** : Nouvelles fonctions (RECHERCHEX, EQUIVX)
- **Versions antérieures** : Alternatives avec SI et ESTNA

### Alternative pour versions anciennes

```excel
=SI(ESTNA(RECHERCHEV(A1;B1:C10;2;FAUX));"Non trouvé";RECHERCHEV(A1;B1:C10;2;FAUX))
```

## RechercheV et validation de données

Combinez SI.NON.DISP avec RECHERCHEV pour créer des systèmes robustes de recherche et validation de données. Cela permet d'éviter que les erreurs n'interrompent vos calculs et analyses.
