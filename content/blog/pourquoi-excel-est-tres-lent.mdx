# Pourquoi Excel Est Très Lent

Un fichier Excel lent peut avoir plusieurs causes. Voici comment diagnostiquer et resoudre.

## Causes courantes

### 1. Fichier trop volumineux

| Problème | Solution |
|----------|----------|
| Trop de données | Supprimer les données inutiles |
| Images lourdes | Compresser les images |
| Mises en forme excessives | Simplifier |

### 2. Formules complexes

| Problème | Solution |
|----------|----------|
| Formules volatiles | Réduire INDIRECT, AUJOURDHUI |
| Colonnes entières | Limiter les plages |
| Formules imbriquées | Simplifier où utiliser LET |

### 3. Calcul automatique

Le recalcul de milliers de formules ralentit Excel.

## Solutions rapides

### Passer en calcul manuel

1. **Formules** > **Options de calcul** > **Manuel**
2. Appuyez sûr **F9** pour recalculer

### Supprimer les lignes/colonnes inutiles

1. Sélectionnez les lignes vides
2. Supprimez-les

### Vérifier la dernière cellule utilisée

1. **Ctrl + Fin**
2. Si elle est loin des données, supprimez les lignes/colonnes

## Optimiser les formules

### Éviter les colonnes entières

```excel
' MAL
=SOMME(A:A)

' BIEN
=SOMME(A1:A1000)
```

### Réduire les formules volatiles

| A éviter | Alternative |
|----------|-------------|
| INDIRECT | Plages nommées |
| DECALER | INDEX |
| AUJOURDHUI | Valeur fixe si possible |

### Utiliser des tableaux structurés

Les tableaux structurés sont plus performants.

## Réduire la taille du fichier

### Compresser les images

1. Sélectionnez une image
2. **Format** > **Compresser les images**
3. Choisissez une resolution adaptee

### Supprimer la mise en forme inutile

1. Sélectionnez les cellules vides mises en forme
2. **Accueil** > **Effacer** > **Effacer les formats**

### Supprimer les objets masques

1. **Accueil** > **Rechercher et sélectionner** > **Volet de sélection**
2. Supprimez les objets inutiles

## Problèmes de mémoire

### Symptomes

- Excel utilisé beaucoup de RAM
- Messages "Mémoire insuffisante"

### Solutions

1. Fermez les autres applications
2. Utilisez Excel 64 bits
3. Divisez le fichier en plusieurs

## Complements et add-ins

### Desactiver les complements

1. **Fichier** > **Options** > **Complements**
2. Desactivez les complements inutiles

### Tester en mode sans echec

1. Maintenez **Ctrl** au démarrage d'Excel
2. Ou lancez : `excel /safe`

## Mise en forme conditionnelle

### Problème

Trop de règles de mise en forme ralentissent Excel.

### Solution

1. **Accueil** > **Mise en forme conditionnelle** > **Gérer les règles**
2. Supprimez les règles inutiles

## Tableaux croisés dynamiques

### Optimiser les TCD

1. Desactivez l'actualisation automatique
2. Limitez les données sources
3. Supprimez les TCD inutilises

## Vérifier la corruption

### Fichier endommage

1. **Fichier** > **Ouvrir**
2. Sélectionnez le fichier
3. Fleche à côté de "Ouvrir" > **Ouvrir et reparer**

## Bonnes pratiques

1. **Limitez** les données au strict nécessaire
2. **Évitez** les formules sûr colonnes entières
3. **Compressez** les images
4. **Supprimez** les objets masques
5. **Divisez** les gros fichiers
