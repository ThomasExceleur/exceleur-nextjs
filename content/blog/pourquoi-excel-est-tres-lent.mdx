# Pourquoi Excel Est Tres Lent

Un fichier Excel lent peut avoir plusieurs causes. Voici comment diagnostiquer et resoudre.

## Causes courantes

### 1. Fichier trop volumineux

| Probleme | Solution |
|----------|----------|
| Trop de donnees | Supprimer les donnees inutiles |
| Images lourdes | Compresser les images |
| Mises en forme excessives | Simplifier |

### 2. Formules complexes

| Probleme | Solution |
|----------|----------|
| Formules volatiles | Reduire INDIRECT, AUJOURDHUI |
| Colonnes entieres | Limiter les plages |
| Formules imbriquees | Simplifier ou utiliser LET |

### 3. Calcul automatique

Le recalcul de milliers de formules ralentit Excel.

## Solutions rapides

### Passer en calcul manuel

1. **Formules** > **Options de calcul** > **Manuel**
2. Appuyez sur **F9** pour recalculer

### Supprimer les lignes/colonnes inutiles

1. Selectionnez les lignes vides
2. Supprimez-les

### Verifier la derniere cellule utilisee

1. **Ctrl + Fin**
2. Si elle est loin des donnees, supprimez les lignes/colonnes

## Optimiser les formules

### Eviter les colonnes entieres

```excel
' MAL
=SOMME(A:A)

' BIEN
=SOMME(A1:A1000)
```

### Reduire les formules volatiles

| A eviter | Alternative |
|----------|-------------|
| INDIRECT | Plages nommees |
| DECALER | INDEX |
| AUJOURDHUI | Valeur fixe si possible |

### Utiliser des tableaux structures

Les tableaux structures sont plus performants.

## Reduire la taille du fichier

### Compresser les images

1. Selectionnez une image
2. **Format** > **Compresser les images**
3. Choisissez une resolution adaptee

### Supprimer la mise en forme inutile

1. Selectionnez les cellules vides mises en forme
2. **Accueil** > **Effacer** > **Effacer les formats**

### Supprimer les objets masques

1. **Accueil** > **Rechercher et selectionner** > **Volet de selection**
2. Supprimez les objets inutiles

## Problemes de memoire

### Symptomes

- Excel utilise beaucoup de RAM
- Messages "Memoire insuffisante"

### Solutions

1. Fermez les autres applications
2. Utilisez Excel 64 bits
3. Divisez le fichier en plusieurs

## Complements et add-ins

### Desactiver les complements

1. **Fichier** > **Options** > **Complements**
2. Desactivez les complements inutiles

### Tester en mode sans echec

1. Maintenez **Ctrl** au demarrage d'Excel
2. Ou lancez : `excel /safe`

## Mise en forme conditionnelle

### Probleme

Trop de regles de mise en forme ralentissent Excel.

### Solution

1. **Accueil** > **Mise en forme conditionnelle** > **Gerer les regles**
2. Supprimez les regles inutiles

## Tableaux croises dynamiques

### Optimiser les TCD

1. Desactivez l'actualisation automatique
2. Limitez les donnees sources
3. Supprimez les TCD inutilises

## Verifier la corruption

### Fichier endommage

1. **Fichier** > **Ouvrir**
2. Selectionnez le fichier
3. Fleche a cote de "Ouvrir" > **Ouvrir et reparer**

## Bonnes pratiques

1. **Limitez** les donnees au strict necessaire
2. **Evitez** les formules sur colonnes entieres
3. **Compressez** les images
4. **Supprimez** les objets masques
5. **Divisez** les gros fichiers
