# Barres de Donnees dans les Cellules

Visualisez instantanement les proportions.

## Ajouter des barres

### Etapes

1. Selectionnez vos donnees
2. **Accueil** > **Mise en forme conditionnelle**
3. **Barres de donnees**
4. Choisissez un style

## Styles disponibles

### Degrade

Les barres vont du clair au fonce.

### Plein

Couleur unie.

### Couleurs

| Couleur | Suggestion d'usage |
|---------|-------------------|
| Bleu | Donnees neutres |
| Vert | Positif |
| Rouge | Alertes |

## Personnaliser

### Regles avancees

1. **Mise en forme conditionnelle** > **Gerer les regles**
2. Selectionnez la regle > **Modifier**

### Options

| Option | Description |
|--------|-------------|
| Minimum | Valeur de depart |
| Maximum | Valeur de fin |
| Couleur negative | Pour les valeurs negatives |
| Direction | Gauche a droite ou inverse |

## Valeurs negatives

### Configuration

1. **Modifier la regle**
2. **Valeurs negatives**
3. Choisissez couleur et bordure

## Afficher uniquement la barre

### Masquer les valeurs

1. **Modifier la regle**
2. Cochez **Afficher la barre uniquement**

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Coherence | Meme style par section |
| Lisibilite | Evitez les couleurs vives |
| Minimum fixe | 0 pour les comparaisons |
