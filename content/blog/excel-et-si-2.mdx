# Fonction ET avec SI

Testez plusieurs conditions simultanement.

## Syntaxe ET

```excel
=ET(condition1;condition2;...)
```

Retourne VRAI si toutes les conditions sont vraies.

## Combinaison SI + ET

```excel
=SI(ET(condition1;condition2);si_vrai;si_faux)
```

## Exemple

### Cas pratique

Bonus si ventes > 10000 ET objectif atteint.

```excel
=SI(ET(A1>10000;B1="Oui");"Bonus";"Pas de bonus")
```

## Plusieurs conditions

```excel
=SI(ET(A1>0;B1>0;C1>0);"Tous positifs";"Au moins un negatif ou nul")
```

## OU alternative

```excel
=SI(OU(A1>100;B1>100);"Au moins un depasse";"Aucun")
```

## Combinaison ET + OU

```excel
=SI(ET(A1>0;OU(B1="Oui";C1="Oui"));"Valide";"Non valide")
```

## Comparaison

| Fonction | Retourne VRAI si |
|----------|------------------|
| ET | Toutes vraies |
| OU | Au moins une vraie |

## Erreurs courantes

| Erreur | Cause |
|--------|-------|
| Parentheses | Mal fermees |
| Point-virgule | Oublie |

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Lisibilite | Evitez trop de conditions |
| Test | Verifiez chaque condition |
| SI.CONDITIONS | Pour cas complexes |
