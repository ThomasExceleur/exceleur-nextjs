# Creer une Heatmap Dynamique dans Excel

Les heatmaps (cartes de chaleur) permettent de visualiser des donnees numeriques avec des couleurs.

## Qu'est-ce qu'une heatmap ?

### Definition

Une heatmap represente des valeurs par des intensites de couleur :
- Valeurs elevees : couleurs chaudes
- Valeurs basses : couleurs froides

### Applications

- Analyse de performance
- Comparaisons temporelles
- Matrices de correlation
- Tableaux de bord

## Methode simple : Mise en forme conditionnelle

### Etape 1 : Selectionnez les donnees

Selectionnez la plage de donnees numeriques.

### Etape 2 : Appliquer l'echelle de couleurs

1. **Accueil** > **Mise en forme conditionnelle**
2. **Nuances de couleurs**
3. Choisissez un schema

### Schemas disponibles

| Schema | Usage |
|--------|-------|
| Vert-Jaune-Rouge | Performance (vert=bon) |
| Rouge-Jaune-Vert | Performance inversee |
| Bleu-Blanc-Rouge | Ecarts (blanc=neutre) |

## Personnaliser l'echelle

### Acces

1. **Mise en forme conditionnelle** > **Gerer les regles**
2. Selectionnez la regle
3. **Modifier la regle**

### Options

| Parametre | Description |
|-----------|-------------|
| Type | Nombre, Pourcentage, Formule, Centile |
| Valeur | Seuils pour chaque couleur |
| Couleur | Personnalisation des teintes |

### Exemple : Echelle 3 couleurs

- Minimum : Rouge (Type: Min)
- Milieu : Jaune (Type: Centile, Valeur: 50)
- Maximum : Vert (Type: Max)

## Heatmap avec ecart-type

### Concept

Colorez selon l'ecart par rapport a la moyenne.

### Formule

```excel
=A1-MOYENNE($A$1:$Z$10)
```

### Application

Appliquez une echelle de couleurs :
- Rouge : valeurs negatives (sous la moyenne)
- Blanc : proche de zero (moyenne)
- Bleu : valeurs positives (au-dessus)

## Heatmap calendrier

### Structure

| | Lun | Mar | Mer | Jeu | Ven | Sam | Dim |
|---|-----|-----|-----|-----|-----|-----|-----|
| S1 | 10 | 15 | 8 | 12 | 20 | 5 | 3 |
| S2 | 12 | 18 | 9 | 14 | 22 | 6 | 4 |

### Application

Parfait pour visualiser l'activite par jour de la semaine.

## Heatmap avec formules conditionnelles

### Regle personnalisee

1. **Mise en forme conditionnelle** > **Nouvelle regle**
2. **Utiliser une formule**
3. Entrez votre condition

### Exemples

```excel
=A1>MOYENNE($A:$A)     ' Au-dessus de la moyenne
=A1=MAX($A:$A)         ' Valeur maximale
=ABS(A1-$B1)>10        ' Ecart significatif
```

## Matrice de correlation

### Structure

|   | A | B | C |
|---|---|---|---|
| A | 1.00 | 0.85 | -0.20 |
| B | 0.85 | 1.00 | 0.45 |
| C | -0.20 | 0.45 | 1.00 |

### Echelle recommandee

- Rouge : -1 (correlation negative)
- Blanc : 0 (pas de correlation)
- Vert : +1 (correlation positive)

## Astuces avancees

### Masquer les valeurs

1. Selectionnez les cellules
2. Format personnalise : `;;;`
3. Seules les couleurs apparaissent

### Bordures pour lisibilite

Ajoutez des bordures blanches fines pour separer les cellules.

### Legende

Creez une legende a cote expliquant les couleurs.

## Cas pratique : Ventes par region et mois

### Donnees

| Region | Jan | Fev | Mar |
|--------|-----|-----|-----|
| Nord | 150 | 180 | 200 |
| Sud | 120 | 140 | 160 |
| Est | 90 | 100 | 110 |
| Ouest | 200 | 220 | 250 |

### Resultat

Les cellules a forte valeur apparaissent en vert, les faibles en rouge.

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Coherence | Meme echelle pour comparaisons |
| Lisibilite | Pas trop de nuances |
| Legende | Expliquez les couleurs |
| Accessibilite | Evitez rouge-vert seul |
