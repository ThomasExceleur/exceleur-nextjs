# Supprimer les Lignes Vides dans Excel

Les lignes vides peuvent perturber vos analyses. Voici comment les supprimer.

## Methode 1 : Atteindre les cellules vides

### Etapes

1. Selectionnez votre plage de donnees
2. **Ctrl + G** (Atteindre)
3. Cliquez sur **Cellules...**
4. Selectionnez **Cellules vides**
5. **OK**
6. Clic droit > **Supprimer** > **Ligne entiere**

### Raccourci

- **Ctrl + G** puis **Cellules...** puis **Cellules vides**

## Methode 2 : Filtrer et supprimer

### Etapes

1. Selectionnez vos donnees
2. **Ctrl + Maj + L** (activer les filtres)
3. Cliquez sur la fleche du filtre
4. Decochez **(Selectionner tout)**
5. Cochez **(Vides)**
6. Selectionnez les lignes filtrees
7. Clic droit > **Supprimer la ligne**
8. Desactivez le filtre

## Methode 3 : Tri pour regrouper

### Etapes

1. Triez vos donnees sur une colonne cle
2. Les lignes vides se retrouvent en bas ou en haut
3. Selectionnez-les
4. Supprimez-les

## Methode 4 : Rechercher-Remplacer

### Pour une colonne specifique

1. Selectionnez la colonne
2. **Ctrl + H**
3. Rechercher : laissez vide
4. Options > Cellules entieres
5. **Rechercher tout**
6. **Ctrl + A** pour selectionner tous les resultats
7. Fermez et supprimez les lignes

## Methode 5 : Power Query

### Pour les grands volumes

1. **Donnees** > **A partir d'un tableau/plage**
2. Dans Power Query, filtrez la colonne
3. Decochez "null" et les vides
4. **Fermer et charger**

## VBA : Macro pour automatiser

### Code

```vba
Sub SupprimerLignesVides()
    Dim i As Long
    For i = Cells(Rows.Count, 1).End(xlUp).Row To 1 Step -1
        If Application.CountA(Rows(i)) = 0 Then
            Rows(i).Delete
        End If
    Next i
End Sub
```

## Quelle methode choisir ?

| Methode | Ideal pour |
|---------|------------|
| Atteindre | Petits tableaux |
| Filtrer | Tableaux avec en-tetes |
| Tri | Donnees triables |
| Power Query | Grands volumes |
| VBA | Automatisation |

## Attention

### Verifier avant de supprimer

Certaines lignes peuvent sembler vides mais contenir :
- Des espaces
- Des formules retournant ""
- Des mises en forme

### Utiliser NB.VIDE

```excel
=NB.VIDE(A1:Z1)=NBVAL($1:$1)    ' VRAI si ligne vide
```

## Prevenir les lignes vides

1. Utilisez des tableaux structures (**Ctrl + T**)
2. Validez les saisies
3. Nettoyez regulierement vos donnees

## Erreurs courantes

### Supprimer des lignes avec des formules

Les formules retournant "" ne sont pas vides.

### Oublier de verifier

Verifiez toujours le resultat apres suppression.
