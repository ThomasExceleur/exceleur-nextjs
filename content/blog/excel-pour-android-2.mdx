# Excel pour Android

Excel sur votre smartphone ou tablette.

## Installation

### Play Store

1. Ouvrez le Play Store
2. Recherchez "Microsoft Excel"
3. Installez

## Gratuit ou payant ?

### Gratuit sur

Appareils de moins de 10,1 pouces.

### Payant

- Tablettes > 10,1 pouces
- Fonctions premium

## Fonctionnalites

### Incluses

| Fonction | Disponible |
|----------|------------|
| Formules de base | Oui |
| Graphiques | Oui |
| Tableaux | Oui |
| Partage | Oui |

### Limitees

- Macros VBA
- Certaines fonctions avancees
- Power Query

## Synchronisation

### OneDrive

Vos fichiers se synchronisent automatiquement.

### Hors ligne

Marquez les fichiers disponibles hors connexion.

## Interface tactile

### Adaptee

- Boutons plus grands
- Gestes tactiles
- Clavier adapte
