# Nettoyer vos Donnees Efficacement

Le nettoyage des donnees est essentiel pour des analyses fiables.

## Problemes courants

### Types de problemes

| Probleme | Description |
|----------|-------------|
| Espaces | Espaces avant/apres le texte |
| Doublons | Lignes identiques |
| Formats | Texte au lieu de nombres |
| Casse | Majuscules/minuscules |
| Vides | Cellules vides |

## Formules de nettoyage

### Supprimer les espaces

```excel
=SUPPRESPACE(A1)
```

Supprime les espaces en debut, fin et doubles.

### Uniformiser la casse

```excel
=MAJUSCULE(A1)        ' TOUT EN MAJUSCULE
=MINUSCULE(A1)        ' tout en minuscule
=NOMPROPRE(A1)        ' Premiere Lettre Majuscule
```

### Nettoyer les caracteres

```excel
=EPURAGE(A1)          ' Supprime les non-imprimables
```

### Combiner le nettoyage

```excel
=SUPPRESPACE(EPURAGE(MINUSCULE(A1)))
```

## Supprimer les doublons

### Methode manuelle

1. Selectionnez les donnees
2. **Donnees** > **Supprimer les doublons**
3. Choisissez les colonnes
4. **OK**

### Avec formule (marquer)

```excel
=SI(NB.SI($A$1:A1;A1)>1;"Doublon";"Unique")
```

## Convertir texte en nombre

### Methode Convertir

1. Selectionnez la colonne
2. **Donnees** > **Convertir**
3. **Terminer**

### Avec formule

```excel
=CNUM(A1)
```

### Multiplier par 1

```excel
=A1*1
```

## Power Query

### Avantages

- Process automatisable
- Interface visuelle
- Transformations multiples

### Etapes courantes

1. **Donnees** > **A partir d'un tableau**
2. Dans l'editeur, utilisez :
   - **Supprimer les lignes** > **Supprimer les doublons**
   - **Transformer** > **Nettoyer**
   - **Transformer** > **Supprimer les espaces**

## Remplir les cellules vides

### Avec Atteindre

1. Selectionnez la plage
2. **Ctrl + G** > **Special** > **Cellules vides**
3. Tapez `=` puis la cellule au-dessus
4. **Ctrl + Entree**

### Avec Power Query

**Transformer** > **Remplir** > **Vers le bas**

## Standardiser les dates

### Format date

```excel
=DATEVAL(A1)
```

### Format texte

```excel
=TEXTE(A1;"JJ/MM/AAAA")
```

## Checklist de nettoyage

1. Supprimer les doublons
2. Nettoyer les espaces
3. Uniformiser la casse
4. Verifier les formats
5. Remplir les vides
6. Valider les resultats

## Bonnes pratiques

1. **Sauvegardez** avant de nettoyer
2. **Documentez** les transformations
3. **Automatisez** avec Power Query
4. **Verifiez** apres chaque etape
