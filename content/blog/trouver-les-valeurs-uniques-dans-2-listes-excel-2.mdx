# Trouver les Valeurs Uniques entre 2 Listes

Identifiez les differences entre vos donnees.

## Valeurs dans Liste1 pas dans Liste2

### FILTRE + ESTNA

```excel
=FILTRE(Liste1;ESTNA(EQUIV(Liste1;Liste2;0)))
```

## Valeurs dans Liste2 pas dans Liste1

```excel
=FILTRE(Liste2;ESTNA(EQUIV(Liste2;Liste1;0)))
```

## Mise en forme conditionnelle

### Surligner les uniques

1. Selectionnez la Liste1
2. **Mise en forme conditionnelle** > **Nouvelle regle**
3. Formule : `=ESTNA(EQUIV(A1;Liste2;0))`

## NB.SI pour compter

### Formule

```excel
=SI(NB.SI(Liste2;A1)=0;"Unique";"Commun")
```

## RECHERCHEV classique

### Test d'existence

```excel
=SIERREUR(RECHERCHEV(A1;Liste2;1;FAUX);"Non trouve")
```

## Toutes les valeurs uniques

### Combiner et dedupliquer

```excel
=UNIQUE(ASSEMB.V(Liste1;Liste2))
```
