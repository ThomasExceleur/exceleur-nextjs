# Optimisez vos Donnees avec FILTRE

Extrayez et analysez vos donnees efficacement.

## Syntaxe rappel

```excel
=FILTRE(tableau;inclure;[si_vide])
```

## Strategies d'optimisation

### Pre-filtrage

Filtrez d'abord les donnees volumineuses.

### Combinaison avec calculs

```excel
=SOMME(FILTRE(Montants;Region=\"Nord\"))
```

### Tableaux structures

Utilisez des tableaux pour des references dynamiques.

## Filtres cascades

```excel
=LET(
  etape1;FILTRE(Donnees;Annee=2024);
  etape2;FILTRE(etape1;Region=\"Nord\");
  etape2
)
```

## Avec agregations

### Somme filtree

```excel
=SOMME(FILTRE(Ventes[Montant];Ventes[Region]=E1))
```

### Moyenne filtree

```excel
=MOYENNE(FILTRE(Notes;Classe=\"A\"))
```

### Compte filtre

```excel
=LIGNES(FILTRE(Donnees;Statut=\"Actif\"))
```
