# Remplir les Cellules Vides

Completez vos donnees automatiquement.

## Methode Atteindre

### Etapes

1. Selectionnez la plage
2. **Ctrl+G** ou **F5**
3. **Cellules** > **Cellules vides**
4. Tapez la formule (ex: `=A1`)
5. **Ctrl+Entree**

## Remplir avec la valeur au-dessus

### Formule

```excel
=A1
```

Puis Ctrl+Entree pour toutes les cellules vides.

## Remplir avec une valeur fixe

### Etapes

1. Selectionnez les cellules vides
2. Tapez la valeur
3. **Ctrl+Entree**

## Power Query

### Methode

1. **Donnees** > **A partir d'un tableau**
2. Selectionnez la colonne
3. **Transformer** > **Remplir** > **Vers le bas**

## Remplacer par zero

### Rechercher/Remplacer

1. **Ctrl+H**
2. **Rechercher** : (laisser vide)
3. **Remplacer** : 0
4. **Options** > **Cellule entiere**

## Cas d'usage

| Situation | Solution |
|-----------|----------|
| Donnees groupees | Valeur au-dessus |
| Calculs | Zero |
| Texte | "N/A" ou "-" |

## Attention

### Formules vs valeurs

Apres Ctrl+Entree, vous avez des formules.
Pour des valeurs : Copier > Collage special > Valeurs.

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Sauvegarde | Avant modification |
| Verification | Controlez le resultat |
