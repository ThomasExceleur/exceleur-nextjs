# Les Filtres Segments

Filtrez visuellement vos donnees.

## Qu'est-ce qu'un segment ?

### Definition

Boutons visuels pour filtrer les TCD et tableaux.

### Avantages

- Visuels et intuitifs
- Multi-selection
- Connectables entre TCD

## Creer un segment

### Pour TCD

1. Cliquez dans le TCD
2. **Analyse** > **Inserer un segment**
3. Selectionnez les champs

### Pour tableaux

1. Cliquez dans le tableau
2. **Creation** > **Inserer un segment**

## Utiliser les segments

### Selection simple

Cliquez sur un bouton.

### Multi-selection

Ctrl+Clic pour plusieurs.

### Tout deselectionner

Cliquez sur l'icone de filtre.

## Personnaliser

### Colonnes

Ajustez le nombre de colonnes de boutons.

### Style

**Outils Segment** > **Styles**

### Taille

Redimensionnez le segment.

## Connecter plusieurs TCD

### Etapes

1. Clic droit sur le segment
2. **Connexions de rapport**
3. Cochez les TCD a connecter

## Chronologie

### Pour les dates

**Analyse** > **Inserer une chronologie**

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Position | En haut ou a gauche |
| Style | Coherent avec le theme |
| Connexion | Liez les TCD connexes |
