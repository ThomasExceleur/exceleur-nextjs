# Les Filtres Segments

Filtrez visuellement vos données.

## Qu'est-ce qu'un segment ?

### Définition

Boutons visuels pour filtrer les TCD et tableaux.

### Avantages

- Visuels et intuitifs
- Multi-sélection
- Connectables entre TCD

## Créer un segment

### Pour TCD

1. Cliquez dans le TCD
2. **Analyse** > **Insérer un segment**
3. Sélectionnez les champs

### Pour tableaux

1. Cliquez dans le tableau
2. **Création** > **Insérer un segment**

## Utiliser les segments

### Sélection simple

Cliquez sûr un bouton.

### Multi-sélection

Ctrl+Clic pour plusieurs.

### Tout deselectionner

Cliquez sûr l'icone de filtre.

## Personnaliser

### Colonnes

Ajustez le nombre de colonnes de boutons.

### Style

**Outils Segment** > **Styles**

### Taille

Redimensionnez le segment.

## Connecter plusieurs TCD

### Étapes

1. Clic droit sûr le segment
2. **Connexions de rapport**
3. Cochez les TCD à connecter

## Chronologie

### Pour les dates

**Analyse** > **Insérer une chronologie**

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Position | En haut où à gauche |
| Style | Cohérent avec le theme |
| Connexion | Liez les TCD connexes |
