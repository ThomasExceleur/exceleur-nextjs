# Les Bases d'Excel pour Debutants

Ce guide vous apprend les fondamentaux pour bien demarrer avec Excel.

## L'interface Excel

### Le ruban

La barre d'outils en haut contenant tous les menus et fonctionnalites.

### Les onglets du ruban

- **Fichier** : Ouvrir, enregistrer, imprimer
- **Accueil** : Mise en forme, copier-coller
- **Insertion** : Graphiques, tableaux, images
- **Formules** : Fonctions et calculs
- **Donnees** : Tri, filtres, validation

### La barre de formule

Affiche le contenu de la cellule active. Vous pouvez y saisir et modifier les formules.

### Les feuilles

En bas de l'ecran, les onglets permettent de naviguer entre les feuilles.

## Navigation de base

### Cellules

L'intersection d'une colonne (A, B, C...) et d'une ligne (1, 2, 3...).

Exemple : A1, B5, Z100

### Selectionner une cellule

Cliquez dessus ou utilisez les fleches du clavier.

### Selectionner une plage

1. Cliquez sur la premiere cellule
2. Maintenez Maj et cliquez sur la derniere
3. Ou faites glisser la souris

### Raccourcis de navigation

| Raccourci | Action |
|-----------|--------|
| Ctrl + Debut | Aller en A1 |
| Ctrl + Fin | Derniere cellule utilisee |
| Ctrl + Fleche | Saut au bord des donnees |
| Ctrl + G | Aller a une cellule |

## Saisie de donnees

### Texte

Tapez directement dans la cellule. Le texte s'aligne a gauche.

### Nombres

Tapez le nombre. Il s'aligne a droite automatiquement.

### Dates

Tapez au format JJ/MM/AAAA (ex: 15/01/2024).

### Valider la saisie

- **Entree** : Valide et descend
- **Tab** : Valide et va a droite
- **Echap** : Annule la saisie

## Mise en forme basique

### Gras, italique, souligne

- **Ctrl + G** : Gras
- **Ctrl + I** : Italique
- **Ctrl + U** : Souligne

### Alignement

Boutons dans l'onglet Accueil pour aligner gauche, centre, droite.

### Couleur de remplissage

Icone pot de peinture dans le ruban.

### Bordures

Icone bordures pour encadrer les cellules.

## Premieres formules

### Structure d'une formule

Toute formule commence par **=**

```excel
=A1+B1
=2*3
=SOMME(A1:A10)
```

### Operations de base

| Operation | Symbole | Exemple |
|-----------|---------|---------|
| Addition | + | =A1+B1 |
| Soustraction | - | =A1-B1 |
| Multiplication | * | =A1*B1 |
| Division | / | =A1/B1 |

### Fonction SOMME

```excel
=SOMME(A1:A10)
```

Additionne les cellules de A1 a A10.

### Fonction MOYENNE

```excel
=MOYENNE(B1:B20)
```

Calcule la moyenne des valeurs.

## Copier-Coller

### Raccourcis

- **Ctrl + C** : Copier
- **Ctrl + X** : Couper
- **Ctrl + V** : Coller

### Copier une formule

1. Copiez la cellule
2. Selectionnez la destination
3. Collez

Les references s'adaptent automatiquement.

### Collage special

**Ctrl + Alt + V** pour choisir :
- Valeurs (sans formules)
- Formats
- Formules

## Trier les donnees

### Tri simple

1. Cliquez dans la colonne a trier
2. **Donnees** > **Trier de A a Z** ou **Z a A**

### Tri multicritere

1. Selectionnez toutes les donnees
2. **Donnees** > **Trier**
3. Ajoutez plusieurs niveaux

## Filtrer les donnees

### Activer les filtres

1. Selectionnez vos donnees
2. **Donnees** > **Filtrer**
3. Des fleches apparaissent sur les en-tetes

### Utiliser un filtre

1. Cliquez sur la fleche
2. Decochez les valeurs a masquer
3. Ou utilisez les options de filtre

## Enregistrer votre travail

### Formats de fichier

- **.xlsx** : Format standard
- **.xls** : Ancien format
- **.xlsm** : Avec macros

### Enregistrer

- **Ctrl + S** : Enregistrer
- **F12** : Enregistrer sous

## Raccourcis essentiels

| Raccourci | Action |
|-----------|--------|
| Ctrl + S | Enregistrer |
| Ctrl + Z | Annuler |
| Ctrl + Y | Retablir |
| Ctrl + C | Copier |
| Ctrl + V | Coller |
| Ctrl + B | Gras |
| F2 | Modifier la cellule |

## Erreurs courantes des debutants

### Oublier le =

Les formules DOIVENT commencer par =.

### Confondre , et ;

En France, utilisez ; comme separateur de formule.

### Ne pas sauvegarder

Sauvegardez regulierement avec Ctrl + S.
