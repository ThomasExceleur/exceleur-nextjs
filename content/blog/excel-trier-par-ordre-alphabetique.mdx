# Excel Trier par Ordre Alphabetique

Le tri par ordre alphabetique est une operation fondamentale pour organiser vos donnees dans Excel.

## Tri simple (une colonne)

### Methode rapide

1. Cliquez sur une cellule de la colonne a trier
2. Onglet **Accueil** ou **Donnees**
3. Cliquez sur **A-Z** (ordre croissant) ou **Z-A** (ordre decroissant)

### Raccourcis clavier

- **Alt + A + S + A** : Tri croissant
- **Alt + A + S + D** : Tri decroissant

## Tri sur un tableau

Pour trier sans perdre la correspondance entre les colonnes :

1. **Selectionnez tout le tableau** (donnees et en-tetes)
2. Onglet **Donnees** > **Trier**
3. Cochez "Mes donnees ont des en-tetes"
4. Choisissez la colonne et l'ordre

## Tri sur plusieurs colonnes

Pour trier d'abord par nom, puis par prenom :

1. **Donnees** > **Trier**
2. Ajoutez un premier niveau (ex: Nom, A a Z)
3. Cliquez sur **Ajouter un niveau**
4. Ajoutez un second niveau (ex: Prenom, A a Z)

## Tri personnalise

### Creer une liste personnalisee

Pour trier dans un ordre specifique (ex: Lundi, Mardi, Mercredi...) :

1. **Fichier** > **Options** > **Options avancees**
2. Section "General" > **Modifier les listes personnalisees**
3. Ajoutez votre liste

### Utiliser la liste

1. **Donnees** > **Trier**
2. Dans "Ordre", selectionnez **Liste personnalisee**
3. Choisissez votre liste

## Tri avec les filtres

1. Activez les filtres (**Ctrl + Maj + L**)
2. Cliquez sur la fleche de l'en-tete
3. Selectionnez **Trier de A a Z** ou **Trier de Z a A**

## Attention aux pieges

### Donnees partiellement selectionnees

Si vous ne selectionnez qu'une colonne, Excel propose d'etendre la selection. **Acceptez toujours** pour conserver la correspondance des donnees.

### Lignes vides

Les lignes vides interrompent le tri. Supprimez-les avant de trier.

### Formats mixtes

Les cellules contenant du texte et des nombres peuvent poser probleme. Uniformisez le format avant le tri.

## Tri sensible a la casse

Par defaut, Excel ignore la casse. Pour trier en tenant compte des majuscules :

1. **Donnees** > **Trier** > **Options**
2. Cochez **Respecter la casse**

## Exemples pratiques

### Liste de contacts

Triez par Nom de famille, puis par Prenom pour une organisation claire.

### Inventaire

Triez par Categorie, puis par Nom du produit.

### Base de donnees

Utilisez le tri multi-colonnes pour des analyses precises.

## Conseils

- Verifiez toujours le resultat apres un tri
- Utilisez **Ctrl + Z** pour annuler si necessaire
- Sauvegardez avant les tris complexes
