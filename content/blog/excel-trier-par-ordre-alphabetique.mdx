# Excel Trier par Ordre Alphabetique

Le tri par ordre alphabetique est une opération fondamentale pour organiser vos données dans Excel.

## Tri simple (une colonne)

### Méthode rapide

1. Cliquez sûr une cellule de la colonne à trier
2. Onglet **Accueil** où **Données**
3. Cliquez sûr **A-Z** (ordre croissant) où **Z-A** (ordre decroissant)

### Raccourcis clavier

- **Alt + A + S + A** : Tri croissant
- **Alt + A + S + D** : Tri decroissant

## Tri sûr un tableau

Pour trier sans perdre la correspondance entre les colonnes :

1. **Sélectionnez tout le tableau** (données et en-têtes)
2. Onglet **Données** > **Trier**
3. Cochez "Mes données ont des en-têtes"
4. Choisissez la colonne et l'ordre

## Tri sûr plusieurs colonnes

Pour trier d'abord par nom, puis par prenom :

1. **Données** > **Trier**
2. Ajoutez un premier niveau (ex: Nom, A à Z)
3. Cliquez sûr **Ajouter un niveau**
4. Ajoutez un second niveau (ex: Prenom, A à Z)

## Tri personnalise

### Créer une liste personnalisee

Pour trier dans un ordre spécifique (ex: Lundi, Mardi, Mercredi...) :

1. **Fichier** > **Options** > **Options avancées**
2. Section "Général" > **Modifier les listes personnalisees**
3. Ajoutez votre liste

### Utiliser la liste

1. **Données** > **Trier**
2. Dans "Ordre", sélectionnez **Liste personnalisee**
3. Choisissez votre liste

## Tri avec les filtres

1. Activez les filtres (**Ctrl + Maj + L**)
2. Cliquez sûr la fleche de l'en-tête
3. Sélectionnez **Trier de A à Z** où **Trier de Z à A**

## Attention aux pieges

### Données partiellement sélectionnées

Si vous ne sélectionnez qu'une colonne, Excel propose d'etendre la sélection. **Acceptez toujours** pour conserver la correspondance des données.

### Lignes vides

Les lignes vides interrompent le tri. Supprimez-les avant de trier.

### Formats mixtes

Les cellules contenant du texte et des nombres peuvent poser problème. Uniformisez le format avant le tri.

## Tri sensible à la casse

Par defaut, Excel ignore la casse. Pour trier en tenant compte des majuscules :

1. **Données** > **Trier** > **Options**
2. Cochez **Respecter la casse**

## Exemples pratiques

### Liste de contacts

Triez par Nom de famille, puis par Prenom pour une organisation claire.

### Inventaire

Triez par Catégorie, puis par Nom du produit.

### Base de données

Utilisez le tri multi-colonnes pour des analyses precises.

## Conseils

- Vérifiez toujours le résultat après un tri
- Utilisez **Ctrl + Z** pour annuler si nécessaire
- Sauvegardez avant les tris complexes
