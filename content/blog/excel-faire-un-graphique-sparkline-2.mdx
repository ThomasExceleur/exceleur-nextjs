# Graphiques Sparkline Excel

Des mini-graphiques directement dans vos cellules.

## Qu'est-ce qu'un Sparkline ?

### Definition

Petit graphique tenant dans une seule cellule.

### Avantages

- Vue rapide des tendances
- Compact
- Contextuel

## Types de Sparklines

| Type | Usage |
|------|-------|
| Courbes | Tendances temporelles |
| Colonnes | Comparaisons |
| Positif/Negatif | Variations |

## Creer un Sparkline

### Etapes

1. Selectionnez la cellule de destination
2. **Insertion** > **Sparklines**
3. Choisissez le type
4. Selectionnez la plage de donnees

## Personnaliser

### Couleurs

1. Cliquez sur le Sparkline
2. **Creation** > **Couleur du sparkline**

### Points speciaux

| Point | Description |
|-------|-------------|
| Point haut | Valeur maximale |
| Point bas | Valeur minimale |
| Premier/Dernier | Debut et fin |
| Negatifs | Valeurs negatives |

### Activer les points

**Creation** > Cochez les points souhaites

## Modifier l'echelle

### Options

1. Clic droit > **Sparkline**
2. **Axe**
3. Personnalisez min/max

## Regrouper les Sparklines

### Avantage

Meme echelle pour comparer.

### Methode

Selectionnez tous les Sparklines > modifiez l'axe.

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Echelle commune | Pour comparaison |
| Points visibles | Mettez en avant les extremes |
| Simplicite | Evitez la surcharge |
