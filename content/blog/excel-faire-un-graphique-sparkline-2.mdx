# Graphiques Sparkline Excel

Des mini-graphiques directement dans vos cellules.

## Qu'est-ce qu'un Sparkline ?

### Définition

Petit graphique tenant dans une seule cellule.

### Avantages

- Vue rapide des tendances
- Compact
- Contextuel

## Types de Sparklines

| Type | Usage |
|------|-------|
| Courbes | Tendances temporelles |
| Colonnes | Comparaisons |
| Positif/Negatif | Variations |

## Créer un Sparkline

### Étapes

1. Sélectionnez la cellule de destination
2. **Insertion** > **Sparklines**
3. Choisissez le type
4. Sélectionnez la plage de données

## Personnaliser

### Couleurs

1. Cliquez sûr le Sparkline
2. **Création** > **Couleur du sparkline**

### Points speciaux

| Point | Description |
|-------|-------------|
| Point haut | Valeur maximale |
| Point bas | Valeur minimale |
| Premier/Dernier | Début et fin |
| Negatifs | Valeurs negatives |

### Activer les points

**Création** > Cochez les points souhaités

## Modifier l'échelle

### Options

1. Clic droit > **Sparkline**
2. **Axe**
3. Personnalisez min/max

## Regrouper les Sparklines

### Avantage

Meme échelle pour comparer.

### Méthode

Sélectionnez tous les Sparklines > modifiez l'axe.

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Échelle commune | Pour comparaison |
| Points visibles | Mettez en avant les extremes |
| Simplicité | Évitez la surcharge |
