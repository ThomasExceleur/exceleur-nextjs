# Fonctions Financieres Excel pour l'Investissement

Evaluez vos projets d'investissement avec les outils Excel.

## VAN - Valeur Actuelle Nette

### Syntaxe

```excel
=VAN(taux;valeur1;valeur2;...)
```

### Exemple

```excel
=VAN(10%;CF1;CF2;CF3;CF4;CF5)+Investissement_initial
```

### Interpretation

VAN > 0 : Projet rentable

## TRI - Taux de Rendement Interne

### Syntaxe

```excel
=TRI(valeurs;[estimation])
```

### Exemple

```excel
=TRI(B1:B6)
```

B1 contient l'investissement initial (negatif).

### Interpretation

Taux auquel VAN = 0.

## VPM - Versement Periodique

### Syntaxe

```excel
=VPM(taux;npm;va;[vc];[type])
```

### Exemple

```excel
=VPM(5%/12;120;100000)
```

Mensualite d'un pret de 100 000 € sur 10 ans a 5%.

## VA - Valeur Actuelle

### Syntaxe

```excel
=VA(taux;npm;vpm;[vc];[type])
```

### Usage

Calculer combien vaut aujourd'hui une serie de paiements futurs.

## VC - Valeur Capitalisee

### Syntaxe

```excel
=VC(taux;npm;vpm;[va];[type])
```

### Exemple

```excel
=VC(8%/12;120;-500)
```

Valeur future d'un placement de 500 €/mois pendant 10 ans a 8%.

## TAUX

### Syntaxe

```excel
=TAUX(npm;vpm;va;[vc];[type];[estimation])
```

### Usage

Trouver le taux d'interet d'un pret ou investissement.

## NPM - Nombre de Periodes

### Syntaxe

```excel
=NPM(taux;vpm;va;[vc];[type])
```

### Usage

Calculer la duree necessaire pour atteindre un objectif.

## TRI.PAIEMENTS

### Pour flux irreguliers

```excel
=TRI.PAIEMENTS(valeurs;dates;[estimation])
```

## VAN.PAIEMENTS

### Pour flux irreguliers

```excel
=VAN.PAIEMENTS(taux;valeurs;dates)
```

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Coherence | Meme periodicite pour taux et npm |
| Signes | Negatif = sortie, Positif = entree |
| Verification | Comparez plusieurs methodes |
