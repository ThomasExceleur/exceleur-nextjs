# Tableaux Croises Dynamiques Expliques

Les TCD synthetisent et analysent vos donnees.

## Qu'est-ce qu'un TCD ?

### Definition

Un tableau croise dynamique resume de grandes quantites de donnees.

### Avantages

- Resume instantane
- Regroupements automatiques
- Mise a jour facile

## Creer un TCD

### Etapes

1. Selectionnez vos donnees
2. **Insertion** > **Tableau croise dynamique**
3. Choisissez l'emplacement

## Zones du TCD

| Zone | Role |
|------|------|
| Filtres | Filtrer l'ensemble |
| Colonnes | En-tetes horizontaux |
| Lignes | En-tetes verticaux |
| Valeurs | Donnees calculees |

## Configurer les valeurs

### Fonctions disponibles

| Fonction | Usage |
|----------|-------|
| Somme | Total des valeurs |
| Nombre | Compter les elements |
| Moyenne | Valeur moyenne |
| Max/Min | Extremes |

### Modifier

1. Clic droit sur une valeur
2. **Parametres des champs de valeurs**

## Actualiser

### Methodes

- **Clic droit** > **Actualiser**
- **Analyse** > **Actualiser**
- **Alt+F5**

## Mise en forme

### Styles

Appliquez des styles predifinis via **Creation**.

### Disposition

- Compacte
- Plan
- Tabulaire

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Donnees propres | Nettoyez avant de creer |
| En-tetes | Nommez vos colonnes |
| Source | Utilisez un tableau structure |
