# Proteger un Fichier Excel

Securisez vos fichiers Excel avec les bonnes méthodes.

## Types de protection

### Chiffrement

Mot de passe pour ouvrir le fichier.

### Modification

Mot de passe pour modifier (lecture seule sinon).

### Structuré

Empeche d'ajouter/supprimer des feuilles.

### Feuille

Empeche de modifier le contenu.

## Chiffrer le fichier

### Étapes

1. **Fichier** > **Informations**
2. **Proteger le classeur**
3. **Chiffrer avec mot de passe**
4. Entrez et confirmez

### Niveau de sécurité

Élève - le fichier est chiffre en AES.

### Attention

Mot de passe perdu = fichier irrecuperable.

## Protection en modification

### Étapes

1. **Fichier** > **Enregistrer sous**
2. **Outils** > **Options générales**
3. Renseignez les mots de passe

### Options

| Champ | Effet |
|-------|-------|
| Mot de passe ouverture | Impossible d'ouvrir |
| Mot de passe modification | Lecture seule sans mdp |
| Lecture seule recommandée | Suggestion à l'ouverture |

## Proteger la structuré

### Étapes

1. **Révision** > **Proteger le classeur**
2. Cochez **Structuré**
3. Mot de passe optionnel

### Effets

- Pas d'ajout/suppression de feuilles
- Pas de renommage
- Pas de masquage/affichage

## Proteger les feuilles

### Étapes

1. **Révision** > **Proteger la feuille**
2. Configurez les options
3. Mot de passe optionnel

### Cellules editables

1. Sélectionnez les cellules editables
2. Format > Protection > Decochez Verrouillee
3. Protégez la feuille

## Marquer comme final

### Usage

Indique que le document est termine.

### Étapes

1. **Fichier** > **Informations**
2. **Proteger le classeur**
3. **Marquer comme final**

### Note

Protection symbolique - facilement desactivable.

## Signature numérique

### Objectif

Garantir l'authenticite et l'integrite.

### Étapes

1. **Fichier** > **Informations**
2. **Proteger le classeur**
3. **Ajouter une signature numérique**

## Permissions IRM

### Information Rights Management

Contrôle avancé des permissions dans un environnement Microsoft 365.

### Fonctionnalités

- Empecher l'impression
- Empecher la copie
- Date d'expiration

## Comparaison des protections

| Type | Sécurité | Usage |
|------|----------|-------|
| Chiffrement | Élevée | Confidentialite |
| Modification | Moyenne | Collaboration |
| Structuré | Faible | Organisation |
| Feuille | Faible | Saisie controllee |
| Final | Très faible | Information |

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Plusieurs niveaux | Combinez les protections |
| Mots de passe forts | Min 12 caracteres varies |
| Documentation | Stockez les mdp securisement |
| Sauvegarde | Gardez une copie accessible |
