# Formule SI Avancee

Maitrisez les conditions complexes.

## SI imbrique

```excel
=SI(A1>=90;"A";SI(A1>=80;"B";SI(A1>=70;"C";"D")))
```

## SI.CONDITIONS (Excel 365)

### Plus lisible

```excel
=SI.CONDITIONS(A1>=90;"A";A1>=80;"B";A1>=70;"C";VRAI;"D")
```

## Combinaison ET/OU

### ET

```excel
=SI(ET(A1>0;B1>0);"Positifs";"Non")
```

### OU

```excel
=SI(OU(A1="Oui";B1="Oui");"Au moins un";"Aucun")
```

## NON

```excel
=SI(NON(A1="Non");"Oui";"Non")
```

## SI avec calculs

```excel
=SI(A1>100;A1*0.9;A1)
```

## CHOISIR alternative

```excel
=CHOISIR(A1;"Lundi";"Mardi";"Mercredi")
```

## Erreurs courantes

| Erreur | Cause |
|--------|-------|
| Parentheses | Compter ouvertures/fermetures |
| Guillemets | "texte" pas texte |
| Ordre | Du plus specifique au general |
