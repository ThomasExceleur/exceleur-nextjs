# Erreurs Communes RECHERCHEV

Évitez les pieges classiques.

## Erreur #N/A

### Causes

| Cause | Solution |
|-------|----------|
| Valeur absente | Vérifiez les données |
| Espaces invisibles | SUPPRESPACE |
| Format différent | Nombre vs Texte |

### Solution

```excel
=SIERREUR(RECHERCHEV(...);"Non trouve")
```

## Erreur #REF!

### Cause

Index de colonne supérieur au nombre de colonnes.

### Exemple

```excel
=RECHERCHEV(A1;B:C;5;FAUX)
```

5 > 2 colonnes = #REF!

## Mauvais résultat

### Cause frequente

4eme argument manquant où VRAI.

### Solution

Toujours specifier FAUX pour correspondance exacte.

```excel
=RECHERCHEV(A1;B:C;2;FAUX)
```

## Espace invisible

### Problème

"Paris " n'est pas "Paris".

### Detection

```excel
=NBCAR(A1)
```

### Correction

```excel
=RECHERCHEV(SUPPRESPACE(A1);...)
```

## Texte vs Nombre

### Problème

"123" (texte) != 123 (nombre).

### Detection

```excel
=ESTNUM(A1)
=ESTTEXTE(A1)
```

### Correction

```excel
=RECHERCHEV(CNUM(A1);...)
=RECHERCHEV(TEXTE(A1;"0");...)
```

## Index dynamique

### Problème

Insertion de colonne casse la formule.

### Solution

```excel
=RECHERCHEV(A1;tableau;EQUIV("Colonne";headers;0);FAUX)
```

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| FAUX toujours | Correspondance exacte |
| SIERREUR | Gérer les erreurs |
| Nettoyage | SUPPRESPACE avant recherche |
| Test | Vérifiez avec des exemples connus |
