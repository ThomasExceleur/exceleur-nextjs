# C'est Quoi INDEX EQUIV

La combinaison puissante pour les recherches.

## Les deux fonctions

### INDEX

Retourne une valeur a une position.

```excel
=INDEX(plage;numero_ligne)
```

### EQUIV

Trouve la position d'une valeur.

```excel
=EQUIV(valeur;plage;0)
```

## Combinaison

```excel
=INDEX(plage_resultat;EQUIV(cherche;plage_recherche;0))
```

## Exemple

### Donnees

| Nom | Salaire |
|-----|---------|
| Jean | 3000 |
| Marie | 3500 |

### Formule

```excel
=INDEX(B:B;EQUIV("Marie";A:A;0))
```

Resultat : 3500

## Vs RECHERCHEV

| Critere | INDEX+EQUIV | RECHERCHEV |
|---------|-------------|------------|
| Direction | Les deux | Droite |
| Flexibilite | Haute | Limitee |

## Avantages

- Recherche bidirectionnelle
- Plus performant
- Plus flexible

| Pratique | Description |
|----------|-------------|
