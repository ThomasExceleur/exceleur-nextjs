# Excel Multiplication : Toutes les Méthodes

La multiplication est l'une des opérations les plus courantes dans Excel. Voici toutes les façons de la realiser.

## Méthode 1 : L'opérateur *

La méthode la plus simple :

```excel
=A1*B1
```

### Exemples

```excel
=5*3           ' Résultat: 15
=A1*10         ' Multiplie A1 par 10
=A1*B1*C1      ' Multiplie trois cellules
```

## Méthode 2 : La fonction PRODUIT

Pour multiplier plusieurs valeurs :

```excel
=PRODUIT(nombre1; nombre2; ...)
```

### Exemples

```excel
=PRODUIT(A1:A10)       ' Multiplie toutes les valeurs de A1 à A10
=PRODUIT(A1;B1;C1)     ' Multiplie trois cellules
=PRODUIT(2;3;4;5)      ' Résultat: 120
```

### Avantage sûr l'opérateur *

PRODUIT ignore les cellules vides et les textes, evitant les erreurs.

## Multiplier une colonne par un nombre fixe

### Avec références absolues

Pour multiplier toute une colonne par un taux fixe (ex: TVA 20%) :

```excel
=A1*$B$1
```

Le signe **$** fige la référence lors de la copie.

## Multiplier des plages

### Méthode SOMMEPROD

Pour multiplier élément par élément et sommer :

```excel
=SOMMEPROD(A1:A10;B1:B10)
```

Equivalent à : `A1*B1 + A2*B2 + ... + A10*B10`

## Multiplication matricielle

### Fonction PRODUITMAT

Pour les calculs matriciels (algebre lineaire) :

```excel
=PRODUITMAT(matrice1;matrice2)
```

**Note** : Le nombre de colonnes de matrice1 doit egaler le nombre de lignes de matrice2.

## Cas pratiques

### Calcul de TVA

```excel
=Prix_HT * 1,20
```

### Prix total

```excel
=Quantité * Prix_unitaire
```

### Conversion de devises

```excel
=Montant_EUR * Taux_change
```

### Pourcentage

```excel
=Valeur * 15%    ' Calcule 15% de la valeur
```

## Erreurs courantes

### #VALEUR!

**Cause** : Multiplication avec du texte

**Solution** : Vérifiez que toutes les cellules contiennent des nombres

### Résultat 0 inattendu

**Cause** : Cellule vide (traitee comme 0)

**Solution** : Utilisez SI pour gérer les cas vides

```excel
=SI(B1="";"";A1*B1)
```

## Collage special : Multiplier des valeurs existantes

Pour multiplier des valeurs en place :

1. Entrez le multiplicateur dans une cellule (ex: 1,1)
2. Copiez cette cellule (**Ctrl + C**)
3. Sélectionnez les cellules à modifier
4. **Collage special** (**Ctrl + Alt + V**)
5. Cochez **Multiplication** > OK
