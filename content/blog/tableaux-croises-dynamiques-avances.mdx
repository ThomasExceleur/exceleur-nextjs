# Tableaux Croises Dynamiques Avances

Allez au-dela des fonctionnalites de base des TCD.

## Champs calcules

### Creer

1. Cliquez dans le TCD
2. **Analyse** > **Champs, elements et jeux**
3. **Champ calcule**

### Exemple

```
Marge = Ventes - Couts
Taux = Marge / Ventes
```

## Elements calcules

### Usage

Creer des regroupements personnalises dans les categories.

## Afficher les valeurs comme

### Options

| Option | Description |
|--------|-------------|
| % du total general | Part du total |
| % du total de la ligne | Part horizontale |
| % du total de la colonne | Part verticale |
| Difference | Ecart vs precedent |
| % de difference | Ecart en % |
| Total cumule | Somme progressive |
| Rang | Classement |

## Grouper les donnees

### Dates

Clic droit > Grouper > Annees, Trimestres, Mois

### Nombres

Grouper par tranches (0-100, 100-200, etc.)

## Segments connectes

### Relier plusieurs TCD

1. Clic droit sur le segment
2. Connexions de rapport
3. Cochez les TCD

## Modele de donnees

### Avantage

Relations entre tables pour analyses croisees.
