# Tableaux Croisés Dynamiques Avancés

Allez au-dela des fonctionnalités de base des TCD.

## Champs calcules

### Créer

1. Cliquez dans le TCD
2. **Analyse** > **Champs, éléments et jeux**
3. **Champ calcule**

### Exemple

```
Marge = Ventes - Couts
Taux = Marge / Ventes
```

## Éléments calcules

### Usage

Créer des regroupements personnalises dans les catégories.

## Afficher les valeurs comme

### Options

| Option | Description |
|--------|-------------|
| % du total général | Part du total |
| % du total de la ligne | Part horizontale |
| % du total de la colonne | Part verticale |
| Différence | Ecart vs précédent |
| % de différence | Ecart en % |
| Total cumule | Somme progressive |
| Rang | Classement |

## Grouper les données

### Dates

Clic droit > Grouper > Années, Trimestres, Mois

### Nombres

Grouper par tranches (0-100, 100-200, etc.)

## Segments connectes

### Relier plusieurs TCD

1. Clic droit sûr le segment
2. Connexions de rapport
3. Cochez les TCD

## Modèle de données

### Avantage

Relations entre tables pour analyses croisées.

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Source tableau | Références dynamiques |
| Actualisation | Configurez l'auto-refresh |
| Noms clairs | Renommez les champs |
