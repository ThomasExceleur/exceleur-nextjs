# Remplacer une Couleur par une Autre

Modifiez les couleurs de vos cellules efficacement.

## Méthode Rechercher/Remplacer

### Étapes

1. **Ctrl+H** (Rechercher et remplacer)
2. **Options** > **Format**
3. Dans **Rechercher** : sélectionnez la couleur d'origine
4. Dans **Remplacer** : sélectionnez la nouvelle couleur
5. **Remplacer tout**

### Accès au format

Cliquez sûr **Format** à côté de chaque champ.

## Rechercher par couleur

### Étapes

1. **Format** > **Arriere-plan**
2. Sélectionnez la couleur
3. OK

## Limites

### Ce qui fonctionne

- Couleur de remplissage
- Couleur de police
- Bordures

### Ce qui ne fonctionne pas

- Mise en forme conditionnelle
- Couleurs de graphiques

## Mise en forme conditionnelle

### Modifier une règle

1. **Accueil** > **Mise en forme conditionnelle**
2. **Gérer les règles**
3. Sélectionnez et **Modifier**
4. Changez la couleur

## VBA pour gros volumes

### Code

```vba
Sub RemplacerCouleur()
    Dim c As Range
    For Each c In Sélection
        If c.Interior.Color = RGB(255, 0, 0) Then
            c.Interior.Color = RGB(0, 255, 0)
        End If
    Next c
End Sub
```

| Situation | Solution |
