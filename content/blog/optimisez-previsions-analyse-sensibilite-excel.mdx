# Optimisez vos Previsions avec l'Analyse de Sensibilite Excel

L'analyse de sensibilite permet d'evaluer l'impact des variations sur vos resultats.

## Qu'est-ce que l'analyse de sensibilite ?

### Definition

L'analyse de sensibilite etudie comment les variations des variables d'entree affectent le resultat final.

### Applications

- Previsions financieres
- Budgetisation
- Gestion de projets
- Decisions d'investissement

## Outils Excel pour l'analyse de sensibilite

### Tables de donnees

| Type | Usage |
|------|-------|
| Une variable | Impact d'un facteur |
| Deux variables | Croisement de facteurs |

### Gestionnaire de scenarios

Comparez plusieurs hypotheses simultanement.

### Valeur cible

Trouvez la valeur d'entree pour un resultat cible.

## Creer une table de donnees a une variable

### Etape 1 : Preparer le modele

```excel
Cellule B1 : Variable (ex: taux d'interet)
Cellule B5 : Resultat (formule utilisant B1)
```

### Etape 2 : Configurer la table

1. Listez les valeurs a tester en colonne
2. Selectionnez la plage
3. **Donnees** > **Analyse de scenarios** > **Table de donnees**

### Etape 3 : Definir la cellule d'entree

Specifiez la cellule de la variable (B1).

## Table de donnees a deux variables

### Configuration

| | Valeur 1 | Valeur 2 | Valeur 3 |
|---------|----------|----------|----------|
| Facteur A | Resultat | Resultat | Resultat |
| Facteur B | Resultat | Resultat | Resultat |

### Utilisation

Analysez l'interaction entre deux variables.

## Gestionnaire de scenarios

### Creer un scenario

1. **Donnees** > **Analyse de scenarios** > **Gestionnaire de scenarios**
2. **Ajouter** pour creer un nouveau scenario
3. Nommez et definissez les cellules variables
4. Entrez les valeurs du scenario

### Synthese des scenarios

Generez un rapport comparatif de tous les scenarios.

## Exemple pratique : Prevision de ventes

### Modele de base

```excel
Prix unitaire : B1 = 100
Quantite : B2 = 1000
Cout variable : B3 = 60
Couts fixes : B4 = 20000
Profit : B5 = (B1-B3)*B2-B4
```

### Scenarios a tester

| Scenario | Prix | Quantite | Resultat |
|----------|------|----------|----------|
| Optimiste | 110 | 1200 | 40000 |
| Base | 100 | 1000 | 20000 |
| Pessimiste | 90 | 800 | 4000 |

## Interpreter les resultats

### Points cles

- Identifiez les variables les plus sensibles
- Evaluez les risques de chaque scenario
- Preparez des plans d'action

### Visualisation

Creez des graphiques pour illustrer l'impact des variations.

|----------|-------------|

| Plages realistes | Utilisez des valeurs probables |
| Validation | Verifiez la coherence |
| Presentation | Resumez les conclusions |
