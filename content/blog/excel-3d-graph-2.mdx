# Graphiques 3D Excel

Ajoutez de la profondeur a vos visualisations.

## Types de graphiques 3D

### Disponibles

| Type | Usage |
|------|-------|
| Colonnes 3D | Comparaisons |
| Barres 3D | Comparaisons horizontales |
| Secteurs 3D | Repartitions |
| Lignes 3D | Tendances |
| Aires 3D | Evolution |

## Creer un graphique 3D

### Etapes

1. Selectionnez vos donnees
2. **Insertion** > **Graphiques**
3. Choisissez une variante 3D

## Personnaliser la 3D

### Rotation

1. Clic droit sur le graphique
2. **Rotation 3D**
3. Ajustez les angles

### Options

| Option | Effet |
|--------|-------|
| X | Rotation horizontale |
| Y | Rotation verticale |
| Perspective | Profondeur |

## 3D Maps

### Pour donnees geographiques

**Insertion** > **3D Map**

## Avantages et inconvenients

### Avantages

- Visuellement attractif
- Impression de profondeur

### Inconvenients

- Peut deformer la perception
- Plus difficile a lire

## Recommandations

| Situation | Recommandation |
|-----------|----------------|
| Precision | Evitez 3D |
| Presentation | 3D acceptable |
| Analyse | 2D preferable |

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Sobriete | N'abusez pas |
| Lisibilite | Priorisez la clarte |
| Contexte | Adaptez au public |
