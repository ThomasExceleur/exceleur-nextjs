# Listes Deroulantes Excel 365

Profitez des nouvelles fonctionnalites.

## Liste simple

### Creation

1. Selectionnez la cellule
2. **Donnees** > **Validation des donnees**
3. **Liste**
4. Entrez les valeurs ou une plage

## Liste dynamique avec UNIQUE

### Avantage Excel 365

```excel
=UNIQUE(Tableau[Colonne])
```

La liste se met a jour automatiquement.

### Configuration

1. Creez la liste avec UNIQUE
2. Nommez la plage (Formules > Definir un nom)
3. Utilisez ce nom dans la validation

## Liste dependante

### Scenario

Pays > Villes

### Methode avec FILTRE

```excel
=FILTRE(Villes[Ville];Villes[Pays]=A1)
```

## INDIRECT pour listes dynamiques

### Syntaxe

```excel
=INDIRECT(A1)
```

Reference une plage nommee selon le choix.

## Recherche dans la liste

### Excel 365+

Tapez pour filtrer les options.

| Option | Description |
|--------|-------------|
