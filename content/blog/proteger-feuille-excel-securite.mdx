# Proteger une Feuille Excel : Securite

Controlez les modifications sur vos feuilles.

## Activer la protection

### Etapes

1. **Revision** > **Proteger la feuille**
2. Definissez un mot de passe (optionnel)
3. Configurez les options
4. **OK**

## Options de protection

| Option | Description |
|--------|-------------|
| Selectionner cellules verrouillees | Naviguer |
| Selectionner cellules deverrouillees | Saisie |
| Mettre en forme | Changer le format |
| Inserer lignes/colonnes | Ajouter |
| Supprimer lignes/colonnes | Retirer |
| Tri | Trier les donnees |
| Filtre auto | Utiliser les filtres |

## Cellules editables

### Deverrouiller

1. Selectionnez les cellules
2. Ctrl+1 > Protection
3. Decochez \"Verrouillee\"
4. Protegez la feuille

## Masquer les formules

1. Selectionnez les cellules
2. Ctrl+1 > Protection
3. Cochez \"Masquee\"
4. Protegez la feuille

## Retirer la protection

**Revision** > **Oter la protection de la feuille**

## Limites

### Attention

- Protection contre erreurs, pas securite absolue
- Des outils peuvent contourner
- Pour confidentialite : chiffrement

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Mot de passe | Fort et documente |
| Test | Verifiez les permissions |
| Sauvegarde | Copie non protegee |
