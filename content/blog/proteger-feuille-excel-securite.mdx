# Proteger une Feuille Excel : Sécurité

Controlez les modifications sûr vos feuilles.

## Activer la protection

### Étapes

1. **Révision** > **Proteger la feuille**
2. Définissez un mot de passe (optionnel)
3. Configurez les options
4. **OK**

## Options de protection

| Option | Description |
|--------|-------------|
| Sélectionner cellules verrouillees | Naviguer |
| Sélectionner cellules deverrouillees | Saisie |
| Mettre en forme | Changer le format |
| Insérer lignes/colonnes | Ajouter |
| Supprimer lignes/colonnes | Retirer |
| Tri | Trier les données |
| Filtre auto | Utiliser les filtres |

## Cellules editables

### Deverrouiller

1. Sélectionnez les cellules
2. Ctrl+1 > Protection
3. Decochez \"Verrouillee\"
4. Protégez la feuille

## Masquer les formules

1. Sélectionnez les cellules
2. Ctrl+1 > Protection
3. Cochez \"Masquee\"
4. Protégez la feuille

## Retirer la protection

**Révision** > **Oter la protection de la feuille**

## Limites

### Attention

- Protection contre erreurs, pas sécurité absolue
- Des outils peuvent contourner
- Pour confidentialite : chiffrement

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Mot de passe | Fort et documente |
| Test | Vérifiez les permissions |
| Sauvegarde | Copie non protegee |
