# Tableau Croise Dynamique Interactif

Rendez vos TCD interactifs avec segments et chronologies.

## Creer un TCD

### Etapes

1. Selectionnez vos donnees
2. **Insertion** > **Tableau croise dynamique**
3. Choisissez l'emplacement

## Ajouter des segments

### Insertion

1. Cliquez dans le TCD
2. **Analyse** > **Inserer un segment**
3. Selectionnez les champs

### Personnalisation

| Option | Description |
|--------|-------------|
| Colonnes | Nombre d'elements par ligne |
| Hauteur | Taille des boutons |
| Style | Theme visuel |

## Ajouter une chronologie

### Pour les dates

1. **Analyse** > **Inserer une chronologie**
2. Selectionnez le champ date

### Niveaux

- Annees
- Trimestres
- Mois
- Jours

## Connecter plusieurs TCD

### Connexions de rapport

1. Clic droit sur segment
2. **Connexions de rapport**
3. Selectionnez les TCD a connecter

## Mise en forme

### Styles de TCD

| Style | Usage |
|-------|-------|
| Clair | Lecture simple |
| Moyen | Professionnel |
| Fonce | Presentation |

### Segments assortis

Utilisez le meme theme pour coherence.

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Disposition | Segments en haut ou a gauche |
| Nommage | Renommez les segments |
| Taille | Ajustez pour lisibilite |
| Connexion | Liez tous les TCD relies |
