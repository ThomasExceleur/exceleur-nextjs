# Tableau de Bord KPI en Temps Reel

Dashboard qui se met a jour automatiquement.

## Sources de donnees

### Connexions temps reel

- Power Query vers base de donnees
- API REST
- SharePoint/OneDrive

### Actualisation

Configurez l'actualisation automatique.

## KPIs essentiels

### Structure

| KPI | Formule |
|-----|---------|
| Total | SOMME |
| Moyenne | MOYENNE |
| Variation | (Actuel-Precedent)/Precedent |
| Objectif | Reel vs Cible |

## Formules dynamiques

### KPI avec periode actuelle

```excel
=SOMME.SI(Dates;">="&DEBUT.MOIS(AUJOURDHUI();0);Valeurs)
```

### Comparaison vs mois precedent

```excel
=LET(
  actuel;SOMME.SI(Dates;">="&DEBUT.MOIS(AUJOURDHUI();0);Valeurs);
  precedent;SOMME.SI.ENS(Dates;">="&DEBUT.MOIS(AUJOURDHUI();-1);Dates;"<"&DEBUT.MOIS(AUJOURDHUI();0);Valeurs);
  variation;(actuel-precedent)/precedent;
  variation
)
```

## Indicateurs visuels

### Mise en forme conditionnelle

- Vert : objectif atteint
- Jaune : proche
- Rouge : en retard
