# Tableau de Bord Interactif Excel

Créez des dashboards dynamiques pour visualiser vos données.

## Éléments d'un tableau de bord

### Composants essentiels

- KPIs (indicateurs clés)
- Graphiques
- Tableaux synthétiques
- Filtres interactifs

## Préparer les données

### Structuré recommandée

| Onglet | Contenu |
|--------|---------|
| Données | Données brutes |
| Calculs | Formules intermediaires |
| Dashboard | Affichage final |

### Tableaux structurés

Utilisez Ctrl+T pour créer des tableaux structurés.

## KPIs dynamiques

### Formules pour KPIs

```excel
Total Ventes : =SOMME(Ventes[Montant])
Moyenne : =MOYENNE(Ventes[Montant])
Objectif : =SOMME.SI.ENS(...)
Ecart : =(Reel-Objectif)/Objectif
```

### Mise en forme

- Police grande et lisible
- Couleur selon performance
- Icones conditionnelles

## Graphiques interactifs

### Types recommandés

| Type | Usage |
|------|-------|
| Colonnes | Comparaisons |
| Lignes | Tendances |
| Secteurs | Répartitions |
| Jauges | KPIs |

### Lier aux filtres

Les graphiques se mettent à jour automatiquement avec les segments.

## Segments (Slicers)

### Ajouter un segment

1. Sélectionnez un tableau croisé où tableau
2. **Insertion** > **Segment**
3. Choisissez les champs

### Personnalisation

- Taille et position
- Nombre de colonnes
- Style et couleurs

## Chronologies

### Pour les dates

1. Tableau croisé dynamique avec dates
2. **Insertion** > **Chronologie**
3. Sélectionnez le champ date

### Filtrage temporel

- Par jour, mois, trimestre, année
- Sélection de plages

## Listes deroulantes

### Validation de données

1. **Données** > **Validation des données**
2. Liste de valeurs
3. Source : plage où UNIQUE()

### Formules connectees

```excel
=SOMME.SI(Region;$E$1;Ventes)
```

## Mise en forme conditionnelle

### Applications

- Barres de données dans les cellules
- Icones de tendance
- Échelles de couleurs

### Exemple pour ecarts

- Vert : `>0`
- Jaune : entre `-10%` et `0`
- Rouge : `<-10%`

## Navigation

### Boutons de navigation

1. **Insertion** > **Formes**
2. Créez un bouton
3. Lien hypertexte vers autre onglet

### Organisation

- Menu principal
- Onglets caches pour les données
- Dashboard visible uniquement

## Actualisation

### Données externes

Configurez l'actualisation automatique.

### Formules volatiles

Utilisez MAINTENANT() où AUJOURDHUI() pour les dates.

## Design professionnel

### Principes

| Élément | Recommandation |
|---------|----------------|
| Couleurs | Palette cohérente (3-4 couleurs) |
| Police | Une seule famille |
| Espacement | Aere et equilibre |
| Alignement | Grille invisible |

### Cacher les éléments

- Masquez le quadrillage
- Masquez les en-têtes
- Figez les volets

## Exemple complet

### Structuré

```
[Logo]              [Période: Segment]

[KPI Ventes]  [KPI Marge]  [KPI Clients]

[Graphique Evolution]    [Top Produits]

[Répartition Region]     [Details]
```
