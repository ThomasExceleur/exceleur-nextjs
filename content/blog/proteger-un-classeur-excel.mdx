# Proteger un Classeur Excel : Guide Complet

Securisez vos classeurs Excel selon vos besoins specifiques.

## Niveaux de protection

### Vue d'ensemble

| Niveau | Protection |
|--------|------------|
| Fichier | Mot de passe a l'ouverture |
| Classeur | Structure (onglets) |
| Feuille | Contenu des cellules |
| Cellules | Verrouillage individuel |

## Protection a l'ouverture

### Chiffrer avec mot de passe

1. **Fichier** > **Informations**
2. **Proteger le classeur**
3. **Chiffrer avec mot de passe**
4. Entrez et confirmez le mot de passe

### Resultat

Impossible d'ouvrir sans le mot de passe.

### Attention

Mot de passe oublie = fichier inaccessible.

## Protection en modification

### Mot de passe pour modifier

1. **Fichier** > **Enregistrer sous**
2. **Outils** > **Options generales**
3. Renseignez "Mot de passe pour modifier"

### Resultat

Le fichier s'ouvre en lecture seule sans mot de passe.

## Protection de la structure

### Activer

1. **Revision** > **Proteger le classeur**
2. Cochez **Structure**
3. Mot de passe optionnel

### Effets

- Empeche d'ajouter/supprimer des feuilles
- Empeche de renommer des feuilles
- Empeche de masquer/afficher des feuilles

## Protection des feuilles

### Activer

1. **Revision** > **Proteger la feuille**
2. Configurez les options
3. Mot de passe optionnel

### Options courantes

| Option | Effet |
|--------|-------|
| Selectionner les cellules | Navigation |
| Mettre en forme | Changement de format |
| Inserer/Supprimer | Modification de structure |
| Tri/Filtre | Utilisation des donnees |

## Permettre certaines cellules

### Deverrouiller des cellules

1. Selectionnez les cellules editables
2. Ctrl+1 (Format de cellule)
3. Onglet **Protection**
4. Decochez **Verrouillee**
5. Protegez ensuite la feuille

## Marquer comme final

### Activer

1. **Fichier** > **Informations**
2. **Proteger le classeur**
3. **Marquer comme final**

### Effet

Indique que le document est termine (facilement modifiable).

## Plages modifiables par utilisateur

### Configurer

1. **Revision** > **Autoriser la modification des plages**
2. **Nouveau**
3. Definissez la plage et les permissions
4. Protegez la feuille

### Usage

Permissions differentes selon les utilisateurs Windows.

## Masquer les formules

### Configurer

1. Selectionnez les cellules avec formules
2. Ctrl+1 > **Protection**
3. Cochez **Masquee**
4. Protegez la feuille

### Resultat

La barre de formule n'affiche pas la formule.

## Protection VBA

### Proteger le projet VBA

1. Alt+F11 (Editeur VBA)
2. Clic droit sur le projet
3. **Proprietes du projet**
4. Onglet **Protection**
5. Cochez et definissez le mot de passe

## Resume des protections

| Type | Empeche | Securite |
|------|---------|----------|
| Chiffrement | Ouverture | Elevee |
| Modification | Modification sans mdp | Moyenne |
| Structure | Gestion des feuilles | Faible |
| Feuille | Modification contenu | Faible |
| VBA | Acces au code | Moyenne |

| Pratique | Description |
|----------|-------------|
| Plusieurs niveaux | Combinez les protections |
| Documentation | Notez les mots de passe |
| Test | Verifiez les restrictions |
| Sauvegarde | Gardez une copie accessible |
