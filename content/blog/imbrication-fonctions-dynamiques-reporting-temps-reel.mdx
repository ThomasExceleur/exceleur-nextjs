# Imbrication de Fonctions Dynamiques pour le Reporting

Creez des rapports qui se mettent a jour automatiquement.

## Fonctions dynamiques principales

| Fonction | Role |
|----------|------|
| FILTRE | Extraire selon criteres |
| TRIER | Ordonner les resultats |
| UNIQUE | Eliminer les doublons |
| SEQUENCE | Generer des series |

## Combinaisons puissantes

### FILTRE + TRIER

```excel
=TRIER(FILTRE(Donnees;Region=\"Nord\");3;-1)
```

Filtre puis trie par la 3eme colonne.

### UNIQUE + TRIER

```excel
=TRIER(UNIQUE(Donnees))
```

Liste unique et triee.

### FILTRE + UNIQUE

```excel
=UNIQUE(FILTRE(Produits;Statut=\"Actif\"))
```

Produits actifs uniques.

## Avec LET pour la lisibilite

```excel
=LET(
  source;Ventes;
  filtrees;FILTRE(source;Region=\"Nord\");
  triees;TRIER(filtrees;Montant;-1);
  top10;INDEX(triees;SEQUENCE(10);SEQUENCE(1;COLONNES(source)));
  top10
)
```

## Reporting automatique

### KPIs dynamiques

```excel
=LET(
  donnees;FILTRE(Ventes;Mois=MoisActuel);
  total;SOMME(INDEX(donnees;;3));
  nb;LIGNES(donnees);
  moyenne;total/nb;
  CONCAT(\"Total: \";total;\" | Moyenne: \";moyenne)
)
```

### Top performers

```excel
=INDEX(TRIER(FILTRE(Donnees;Annee=2024);Montant;-1);SEQUENCE(5);SEQUENCE(1;4))
```
