# Excel Quelle Version Choisir

Plusieurs versions d'Excel sont disponibles. Voici comment choisir celle qui vous convient.

## Les differentes versions

### Microsoft 365 (Abonnement)

La version la plus complete et toujours a jour.

**Inclus :**
- Excel desktop (Windows/Mac)
- Excel Online
- Excel mobile
- Mises a jour continues
- Nouvelles fonctionnalites en priorite
- 1 To OneDrive

**Prix :** ~7-12 EUR/mois selon la formule

### Excel 2021 (Achat unique)

Version perpetuelle sans abonnement.

**Inclus :**
- Excel desktop
- Support pendant 5 ans
- Fonctionnalites a la date d'achat

**Prix :** ~150 EUR (une seule fois)

### Excel Online (Gratuit)

Version web accessible gratuitement.

**Inclus :**
- Fonctionnalites de base
- Collaboration en temps reel
- Sauvegarde sur OneDrive

**Prix :** Gratuit

### Excel Mobile (Gratuit/Premium)

Applications pour smartphone et tablette.

**Version gratuite :**
- Visualisation et edition basique
- Ecrans < 10.1 pouces

**Version premium (avec Microsoft 365) :**
- Toutes les fonctionnalites
- Grands ecrans

## Quelle version pour quel usage ?

### Usage personnel occasionnel

**Recommandation : Excel Online (gratuit)**

- Suffisant pour les taches basiques
- Pas de cout
- Accessible partout

### Etudiant

**Recommandation : Microsoft 365 Education (souvent gratuit)**

- Verifiez l'offre de votre etablissement
- Toutes les fonctionnalites
- Gratuit pour les etudiants eligibles

### Professionnel

**Recommandation : Microsoft 365**

- Toujours a jour
- Support inclus
- Collaboration facile
- Nouvelles fonctionnalites

### Entreprise

**Recommandation : Microsoft 365 Business**

- Administration centralisee
- Securite renforcee
- Teams inclus
- Support entreprise

### Budget limite, usage regulier

**Recommandation : Excel 2021**

- Paiement unique
- Fonctionnalites completes
- Pas de frais recurrents

## Microsoft 365 vs Excel 2021

### Choisissez Microsoft 365 si :

- Vous voulez les dernieres fonctionnalites
- Vous travaillez en equipe
- Vous utilisez plusieurs appareils
- Le cloud est important pour vous

### Choisissez Excel 2021 si :

- Vous preferez payer une fois
- Vous n'avez pas besoin des nouveautes
- Vous travaillez principalement hors ligne
- Votre budget est limite

## Compatibilite des fichiers

### Format .xlsx

Toutes les versions recentes lisent et ecrivent le format .xlsx.

### Formules nouvelles

Les nouvelles formules (RECHERCHEX, FILTRE) ne fonctionnent pas dans les anciennes versions.

**Solution :** Utilisez RECHERCHEV ou partagez en PDF.

### Macros

Les macros fonctionnent uniquement dans les versions desktop (pas Online ni Mobile).

## Comment mettre a niveau

### D'Excel 2016/2019 vers 2021

1. Achetez la nouvelle licence
2. Desinstallez l'ancienne version
3. Installez la nouvelle

### D'Excel 2021 vers Microsoft 365

1. Souscrivez a l'abonnement
2. Desinstallez Excel 2021
3. Telechargez Microsoft 365

### Conserver ses fichiers

Vos fichiers Excel sont compatibles entre toutes les versions. Pas de perte de donnees.

## Notre recommandation

Pour la plupart des utilisateurs, **Microsoft 365** offre le meilleur rapport qualite/prix avec ses mises a jour continues et son stockage cloud.

Si le budget est une contrainte, **Excel Online** est une excellente option gratuite pour debuter.
