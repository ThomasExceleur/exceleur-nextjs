# Excel Affiche la Formule

Solutions a ce probleme courant.

## Causes

### Mode affichage formules

Le mode affichage formules est active.

### Solution

**Ctrl+`** (accent grave) pour basculer.

Ou : **Formules** > **Afficher les formules**

## Apostrophe devant

### Cause

La formule commence par ' (apostrophe).

### Solution

1. F2 pour editer
2. Supprimez l'apostrophe
3. Entree

## Format texte

### Cause

La cellule est formatee en texte.

### Solution

1. Changez le format en Nombre/Standard
2. F2 puis Entree pour recalculer

## Verifications

### Mode formules

Regardez si toutes les cellules affichent des formules.

### Format cellule

**Ctrl+1** > Verifiez le format.

## Prevention

| Action | Effet |
|--------|-------|
| Evitez ' | Ne precede pas par apostrophe |
| Format | Standard par defaut |
| Ctrl+` | Desactivez le mode formules |

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Verification | Ctrl+` pour verifier |
| Format | Appliquez avant saisie |
| Nettoyage | Supprimez les apostrophes |
