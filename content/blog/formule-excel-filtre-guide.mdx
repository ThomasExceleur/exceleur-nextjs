# Formule FILTRE : Guide Complet

Tout ce que vous devez savoir sur la fonction FILTRE.

## Syntaxe

```excel
=FILTRE(tableau;inclure;[si_vide])
```

## Comprendre le parametre inclure

### Ce que c'est

Un tableau de valeurs VRAI/FAUX de meme dimension que le tableau source.

### Creation automatique

```excel
B2:B100="Paris"
```

Cree un tableau de VRAI/FAUX pour chaque cellule de B2:B100.

## Tous les types de criteres

### Egal a

```excel
=FILTRE(A:C;B:B="Valeur")
```

### Different de

```excel
=FILTRE(A:C;B:B<>"Exclu")
```

### Superieur

```excel
=FILTRE(A:C;C:C>100)
```

### Entre deux valeurs

```excel
=FILTRE(A:C;(C:C>=100)*(C:C<=500))
```

### Contient du texte

```excel
=FILTRE(A:C;ESTNUM(CHERCHE("mot";A:A)))
```

## Criteres multiples

### ET logique (tous les criteres)

Multiplication :
```excel
=FILTRE(A:C;(B:B="Paris")*(C:C>100))
```

### OU logique (au moins un critere)

Addition :
```excel
=FILTRE(A:C;(B:B="Paris")+(B:B="Lyon"))
```

### Combinaison

```excel
=FILTRE(A:C;((B:B="Paris")+(B:B="Lyon"))*(C:C>100))
```

## Gestion si_vide

### Message texte

```excel
=FILTRE(A:C;B:B="Berlin";"Aucune correspondance")
```

### Cellule vide

```excel
=FILTRE(A:C;B:B="Berlin";"")
```

### Tableau vide

```excel
=FILTRE(A:C;B:B="Berlin";{""})
```

## Une seule colonne

### Extraire colonne A

```excel
=FILTRE(A:A;B:B="Paris")
```

### Extraire colonne C

```excel
=FILTRE(C:C;B:B="Paris")
```

## Combinaisons avancees

### FILTRE + TRIER

```excel
=TRIER(FILTRE(A:C;B:B="Paris");3;-1)
```

### FILTRE + UNIQUE

```excel
=UNIQUE(FILTRE(A:A;B:B="Actif"))
```

### FILTRE + INDEX

```excel
=INDEX(FILTRE(A:C;B:B="Paris");1;3)
```

Premiere ligne, troisieme colonne du resultat.

### FILTRE + SOMME

```excel
=SOMME(FILTRE(C:C;B:B="Paris"))
```

## Plages dynamiques avec LET

```excel
=LET(
  donnees;A2:D100;
  region;B2:B100;
  montant;D2:D100;
  critere_region;"Paris";
  critere_montant;100;
  FILTRE(donnees;(region=critere_region)*(montant>critere_montant))
)
```

## Erreurs et solutions

| Erreur | Cause | Solution |
|--------|-------|----------|
| #CALC! | Aucun resultat | Ajoutez si_vide |
| #VALEUR! | Plages differentes | Alignez les tailles |
| #PROPAGATION! | Pas d'espace | Liberez les cellules |

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Tableaux structures | References dynamiques |
| LET | Clarifiez les formules |
| si_vide | Toujours le definir |
