# Excel ne Reconnait Pas les Chiffres

Solutions au probleme texte/nombre.

## Symptomes

- Triangle vert dans la cellule
- Calculs qui ne fonctionnent pas
- Alignement a gauche (comme du texte)

## Causes

| Cause | Frequence |
|-------|-----------|
| Import CSV | Haute |
| Copier-coller | Haute |
| Format texte | Moyenne |
| Espaces invisibles | Moyenne |

## Solutions

### Conversion rapide

1. Selectionnez les cellules
2. Cliquez sur l'icone d'avertissement
3. **Convertir en nombre**

### Formule CNUM

```excel
=CNUM(A1)
```

### Collage special

1. Tapez 1 dans une cellule
2. Copiez
3. Selectionnez les "faux nombres"
4. Collage special > Multiplier

## Supprimer les espaces

```excel
=SUPPRESPACE(A1)
```

## Nettoyer

```excel
=EPURAGE(A1)
```

Supprime les caracteres non imprimables.

## Verification

```excel
=ESTNUM(A1)
=ESTTEXTE(A1)
```

## Prevention

### A l'import

Specifiez les types de colonnes.

### Format de cellule

Appliquez le format Nombre avant la saisie.

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Import | Configurez les types |
| Verification | ESTNUM pour controler |
| Nettoyage | SUPPRESPACE systematique |
