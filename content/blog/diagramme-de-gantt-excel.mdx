# Le Diagramme de Gantt dans Excel

Un outil essentiel pour visualiser et gérer vos projets.

## Qu'est-ce qu'un diagramme de Gantt ?

### Origine

Invente par Henry Gantt au début du XXe siecle pour la planification industrielle.

### Principe

Represente les taches d'un projet sous forme de barres horizontales sûr une échelle de temps.

### Éléments

- Axe horizontal : temps
- Axe vertical : taches
- Barres : duree des taches

## Pourquoi utiliser un Gantt ?

### Avantages

| Avantage | Description |
|----------|-------------|
| Visualisation | Vue d'ensemble du projet |
| Planification | Identification des délais |
| Communication | Partage avec l'equipe |
| Suivi | Progression visible |

### Cas d'usage

- Gestion de projet
- Planification d'evenements
- Lancement de produit
- Campagnes marketing

## Créer un Gantt simple

### Structuré des données

| Tache | Début | Duree |
|-------|-------|-------|
| Etude | 1 | 5 |
| Design | 6 | 8 |
| Dev | 14 | 15 |
| Tests | 29 | 5 |

### Étapes

1. Sélectionnez vos données
2. **Insertion** > **Barres empilees**
3. Masquez la serie "Début"
4. Inversez l'ordre des taches

## Éléments d'un bon Gantt

### Informations essentielles

- Nom des taches
- Dates de début et fin
- Durees
- Responsables
- Dependances (optionnel)

### Indicateurs visuels

- Progression (barres superposees)
- Jalons (losanges)
- Ligne de date actuelle

## Limites d'Excel

### Ce qu'Excel ne fait pas nativement

- Dependances avec fleches
- Gestion des ressources
- Chemin critique automatique
- Calendrier intégré

### Quand passer à un outil specialise

- Plus de 50 taches
- Équipe nombreuse
- Dependances complexes
- Reporting avancé

## Alternatives à Excel

### Outils specialises

| Outil | Avantage |
|-------|----------|
| Microsoft Project | Complet, intégré Office |
| Smartsheet | Collaboratif, cloud |
| Monday.com | Interface moderne |
| Asana | Gratuit pour petites equipes |

## Bonnes pratiques

### Planification

1. **Définissez** clairement les taches
2. **Estimez** realiste les durees
3. **Identifiez** les dependances
4. **Assignez** les responsables

### Mise à jour

1. **Actualisez** régulièrement la progression
2. **Communiquez** les changements
3. **Documentez** les ecarts

### Présentation

1. **Utilisez** des couleurs significatives
2. **Ajoutez** une legende claire
3. **Incluez** la date de mise à jour

## Modèle de base

### Structuré recommandée

| Colonne | Contenu |
|---------|---------|
| A | ID |
| B | Tache |
| C | Responsable |
| D | Date début |
| E | Duree |
| F | Date fin |
| G | Progression |
| H | Statut |

### Formules utiles

```excel
' Date fin
=[@Début]+[@Duree]-1

' Statut automatique
=SI([@Progression]=1;"Termine";"En cours")
```
