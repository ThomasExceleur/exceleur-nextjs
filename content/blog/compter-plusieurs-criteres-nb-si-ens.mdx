# Compter avec Plusieurs Criteres : NB.SI.ENS

NB.SI.ENS compte les cellules repondant a plusieurs conditions.

## Syntaxe

```excel
=NB.SI.ENS(plage1;critere1;plage2;critere2;...)
```

## Difference avec NB.SI

### NB.SI : Un seul critere

```excel
=NB.SI(A:A;"Paris")
```

### NB.SI.ENS : Plusieurs criteres

```excel
=NB.SI.ENS(A:A;"Paris";B:B;">100")
```

Compte ou A="Paris" ET B>100.

## Exemples de base

### Deux criteres

```excel
=NB.SI.ENS(Region;"Nord";Statut;"Actif")
```

### Trois criteres

```excel
=NB.SI.ENS(Region;"Nord";Statut;"Actif";Annee;2024)
```

### Avec operateurs

```excel
=NB.SI.ENS(Montant;">1000";Date;">="&DATE(2024;1;1))
```

## Operateurs disponibles

| Operateur | Signification |
|-----------|---------------|
| `"="` | Egal (par defaut) |
| `"<>"` | Different |
| `">"` | Superieur |
| `"<"` | Inferieur |
| `">="` | Superieur ou egal |
| `"<="` | Inferieur ou egal |

## Avec des references

### Critere dans une cellule

```excel
=NB.SI.ENS(Region;E1;Statut;E2)
```

### Avec concatenation

```excel
=NB.SI.ENS(Montant;">"&F1;Date;">="&G1)
```

## Criteres de texte

### Texte exact

```excel
=NB.SI.ENS(Nom;"Dupont")
```

### Contient (joker)

```excel
=NB.SI.ENS(Email;"*@gmail.com")
```

### Commence par

```excel
=NB.SI.ENS(Code;"AB*")
```

### Caractere unique

```excel
=NB.SI.ENS(Code;"A?1")   ' A suivi d'un caractere puis 1
```

## Criteres de dates

### Annee specifique

```excel
=NB.SI.ENS(Date;">="&DATE(2024;1;1);Date;"<="&DATE(2024;12;31))
```

### Mois en cours

```excel
=NB.SI.ENS(Date;">="&EOMONTH(AUJOURDHUI();-1)+1;Date;"<="&EOMONTH(AUJOURDHUI();0))
```

### Dernieres 30 jours

```excel
=NB.SI.ENS(Date;">="&AUJOURDHUI()-30)
```

## Cas pratiques

### 1. Ventes par region et produit

```excel
=NB.SI.ENS(Region;"Ile-de-France";Produit;"Widget")
```

### 2. Commandes en attente > 1000â‚¬




## Fonctions associees

| Fonction | Usage |
|----------|-------|
| SOMME.SI.ENS | Somme multicritere |
| MOYENNE.SI.ENS | Moyenne multicritere |
| MAX.SI.ENS | Maximum multicritere |
| MIN.SI.ENS | Minimum multicritere |

### Exemple SOMME.SI.ENS

```excel
=SOMME.SI.ENS(Montant;Region;"Paris";Statut;"Valide")
```

## Alternatives

### Avec SOMMEPROD

```excel
=SOMMEPROD((Region="Paris")*(Statut="Actif")*1)
```

### Avec FILTRE et LIGNES

```excel
=LIGNES(FILTRE(Donnees;(Region="Paris")*(Statut="Actif")))
```

## Erreurs courantes

### Plages de tailles differentes

```excel
=NB.SI.ENS(A1:A10;"Oui";B1:B5;">10")   ' ERREUR
```

Les plages doivent avoir la meme taille.

### Operateur mal place

```excel
=NB.SI.ENS(A:A;>10)     ' INCORRECT
=NB.SI.ENS(A:A;">10")   ' CORRECT
```

## Optimisation

### Plages limitees

```excel
=NB.SI.ENS(A2:A1000;"Paris";B2:B1000;">100")
```

Au lieu de A:A pour de meilleures performances.

### Tableaux structures

```excel
=NB.SI.ENS(Ventes[Region];"Paris";Ventes[Montant];">100")
```
