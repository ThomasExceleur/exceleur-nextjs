# Erreur #N/A Excel

Comprendre et corriger cette erreur courante.

## Signification

### Definition

#N/A = "Non Applicable" ou "Non Disponible"

### Cause principale

Une valeur recherchee n'a pas ete trouvee.

## Causes frequentes

| Cause | Fonction concernee |
|-------|-------------------|
| Valeur absente | RECHERCHEV, RECHERCHEX |
| Espaces invisibles | Toutes les recherches |
| Format different | Recherches |
| Plage incorrecte | Toutes |

## Solutions

### SIERREUR

```excel
=SIERREUR(RECHERCHEV(...);"Non trouve")
```

### SI.NON.DISP

```excel
=SI.NON.DISP(RECHERCHEV(...);"Valeur par defaut")
```

### RECHERCHEX natif

```excel
=RECHERCHEX(val;plage;result;"Non trouve")
```

## Diagnostic

### Verifier les espaces

```excel
=SUPPRESPACE(A1)=A1
```

FAUX = il y a des espaces.

### Verifier le format

```excel
=ESTNUM(A1)
=ESTTEXTE(A1)
```

### Comparer manuellement

```excel
=A1=B1
```

## Corrections

### Supprimer les espaces

```excel
=SUPPRESPACE(A1)
```

### Convertir en nombre

```excel
=CNUM(A1)
```

### Convertir en texte

```excel
=TEXTE(A1;"0")
```

## Prevention

| Action | Methode |
|--------|---------|
