# Reduire la Taille de l'Ascenseur

Corrigez le probleme d'ascenseur minuscule.

## Le probleme

### Symptome

L'ascenseur (scrollbar) est tres petit, navigation difficile.

### Cause

Excel croit que les donnees s'etendent tres loin.

## Solution 1 : Supprimer les lignes/colonnes

### Etapes

1. Trouvez la derniere cellule utilisee (**Ctrl+Fin**)
2. Selectionnez les lignes/colonnes vides en dessous/a droite
3. Clic droit > **Supprimer**
4. **Enregistrez** le fichier

## Solution 2 : Definir la zone d'impression

### Etapes

1. Selectionnez vos donnees reelles
2. **Mise en page** > **Zone d'impression** > **Definir**

## Solution 3 : Nettoyer via VBA

### Code

```vba
Sub NettoierFeuille()
    Dim derniereLigne As Long
    Dim derniereCol As Long

    derniereLigne = Cells.Find("*", SearchOrder:=xlByRows, SearchDirection:=xlPrevious).Row
    derniereCol = Cells.Find("*", SearchOrder:=xlByColumns, SearchDirection:=xlPrevious).Column

    Range(Cells(derniereLigne + 1, 1), Cells(Rows.Count, Columns.Count)).Delete
    Range(Cells(1, derniereCol + 1), Cells(Rows.Count, Columns.Count)).Delete
End Sub
```

## Verification

### Trouver la "fausse" fin

1. **Ctrl+Fin**
2. Si loin de vos donnees = probleme

### Apres nettoyage

Enregistrez, fermez et rouvrez le fichier.
