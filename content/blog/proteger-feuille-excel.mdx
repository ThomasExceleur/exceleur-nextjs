# Proteger une Feuille Excel

La protection de feuille empeche les modifications non autorisees.

## Pourquoi proteger une feuille ?

### Cas d'usage

- Empecher la suppression de formules
- Verrouiller les cellules de structure
- Permettre la saisie uniquement dans certaines cellules
- Securiser les tableaux de bord

## Protection de base

### Activer la protection

1. **Revision** > **Proteger la feuille**
2. Definissez un mot de passe (optionnel)
3. Cochez les actions autorisees
4. Cliquez **OK**

### Desactiver la protection

**Revision** > **Oter la protection de la feuille**

## Verrouillage des cellules

### Par defaut

Toutes les cellules sont verrouillees par defaut.

### Deverrouiller des cellules

1. Selectionnez les cellules a laisser editables
2. **Clic droit** > **Format de cellule**
3. Onglet **Protection**
4. Decochez **Verrouillee**
5. Protegez ensuite la feuille

## Cellules masquees

### Masquer les formules

1. Selectionnez les cellules avec formules
2. **Format de cellule** > **Protection**
3. Cochez **Masquee**
4. Protegez la feuille

La barre de formule n'affichera plus la formule.

## Protection par VBA

### Proteger avec macro

```vba
Sub ProtegerFeuille()
    ActiveSheet.Protect Password:="MonMotDePasse", _
        AllowFormattingCells:=True, _
        AllowFiltering:=True
End Sub
```

### Deproteger avec macro

```vba
Sub DeprotegerFeuille()
    ActiveSheet.Unprotect Password:="MonMotDePasse"
End Sub
```

## Permettre certaines plages

### Autoriser des utilisateurs specifiques

1. **Revision** > **Autoriser la modification des plages**
2. **Nouveau**
3. Definissez la plage et les permissions
4. Protegez ensuite la feuille

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Mot de passe fort | Combinez lettres, chiffres, symboles |
| Documentation | Notez les mots de passe en lieu sur |
| Test | Verifiez que la saisie fonctionne |
| Sauvegarde | Gardez une copie non protegee |

## Limites de la protection

### Attention

- Protection contre modifications accidentelles
- Pas une securite absolue
- Des outils peuvent contourner la protection
- Pour securite maximale, utilisez le chiffrement
