# La Touche F4 dans Excel : Deux Fonctions Essentielles

F4 a deux usages majeurs dans Excel : repeter une action et basculer les references.

## Fonction 1 : Repeter la derniere action

### Comment ca marche

1. Effectuez une action (mise en forme, suppression, etc.)
2. Selectionnez d'autres cellules
3. Appuyez sur **F4**

### Exemples

#### Repeter une mise en forme

1. Mettez une cellule en gras
2. Selectionnez une autre cellule
3. **F4** : elle passe en gras

#### Repeter une suppression

1. Supprimez une ligne
2. Selectionnez une autre ligne
3. **F4** : elle est supprimee

#### Repeter une insertion

1. Inserez une colonne
2. **F4** : insere une autre colonne

### Actions compatibles

- Mise en forme (couleur, police, bordures)
- Insertion/suppression de lignes/colonnes
- Largeur/hauteur de lignes/colonnes
- Collage

### Actions non repetables

- Saisie de texte
- Formules
- Operations multiples

## Fonction 2 : Basculer les references

### Dans une formule

Lors de l'edition d'une formule, F4 bascule entre les types de references.

### Cycle de basculement

| Appui | Resultat | Type |
|-------|----------|------|
| 1 | $A$1 | Absolu |
| 2 | A$1 | Mixte (ligne fixe) |
| 3 | $A1 | Mixte (colonne fixe) |
| 4 | A1 | Relatif |

### Exemple pratique

1. Tapez une formule : `=A1*B1`
2. Cliquez sur A1 dans la barre de formule
3. Appuyez sur **F4** : `=$A$1*B1`
4. Appuyez encore : `=A$1*B1`

### Quand utiliser

| Reference | Utilisation |
|-----------|-------------|
| $A$1 | Toujours la meme cellule |
| A$1 | Ligne fixe, colonne variable |
| $A1 | Colonne fixe, ligne variable |
| A1 | Tout variable |

## Combinaison des deux fonctions

### Contexte

- **Hors formule** : repete l'action
- **En mode edition** : bascule les references

### Comment savoir

Regardez si vous etes en mode edition (curseur dans la barre de formule).

## Sur Mac

### Raccourcis equivalents

| Action | Mac |
|--------|-----|
| Repeter | Cmd + Y |
| Basculer references | Fn + F4 ou Cmd + T |

## Alternatives

### Pour repeter une action

- **Ctrl + Y** : equivalent de F4

### Pour les references absolues

- Tapez manuellement les $

## Bonnes pratiques

### Pour F4 repetition

1. Effectuez l'action une fois proprement
2. Verifiez qu'elle est correcte
3. Utilisez F4 pour les autres cellules

### Pour F4 references

1. Selectionnez la reference a modifier
2. Appuyez sur F4 jusqu'au type souhaite
3. Verifiez en copiant la formule

## Exemples d'utilisation avancee

### Mise en forme rapide

1. Appliquez le format souhaite a une cellule
2. Selectionnez toutes les autres cellules concernees
3. **F4** pour appliquer le meme format

### Formule avec taux fixe

```excel
=A2*$B$1    ' B1 contient le taux, fixe avec F4
```
