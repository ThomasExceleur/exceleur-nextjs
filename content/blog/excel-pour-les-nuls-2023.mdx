# Excel pour les Debutants

Ce guide vous accompagne dans vos premiers pas avec Excel.

## Qu'est-ce qu'Excel ?

### Definition

Excel est un tableur qui permet de :
- Organiser des donnees en lignes et colonnes
- Faire des calculs automatiques
- Creer des graphiques
- Analyser des donnees

### Vocabulaire de base

| Terme | Definition |
|-------|------------|
| Classeur | Fichier Excel |
| Feuille | Onglet dans un classeur |
| Cellule | Intersection ligne/colonne |
| Plage | Groupe de cellules |

## L'interface Excel

### Elements principaux

1. **Ruban** : barre d'outils en haut
2. **Barre de formule** : affiche le contenu de la cellule
3. **Zone de travail** : grille de cellules
4. **Onglets de feuilles** : en bas

### Se reperer

- **Colonnes** : lettres (A, B, C...)
- **Lignes** : numeros (1, 2, 3...)
- **Cellule** : combinaison (A1, B2, C3...)

## Saisir des donnees

### Types de donnees

| Type | Exemple | Alignement |
|------|---------|------------|
| Texte | "Paris" | Gauche |
| Nombre | 123 | Droite |
| Date | 15/01/2024 | Droite |
| Formule | =A1+B1 | Selon resultat |

### Saisie

1. Cliquez sur une cellule
2. Tapez votre donnee
3. Appuyez sur **Entree**

## Premieres formules

### Structure

Toute formule commence par **=**

### Formules de base

| Formule | Action |
|---------|--------|
| =A1+B1 | Addition |
| =A1-B1 | Soustraction |
| =A1*B1 | Multiplication |
| =A1/B1 | Division |

### Fonctions simples

```excel
=SOMME(A1:A10)     ' Additionne A1 a A10
=MOYENNE(A1:A10)   ' Moyenne de A1 a A10
=MAX(A1:A10)       ' Plus grande valeur
=MIN(A1:A10)       ' Plus petite valeur
=NB(A1:A10)        ' Compte les nombres
```

## Mise en forme

### Texte

- **Ctrl + G** : Gras
- **Ctrl + I** : Italique
- **Ctrl + U** : Souligne

### Cellules

1. Selectionnez les cellules
2. Utilisez le ruban **Accueil**
3. Modifiez police, couleur, bordures

### Format des nombres

1. **Ctrl + 1** (Format de cellule)
2. Choisissez le format (nombre, monnaie, date)

## Raccourcis essentiels

| Raccourci | Action |
|-----------|--------|
| Ctrl + C | Copier |
| Ctrl + V | Coller |
| Ctrl + X | Couper |
| Ctrl + Z | Annuler |
| Ctrl + S | Enregistrer |
| Ctrl + N | Nouveau classeur |

## Selectionner des cellules

### Methodes

| Action | Comment |
|--------|---------|
| Une cellule | Cliquer |
| Plage | Cliquer-glisser |
| Colonne | Cliquer sur la lettre |
| Ligne | Cliquer sur le numero |
| Tout | Ctrl + A |

## Creer un tableau simple

### Exemple de budget

| Description | Montant |
|-------------|---------|
| Salaire | 2000 |
| Loyer | -800 |
| Courses | -400 |
| **Total** | =SOMME(B2:B4) |

## Enregistrer son travail

### Formats

| Extension | Description |
|-----------|-------------|
| .xlsx | Format standard |
| .xls | Ancien format |
| .xlsm | Avec macros |
| .csv | Texte simple |

### Enregistrer

1. **Ctrl + S** ou **Fichier** > **Enregistrer**
2. Choisissez l'emplacement
3. Donnez un nom

## Conseils pour debuter

1. **Pratiquez** regulierement
2. **Explorez** les menus
3. **Utilisez** les raccourcis
4. **Commencez** par des projets simples
5. **N'hesitez pas** a faire des erreurs
