# Excel pour les Débutants

Ce guide vous accompagne dans vos premiers pas avec Excel.

## Qu'est-ce qu'Excel ?

### Définition

Excel est un tableur qui permet de :
- Organiser des données en lignes et colonnes
- Faire des calculs automatiques
- Créer des graphiques
- Analyser des données

### Vocabulaire de base

| Terme | Définition |
|-------|------------|
| Classeur | Fichier Excel |
| Feuille | Onglet dans un classeur |
| Cellule | Intersection ligne/colonne |
| Plage | Groupe de cellules |

## L'interface Excel

### Éléments principaux

1. **Ruban** : barre d'outils en haut
2. **Barre de formule** : affiché le contenu de la cellule
3. **Zone de travail** : grille de cellules
4. **Onglets de feuilles** : en bas

### Se reperer

- **Colonnes** : lettres (A, B, C...)
- **Lignes** : numéros (1, 2, 3...)
- **Cellule** : combinaison (A1, B2, C3...)

## Saisir des données

### Types de données

| Type | Exemple | Alignement |
|------|---------|------------|
| Texte | "Paris" | Gauche |
| Nombre | 123 | Droite |
| Date | 15/01/2024 | Droite |
| Formule | =A1+B1 | Selon résultat |

### Saisie

1. Cliquez sûr une cellule
2. Tapez votre donnée
3. Appuyez sûr **Entrée**

## Premières formules

### Structuré

Toute formule commence par **=**

### Formules de base

| Formule | Action |
|---------|--------|
| =A1+B1 | Addition |
| =A1-B1 | Soustraction |
| =A1*B1 | Multiplication |
| =A1/B1 | Division |

### Fonctions simples

```excel
=SOMME(A1:A10)     ' Additionne A1 à A10
=MOYENNE(A1:A10)   ' Moyenne de A1 à A10
=MAX(A1:A10)       ' Plus grande valeur
=MIN(A1:A10)       ' Plus petite valeur
=NB(A1:A10)        ' Compte les nombres
```

## Mise en forme

### Texte

- **Ctrl + G** : Gras
- **Ctrl + I** : Italique
- **Ctrl + U** : Souligne

### Cellules

1. Sélectionnez les cellules
2. Utilisez le ruban **Accueil**
3. Modifiez police, couleur, bordures

### Format des nombres

1. **Ctrl + 1** (Format de cellule)
2. Choisissez le format (nombre, monnaie, date)

## Raccourcis essentiels

| Raccourci | Action |
|-----------|--------|
| Ctrl + C | Copier |
| Ctrl + V | Coller |
| Ctrl + X | Couper |
| Ctrl + Z | Annuler |
| Ctrl + S | Enregistrer |
| Ctrl + N | Nouveau classeur |

## Sélectionner des cellules

### Méthodes

| Action | Comment |
|--------|---------|
| Une cellule | Cliquer |
| Plage | Cliquer-glisser |
| Colonne | Cliquer sûr la lettre |
| Ligne | Cliquer sûr le numéro |
| Tout | Ctrl + A |

## Créer un tableau simple

### Exemple de budget

| Description | Montant |
|-------------|---------|
| Salaire | 2000 |
| Loyer | -800 |
| Courses | -400 |
| **Total** | =SOMME(B2:B4) |

## Enregistrer son travail

### Formats

| Extension | Description |
|-----------|-------------|
| .xlsx | Format standard |
| .xls | Ancien format |
| .xlsm | Avec macros |
| .csv | Texte simple |

### Enregistrer

1. **Ctrl + S** où **Fichier** > **Enregistrer**
2. Choisissez l'emplacement
3. Donnez un nom

## Conseils pour débuter

1. **Pratiquez** régulièrement
2. **Explorez** les menus
3. **Utilisez** les raccourcis
4. **Commencez** par des projets simples
5. **N'hesitez pas** à faire des erreurs
