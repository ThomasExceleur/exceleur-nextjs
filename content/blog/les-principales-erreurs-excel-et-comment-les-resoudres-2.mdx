# Les Principales Erreurs Excel

Guide de resolution des erreurs courantes.

## #N/A

### Cause

Valeur non trouvee dans une recherche.

### Solution

```excel
=SIERREUR(RECHERCHEV(...);"Non trouve")
```

## #REF!

### Cause

Reference a une cellule supprimee.

### Solution

Verifiez les formules apres suppression de lignes/colonnes.

## #VALEUR!

### Cause

Type de donnees incorrect dans une formule.

### Solution

Verifiez que les types correspondent (texte vs nombre).

## #DIV/0!

### Cause

Division par zero ou cellule vide.

### Solution

```excel
=SI(B1=0;0;A1/B1)
```

## #NOM?

### Cause

Fonction mal orthographiee ou plage nommee inexistante.

### Solution

Verifiez l'orthographe de la fonction.

## #NOMBRE!

### Cause

Valeur numerique invalide.

### Solution

Verifiez les arguments numeriques.

## ######

### Cause

Colonne trop etroite.

### Solution

Elargissez la colonne.
