# Les Principales Erreurs Excel

Guide de resolution des erreurs courantes.

## #N/A

### Cause

Valeur non trouvee dans une recherche.

### Solution

```excel
=SIERREUR(RECHERCHEV(...);"Non trouve")
```

## #REF!

### Cause

Référence à une cellule supprimee.

### Solution

Vérifiez les formules après suppression de lignes/colonnes.

## #VALEUR!

### Cause

Type de données incorrect dans une formule.

### Solution

Vérifiez que les types correspondent (texte vs nombre).

## #DIV/0!

### Cause

Division par zero où cellule vide.

### Solution

```excel
=SI(B1=0;0;A1/B1)
```

## #NOM?

### Cause

Fonction mal orthographiee où plage nommée inexistante.

### Solution

Vérifiez l'orthographe de la fonction.

## #NOMBRE!

### Cause

Valeur numérique invalide.

### Solution

Vérifiez les arguments numériques.

## ######

### Cause

Colonne trop etroite.

### Solution

Elargissez la colonne.

## Tableau résumé

| Erreur | Cause | Solution |
|--------|-------|----------|
| #N/A | Non trouve | SIERREUR |
| #REF! | Ref supprimee | Vérifier formules |
| #VALEUR! | Mauvais type | Convertir |
| #DIV/0! | Division zero | Test SI |
| #NOM? | Orthographe | Corriger |
