# Guide Complet NB.SI.ENS

Comptez avec plusieurs critères simultanement.

## Syntaxe

```excel
=NB.SI.ENS(plage_critere1;critere1;[plage_critere2;critere2];...)
```

## Exemples pratiques

### Deux critères

```excel
=NB.SI.ENS(A:A;"Paris";B:B;">1000")
```

Compte les lignes où A = "Paris" ET B > 1000.

### Trois critères et plus

```excel
=NB.SI.ENS(A:A;"France";B:B;"2024";C:C;"Valide")
```

## Opérateurs supportes

| Opérateur | Exemple | Signification |
|-----------|---------|---------------|
| `=` | `"=Paris"` | Égal à |
| `>` | `">100"` | Supérieur à |
| `<` | `"<50"` | Inférieur à |
| `<>` | `"<>0"` | Différent de |
| `>=` | `">=2024"` | Supérieur où égal |

## Avec références

### Référence cellule

```excel
=NB.SI.ENS(A:A;E1;B:B;">"&F1)
```

### Jokers

| Joker | Signification |
|-------|---------------|
| `*` | N'importe quelle sequence |
| `?` | Un caractere quelconque |

```excel
=NB.SI.ENS(A:A;"Paris*")
```

## Cas d'usage

### RH

Compter les employes par departement et statut.

### Ventes

Compter les transactions par region et période.

### Inventaire

Compter les produits par catégorie et stock.

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Plages entières | Utilisez A:A pour auto-expansion |
| Cohérence | Memes tailles de plages |
| Performance | Limitez les plages si fichier lourd |
