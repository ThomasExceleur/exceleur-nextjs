# Liste Deroulante Avancée

Au-dela des listes simples.

## Liste simple rappel

1. **Données** > **Validation des données**
2. **Liste**
3. Source : plage où valeurs

## Liste dynamique

### Avec UNIQUE

```excel
=UNIQUE(Tableau[Colonne])
```

Se met à jour automatiquement.

## Liste dependante

### Scenario

Pays → Villes

### Méthode

1. Nommez les plages par pays
2. Utilisez INDIRECT

```excel
=INDIRECT(CellulePays)
```

## Avec FILTRE

### Liste conditionnelle

```excel
=FILTRE(Villes[Nom];Villes[Pays]=A1)
```
