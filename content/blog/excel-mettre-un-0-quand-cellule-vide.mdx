# Mettre un 0 Quand la Cellule est Vide

Plusieurs methodes permettent de remplacer les cellules vides par 0 dans Excel.

## Methode 1 : Formule SI

### Syntaxe

```excel
=SI(A1=""; 0; A1)
```

### Explication

- Si A1 est vide, affiche 0
- Sinon, affiche la valeur de A1

### Variante avec ESTVIDE

```excel
=SI(ESTVIDE(A1); 0; A1)
```

## Methode 2 : Rechercher-Remplacer

### Etapes

1. Selectionnez la plage
2. **Ctrl + H** (Rechercher et remplacer)
3. Laissez "Rechercher" vide
4. Dans "Remplacer par" : 0
5. **Options** > Cochez **Totalite du contenu de la cellule**
6. **Remplacer tout**

### Attention

Cette methode modifie definitivement les cellules.

## Methode 3 : Collage special

### Etapes

1. Tapez 0 dans une cellule
2. Copiez cette cellule
3. Selectionnez les cellules vides
4. **Accueil** > **Rechercher** > **Atteindre** > **Cellules** > **Cellules vides**
5. **Collage special** > **Valeurs**
6. **OK**

## Methode 4 : Format personnalise

### Cette methode n'affiche que le 0

Les cellules restent vides mais affichent 0.

1. Selectionnez les cellules
2. **Ctrl + 1** (Format de cellule)
3. **Personnalisee**
4. Tapez : `0;-0;0;@`
5. **OK**

### Avantage

Les cellules restent vides pour les calculs.

## Methode 5 : Formule matricielle

### Remplacer toute une colonne

```excel
=SI(A1:A100=""; 0; A1:A100)
```

Validez avec **Ctrl + Maj + Entree** (versions anciennes).

### Excel 365

Les formules sont automatiquement dynamiques.

## Methode 6 : Power Query

### Etapes

1. **Donnees** > **Obtenir des donnees** > **A partir d'une table**
2. **Transformer** > **Remplacer les valeurs**
3. "Valeur a rechercher" : laisser vide
4. "Remplacer par" : 0

### Avantage

Transformation automatique a chaque actualisation.

## Formule avec calcul

### Si le resultat d'un calcul est vide

```excel
=SI(B1-C1=""; 0; B1-C1)
```

### Version simplifiee

```excel
=B1-C1+0
```

L'ajout de 0 force un resultat numerique.

### Avec SIERREUR

```excel
=SIERREUR(B1/C1; 0)
```

Retourne 0 en cas d'erreur.

## Afficher 0 au lieu de vide dans les formules

### Formule retournant vide

```excel
=SI(A1>10; A1; "")
```

### Formule retournant 0

```excel
=SI(A1>10; A1; 0)
```

## Ne pas afficher les zeros

### Si vous voulez l'inverse

1. **Fichier** > **Options**
2. **Options avancees**
3. Decochez **Afficher un zero dans les cellules ayant une valeur nulle**

### Par mise en forme

Format personnalise : `#;-#;;@`

Les zeros ne s'affichent pas.

## Cas pratiques

### Tableau de donnees

Avant :
| A | B | C |
|---|---|---|
| 1 |   | 3 |
|   | 2 |   |

Apres :
| A | B | C |
|---|---|---|
| 1 | 0 | 3 |
| 0 | 2 | 0 |

### Somme avec cellules vides

Les cellules vides sont ignorees par SOMME. Si vous voulez les compter comme 0, c'est deja le cas.

## Choisir la bonne methode

| Methode | Modification | Dynamique | Complexite |
|---------|--------------|-----------|------------|
| Formule SI | Non | Oui | Simple |
| Rechercher-Remplacer | Oui | Non | Tres simple |
| Collage special | Oui | Non | Simple |
| Format personnalise | Non | Visuel seulement | Moyen |
| Power Query | Non | Oui | Moyen |

## Recommandation

- **Pour un affichage** : Format personnalise
- **Pour les calculs** : Formule SI
- **Pour une modification definitive** : Rechercher-Remplacer
