# Quelle est la Meilleure Version d'Excel

Choisir la bonne version d'Excel selon vos besoins.

## Les versions actuelles

### Microsoft 365 (abonnement)

La version la plus complete et toujours à jour.

**Avantages** :
- Nouvelles fonctions des leur sortie
- FILTRE, UNIQUE, RECHERCHEX, LAMBDA
- 1 To de stockage OneDrive
- Mises à jour automatiques

**Inconvenient** :
- Abonnement mensuel/annuel

### Excel 2021 (achat unique)

Version perpetuelle sans abonnement.

**Avantages** :
- Paiement unique
- Fonctions modernes incluses
- RECHERCHEX, FILTRE, UNIQUE disponibles

**Inconvenient** :
- Pas de nouvelles fonctions après l'achat
- Pas de stockage cloud inclus

### Excel 2019

Version précédente, encore fonctionnelle.

**Avantages** :
- Prix plus bas
- Fonctions essentielles présentés

**Inconvenient** :
- Pas de FILTRE, UNIQUE, RECHERCHEX
- Plus de support etendu

## Comparatif des fonctions

| Fonction | 365 | 2021 | 2019 |
|----------|-----|------|------|
| RECHERCHEX | Oui | Oui | Non |
| FILTRE | Oui | Oui | Non |
| UNIQUE | Oui | Oui | Non |
| LAMBDA | Oui | Non | Non |
| LET | Oui | Oui | Non |
| Co-edition | Oui | Limitee | Non |

## Quelle version choisir ?

### Pour les particuliers

**Microsoft 365 Personnel** si :
- Vous voulez les dernières fonctions
- Vous utilisez OneDrive
- Budget : ~7 euros/mois

**Excel 2021** si :
- Vous préférez l'achat unique
- Pas besoin des futures nouveautes

### Pour les professionnels

**Microsoft 365 Business** si :
- Collaboration en equipe
- Sécurité avancée
- Outils d'administration

**Office LTSC 2021** si :
- Environnement hors ligne
- Reglementations strictes
- Pas de cloud

### Pour les etudiants

Vérifiez d'abord si votre etablissement offre Microsoft 365 gratuitement.

## Recommandation

| Profil | Version recommandée |
|--------|-------------------|
| Particulier frequent | Microsoft 365 |
| Utilisation occasionnelle | Excel Online (gratuit) |
| Entreprise | Microsoft 365 Business |
| Hors ligne obligatoire | Excel 2021 |

## Prix indicatifs

| Version | Type | Prix approximatif |
|---------|------|-------------------|
| 365 Personnel | Abonnement | 69 euros/an |
| 365 Famille | Abonnement | 99 euros/an |
| Office 2021 | Achat unique | 150 euros |

## Conclusion

Microsoft 365 offre le meilleur rapport fonctionnalités/prix pour un usage régulier. Excel 2021 reste une bonne option pour ceux qui preferent l'achat unique.
