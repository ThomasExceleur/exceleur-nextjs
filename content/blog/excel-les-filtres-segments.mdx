# Les Filtres Segments (Slicers)

Les segments sont des boutons visuels pour filtrer vos donnees.

## Qu'est-ce qu'un segment ?

Un segment est un filtre graphique permettant de :
- Filtrer un tableau ou TCD en un clic
- Voir les filtres actifs clairement
- Offrir une interface intuitive

## Creer un segment

### Pour un tableau

1. Cliquez dans votre tableau
2. **Insertion** > **Segment**
3. Selectionnez les colonnes souhaitees
4. **OK**

### Pour un TCD

1. Cliquez dans le TCD
2. **Analyse de tableau croise dynamique** > **Inserer un segment**
3. Choisissez les champs
4. **OK**

## Utiliser les segments

### Filtrer

Cliquez sur une valeur pour filtrer.

### Multi-selection

**Ctrl + clic** pour selectionner plusieurs valeurs.

### Effacer le filtre

Cliquez sur l'icone d'effacement (en haut a droite du segment).

## Personnalisation

### Redimensionner

Faites glisser les poignees du segment.

### Nombre de colonnes

1. Clic droit sur le segment
2. **Parametres du segment**
3. Modifiez le nombre de colonnes

### Style

1. Selectionnez le segment
2. **Outils Segment** > **Options**
3. Choisissez un style

## Connecter plusieurs segments

### A un seul TCD

Par defaut, un segment filtre son TCD d'origine.

### A plusieurs TCD

1. Clic droit sur le segment
2. **Connexions de rapport**
3. Cochez les TCD a connecter

## Segment de chronologie

### Pour les dates

1. Selectionnez le TCD
2. **Insertion** > **Chronologie**
3. Choisissez le champ date

### Niveaux disponibles

- Annees
- Trimestres
- Mois
- Jours

## Avantages des segments

| Avantage | Description |
|----------|-------------|
| Visibilite | Filtres actifs visibles |
| Rapidite | Un clic pour filtrer |
| Multi-selection | Plusieurs valeurs simultanees |
| Connexion | Un segment, plusieurs TCD |

## Bonnes pratiques

1. **Placez** les segments pres des donnees
2. **Limitez** le nombre de segments (3-5 max)
3. **Utilisez** des couleurs coherentes
4. **Connectez** les segments aux TCD lies

## Exemple de tableau de bord

Combinez :
- Segment Region
- Segment Produit
- Chronologie Date
- TCD de synthese

Tous les elements se filtrent ensemble.

## Limites

- Pas disponible pour les plages simples (avant 2013)
- Peut ralentir les grands fichiers
- Necessite un tableau ou TCD
