# Excel en PDF

Convertissez vos feuilles Excel en documents PDF professionnels.

## Methode 1 : Enregistrer sous

### Etapes

1. **Fichier** > **Enregistrer sous**
2. Choisissez l'emplacement
3. **Type** : PDF
4. **Enregistrer**

### Raccourci

**F12** ouvre directement "Enregistrer sous".

## Methode 2 : Exporter

### Etapes

1. **Fichier** > **Exporter**
2. **Creer un document PDF/XPS**
3. **Creer PDF/XPS**
4. Choisissez l'emplacement
5. **Publier**

## Options d'export

### Avant de publier

Cliquez sur **Options** pour configurer :

| Option | Description |
|--------|-------------|
| Feuilles selectionnees | Seules les feuilles actives |
| Classeur entier | Toutes les feuilles |
| Selection | Seulement les cellules selectionnees |
| Tableau | Un tableau specifique |

## Mise en page avant export

### Verifier l'apercu

**Fichier** > **Imprimer** pour voir le rendu.

### Ajuster la zone

1. **Mise en page** > **Zone d'impression** > **Definir**
2. Selectionnez les cellules a exporter

### Orientation

1. **Mise en page** > **Orientation**
2. Portrait ou Paysage selon le contenu

### Ajuster a la page

1. **Mise en page** > **Largeur** : 1 page
2. **Mise en page** > **Hauteur** : Automatique

## En-tetes et pieds de page

### Configuration

1. **Insertion** > **En-tete et pied de page**
2. Ajoutez titre, date, numero de page

### Codes utiles

| Code | Resultat |
|------|----------|
| &[Page] | Numero de page |
| &[Pages] | Nombre total de pages |
| &[Date] | Date du jour |
| &[Fichier] | Nom du fichier |

## Qualite du PDF

### Standard

Qualite equilibree, taille reduite.

### Minimale

Qualite reduite, fichier leger. Pour partage web.

## Exporter plusieurs feuilles

### Classeur entier

Dans les options, selectionnez "Classeur entier".

### Feuilles specifiques

1. Maintenez **Ctrl** et cliquez sur les onglets
2. Exportez les "Feuilles selectionnees"

## Problemes courants

### Colonnes coupees

1. Reduisez les marges
2. Passez en paysage
3. Ajustez la mise a l'echelle

### Lignes sur plusieurs pages

Definissez les lignes a repeter :
**Mise en page** > **Imprimer les titres**

### PDF trop lourd

Reduisez la qualite des images.

## Bonnes pratiques

1. **Verifiez** l'apercu avant d'exporter
2. **Definissez** la zone d'impression
3. **Ajustez** les marges et l'orientation
4. **Testez** sur plusieurs pages
