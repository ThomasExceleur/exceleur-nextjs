# Power Pivot : Modele de Donnees et DAX

Power Pivot permet d'analyser des millions de lignes et de creer des modeles de donnees relationnels.

## Qu'est-ce que Power Pivot ?

### Definition

Power Pivot est un complement Excel pour :
- Importer des donnees volumineuses
- Creer des relations entre tables
- Ecrire des formules DAX avancees
- Alimenter des tableaux croises dynamiques

### Disponibilite

- Excel 2016+ Professionnel
- Excel 365

## Activer Power Pivot

### Etapes

1. **Fichier** > **Options** > **Complements**
2. **Gerer** : Complements COM
3. **Atteindre**
4. Cochez **Microsoft Power Pivot for Excel**
5. **OK**

## Le modele de donnees

### Concept

Plusieurs tables reliees par des cles, comme une base de donnees.

### Exemple

- Table Ventes (ID, DateVente, ProduitID, Montant)
- Table Produits (ProduitID, Nom, Categorie)
- Table Dates (Date, Mois, Annee)

### Creer une relation

1. **Power Pivot** > **Gerer**
2. Vue **Diagramme**
3. Faites glisser un champ sur un autre

## Introduction au DAX

### Qu'est-ce que DAX ?

Data Analysis Expressions - langage de formules pour Power Pivot.

### Mesures de base

```dax
Total Ventes = SUM(Ventes[Montant])
```

```dax
Nb Commandes = COUNTROWS(Ventes)
```

```dax
Moyenne = AVERAGE(Ventes[Montant])
```

## Fonctions DAX essentielles

### Agregation

| Fonction | Description |
|----------|-------------|
| SUM | Somme |
| AVERAGE | Moyenne |
| COUNT | Compte les nombres |
| COUNTROWS | Compte les lignes |
| MIN / MAX | Minimum / Maximum |

### Logique

```dax
Statut = IF(Ventes[Montant] > 1000; "VIP"; "Standard")
```

### Temps

```dax
Ventes YTD = TOTALYTD(SUM(Ventes[Montant]); Dates[Date])
```

## Creer une mesure

### Etapes

1. Dans Power Pivot, selectionnez une table
2. Zone de calcul en bas
3. Tapez la formule DAX
4. Appuyez sur Entree

### Exemple

```dax
Marge = SUM(Ventes[CA]) - SUM(Ventes[Cout])
```
