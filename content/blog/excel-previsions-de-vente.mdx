# Previsions de Vente dans Excel

Des previsions de vente fiables sont essentielles pour piloter votre activité.

## Méthodes de prevision

### Vue d'ensemble

| Méthode | Utilisation | Precision |
|---------|-------------|-----------|
| Moyenne mobile | Données stables | Moyenne |
| Tendance lineaire | Croissance régulière | Bonne |
| Saisonnalite | Cycles recurrents | Élevée |
| PREVISION.ETS | Données complexes | Élevée |

## Moyenne mobile

### Formule simple

```excel
=MOYENNE(B2:B4)              ' Moyenne des 3 derniers mois
```

### Moyenne mobile ponderee

```excel
=(B2*0.5)+(B3*0.3)+(B4*0.2)  ' Poids decroissants
```

### Moyenne glissante

```excel
=MOYENNE(DECALER(B5;-2;0;3;1))    ' 3 périodes avant la cellule
```

## Tendance lineaire

### Fonction TENDANCE

```excel
=TENDANCE(Ventes;Périodes;Nouvelle_periode)
```

### Exemple pratique

```excel
=TENDANCE(B2:B13;A2:A13;A14)    ' Prevision du mois 13
```

### Fonction DROITEREG

```excel
=DROITEREG(Ventes;Périodes)     ' Retourne pente et ordonnee
```

## Regression lineaire

### Equation de la droite

```
Y = à*X + b
```

### Calcul des coefficients

```excel
=PENTE(Ventes;Périodes)         ' Coefficient à
=ORDONNEE.ORIGINE(Ventes;Périodes)  ' Coefficient b
```

### Prevision

```excel
=PENTE(B2:B13;A2:A13)*A14 + ORDONNEE.ORIGINE(B2:B13;A2:A13)
```

## Coefficient de determination (R2)

```excel
=COEFFICIENT.DETERMINATION(Ventes;Périodes)
```

Plus R2 est proche de 1, meilleure est la prevision.

## Saisonnalite

### Identifier les indices saisonniers

| Mois | Ventes | Moyenne | Indice |
|------|--------|---------|--------|
| Jan | 80 | 100 | 0.80 |
| Fev | 85 | 100 | 0.85 |
| ... | ... | ... | ... |
| Dec | 130 | 100 | 1.30 |

### Formule indice saisonnier

```excel
=Ventes_mois / Moyenne_annuelle
```

### Prevision avec saisonnalite

```excel
=Tendance_prevue * Indice_saisonnier
```

## Fonction PREVISION.ETS

### Syntaxe

```excel
=PREVISION.ETS(Date_cible;Valeurs;Dates;[Saisonnalite];[Completion];[Agregation])
```

### Exemple

```excel
=PREVISION.ETS(DATE(2024;1;1);B2:B25;A2:A25)
```

### Intervalle de confiance

```excel
=PREVISION.ETS.CONFINT(Date_cible;Valeurs;Dates;Niveau_confiance)
```

## Graphique de prevision

### Créer un graphique avec prevision

1. Sélectionnez vos données historiques
2. **Insertion** > **Graphique** > **Courbes**
3. **Conception** > **Ajouter un élément** > **Courbe de tendance**

### Options de courbe de tendance

- Lineaire
- Exponentielle
- Polynomiale
- Moyenne mobile

## Ajustement des previsions

### Facteurs d'ajustement

| Facteur | Impact |
|---------|--------|
| Nouveau produit | +X% |
| Perte client majeur | -Y% |
| Saisonnalite | Indice |
| Économie | +/- Z% |

### Formule ajustee

```excel
=Prevision_base * (1 + Ajustement_total)
```

## Precision des previsions

### Mesure de l'erreur (MAPE)

```excel
=MOYENNE(ABS((Reel-Prevision)/Reel))
```

### Suivi de la precision

| Mois | Prevision | Reel | Erreur |
|------|-----------|------|--------|
| Jan | 100 | 95 | 5% |
| Fev | 105 | 108 | 3% |
