# Guide Pratique Power Pivot Excel

Mettez Power Pivot en pratique avec des exemples concrets.

## Cas d'usage : Analyse des ventes

### Donnees

Tables :
- Ventes (Date, ProduitID, ClientID, Montant)
- Produits (ProduitID, Nom, Categorie)
- Clients (ClientID, Nom, Region)
- Calendrier (Date, Annee, Mois, Trimestre)

### Objectif

Analyser les ventes par produit, client, region et periode.

## Etape 1 : Importer les donnees

### Depuis Excel

1. **Power Pivot** > **Gerer**
2. **Accueil** > **Obtenir des donnees externes** > **Excel**
3. Importez chaque table

### Conseil

Importez des tableaux structures (Ctrl+T).

## Etape 2 : Creer les relations

### Vue Diagramme

1. **Affichage** > **Vue Diagramme**
2. Glissez ProduitID vers ProduitID
3. Glissez ClientID vers ClientID
4. Glissez Date vers Date

### Schema

```
Ventes (fait) -> Produits (dimension)
Ventes (fait) -> Clients (dimension)
Ventes (fait) -> Calendrier (dimension)
```

## Etape 3 : Mesures de base

### Total des ventes

```dax
Total Ventes := SUM(Ventes[Montant])
```

### Nombre de transactions

```dax
Nb Transactions := COUNT(Ventes[ID])
```

### Panier moyen

```dax
Panier Moyen := DIVIDE([Total Ventes]; [Nb Transactions])
```

## Etape 4 : Mesures temporelles

### Table calendrier

Essentielle pour les calculs temporels.

### Ventes annee precedente

```dax
Ventes N-1 := CALCULATE(
    [Total Ventes];
    SAMEPERIODLASTYEAR(Calendrier[Date])
)
```

### Variation

```dax
Variation := DIVIDE([Total Ventes] - [Ventes N-1]; [Ventes N-1])
```

### Cumul annuel

```dax
YTD := TOTALYTD([Total Ventes]; Calendrier[Date])
```

## Etape 5 : Mesures analytiques

### Part du total

```dax
% Total := DIVIDE(
    [Total Ventes];
    CALCULATE([Total Ventes]; ALL(Ventes))
)
```

### Classement

```dax
Rang := RANKX(ALL(Produits); [Total Ventes]; ; DESC)
```

## Etape 6 : Creer le TCD

### Ajouter au classeur

1. **Power Pivot** > **Tableau croise dynamique**
2. Placez les champs
3. Utilisez les mesures dans Valeurs

### Configuration

- Lignes : Categorie, Produit
- Colonnes : Annee, Trimestre
- Valeurs : Total Ventes, Variation

## Etape 7 : Visualiser

### Graphiques croises

Ajoutez des graphiques lies au TCD.

### Segments

Ajoutez des filtres interactifs.

## Optimisations

### Colonnes calculees vs Mesures

| Colonnes calculees | Mesures |
|-------------------|---------|
| Categories fixes | Valeurs dynamiques |
| Stockees | Calculees a la volee |
| Contexte de ligne | Contexte de filtre |

### Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Mesures preferees | Plus flexibles que colonnes |
| Nommage | Prefixes par type (Mes_, Col_) |
| Documentation | Commentez les formules DAX |
| Test | Verifiez les resultats |
