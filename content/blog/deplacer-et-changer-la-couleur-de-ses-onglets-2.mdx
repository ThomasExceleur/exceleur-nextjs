# Deplacer et Colorer ses Onglets

Organisez visuellement votre classeur.

## Deplacer un onglet

### Methode glisser

1. Cliquez sur l'onglet
2. Maintenez et glissez
3. Deposez a la nouvelle position

### Via le menu

1. Clic droit sur l'onglet
2. **Deplacer ou copier**
3. Choisissez la position

## Changer la couleur

### Etapes

1. Clic droit sur l'onglet
2. **Couleur d'onglet**
3. Selectionnez une couleur

### Palette

- Couleurs du theme
- Couleurs standard
- Plus de couleurs (personnalise)

## Organisation par couleur

### Suggestions

| Couleur | Usage |
|---------|-------|
| Vert | Donnees validees |
| Jaune | En cours |
| Rouge | A verifier |
| Bleu | Parametres |
| Gris | Archives |

## Copier un onglet

### Dans le meme classeur

1. **Ctrl + Glisser** l'onglet
2. Ou clic droit > **Deplacer ou copier** > **Creer une copie**

### Vers un autre classeur

1. Clic droit > **Deplacer ou copier**
2. Selectionnez le classeur destination
3. Cochez **Creer une copie** si besoin

## Raccourcis navigation

| Raccourci | Action |
|-----------|--------|
| Ctrl+Page Up | Onglet precedent |
| Ctrl+Page Down | Onglet suivant |
| Clic droit sur fleches | Liste des onglets |

## Renommer

Double-cliquez sur l'onglet et tapez le nouveau nom.

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Convention | Code couleur standard |
| Noms courts | Facilitez la lecture |
| Ordre logique | Workflow naturel |
