# Introduction a Power Pivot

Analysez des millions de lignes avec Power Pivot.

## Qu'est-ce que Power Pivot ?

### Definition

Complement Excel pour :

- Importer de gros volumes
- Creer des relations
- Calculer avec DAX

### Avantages

| Avantage | Description |
|----------|-------------|
| Volume | Millions de lignes |
| Sources multiples | SQL, CSV, Web, etc. |
| Relations | Modele de donnees |
| DAX | Calculs avances |

## Activer Power Pivot

### Etapes

1. **Fichier** > **Options**
2. **Complements**
3. **Complements COM** > **Atteindre**
4. Cochez **Microsoft Power Pivot for Excel**

## Importer des donnees

### Sources

- Fichiers Excel
- Bases SQL
- Fichiers CSV
- Services web

### Processus

1. **Power Pivot** > **Gerer**
2. **Accueil** > **Obtenir des donnees externes**

## Creer des relations

### Vue Diagramme

1. Passez en vue Diagramme
2. Glissez un champ vers un autre
3. Definissez la cardinalite

## Introduction au DAX

### Mesures de base

```dax
Total Ventes := SUM(Ventes[Montant])
```

### Mesures calculees

```dax
Marge := [Total Ventes] - [Total Couts]
```

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Nommage | Tables et colonnes explicites |
| Relations | Une cle par relation |
| Mesures | Creez des mesures, pas des colonnes |
