# Introduction à Power Pivot

Analysez des millions de lignes avec Power Pivot.

## Qu'est-ce que Power Pivot ?

### Définition

Complement Excel pour :

- Importer de gros volumes
- Créer des relations
- Calculer avec DAX

### Avantages

| Avantage | Description |
|----------|-------------|
| Volume | Millions de lignes |
| Sources multiples | SQL, CSV, Web, etc. |
| Relations | Modèle de données |
| DAX | Calculs avancés |

## Activer Power Pivot

### Étapes

1. **Fichier** > **Options**
2. **Complements**
3. **Complements COM** > **Atteindre**
4. Cochez **Microsoft Power Pivot for Excel**

## Importer des données

### Sources

- Fichiers Excel
- Bases SQL
- Fichiers CSV
- Services web

### Processus

1. **Power Pivot** > **Gérer**
2. **Accueil** > **Obtenir des données externes**

## Créer des relations

### Vue Diagramme

1. Passez en vue Diagramme
2. Glissez un champ vers un autre
3. Définissez la cardinalite

## Introduction au DAX

### Mesures de base

```dax
Total Ventes := SUM(Ventes[Montant])
```

### Mesures calculees

```dax
Marge := [Total Ventes] - [Total Couts]
```

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Nommage | Tables et colonnes explicites |
| Relations | Une clé par relation |
| Mesures | Créez des mesures, pas des colonnes |
