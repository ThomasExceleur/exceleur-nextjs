# Erreur #VALEUR! dans Excel : Solutions

L'erreur #VALEUR! indique que quelque chose ne va pas avec les types de donnees dans votre formule.

## Signification

#VALEUR! signifie qu'Excel a trouve un type de donnees inattendu.

## Causes courantes

### Cause 1 : Operation sur du texte

```excel
="abc" + 5    ' Erreur #VALEUR!
```

On ne peut pas additionner du texte et un nombre.

### Solution

Verifiez que vos cellules contiennent bien des nombres.

### Cause 2 : Espace dans une cellule "vide"

### Symptome

La cellule semble vide mais contient un espace.

### Detection

```excel
=NBCAR(A1)
```

Retourne 1 ou plus si la cellule contient des espaces.

### Solution

```excel
=SUPPRESPACE(A1)
```

### Cause 3 : Nombres stockes comme texte

### Symptome

Les nombres sont alignes a gauche.

### Detection

```excel
=TYPE(A1)
```

Retourne 2 si c'est du texte.

### Solutions

```excel
=CNUM(A1)
=A1*1
=A1+0
```

### Cause 4 : Arguments de fonction incorrects

```excel
=RECHERCHEV(A1; B1; 2; FAUX)    ' B1 doit etre une plage
```

### Solution

Verifiez que chaque argument a le bon type.

### Cause 5 : Plages de tailles differentes

```excel
=A1:A5 + B1:B3    ' Erreur car tailles differentes
```

### Solution

Utilisez des plages de meme taille.

### Cause 6 : Caracteres invisibles

### Detection

Des caracteres non imprimables peuvent causer des erreurs.

### Solution

```excel
=EPURAGE(A1)
```

### Cause 7 : Dates en texte

```excel
=A1 - B1    ' Erreur si les dates sont en texte
```

### Solution

```excel
=DATEVAL(A1) - DATEVAL(B1)
```

## Gerer l'erreur

### SIERREUR

```excel
=SIERREUR(formule; "Erreur de valeur")
=SIERREUR(formule; 0)
=SIERREUR(formule; "")
```

### Verification prealable

```excel
=SI(ESTNUM(A1); A1+B1; "Verifier A1")
```

## Diagnostic

### Etape par etape

1. Identifiez quelle cellule cause l'erreur
2. Verifiez le contenu reel (barre de formule)
3. Verifiez le type avec TYPE()
4. Nettoyez les donnees

### Formule de test

```excel
=SI(ESTERREUR(formule); "Erreur"; formule)
```

## Exemples de correction

### Addition avec texte potentiel

**Avant** :
```excel
=A1 + B1
```

**Apres** :
```excel
=CNUM(A1) + CNUM(B1)
```

### RECHERCHEV protegee

**Avant** :
```excel
=RECHERCHEV(A1; B:D; 2; FAUX)
```

**Apres** :
```excel
=SIERREUR(RECHERCHEV(SUPPRESPACE(A1); B:D; 2; FAUX); "")
```

## Cas particuliers

### Formules matricielles

Les anciennes formules matricielles peuvent generer #VALEUR! si mal validees.

**Solution** : Validez avec Ctrl + Maj + Entree (anciennes versions).

### Fonctions de date

```excel
=JOUR("abc")    ' Erreur #VALEUR!
```

**Solution** : Verifiez que la cellule contient une vraie date.

## Prevention

### Validation des donnees

1. **Donnees** > **Validation des donnees**
2. Limitez les saisies aux types corrects

### Mise en forme conditionnelle

Colorez les cellules qui pourraient causer des erreurs.

### Verification automatique

```excel
=SI(ESTNUM(A1); "OK"; "Verifier")
```

1. **Nettoyez** les donnees importees
2. **Verifiez** les types avant calcul
3. **Utilisez** SIERREUR pour gerer les erreurs
4. **Validez** les saisies utilisateur
