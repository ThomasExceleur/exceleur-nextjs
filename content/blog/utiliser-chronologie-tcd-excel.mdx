# Chronologies dans les TCD Excel

Filtrez visuellement vos donnees par periode.

## Qu'est-ce qu'une chronologie ?

### Definition

Un filtre visuel pour les dates dans les tableaux croises dynamiques.

### Avantages

- Selection intuitive des periodes
- Filtrage rapide
- Visualisation claire

## Creer une chronologie

### Etapes

1. Cliquez dans le TCD
2. **Insertion** > **Chronologie**
3. Selectionnez le champ de date
4. **OK**

## Niveaux de filtrage

| Niveau | Granularite |
|--------|-------------|
| Annees | Par annee |
| Trimestres | Par trimestre |
| Mois | Par mois |
| Jours | Par jour |

## Selectionner une periode

### Un element

Cliquez sur un mois/trimestre/annee.

### Plage continue

Cliquez-glissez pour selectionner plusieurs periodes.

### Plusieurs elements non contigus

Ctrl + Clic.

## Personnalisation

### Style

**Outils Chronologie** > **Options** > **Styles**

### Taille

Redimensionnez en glissant les bords.

### Position

Glissez la chronologie ou elle doit apparaitre.

## Connecter plusieurs TCD

### Methode

1. Clic droit sur la chronologie
2. **Connexions de rapport**
3. Cochez les TCD a connecter

### Effet

Un seul filtre pour plusieurs tableaux.

## Effacer le filtre

Cliquez sur le bouton **Effacer le filtre** dans la chronologie.

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Niveau adapte | Jour pour recent, annee pour historique |
| Position | Proche des TCD filtres |
| Connexion | Reliez tous les TCD pertinents |
