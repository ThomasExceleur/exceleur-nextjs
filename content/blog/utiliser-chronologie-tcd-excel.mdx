# Chronologies dans les TCD Excel

Filtrez visuellement vos données par période.

## Qu'est-ce qu'une chronologie ?

### Définition

Un filtre visuel pour les dates dans les tableaux croisés dynamiques.

### Avantages

- Sélection intuitive des périodes
- Filtrage rapide
- Visualisation claire

## Créer une chronologie

### Étapes

1. Cliquez dans le TCD
2. **Insertion** > **Chronologie**
3. Sélectionnez le champ de date
4. **OK**

## Niveaux de filtrage

| Niveau | Granularite |
|--------|-------------|
| Années | Par année |
| Trimestres | Par trimestre |
| Mois | Par mois |
| Jours | Par jour |

## Sélectionner une période

### Un élément

Cliquez sûr un mois/trimestre/année.

### Plage continue

Cliquez-glissez pour sélectionner plusieurs périodes.

### Plusieurs éléments non contigus

Ctrl + Clic.

## Personnalisation

### Style

**Outils Chronologie** > **Options** > **Styles**

### Taille

Redimensionnez en glissant les bords.

### Position

Glissez la chronologie où elle doit apparaitre.

## Connecter plusieurs TCD

### Méthode

1. Clic droit sûr la chronologie
2. **Connexions de rapport**
3. Cochez les TCD à connecter

### Effet

Un seul filtre pour plusieurs tableaux.

## Effacer le filtre

Cliquez sûr le bouton **Effacer le filtre** dans la chronologie.

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Niveau adapte | Jour pour recent, année pour historique |
| Position | Proche des TCD filtres |
| Connexion | Reliez tous les TCD pertinents |
