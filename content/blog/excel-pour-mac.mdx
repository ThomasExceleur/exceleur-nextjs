# Excel pour Mac : Guide Complet

Excel pour Mac offre une experience similaire a Windows avec quelques differences importantes a connaitre.

## Differences principales avec Windows

### Interface

- Ruban plus compact sur Mac
- Preferences au lieu d'Options
- Menus differemment organises

### Fonctionnalites absentes ou limitees

- Power Query : Disponible mais moins complet
- Power Pivot : Non disponible
- Certains graphiques 3D
- Formulaires VBA (UserForms)

### Fonctionnalites identiques

- Toutes les formules principales
- Tableaux croises dynamiques
- Mise en forme conditionnelle
- Graphiques standards

## Raccourcis clavier Mac

### Equivalences de base

| Windows | Mac | Action |
|---------|-----|--------|
| Ctrl + C | Cmd + C | Copier |
| Ctrl + V | Cmd + V | Coller |
| Ctrl + Z | Cmd + Z | Annuler |
| Ctrl + S | Cmd + S | Enregistrer |
| Ctrl + B | Cmd + B | Gras |
| Ctrl + I | Cmd + I | Italique |

### Navigation

| Windows | Mac | Action |
|---------|-----|--------|
| Ctrl + Fleche | Cmd + Fleche | Aller au bord |
| Ctrl + Home | Fn + Cmd + Gauche | Aller en A1 |
| Ctrl + End | Fn + Cmd + Droite | Derniere cellule |
| F5 | Ctrl + G | Atteindre |

### Formules

| Windows | Mac | Action |
|---------|-----|--------|
| F4 | Cmd + T | Figer reference |
| F2 | Ctrl + U | Modifier cellule |
| Ctrl + ` | Ctrl + ` | Afficher formules |
| Alt + = | Cmd + Maj + T | Somme auto |

### Selection

| Windows | Mac | Action |
|---------|-----|--------|
| Ctrl + Maj + End | Cmd + Maj + End | Selectionner jusqu'a la fin |
| Ctrl + A | Cmd + A | Selectionner tout |
| Ctrl + Espace | Ctrl + Espace | Selectionner colonne |
| Maj + Espace | Maj + Espace | Selectionner ligne |

## Parametres et preferences

### Acceder aux preferences

**Excel** > **Preferences** (ou **Cmd + ,**)

### Sections importantes

- **General** : Options de base
- **Modification** : Comportement des cellules
- **Calcul** : Mode de calcul
- **Affichage** : Apparence

## VBA sur Mac

### Differences

- L'editeur VBA fonctionne sur Mac
- Certaines fonctionnalites Windows-only ne marchent pas
- Les UserForms ne sont pas supportes
- Certaines APIs systeme different

### Code compatible Mac

Utilisez des conditions pour adapter votre code :

```vba
#If Mac Then
    ' Code specifique Mac
#Else
    ' Code Windows
#End If
```

## Problemes courants et solutions

### Fichier qui ne s'ouvre pas

1. Verifiez l'extension du fichier
2. Essayez d'ouvrir depuis Excel (Fichier > Ouvrir)
3. Verifiez les permissions du fichier

### Problemes de polices

Certaines polices Windows ne sont pas disponibles sur Mac. Excel substitue automatiquement.

### Macros qui ne fonctionnent pas

1. Verifiez que les macros sont activees
2. Adaptez le code pour Mac si necessaire

## Excel Online comme alternative

Si une fonctionnalite manque sur Mac, essayez Excel Online :
- Fonctionne dans Safari ou Chrome
- Certaines fonctionnalites Windows disponibles
- Sauvegarde automatique sur OneDrive

## Mise a jour d'Excel

### Via App Store

1. Ouvrez l'App Store
2. Cliquez sur Mises a jour
3. Mettez a jour Microsoft Excel

### Via Microsoft AutoUpdate

1. Ouvrez Excel
2. **Aide** > **Rechercher les mises a jour**

## Conseils pour utilisateurs Mac

- Utilisez les gestes du trackpad pour zoomer
- Activez le scroll naturel dans Preferences Systeme
- Utilisez Mission Control pour basculer entre classeurs
- Configurez Hot Corners pour un acces rapide
