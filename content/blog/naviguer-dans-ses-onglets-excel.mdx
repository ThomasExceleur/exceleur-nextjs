# Naviguer entre les Onglets Excel

Gerer plusieurs feuilles dans un classeur est plus facile avec ces astuces.

## Raccourcis de navigation

### Changer de feuille

| Raccourci | Action |
|-----------|--------|
| Ctrl + Page suivante | Feuille suivante |
| Ctrl + Page precedente | Feuille precedente |

### Sur Mac

| Raccourci | Action |
|-----------|--------|
| Option + Fleche droite | Feuille suivante |
| Option + Fleche gauche | Feuille precedente |

## Boutons de navigation

### Fleches en bas a gauche

Les fleches a cote des onglets permettent de :
- Aller au premier/dernier onglet
- Defiler vers la gauche/droite

### Clic droit sur les fleches

Affiche la liste de toutes les feuilles pour un acces direct.

## Gerer les onglets

### Renommer une feuille

1. Double-cliquez sur l'onglet
2. Tapez le nouveau nom
3. **Entree**

### Changer la couleur

1. Clic droit sur l'onglet
2. **Couleur d'onglet**
3. Choisissez une couleur

### Deplacer une feuille

1. Cliquez sur l'onglet et maintenez
2. Faites glisser a la nouvelle position
3. Relachez

### Copier une feuille

1. **Ctrl + Glisser** l'onglet
2. Ou clic droit > **Deplacer ou copier** > **Creer une copie**

## Inserer et supprimer

### Nouvelle feuille

- **Maj + F11** : nouvelle feuille
- Ou clic sur le **+** a cote des onglets

### Supprimer une feuille

1. Clic droit sur l'onglet
2. **Supprimer**
3. Confirmez

## Selectionner plusieurs feuilles

### Selection continue

1. Cliquez sur le premier onglet
2. **Maj + Clic** sur le dernier

### Selection non continue

1. Cliquez sur le premier onglet
2. **Ctrl + Clic** sur les autres

### Desactiver le groupe

Cliquez sur un onglet non selectionne.

## Masquer des feuilles

### Masquer

1. Clic droit sur l'onglet
2. **Masquer**

### Afficher

1. Clic droit sur un onglet visible
2. **Afficher**
3. Selectionnez la feuille

## Proteger les onglets

### Proteger la structure

1. **Revision** > **Proteger le classeur**
2. Cochez **Structure**
3. Definissez un mot de passe (optionnel)

Cela empeche l'ajout, la suppression ou le renommage de feuilles.

## Organiser ses feuilles

### Conventions de nommage

- Noms courts et explicites
- Utilisez des prefixes (01_, 02_)
- Evitez les caracteres speciaux

### Couleurs par type

| Couleur | Type de feuille |
|---------|-----------------|
| Bleu | Donnees |
| Vert | Calculs |
| Rouge | Parametres |
| Gris | Archive |

## Atteindre une feuille specifique

### Avec la boite de dialogue

1. **Ctrl + G** (Atteindre)
2. Tapez le nom de la feuille et une cellule
3. Exemple : `Budget!A1`

### Avec un lien hypertexte

```excel
=LIEN_HYPERTEXTE("#Budget!A1";"Aller au Budget")
```

## Sommaire des feuilles

### Creer un index

1. Listez tous les noms de feuilles
2. Ajoutez des liens hypertexte vers chaque feuille

## Bonnes pratiques

1. **Nommez** vos feuilles clairement
2. **Colorez** par type de contenu
3. **Ordonnez** logiquement
4. **Masquez** les feuilles techniques
5. **Creez** un sommaire si nombreuses feuilles
