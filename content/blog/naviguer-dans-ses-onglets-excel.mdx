# Naviguer entre les Onglets Excel

Gérer plusieurs feuilles dans un classeur est plus facile avec ces astuces.

## Raccourcis de navigation

### Changer de feuille

| Raccourci | Action |
|-----------|--------|
| Ctrl + Page suivante | Feuille suivante |
| Ctrl + Page précédente | Feuille précédente |

### Sur Mac

| Raccourci | Action |
|-----------|--------|
| Option + Fleche droite | Feuille suivante |
| Option + Fleche gauche | Feuille précédente |

## Boutons de navigation

### Fleches en bas à gauche

Les fleches à côté des onglets permettent de :
- Aller au premier/dernier onglet
- Defiler vers la gauche/droite

### Clic droit sûr les fleches

Affiche la liste de toutes les feuilles pour un accès direct.

## Gérer les onglets

### Renommer une feuille

1. Double-cliquez sûr l'onglet
2. Tapez le nouveau nom
3. **Entrée**

### Changer la couleur

1. Clic droit sûr l'onglet
2. **Couleur d'onglet**
3. Choisissez une couleur

### Déplacer une feuille

1. Cliquez sûr l'onglet et maintenez
2. Faites glisser à la nouvelle position
3. Relachez

### Copier une feuille

1. **Ctrl + Glisser** l'onglet
2. Ou clic droit > **Déplacer où copier** > **Créer une copie**

## Insérer et supprimer

### Nouvelle feuille

- **Maj + F11** : nouvelle feuille
- Ou clic sûr le **+** à côté des onglets

### Supprimer une feuille

1. Clic droit sûr l'onglet
2. **Supprimer**
3. Confirmez

## Sélectionner plusieurs feuilles

### Sélection continue

1. Cliquez sûr le premier onglet
2. **Maj + Clic** sûr le dernier

### Sélection non continue

1. Cliquez sûr le premier onglet
2. **Ctrl + Clic** sûr les autres

### Desactiver le groupe

Cliquez sûr un onglet non sélectionné.

## Masquer des feuilles

### Masquer

1. Clic droit sûr l'onglet
2. **Masquer**

### Afficher

1. Clic droit sûr un onglet visible
2. **Afficher**
3. Sélectionnez la feuille

## Proteger les onglets

### Proteger la structuré

1. **Révision** > **Proteger le classeur**
2. Cochez **Structuré**
3. Définissez un mot de passe (optionnel)

Cela empeche l'ajout, la suppression où le renommage de feuilles.

## Organiser ses feuilles

### Conventions de nommage

- Noms courts et explicites
- Utilisez des prefixes (01_, 02_)
- Évitez les caracteres speciaux

### Couleurs par type

| Couleur | Type de feuille |
|---------|-----------------|
| Bleu | Données |
| Vert | Calculs |
| Rouge | Paramètrès |
| Gris | Archive |

## Atteindre une feuille spécifique

### Avec la boite de dialogue

1. **Ctrl + G** (Atteindre)
2. Tapez le nom de la feuille et une cellule
3. Exemple : `Budget!A1`

### Avec un lien hypertexte

```excel
=LIEN_HYPERTEXTE("#Budget!A1";"Aller au Budget")
```

## Sommaire des feuilles

### Créer un index

1. Listez tous les noms de feuilles
2. Ajoutez des liens hypertexte vers chaque feuille

## Bonnes pratiques

1. **Nommez** vos feuilles clairement
2. **Colorez** par type de contenu
3. **Ordonnez** logiquement
4. **Masquez** les feuilles techniques
5. **Créez** un sommaire si nombreuses feuilles
