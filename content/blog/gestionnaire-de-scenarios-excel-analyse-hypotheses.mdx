# Gestionnaire de Scenarios : Analyse d'Hypotheses

Comparez facilement différentes hypotheses de calcul.

## Qu'est-ce que le gestionnaire de scenarios ?

### Définition

Outil permettant de créer et comparer plusieurs jeux de valeurs.

### Usage

- Previsions financieres
- Budgetisation
- Planification
- Prise de decision

## Accès

**Données** > **Analyse de scenarios** > **Gestionnaire de scenarios**

## Créer un scenario

### Étapes

1. Ouvrez le gestionnaire de scenarios
2. Cliquez **Ajouter**
3. Nommez le scenario
4. Sélectionnez les cellules variables
5. Entrez les valeurs

### Exemple

| Cellule | Description |
|---------|-------------|
| B1 | Prix de vente |
| B2 | Quantité |
| B3 | Cout unitaire |
| B5 | Profit (formule) |

## Scenarios types

### Optimiste

Meilleures hypotheses :
- Ventes élevées
- Couts faibles
- Prix optimal

### Base

Hypotheses realistes.

### Pessimiste

Hypotheses defavorables :
- Ventes faibles
- Couts élevés
- Pression prix

## Afficher un scenario

### Méthode

1. Ouvrez le gestionnaire
2. Sélectionnez le scenario
3. Cliquez **Afficher**

### Effet

Les valeurs du scenario sont appliquees aux cellules.

## Synthèse des scenarios

### Générer un rapport

1. **Gestionnaire de scenarios** > **Synthèse**
2. Choisissez les cellules de résultat
3. **OK**

### Résultat

Un tableau comparatif est créé sûr une nouvelle feuille.

## Scenarios et protection

### Empecher les modifications

1. Lors de la création du scenario
2. Cochez **Empecher les modifications**

### Masquer un scenario

Cochez **Masquer** dans les options.

## Combiner avec d'autres outils

### Tables de données

Pour des variations systematiques.

### Valeur cible

Pour trouver la valeur d'entrée nécessaire.

### Solveur

Pour l'optimisation sous contraintes.

## Cas pratique : Business plan

### Variables

- Chiffre d'affaires
- Marge brute
- Charges fixes
- Investissements

### Scenarios

| Scenario | CA | Marge | Charges |
|----------|----|----|---------|
| Pessimiste | 100k | 30% | 50k |
| Base | 150k | 35% | 50k |
| Optimiste | 200k | 40% | 50k |

## Limitations

### Contraintes

- 32 cellules variables maximum par scenario
- Pas de calcul automatique entre scenarios
- Synthèse statique (pas mise à jour)

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Noms explicites | "Optimiste_2024" vs "Scenario1" |
| Documentation | Notez les hypotheses |
| Cohérence | Memes cellules pour tous les scenarios |
| Sauvegarde | Avant de modifier |
