# Gestionnaire de Scenarios : Analyse d'Hypotheses

Comparez facilement differentes hypotheses de calcul.

## Qu'est-ce que le gestionnaire de scenarios ?

### Definition

Outil permettant de creer et comparer plusieurs jeux de valeurs.

### Usage

- Previsions financieres
- Budgetisation
- Planification
- Prise de decision

## Acces

**Donnees** > **Analyse de scenarios** > **Gestionnaire de scenarios**

## Creer un scenario

### Etapes

1. Ouvrez le gestionnaire de scenarios
2. Cliquez **Ajouter**
3. Nommez le scenario
4. Selectionnez les cellules variables
5. Entrez les valeurs

### Exemple

| Cellule | Description |
|---------|-------------|
| B1 | Prix de vente |
| B2 | Quantite |
| B3 | Cout unitaire |
| B5 | Profit (formule) |

## Scenarios types

### Optimiste

Meilleures hypotheses :
- Ventes elevees
- Couts faibles
- Prix optimal

### Base

Hypotheses realistes.

### Pessimiste

Hypotheses defavorables :
- Ventes faibles
- Couts eleves
- Pression prix

## Afficher un scenario

### Methode

1. Ouvrez le gestionnaire
2. Selectionnez le scenario
3. Cliquez **Afficher**

### Effet

Les valeurs du scenario sont appliquees aux cellules.

## Synthese des scenarios

### Generer un rapport

1. **Gestionnaire de scenarios** > **Synthese**
2. Choisissez les cellules de resultat
3. **OK**

### Resultat

Un tableau comparatif est cree sur une nouvelle feuille.

## Scenarios et protection

### Empecher les modifications

1. Lors de la creation du scenario
2. Cochez **Empecher les modifications**

### Masquer un scenario

Cochez **Masquer** dans les options.

## Combiner avec d'autres outils

### Tables de donnees

Pour des variations systematiques.

### Valeur cible

Pour trouver la valeur d'entree necessaire.

### Solveur

Pour l'optimisation sous contraintes.

## Cas pratique : Business plan

### Variables

- Chiffre d'affaires
- Marge brute
- Charges fixes
- Investissements

### Scenarios

| Scenario | CA | Marge | Charges |
|----------|----|----|---------|
| Pessimiste | 100k | 30% | 50k |
| Base | 150k | 35% | 50k |
| Optimiste | 200k | 40% | 50k |

## Limitations

### Contraintes

- 32 cellules variables maximum par scenario
- Pas de calcul automatique entre scenarios
- Synthese statique (pas mise a jour)

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Noms explicites | "Optimiste_2024" vs "Scenario1" |
| Documentation | Notez les hypotheses |
| Coherence | Memes cellules pour tous les scenarios |
| Sauvegarde | Avant de modifier |
