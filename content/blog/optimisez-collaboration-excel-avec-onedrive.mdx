# Collaboration Excel avec OneDrive

Travaillez à plusieurs sûr vos fichiers Excel en temps reel.

## Enregistrer sûr OneDrive

### Méthode 1 : Enregistrer sous

1. **Fichier** > **Enregistrer sous**
2. Sélectionnez OneDrive
3. Choisissez l'emplacement

### Méthode 2 : Déplacer

Déplacez le fichier vers un dossier OneDrive synchronise.

## Co-edition en temps reel

### Fonctionnement

Plusieurs utilisateurs editent le meme fichier simultanement.

### Indicateurs

- Presence des autres utilisateurs visible
- Cellules colorees selon l'utilisateur
- Modifications en temps reel

## Partager un fichier

### Depuis Excel

1. **Fichier** > **Partager**
2. Ou bouton **Partager** en haut à droite
3. Entrez les adresses email
4. Définissez les permissions

### Permissions

| Permission | Description |
|------------|-------------|
| Peut modifier | Lecture et ecriture |
| Peut afficher | Lecture seule |

## Lien de partage

### Créer un lien

1. **Partager** > **Copier le lien**
2. Configurez les options

### Options

- Tout le monde avec le lien
- Personnes de l'organisation
- Personnes spécifiques

## Commentaires et @mentions

### Ajouter un commentaire

1. Clic droit sûr la cellule
2. **Nouveau commentaire**
3. Redigez le commentaire

### Mentionner quelqu'un

Tapez @ suivi du nom de la personne.

## Historique des versions

### Accès

1. **Fichier** > **Informations**
2. **Historique des versions**

### Restaurer

Sélectionnez une version et restaurez si nécessaire.

## Excel pour le web

### Avantages

- Accessible depuis un navigateur
- Pas d'installation requise
- Fonctionnalités collaboratives completes

### Limitations

Certaines fonctionnalités avancées non disponibles.

## Synchronisation

### OneDrive Desktop

- Synchronisation automatique
- Accès hors ligne
- Fichiers à la demande

### Conflits

En cas de conflit, OneDrive créé une copie avec mention de conflit.

## Teams + Excel

### Intégration

- Partagez des fichiers Excel dans les canaux
- Éditez directement dans Teams
- Discussions liees au fichier

## Sécurité

### Options

| Option | Description |
|--------|-------------|
| Date d'expiration | Lien expire après X jours |
| Mot de passe | Protection supplémentaire |
| Bloquer le téléchargement | Consultation uniquement |

| Pratique | Description |
|----------|-------------|
| Nommage | Noms de fichiers clairs |
| Structuré | Dossiers bien organises |
| Communication | Utilisez les commentaires |
| Sauvegarde | Versions automatiques |
