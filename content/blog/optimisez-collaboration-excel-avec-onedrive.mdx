# Collaboration Excel avec OneDrive

Travaillez a plusieurs sur vos fichiers Excel en temps reel.

## Enregistrer sur OneDrive

### Methode 1 : Enregistrer sous

1. **Fichier** > **Enregistrer sous**
2. Selectionnez OneDrive
3. Choisissez l'emplacement

### Methode 2 : Deplacer

Deplacez le fichier vers un dossier OneDrive synchronise.

## Co-edition en temps reel

### Fonctionnement

Plusieurs utilisateurs editent le meme fichier simultanement.

### Indicateurs

- Presence des autres utilisateurs visible
- Cellules colorees selon l'utilisateur
- Modifications en temps reel

## Partager un fichier

### Depuis Excel

1. **Fichier** > **Partager**
2. Ou bouton **Partager** en haut a droite
3. Entrez les adresses email
4. Definissez les permissions

### Permissions

| Permission | Description |
|------------|-------------|
| Peut modifier | Lecture et ecriture |
| Peut afficher | Lecture seule |

## Lien de partage

### Creer un lien

1. **Partager** > **Copier le lien**
2. Configurez les options

### Options

- Tout le monde avec le lien
- Personnes de l'organisation
- Personnes specifiques

## Commentaires et @mentions

### Ajouter un commentaire

1. Clic droit sur la cellule
2. **Nouveau commentaire**
3. Redigez le commentaire

### Mentionner quelqu'un

Tapez @ suivi du nom de la personne.

## Historique des versions

### Acces

1. **Fichier** > **Informations**
2. **Historique des versions**

### Restaurer

Selectionnez une version et restaurez si necessaire.

## Excel pour le web

### Avantages

- Accessible depuis un navigateur
- Pas d'installation requise
- Fonctionnalites collaboratives completes

### Limitations

Certaines fonctionnalites avancees non disponibles.

## Synchronisation

### OneDrive Desktop

- Synchronisation automatique
- Acces hors ligne
- Fichiers a la demande

### Conflits

En cas de conflit, OneDrive cree une copie avec mention de conflit.

## Teams + Excel

### Integration

- Partagez des fichiers Excel dans les canaux
- Editez directement dans Teams
- Discussions liees au fichier

## Securite

### Options

| Option | Description |
|--------|-------------|
| Date d'expiration | Lien expire apres X jours |
| Mot de passe | Protection supplementaire |
| Bloquer le telechargement | Consultation uniquement |

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Nommage | Noms de fichiers clairs |
| Structure | Dossiers bien organises |
| Communication | Utilisez les commentaires |
| Sauvegarde | Versions automatiques |
