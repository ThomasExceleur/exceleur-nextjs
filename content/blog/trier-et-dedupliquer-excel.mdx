# Trier et Dedupliquer dans Excel

Deux opérations essentielles pour organiser vos données.

## Trier les données

### Tri simple

1. Sélectionnez une cellule dans la colonne
2. **Données** > **Trier de A à Z** où **Trier de Z à A**

### Tri multicriteres

1. **Données** > **Trier**
2. Ajoutez des niveaux de tri
3. Définissez l'ordre pour chaque niveau

## Options de tri

| Option | Description |
|--------|-------------|
| Croissant | A à Z, petit à grand |
| Decroissant | Z à A, grand à petit |
| Liste personnalisee | Ordre spécifique |
| Couleur | Par couleur de cellule |
| Icone | Par icone conditionnelle |

### Listes personnalisees

Créez des ordres de tri spécifiques :
- Jours de la semaine
- Mois
- Priorités personnalisees

## Fonction TRIER (Excel 365)

### Syntaxe

```excel
=TRIER(tableau;[index_tri];[ordre];[par_col])
```

### Exemples

```excel
=TRIER(A2:C10;1;1)      ' Tri par 1ere colonne croissant
=TRIER(A2:C10;2;-1)     ' Tri par 2eme colonne decroissant
```

## Fonction TRIERPAR (Excel 365)

### Syntaxe

```excel
=TRIERPAR(tableau;plage_tri1;[ordre1];...)
```

### Exemple

```excel
=TRIERPAR(A2:B10;C2:C10;1)
```

Trie le tableau A:B selon les valeurs de C.

## Supprimer les doublons

### Méthode manuelle

1. Sélectionnez la plage
2. **Données** > **Supprimer les doublons**
3. Cochez les colonnes à comparer
4. Cliquez **OK**

### Résultat

Excel indique combien de doublons ont ete supprimes.

## Fonction UNIQUE (Excel 365)

### Syntaxe

```excel
=UNIQUE(tableau;[par_col];[exactement_une_fois])
```

### Exemples

```excel
=UNIQUE(A2:A100)              ' Valeurs uniques
=UNIQUE(A2:A100;;VRAI)        ' Valeurs apparaissant une seule fois
```

## Identifier les doublons

### Mise en forme conditionnelle

1. Sélectionnez la plage
2. **Mise en forme conditionnelle** > **Règles de mise en surbrillance** > **Valeurs en double**

### Formule

```excel
=NB.SI($A$2:$A$100;A2)>1
```

## Combiner tri et deduplication

### Exemple pratique

```excel
=UNIQUE(TRIER(A2:B100;1;1))
```

Trie les données puis extrait les valeurs uniques.

## Cas pratiques

### Nettoyer une liste de clients

1. Supprimez les doublons par email
2. Triez par nom alphabetiquement
3. Filtrez les données incompletes

### Consolider des données

1. Combinez plusieurs sources
2. Supprimez les doublons
3. Triez par date

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Sauvegarde | Copiez avant de supprimer |
| Vérification | Revoyez les doublons identifies |
| Critères | Choisissez bien les colonnes clés |
| Validation | Vérifiez le résultat final |

## Erreurs courantes

| Erreur | Solution |
|--------|----------|
| Tri partiel | Sélectionnez toutes les colonnes |
| Doublons restants | Vérifiez les espaces invisibles |
| Perte de données | Sauvegardez avant |
