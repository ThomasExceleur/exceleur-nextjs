# Excel DATEDIF : Calculer les Ecarts de Dates

DATEDIF calcule la difference entre deux dates en annees, mois ou jours.

## Particularite

DATEDIF est une fonction "cachee" d'Excel. Elle n'apparait pas dans l'auto-completion mais fonctionne parfaitement.

## Syntaxe

```excel
=DATEDIF(date_debut; date_fin; unite)
```

### Arguments

- **date_debut** : Premiere date (la plus ancienne)
- **date_fin** : Deuxieme date (la plus recente)
- **unite** : Type de resultat souhaite

## Unites disponibles

| Unite | Description |
|-------|-------------|
| "Y" | Annees completes |
| "M" | Mois complets |
| "D" | Jours |
| "MD" | Jours (apres soustraction mois) |
| "YM" | Mois (apres soustraction annees) |
| "YD" | Jours (apres soustraction annees) |

## Exemples de base

### Annees entre deux dates

```excel
=DATEDIF(A1; B1; "Y")
```

### Mois entre deux dates

```excel
=DATEDIF(A1; B1; "M")
```

### Jours entre deux dates

```excel
=DATEDIF(A1; B1; "D")
```

## Calculer l'age

### Age en annees

```excel
=DATEDIF(DateNaissance; AUJOURDHUI(); "Y")
```

### Age complet (ans, mois, jours)

```excel
=DATEDIF(A1;AUJOURDHUI();"Y")&" ans, "&DATEDIF(A1;AUJOURDHUI();"YM")&" mois, "&DATEDIF(A1;AUJOURDHUI();"MD")&" jours"
```

Resultat : "35 ans, 6 mois, 15 jours"

## Calculer l'anciennete

### En annees

```excel
=DATEDIF(DateEmbauche; AUJOURDHUI(); "Y")
```

### Detaillee

```excel
=DATEDIF(A1;AUJOURDHUI();"Y")&" an(s) et "&DATEDIF(A1;AUJOURDHUI();"YM")&" mois"
```

## Difference entre unites

### Y vs M vs D

Dates : 15/01/2020 au 20/06/2024

- "Y" = 4 (annees completes)
- "M" = 53 (mois complets)
- "D" = 1618 (jours)

### MD, YM, YD

- "MD" = 5 (jours restants apres les mois)
- "YM" = 5 (mois restants apres les annees)
- "YD" = 157 (jours depuis l'anniversaire)

## Cas pratiques

### Duree d'un projet

```excel
=DATEDIF(DateDebut; DateFin; "D")&" jours"
```

### Echeance de contrat

```excel
=SI(DATEDIF(AUJOURDHUI(); DateFin; "M")<3; "Renouveler bientot"; "OK")
```

### Periode de garantie

```excel
=SI(DATEDIF(DateAchat; AUJOURDHUI(); "Y")<2; "Sous garantie"; "Garantie expiree")
```

## Gestion des erreurs

### Date debut > Date fin

DATEDIF retourne #NOMBRE! si la date de debut est posterieure.

### Solution

```excel
=SIERREUR(DATEDIF(A1; B1; "Y"); "Dates inversees")
```

### Verification prealable

```excel
=SI(A1>B1; "Erreur"; DATEDIF(A1; B1; "Y"))
```

## Alternatives a DATEDIF

### Jours simples

```excel
=B1-A1
```

### Mois approximatifs

```excel
=MOIS.DECALER(B1;-A1)
```

Non recommande car imprecis.

### Annees approximatives

```excel
=(B1-A1)/365.25
```

Approximation avec annees bissextiles.

## Combiner avec d'autres fonctions

### Avec AUJOURD'HUI

```excel
=DATEDIF(A1; AUJOURDHUI(); "Y")
```

### Avec DATE

```excel
=DATEDIF(DATE(2020;1;1); DATE(2024;6;15); "M")
```

### Dans SI

```excel
=SI(DATEDIF(A1; AUJOURDHUI(); "Y")>=18; "Majeur"; "Mineur")
```

## Problemes connus

### Bug avec "MD"

Dans certaines versions, "MD" peut donner des resultats incorrects. Alternative :

```excel
=JOUR(B1)-JOUR(A1)+SI(JOUR(B1)<JOUR(A1);JOUR(FIN.MOIS(A1;0));0)
```

### Dates en texte

Convertissez d'abord :

```excel
=DATEDIF(DATEVAL(A1); DATEVAL(B1); "Y")
```

## Bonnes pratiques

1. **Verifiez l'ordre** des dates (debut avant fin)
2. **Utilisez SIERREUR** pour gerer les anomalies
3. **Testez** avec des dates connues
4. **Evitez "MD"** si possible (bug potentiel)
