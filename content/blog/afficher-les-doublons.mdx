# Afficher les Doublons dans Excel

Identifier les doublons est essentiel pour maintenir des données propres et fiables. Voici toutes les méthodes pour les reperer.

## Méthode 1 : Mise en forme conditionnelle

### Étapes

1. Sélectionnez votre plage de données
2. **Accueil** > **Mise en forme conditionnelle**
3. **Règles de mise en surbrillance** > **Valeurs en double**
4. Choisissez un format (couleur de remplissage)
5. **OK**

### Résultat

Les cellules contenant des doublons sont automatiquement colorees.

## Méthode 2 : Formule NB.SI

### Compter les occurrences

```excel
=NB.SI(A:A; A1)
```

Cette formule compte combien de fois la valeur de A1 apparait dans la colonne A.

### Identifier les doublons

```excel
=SI(NB.SI($A$1:$A$100; A1)>1; "Doublon"; "Unique")
```

Affiche "Doublon" si la valeur apparait plus d'une fois.

### Marquer la première occurrence

```excel
=SI(NB.SI($A$1:A1; A1)>1; "Doublon"; "")
```

Laisse la première occurrence vide, marque les suivantes.

## Méthode 3 : Mise en forme conditionnelle avancée

### Règle personnalisee

1. Sélectionnez la plage
2. **Mise en forme conditionnelle** > **Nouvelle règle**
3. **Utiliser une formule**
4. Entrez : `=NB.SI($A$1:$A$100;A1)>1`
5. Définissez le format
6. **OK**

### Avantage

Permet de personnaliser complètement le format des doublons.

## Méthode 4 : Filtre avancé

### Afficher uniquement les doublons

1. Copiez vos données dans une zone de critères
2. **Données** > **Avancé** (dans Trier et filtrer)
3. Sélectionnez **Copier vers un autre emplacement**
4. Cochez **Uniquement des enregistrements uniques**

### Comparer les listes

La liste filtree ne contient que les valeurs uniques. Comparez avec l'original pour identifier les doublons.

## Méthode 5 : Tableau croisé dynamique

### Étapes

1. Insérez un tableau croisé dynamique
2. Placez le champ à analyser en **Lignes**
3. Placez le meme champ en **Valeurs** (Nombre)
4. Filtrez les valeurs > 1

### Résultat

Vous obtenez la liste des valeurs qui apparaissent plusieurs fois et leur frequence.

## Doublons sûr plusieurs colonnes

### Concatener pour comparer

```excel
=A1&B1&C1
```

Créé une valeur unique basee sûr plusieurs colonnes.

### Formule complete

```excel
=SI(NB.SI($D$1:$D$100; D1)>1; "Doublon"; "Unique")
```

Ou D contient les valeurs concatenees.

### Mise en forme conditionnelle multi-colonnes

Formule : `=NB.SI.ENS($A$1:$A$100;$A1;$B$1:$B$100;$B1)>1`

## Cas pratiques

### Liste de clients

Identifier les clients enregistres plusieurs fois.

### Factures

Reperer les numéros de factures en double.

### Emails

Trouver les adresses email dupliquees.

## Supprimer les doublons

### Fonctionnalité intégrée

1. Sélectionnez vos données
2. **Données** > **Supprimer les doublons**
3. Cochez les colonnes à considerer
4. **OK**

### Attention

Cette action est irreversible. Faites une copie avant.

## Couleurs différentes par type

### Première occurrence en vert

Formule : `=NB.SI($A$1:A1;A1)=1`

### Doublons en rouge

Formule : `=NB.SI($A$1:A1;A1)>1`

## Compter les doublons

### Total des valeurs en double

```excel
=SOMME(SI(NB.SI(A1:A100;A1:A100)>1;1/NB.SI(A1:A100;A1:A100)))
```

(Valider avec Ctrl + Maj + Entrée dans les anciennes versions)

### Nombre de valeurs uniques

```excel
=SOMMEPROD((A1:A100<>"")/NB.SI(A1:A100;A1:A100&""))
```

## Bonnes pratiques

1. **Sauvegardez** avant toute modification
2. **Vérifiez** les espaces invisibles (utilisez SUPPRESPACE)
3. **Uniformisez** la casse si nécessaire (MAJUSCULE/MINUSCULE)
4. **Attention** aux formats (texte vs nombre)

## Erreurs courantes

### Espaces invisibles

```excel
=SUPPRESPACE(A1)
```

### Problèmes de casse

```excel
=EXACT(A1; B1)
```

Comparaison sensible à la casse.

### Format texte vs nombre

Convertissez avec :
```excel
=CNUM(A1)
```
