# Recherche Multicriteres dans Excel

Trouvez des données avec plusieurs conditions.

## Méthodes disponibles

| Méthode | Version |
|---------|---------|
| INDEX/EQUIV concatene | Toutes versions |
| RECHERCHEX concatene | Excel 365 |
| FILTRE | Excel 365 |

## INDEX/EQUIV multicriteres

### Syntaxe

```excel
=INDEX(Résultat;EQUIV(1;(Critere1=Plage1)*(Critere2=Plage2);0))
```

### Validation matricielle

Ctrl+Maj+Entrée (versions anciennes).

## RECHERCHEX multicriteres

### Avec concatenation

```excel
=RECHERCHEX(A1&B1;Colonne1&Colonne2;ColonneResultat)
```

### Exemple

```excel
=RECHERCHEX(Region&Produit;Table[Region]&Table[Produit];Table[Prix])
```

## Avec FILTRE

### Méthode moderne

```excel
=FILTRE(Résultat;(Critere1=Plage1)*(Critere2=Plage2))
```

### Premier résultat

```excel
=INDEX(FILTRE(Résultat;(Critere1=Plage1)*(Critere2=Plage2));1)
```

## Cas pratique

### Trouver le prix

| Region | Produit | Prix |
|--------|---------|------|
| Nord | A | 100 |
| Nord | B | 120 |
| Sud | A | 95 |

```excel
=RECHERCHEX(E1&F1;A:A&B:B;C:C)
```
