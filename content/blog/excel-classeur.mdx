# Le Classeur Excel : Guide Complet

Un classeur est le fichier de travail principal dans Excel. Comprenez sa structure pour mieux l'utiliser.

## Structure d'un classeur

### Definition

Un classeur Excel est un fichier contenant une ou plusieurs feuilles de calcul.

### Composants

- **Feuilles de calcul** : Onglets en bas de l'ecran
- **Cellules** : Intersection lignes/colonnes
- **Donnees** : Contenu des cellules
- **Formules** : Calculs automatiques
- **Mise en forme** : Apparence visuelle

## Creer un classeur

### Nouveau classeur vide

1. **Fichier** > **Nouveau**
2. **Classeur vide**

### Raccourci

**Ctrl + N**

### A partir d'un modele

1. **Fichier** > **Nouveau**
2. Choisissez un modele

## Formats de fichiers

### .xlsx (Standard)

Format par defaut depuis Excel 2007.

- Compatible avec les versions modernes
- Ne supporte pas les macros
- Fichier compresse (leger)

### .xlsm (Avec macros)

- Contient du code VBA
- Alertes de securite a l'ouverture
- Necessaire pour l'automatisation

### .xls (Ancien)

Format Excel 97-2003.

- Limite a 65 536 lignes
- Compatibilite anciens systemes
- Evitez si possible

### .xlsb (Binaire)

- Fichiers tres volumineux
- Plus rapide a ouvrir
- Moins compatible

### .csv (Texte)

- Donnees uniquement
- Pas de formules ni mise en forme
- Echange entre logiciels

## Gerer les feuilles

### Ajouter une feuille

- Cliquez sur **+** a cote des onglets
- Ou **Ctrl + Maj + F11**

### Renommer une feuille

Double-cliquez sur l'onglet et tapez le nouveau nom.

### Supprimer une feuille

Clic droit sur l'onglet > **Supprimer**

**Attention** : Action irreversible !

### Deplacer une feuille

Faites glisser l'onglet vers la nouvelle position.

### Copier une feuille

Maintenez **Ctrl** et faites glisser l'onglet.

## Organiser les feuilles

### Couleurs d'onglets

1. Clic droit sur l'onglet
2. **Couleur d'onglet**
3. Choisissez une couleur

### Grouper les feuilles

1. Cliquez sur le premier onglet
2. Maintenez Maj
3. Cliquez sur le dernier onglet

Les modifications s'appliquent a toutes les feuilles selectionnees.

### Masquer une feuille

Clic droit sur l'onglet > **Masquer**

Pour afficher : Clic droit > **Afficher**

## Enregistrer un classeur

### Premiere sauvegarde

1. **Fichier** > **Enregistrer sous**
2. Choisissez l'emplacement
3. Nommez le fichier
4. Selectionnez le format

### Sauvegarde rapide

**Ctrl + S**

### Sauvegarde automatique

OneDrive/SharePoint sauvegarde automatiquement.

## Proteger un classeur

### Protection par mot de passe

1. **Fichier** > **Informations**
2. **Proteger le classeur**
3. **Chiffrer avec mot de passe**

### Protection de structure

Empeche l'ajout/suppression de feuilles.

1. **Revision** > **Proteger le classeur**
2. Cochez **Structure**

## Proprietes du classeur

### Acces

**Fichier** > **Informations**

### Proprietes disponibles

- Titre
- Auteur
- Mots-cles
- Commentaires

## Recuperer un classeur

### Fichiers recents

**Fichier** > **Ouvrir** > **Recent**

### Fichiers non enregistres

1. **Fichier** > **Informations**
2. **Gerer le classeur**
3. **Recuperer les classeurs non enregistres**

### Versions precedentes

1. **Fichier** > **Informations**
2. **Historique des versions**

## Optimiser un classeur

### Reduire la taille

- Supprimez les donnees inutiles
- Compressez les images
- Supprimez les feuilles vides

### Ameliorer les performances

- Limitez les formules volatiles
- Evitez les formules sur colonnes entieres
- Utilisez des plages definies

## Partager un classeur

### Par email

**Fichier** > **Partager** > **Joindre une copie**

### Via OneDrive

1. Enregistrez sur OneDrive
2. **Partager** > Invitez des personnes

### Exportation

- PDF : Pour lecture seule
- CSV : Pour donnees uniquement

## Bonnes pratiques

1. **Nommez clairement** vos classeurs et feuilles
2. **Sauvegardez regulierement**
3. **Organisez** avec des couleurs d'onglets
4. **Documentez** avec une feuille "Lisez-moi"
