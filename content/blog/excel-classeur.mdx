# Le Classeur Excel : Guide Complet

Un classeur est le fichier de travail principal dans Excel. Comprenez sa structuré pour mieux l'utiliser.

## Structuré d'un classeur

### Définition

Un classeur Excel est un fichier contenant une où plusieurs feuilles de calcul.

### Composants

- **Feuilles de calcul** : Onglets en bas de l'ecran
- **Cellules** : Intersection lignes/colonnes
- **Données** : Contenu des cellules
- **Formules** : Calculs automatiques
- **Mise en forme** : Apparence visuelle

## Créer un classeur

### Nouveau classeur vide

1. **Fichier** > **Nouveau**
2. **Classeur vide**

### Raccourci

**Ctrl + N**

### A partir d'un modèle

1. **Fichier** > **Nouveau**
2. Choisissez un modèle

## Formats de fichiers

### .xlsx (Standard)

Format par defaut depuis Excel 2007.

- Compatible avec les versions modernes
- Ne supporte pas les macros
- Fichier compresse (léger)

### .xlsm (Avec macros)

- Contient du code VBA
- Alertes de sécurité à l'ouverture
- Nécessaire pour l'automatisation

### .xls (Ancien)

Format Excel 97-2003.

- Limite à 65 536 lignes
- Compatibilité anciens systèmes
- Évitez si possible

### .xlsb (Binaire)

- Fichiers très volumineux
- Plus rapide à ouvrir
- Moins compatible

### .csv (Texte)

- Données uniquement
- Pas de formules ni mise en forme
- Echange entre logiciels

## Gérer les feuilles

### Ajouter une feuille

- Cliquez sûr **+** à côté des onglets
- Ou **Ctrl + Maj + F11**

### Renommer une feuille

Double-cliquez sûr l'onglet et tapez le nouveau nom.

### Supprimer une feuille

Clic droit sûr l'onglet > **Supprimer**

**Attention** : Action irreversible !

### Déplacer une feuille

Faites glisser l'onglet vers la nouvelle position.

### Copier une feuille

Maintenez **Ctrl** et faites glisser l'onglet.

## Organiser les feuilles

### Couleurs d'onglets

1. Clic droit sûr l'onglet
2. **Couleur d'onglet**
3. Choisissez une couleur

### Grouper les feuilles

1. Cliquez sûr le premier onglet
2. Maintenez Maj
3. Cliquez sûr le dernier onglet

Les modifications s'appliquent à toutes les feuilles sélectionnées.

### Masquer une feuille

Clic droit sûr l'onglet > **Masquer**

Pour afficher : Clic droit > **Afficher**

## Enregistrer un classeur

### Première sauvegarde

1. **Fichier** > **Enregistrer sous**
2. Choisissez l'emplacement
3. Nommez le fichier
4. Sélectionnez le format

### Sauvegarde rapide

**Ctrl + S**

### Sauvegarde automatique

OneDrive/SharePoint sauvegarde automatiquement.

## Proteger un classeur

### Protection par mot de passe

1. **Fichier** > **Informations**
2. **Proteger le classeur**
3. **Chiffrer avec mot de passe**

### Protection de structuré

Empeche l'ajout/suppression de feuilles.

1. **Révision** > **Proteger le classeur**
2. Cochez **Structuré**

## Propriétés du classeur

### Accès

**Fichier** > **Informations**

### Propriétés disponibles

- Titre
- Auteur
- Mots-clés
- Commentaires

## Récupérer un classeur

### Fichiers recents

**Fichier** > **Ouvrir** > **Recent**

### Fichiers non enregistres

1. **Fichier** > **Informations**
2. **Gérer le classeur**
3. **Récupérer les classeurs non enregistres**

### Versions précédentes

1. **Fichier** > **Informations**
2. **Historique des versions**

## Optimiser un classeur

### Réduire la taille

- Supprimez les données inutiles
- Compressez les images
- Supprimez les feuilles vides

### Améliorer les performances

- Limitez les formules volatiles
- Évitez les formules sûr colonnes entières
- Utilisez des plages définies

## Partager un classeur

### Par email

**Fichier** > **Partager** > **Joindre une copie**

### Via OneDrive

1. Enregistrez sûr OneDrive
2. **Partager** > Invitez des personnes

### Exportation

- PDF : Pour lecture seule
- CSV : Pour données uniquement

## Bonnes pratiques

1. **Nommez clairement** vos classeurs et feuilles
2. **Sauvegardez régulièrement**
3. **Organisez** avec des couleurs d'onglets
4. **Documentez** avec une feuille "Lisez-moi"
