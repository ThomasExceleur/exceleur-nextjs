# Automatiser les Reports avec Power Pivot

Guide debutant pour creer des reports automatises avec Power Pivot.

## Qu'est-ce que Power Pivot ?

### Definition

Power Pivot est un complement Excel pour :

- Gerer de grands volumes de donnees
- Creer des modeles relationnels
- Calculer avec DAX
- Alimenter des TCD puissants

### Activer Power Pivot

1. **Fichier** > **Options** > **Complements**
2. Gerer : **Complements COM** > **Atteindre**
3. Cochez **Microsoft Power Pivot for Excel**

## Importer vos donnees

### Depuis Excel

1. Selectionnez votre tableau
2. **Power Pivot** > **Ajouter au modele de donnees**

### Depuis d'autres sources

1. **Power Pivot** > **Gerer**
2. **Obtenir des donnees externes**
3. Choisissez votre source (SQL, Access, etc.)

## Creer des relations

### Pourquoi des relations ?

Pour lier plusieurs tables entre elles (ventes avec produits, par exemple).

### Comment faire

1. Ouvrez la fenetre Power Pivot
2. Vue **Diagramme**
3. Glissez un champ d'une table vers l'autre
4. La relation est creee

### Exemple

| Table Ventes | Table Produits |
|--------------|----------------|
| ProduitID | ProduitID (cle) |
| Date | Nom |
| Montant | Categorie |

## Vos premieres mesures DAX

### Qu'est-ce qu'une mesure ?

Un calcul dynamique qui s'adapte au contexte (filtres, lignes, colonnes).

### Mesures de base

```dax
Total Ventes = SUM(Ventes[Montant])
```

```dax
Nombre Transactions = COUNTROWS(Ventes)
```

```dax
Moyenne Vente = AVERAGE(Ventes[Montant])
```

### Creer une mesure

1. Cliquez dans la zone de calcul Power Pivot
2. Tapez votre formule
3. Appuyez sur Entree

## Creer un TCD automatise

### Etapes

1. **Insertion** > **Tableau croise dynamique**
2. Cochez **Utiliser le modele de donnees**
3. Placez vos champs
4. Utilisez vos mesures

### Avantages

- Donnees de plusieurs tables
- Mesures personnalisees
- Grande performance

## Mesures pour reports

### Comparaison avec periode precedente

```dax
Ventes Mois Precedent =
CALCULATE(
    [Total Ventes];
    DATEADD(Calendrier[Date];-1;MONTH)
)
```

### Cumul annuel (YTD)

```dax
Ventes YTD =
TOTALYTD(
    [Total Ventes];
    Calendrier[Date]
)
```

### Variation en pourcentage

```dax
Variation % =
DIVIDE(
    [Total Ventes] - [Ventes Mois Precedent];
    [Ventes Mois Precedent];
    0
)
```

## Table de dates

### Pourquoi une table de dates ?

Indispensable pour les calculs temporels (YTD, comparaisons).

### Creer automatiquement

```dax
Calendrier = CALENDAR(DATE(2020;1;1);DATE(2025;12;31))
```

### Colonnes utiles

```dax
Annee = YEAR(Calendrier[Date])
Mois = MONTH(Calendrier[Date])
NomMois = FORMAT(Calendrier[Date];"MMMM")
```

## Automatiser l'actualisation

### Depuis Excel

1. Actualisez le TCD : **Actualiser**
2. Les donnees Power Pivot se mettent a jour

### Connexions externes

1. **Donnees** > **Actualiser tout**
2. Toutes les sources sont actualisees

## Erreurs courantes

### Relation manquante

**Symptome** : Donnees incorrectes ou dupliquees.

**Solution** : Verifiez les relations dans la vue Diagramme.

### Formule DAX incorrecte

**Symptome** : Erreur ou valeur vide.

**Solution** : Testez la formule etape par etape.
