# 50 Raccourcis Excel Essentiels

Gagnez du temps avec ces raccourcis clavier.

## Navigation

| Raccourci | Action |
|-----------|--------|
| Ctrl+Debut | Aller en A1 |
| Ctrl+Fin | Derniere cellule utilisee |
| Ctrl+Fleche | Sauter au bord des donnees |
| Ctrl+Page Up/Down | Changer d'onglet |

## Selection

| Raccourci | Action |
|-----------|--------|
| Ctrl+Maj+Fin | Selectionner jusqu'a la fin |
| Ctrl+Espace | Selectionner la colonne |
| Maj+Espace | Selectionner la ligne |
| Ctrl+A | Tout selectionner |

## Edition

| Raccourci | Action |
|-----------|--------|
| F2 | Editer la cellule |
| Ctrl+C / Ctrl+V | Copier / Coller |
| Ctrl+X | Couper |
| Ctrl+Z | Annuler |
| Ctrl+Y | Retablir |
| Suppr | Effacer le contenu |

## Mise en forme

| Raccourci | Action |
|-----------|--------|
| Ctrl+G | Gras |
| Ctrl+I | Italique |
| Ctrl+U | Souligne |
| Ctrl+1 | Format de cellule |
| Ctrl+Maj+$ | Format monnaie |
| Ctrl+Maj+% | Format pourcentage |

## Formules

| Raccourci | Action |
|-----------|--------|
| F4 | Reference absolue |
| Ctrl+` | Afficher les formules |
| Tab | Valider suggestion |
| Ctrl+Maj+Entree | Formule matricielle |

## Fichier

| Raccourci | Action |
|-----------|--------|
| Ctrl+N | Nouveau classeur |
| Ctrl+O | Ouvrir |
| Ctrl+S | Enregistrer |
| F12 | Enregistrer sous |
| Ctrl+P | Imprimer |
| Ctrl+W | Fermer |

## Divers

| Raccourci | Action |
|-----------|--------|
| Ctrl+E | Remplissage instantane |
| Ctrl+T | Creer un tableau |
| Ctrl+Maj+L | Activer les filtres |
| Alt+= | Somme automatique |
| F11 | Creer un graphique |
| Ctrl+; | Date du jour |
