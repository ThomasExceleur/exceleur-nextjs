# Utiliser les Filtres dans Excel

Les filtres permettent d'afficher uniquement les donnees qui vous interessent.

## Activer les filtres

### Raccourci

**Ctrl + Maj + L** : active/desactive les filtres

### Via le menu

1. Selectionnez vos donnees
2. **Donnees** > **Filtrer**

### Resultat

Des fleches apparaissent dans les en-tetes de colonnes.

## Filtrer des donnees

### Filtre simple

1. Cliquez sur la fleche d'une colonne
2. Decochez **(Selectionner tout)**
3. Cochez les valeurs souhaitees
4. **OK**

### Filtrer plusieurs colonnes

Repetez l'operation sur chaque colonne.

## Filtres de texte

### Options disponibles

| Filtre | Description |
|--------|-------------|
| Est egal a | Correspondance exacte |
| N'est pas egal a | Tout sauf |
| Commence par | Debut du texte |
| Se termine par | Fin du texte |
| Contient | Presence dans le texte |
| Ne contient pas | Absence dans le texte |

### Utiliser les jokers

| Joker | Signification |
|-------|---------------|
| * | N'importe quels caracteres |
| ? | Un seul caractere |

## Filtres de nombres

### Options

| Filtre | Description |
|--------|-------------|
| Egal a | Exactement |
| Superieur a | Plus grand |
| Inferieur a | Plus petit |
| Entre | Intervalle |
| 10 premiers | Top N |

### Exemple

Filtrer les ventes superieures a 1000 :
1. Fleche > **Filtres numeriques** > **Superieur a**
2. Entrez 1000
3. **OK**

## Filtres de dates

### Options

| Filtre | Description |
|--------|-------------|
| Aujourd'hui | Date du jour |
| Cette semaine | Semaine en cours |
| Ce mois-ci | Mois en cours |
| Trimestre | Q1, Q2, Q3, Q4 |
| Annee | Annee specifique |
| Avant/Apres | Date limite |

## Filtres personnalises

### Creer un filtre complexe

1. Fleche > **Filtres [type]** > **Filtre personnalise**
2. Definissez les conditions
3. Utilisez ET/OU pour combiner

### Exemple

Valeurs entre 100 et 500 :
- Superieur ou egal a 100
- ET Inferieur ou egal a 500

## Filtrer par couleur

### Si mise en forme conditionnelle

1. Cliquez sur la fleche
2. **Filtrer par couleur**
3. Selectionnez la couleur de cellule ou de police

## Segments (Slicers)

### Pour un filtrage visuel

1. Convertissez en tableau (**Ctrl + T**)
2. **Insertion** > **Segment**
3. Selectionnez les colonnes
4. Cliquez sur les boutons pour filtrer

### Avantages

- Visuel et intuitif
- Multi-selection facile
- Peut controler plusieurs tableaux

## Effacer les filtres

### Une colonne

1. Cliquez sur la fleche
2. **Effacer le filtre de [colonne]**

### Tous les filtres

- **Donnees** > **Effacer**
- Ou cliquez sur l'icone d'entonnoir

## Copier les donnees filtrees

### Methode

1. Filtrez vos donnees
2. Selectionnez les cellules visibles
3. **Ctrl + C**
4. Collez ailleurs

### Astuce

**Alt + ;** : selectionne uniquement les cellules visibles

## Filtres avances

### Pour des criteres complexes

1. Creez une zone de criteres
2. **Donnees** > **Avance**
3. Definissez la plage de criteres

### Zone de criteres

| Colonne1 | Colonne2 |
|----------|----------|
| Critere1 | Critere2 |

## Bonnes pratiques

1. **Verifiez** que les filtres sont actifs (icone d'entonnoir)
2. **Effacez** les filtres avant de creer des totaux
3. **Utilisez** les segments pour les analyses repetitives
4. **Sauvegardez** avant de supprimer des donnees filtrees
