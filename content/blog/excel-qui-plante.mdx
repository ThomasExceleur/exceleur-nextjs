# Excel qui Plante : Solutions

Excel peut planter pour diverses raisons. Voici comment diagnostiquer et resoudre ces problèmes.

## Causes courantes

### Fichier corrompu

Le fichier Excel lui-meme est endommage.

### Complements defaillants

Un complement cause des conflits.

### Manque de mémoire

Le système n'à plus assez de RAM.

### Fichier trop volumineux

Le fichier dépasse les capacités.

### Formules complexes

Trop de calculs simultanement.

## Solutions immediates

### Forcer la fermeture

1. **Ctrl + Alt + Suppr**
2. **Gestionnaire des taches**
3. Sélectionnez Excel
4. **Fin de tache**

### Récupérer le travail

1. Rouvrez Excel
2. Le volet de récupération apparait
3. Sélectionnez la version à récupérer

## Démarrer en mode sans echec

### Méthode

1. Fermez Excel
2. Maintenez **Ctrl** en lancant Excel
3. Confirmez le mode sans echec

### Ce qui est desactive

- Complements
- Personnalisations
- Certaines fonctionnalités

### Diagnostic

Si Excel fonctionne en mode sans echec, un complement cause le problème.

## Desactiver les complements

### Étapes

1. **Fichier** > **Options**
2. **Complements**
3. En bas : **Gérer** > **Complements COM** > **Atteindre**
4. Decochez tous les complements
5. Reactivez un par un pour identifier le coupable

## Reparer Office

### Reparation rapide

1. **Paramètrès Windows** > **Applications**
2. Trouvez Microsoft Office
3. **Modifier** > **Reparation rapide**

### Reparation en ligne

Si la reparation rapide ne suffit pas :
1. Memes étapes
2. Choisissez **Reparation en ligne**
3. Nécessité une connexion internet

## Optimiser le fichier

### Supprimer les éléments inutiles

- Feuilles vides
- Lignes/colonnes superflues
- Objets caches

### Réduire les formules

- Convertissez les formules en valeurs si possible
- Simplifiez les formules complexes
- Évitez les colonnes entières (A:A)

### Compresser les images

1. Sélectionnez une image
2. **Format** > **Compresser les images**
3. Choisissez la resolution

## Liberer de la mémoire

### Fermer les applications

Fermez les programmes inutilises.

### Fermer les fichiers Excel

Ne gardez ouverts que les fichiers nécessaires.

### Redemarrer l'ordinateur

Un redemarrage libere la mémoire.

## Mettre à jour

### Office

1. **Fichier** > **Compte**
2. **Options de mise à jour** > **Mettre à jour**

### Windows

1. **Paramètrès** > **Mise à jour Windows**
2. Installez les mises à jour

### Pilotes graphiques

Mettez à jour les pilotes de votre carte graphique.

## Récupérer un fichier corrompu

### Méthode 1 : Ouvrir et reparer

1. **Fichier** > **Ouvrir**
2. Sélectionnez le fichier
3. Fleche à côté de Ouvrir > **Ouvrir et reparer**

### Méthode 2 : Extraction de données

1. Meme procedure
2. Choisissez **Extraire les données**

### Méthode 3 : Copier dans un nouveau fichier

1. Ouvrez le fichier problematique
2. Sélectionnez les données
3. Copiez dans un nouveau fichier

## Prevenir les crashes

### Sauvegardes automatiques

1. **Fichier** > **Options** > **Enregistrement**
2. Activez **Enregistrer les informations de récupération**
3. Reduisez l'intervalle (ex: 5 minutes)

### Bonnes pratiques

- Sauvegardez régulièrement (Ctrl + S)
- Évitez les fichiers trop lourds
- Limitez les formules volatiles
- Fermez les fichiers inutilises

## Tableau de diagnostic

| Symptome | Cause probable | Solution |
|----------|----------------|----------|
| Crash à l'ouverture | Fichier corrompu | Ouvrir et reparer |
| Crash aleatoire | Complement | Mode sans echec |
| Lenteur puis crash | Mémoire | Fermer applications |
| Crash sûr formule | Calcul complexe | Simplifier |
| Crash frequent | Office | Reparer Office |

## Quand contacter le support

- Les solutions ci-dessus ne fonctionnent pas
- Le problème persiste après reparation
- Plusieurs fichiers sont affectes
- Erreurs système associees
