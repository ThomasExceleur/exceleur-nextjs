# Excel qui Plante : Solutions

Excel peut planter pour diverses raisons. Voici comment diagnostiquer et resoudre ces problemes.

## Causes courantes

### Fichier corrompu

Le fichier Excel lui-meme est endommage.

### Complements defaillants

Un complement cause des conflits.

### Manque de memoire

Le systeme n'a plus assez de RAM.

### Fichier trop volumineux

Le fichier depasse les capacites.

### Formules complexes

Trop de calculs simultanement.

## Solutions immediates

### Forcer la fermeture

1. **Ctrl + Alt + Suppr**
2. **Gestionnaire des taches**
3. Selectionnez Excel
4. **Fin de tache**

### Recuperer le travail

1. Rouvrez Excel
2. Le volet de recuperation apparait
3. Selectionnez la version a recuperer

## Demarrer en mode sans echec

### Methode

1. Fermez Excel
2. Maintenez **Ctrl** en lancant Excel
3. Confirmez le mode sans echec

### Ce qui est desactive

- Complements
- Personnalisations
- Certaines fonctionnalites

### Diagnostic

Si Excel fonctionne en mode sans echec, un complement cause le probleme.

## Desactiver les complements

### Etapes

1. **Fichier** > **Options**
2. **Complements**
3. En bas : **Gerer** > **Complements COM** > **Atteindre**
4. Decochez tous les complements
5. Reactivez un par un pour identifier le coupable

## Reparer Office

### Reparation rapide

1. **Parametres Windows** > **Applications**
2. Trouvez Microsoft Office
3. **Modifier** > **Reparation rapide**

### Reparation en ligne

Si la reparation rapide ne suffit pas :
1. Memes etapes
2. Choisissez **Reparation en ligne**
3. Necessite une connexion internet

## Optimiser le fichier

### Supprimer les elements inutiles

- Feuilles vides
- Lignes/colonnes superflues
- Objets caches

### Reduire les formules

- Convertissez les formules en valeurs si possible
- Simplifiez les formules complexes
- Evitez les colonnes entieres (A:A)

### Compresser les images

1. Selectionnez une image
2. **Format** > **Compresser les images**
3. Choisissez la resolution

## Liberer de la memoire

### Fermer les applications

Fermez les programmes inutilises.

### Fermer les fichiers Excel

Ne gardez ouverts que les fichiers necessaires.

### Redemarrer l'ordinateur

Un redemarrage libere la memoire.

## Mettre a jour

### Office

1. **Fichier** > **Compte**
2. **Options de mise a jour** > **Mettre a jour**

### Windows

1. **Parametres** > **Mise a jour Windows**
2. Installez les mises a jour

### Pilotes graphiques

Mettez a jour les pilotes de votre carte graphique.

## Recuperer un fichier corrompu

### Methode 1 : Ouvrir et reparer

1. **Fichier** > **Ouvrir**
2. Selectionnez le fichier
3. Fleche a cote de Ouvrir > **Ouvrir et reparer**

### Methode 2 : Extraction de donnees

1. Meme procedure
2. Choisissez **Extraire les donnees**

### Methode 3 : Copier dans un nouveau fichier

1. Ouvrez le fichier problematique
2. Selectionnez les donnees
3. Copiez dans un nouveau fichier

## Prevenir les crashes

### Sauvegardes automatiques

1. **Fichier** > **Options** > **Enregistrement**
2. Activez **Enregistrer les informations de recuperation**
3. Reduisez l'intervalle (ex: 5 minutes)

### Bonnes pratiques

- Sauvegardez regulierement (Ctrl + S)
- Evitez les fichiers trop lourds
- Limitez les formules volatiles
- Fermez les fichiers inutilises

## Tableau de diagnostic

| Symptome | Cause probable | Solution |
|----------|----------------|----------|
| Crash a l'ouverture | Fichier corrompu | Ouvrir et reparer |
| Crash aleatoire | Complement | Mode sans echec |
| Lenteur puis crash | Memoire | Fermer applications |
| Crash sur formule | Calcul complexe | Simplifier |
| Crash frequent | Office | Reparer Office |

## Quand contacter le support

- Les solutions ci-dessus ne fonctionnent pas
- Le probleme persiste apres reparation
- Plusieurs fichiers sont affectes
- Erreurs systeme associees
