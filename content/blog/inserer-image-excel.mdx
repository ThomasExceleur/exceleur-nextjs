# Inserer et Gerer les Images dans Excel

Integrez des images efficacement dans vos classeurs.

## Inserer une image

### Depuis un fichier

1. **Insertion** > **Images** > **Cet appareil**
2. Selectionnez le fichier
3. **Inserer**

### Depuis le web

1. **Insertion** > **Images** > **Images en ligne**
2. Recherchez ou collez l'URL

### Copier-coller

Ctrl+C depuis une source, Ctrl+V dans Excel.

## Redimensionner

### Manuellement

Glissez les poignees de coin (maintenir Maj pour conserver les proportions).

### Precisement

1. Selectionnez l'image
2. **Format** > **Taille**
3. Entrez les dimensions

## Positionner dans une cellule

### Methode 1 : Alignement manuel

1. Positionnez l'image sur la cellule
2. Ajustez la taille

### Methode 2 : Proprietes

1. Clic droit > **Taille et proprietes**
2. **Proprietes**
3. **Deplacer et dimensionner avec les cellules**

## IMAGE() - Fonction Excel 365

### Syntaxe

```excel
=IMAGE(url;[texte_alt];[taille];[hauteur];[largeur])
```

### Exemple

```excel
=IMAGE("https://exemple.com/photo.jpg";"Logo")
```

### Avantage

L'image est liee a une cellule et se comporte comme une formule.

## Images dans les cellules (Excel 365)

### Insertion

1. Selectionnez la cellule
2. **Insertion** > **Image** > **Placer dans la cellule**

### Comportement

L'image se redimensionne avec la cellule.

## Compresser les images

### Reduire la taille du fichier

1. Selectionnez une image
2. **Format** > **Compresser les images**
3. Choisissez la resolution

### Options

| Resolution | Usage |
|------------|-------|
| Haute fidelite | Qualite max |
| 220 ppp | Impression |
| 150 ppp | Ecran |
| 96 ppp | Email |

## Rogner une image

### Rogner simplement

1. Selectionnez l'image
2. **Format** > **Rogner**
3. Ajustez les bords

### Rogner selon une forme

1. **Rogner** > **Rogner a la forme**
2. Choisissez la forme

## Effets visuels

### Options disponibles

| Effet | Description |
|-------|-------------|
| Ombre | Ajoute une ombre portee |
| Reflexion | Effet miroir |
| Lumiere | Halo lumineux |
| Bordures | Cadre autour |
| 3D | Rotation perspective |

### Acces

**Format** > **Effets d'image**

## Arriere-plan transparent

### Supprimer l'arriere-plan

1. Selectionnez l'image
2. **Format** > **Supprimer l'arriere-plan**
3. Ajustez les zones

## Liens hypertexte sur image

### Ajouter un lien

1. Clic droit sur l'image
2. **Lien**
3. Entrez l'URL ou la reference

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Compression | Reduisez la taille fichier |
| Format | PNG pour logos, JPG pour photos |
| Proportions | Conservez le ratio |
| Accessibilite | Ajoutez du texte alternatif |

## Erreurs courantes

| Probleme | Solution |
|----------|----------|
| Image floue | Utilisez une meilleure resolution |
| Fichier lourd | Compressez les images |
| Image qui bouge | Verrouillez la position |
