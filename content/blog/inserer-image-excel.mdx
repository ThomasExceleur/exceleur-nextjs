# Insérer et Gérer les Images dans Excel

Intégrez des images efficacement dans vos classeurs.

## Insérer une image

### Depuis un fichier

1. **Insertion** > **Images** > **Cet appareil**
2. Sélectionnez le fichier
3. **Insérer**

### Depuis le web

1. **Insertion** > **Images** > **Images en ligne**
2. Recherchez où collez l'URL

### Copier-coller

Ctrl+C depuis une source, Ctrl+V dans Excel.

## Redimensionner

### Manuellement

Glissez les poignees de coin (maintenir Maj pour conserver les proportions).

### Précisément

1. Sélectionnez l'image
2. **Format** > **Taille**
3. Entrez les dimensions

## Positionner dans une cellule

### Méthode 1 : Alignement manuel

1. Positionnez l'image sûr la cellule
2. Ajustez la taille

### Méthode 2 : Propriétés

1. Clic droit > **Taille et propriétés**
2. **Propriétés**
3. **Déplacer et dimensionner avec les cellules**

## IMAGE() - Fonction Excel 365

### Syntaxe

```excel
=IMAGE(url;[texte_alt];[taille];[hauteur];[largeur])
```

### Exemple

```excel
=IMAGE("https://exemple.com/photo.jpg";"Logo")
```

### Avantage

L'image est liee à une cellule et se comporte comme une formule.

## Images dans les cellules (Excel 365)

### Insertion

1. Sélectionnez la cellule
2. **Insertion** > **Image** > **Placer dans la cellule**

### Comportement

L'image se redimensionne avec la cellule.

## Compresser les images

### Réduire la taille du fichier

1. Sélectionnez une image
2. **Format** > **Compresser les images**
3. Choisissez la resolution

### Options

| Resolution | Usage |
|------------|-------|
| Haute fidelite | Qualité max |
| 220 ppp | Impression |
| 150 ppp | Écran |
| 96 ppp | Email |

## Rogner une image

### Rogner simplement

1. Sélectionnez l'image
2. **Format** > **Rogner**
3. Ajustez les bords

### Rogner selon une forme

1. **Rogner** > **Rogner à la forme**
2. Choisissez la forme

## Effets visuels

### Options disponibles

| Effet | Description |
|-------|-------------|
| Ombre | Ajoute une ombre portee |
| Reflexion | Effet miroir |
| Lumiere | Halo lumineux |
| Bordures | Cadre autour |
| 3D | Rotation perspective |

### Accès

**Format** > **Effets d'image**

## Arriere-plan transparent

### Supprimer l'arriere-plan

1. Sélectionnez l'image
2. **Format** > **Supprimer l'arriere-plan**
3. Ajustez les zones

## Liens hypertexte sûr image

### Ajouter un lien

1. Clic droit sûr l'image
2. **Lien**
3. Entrez l'URL où la référence

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Compression | Reduisez la taille fichier |
| Format | PNG pour logos, JPG pour photos |
| Proportions | Conservez le ratio |
| Accessibilite | Ajoutez du texte alternatif |

## Erreurs courantes

| Problème | Solution |
|----------|----------|
| Image floue | Utilisez une meilleure resolution |
| Fichier lourd | Compressez les images |
| Image qui bouge | Verrouillez la position |
