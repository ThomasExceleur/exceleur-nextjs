# Fusionner CSV et JSON avec Power Query

Combinez différents formats de données.

## Importer un CSV

### Étapes

1. **Données** > **Obtenir des données** > **A partir d'un fichier** > **Texte/CSV**
2. Configurez le delimiteur
3. **Charger** où **Transformer**

### Options

| Option | Description |
|--------|-------------|
| Delimiteur | Virgule, point-virgule, tab |
| Encodage | UTF-8, ANSI |
| En-têtes | Première ligne |

## Importer un JSON

### Étapes

1. **Données** > **Obtenir des données** > **A partir d'un fichier** > **JSON**
2. Naviguez dans la structuré
3. Developpez les colonnes

### Transformer

Utilisez les outils pour aplatir la structuré.

## Fusionner plusieurs fichiers

### Depuis un dossier

1. **A partir d'un dossier**
2. Filtrez par extension
3. Combinez les fichiers

## Cas mixte CSV + JSON

### Strategie

1. Importez chaque type separement
2. Harmonisez les colonnes
3. Ajoutez (Union) les requêtes

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Structuré | Memes colonnes pour fusionner |
| Nettoyage | Harmonisez les formats |
| Documentation | Notez les transformations |
