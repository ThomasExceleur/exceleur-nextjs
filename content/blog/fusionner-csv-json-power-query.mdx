# Fusionner CSV et JSON avec Power Query

Combinez differents formats de donnees.

## Importer un CSV

### Etapes

1. **Donnees** > **Obtenir des donnees** > **A partir d'un fichier** > **Texte/CSV**
2. Configurez le delimiteur
3. **Charger** ou **Transformer**

### Options

| Option | Description |
|--------|-------------|
| Delimiteur | Virgule, point-virgule, tab |
| Encodage | UTF-8, ANSI |
| En-tetes | Premiere ligne |

## Importer un JSON

### Etapes

1. **Donnees** > **Obtenir des donnees** > **A partir d'un fichier** > **JSON**
2. Naviguez dans la structure
3. Developpez les colonnes

### Transformer

Utilisez les outils pour aplatir la structure.

## Fusionner plusieurs fichiers

### Depuis un dossier

1. **A partir d'un dossier**
2. Filtrez par extension
3. Combinez les fichiers

## Cas mixte CSV + JSON

### Strategie

1. Importez chaque type separement
2. Harmonisez les colonnes
3. Ajoutez (Union) les requetes

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Structure | Memes colonnes pour fusionner |
| Nettoyage | Harmonisez les formats |
| Documentation | Notez les transformations |
