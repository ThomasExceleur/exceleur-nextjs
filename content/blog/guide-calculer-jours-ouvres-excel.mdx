# Calculer les Jours Ouvres dans Excel

Maitrisez le calcul des jours travailles pour la gestion de projets et RH.

## Fonctions principales

### NB.JOURS.OUVRES

Compte les jours ouvres entre deux dates.

```excel
=NB.JOURS.OUVRES(date_debut;date_fin;[jours_feries])
```

### SERIE.JOUR.OUVRE

Trouve la date apres N jours ouvres.

```excel
=SERIE.JOUR.OUVRE(date_debut;jours;[jours_feries])
```

## NB.JOURS.OUVRES en detail

### Parametres

| Parametre | Description |
|-----------|-------------|
| date_debut | Date de depart |
| date_fin | Date d'arrivee |
| jours_feries | Liste des jours feries (optionnel) |

### Exemple simple

```excel
=NB.JOURS.OUVRES("01/01/2024";"31/01/2024")
```

Compte les jours ouvrables de janvier 2024.

## Gerer les jours feries

### Liste des jours feries

| Date | Ferie |
|------|-------|
| 01/01/2024 | Jour de l'an |
| 01/05/2024 | Fete du travail |
| 14/07/2024 | Fete nationale |
| 25/12/2024 | Noel |

### Formule avec feries

```excel
=NB.JOURS.OUVRES(A1;B1;$E$2:$E$15)
```

## NB.JOURS.OUVRES.INTL

### Syntaxe

```excel
=NB.JOURS.OUVRES.INTL(date_debut;date_fin;[weekend];[jours_feries])
```

### Codes weekend

| Code | Weekend |
|------|---------|
| 1 | Samedi, Dimanche |
| 2 | Dimanche, Lundi |
| 11 | Dimanche seul |
| "0000011" | Personnalise |

### Exemple : 6 jours par semaine

```excel
=NB.JOURS.OUVRES.INTL(A1;B1;11;Feries)
```

Dimanche seul comme jour de repos.

## SERIE.JOUR.OUVRE

### Usage

Trouver la date de fin d'un projet.

### Exemple

```excel
=SERIE.JOUR.OUVRE("01/01/2024";20;Feries)
```

Date apres 20 jours ouvres.

## Cas pratiques

### Calculer une echeance projet

```excel
=SERIE.JOUR.OUVRE(Date_debut;Duree_jours;Feries)
```

### Compter les jours de conge

```excel
=NB.JOURS.OUVRES(Date_debut_conge;Date_fin_conge;Feries)+1
```

### Calcul de penalites de retard

```excel
=MAX(0;NB.JOURS.OUVRES(Date_prevue;AUJOURDHUI();Feries))*Montant_journalier
```

### Feries mobiles

Paques, Ascension, Pentecote varient chaque annee.

## Calculer Paques

### Formule approximative

```excel
=PLANCHER.MATH(DATE(Annee;4;1)+MOD(19*MOD(Annee;19)+15;30)-MOD(Annee+QUOTIENT(Annee;4)+MOD(19*MOD(Annee;19)+15;30);7)+25;7)
```
