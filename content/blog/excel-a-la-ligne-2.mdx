# Retour a la Ligne dans une Cellule

Plusieurs lignes dans une seule cellule.

## Raccourci clavier

### Alt + Entree

En mode edition (F2), appuyez sur **Alt+Entree**.

## Retour automatique

### Activer

1. Selectionnez les cellules
2. **Accueil** > **Renvoyer a la ligne automatiquement**

### Effet

Le texte s'adapte a la largeur de la colonne.

## Differences

| Methode | Comportement |
|---------|--------------|
| Alt+Entree | Saut manuel |
| Renvoyer auto | S'adapte a la largeur |

## Formule avec saut de ligne

### CAR(10)

```excel
=A1&CAR(10)&B1
```

CAR(10) insere un saut de ligne.

## Exemple pratique

### Adresse

```excel
=Nom&CAR(10)&Rue&CAR(10)&CP&" "&Ville
```

## Supprimer les sauts

### SUBSTITUE

```excel
=SUBSTITUE(A1;CAR(10);" ")
```

## Hauteur de ligne

### Ajuster automatiquement

Double-cliquez sur la bordure de la ligne.

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Coherence | Meme methode partout |
| Hauteur | Ajustez les lignes |
| Lisibilite | N'abusez pas des sauts |
