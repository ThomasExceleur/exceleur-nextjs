# Analyse de Sensibilite dans Excel

L'analyse de sensibilite évalué l'impact des variations de paramètrès sûr vos résultats.

## Qu'est-ce que l'analyse de sensibilite ?

### Définition

Etudier comment les changements de variables d'entrée affectent le résultat d'un modèle.

### Usage

- Évaluer les risques
- Identifier les variables critiques
- Tester des scenarios

## Méthode 1 : Table de données à une variable

### Principe

Varier un paramèêtre et observer l'impact.

### Création

1. Listez les valeurs à tester verticalement
2. En haut de la colonne suivante, referencez votre formule de résultat
3. Sélectionnez la plage (valeurs + formule)
4. **Données** > **Analyse de simulation** > **Table de données**
5. **Cellule d'entrée en colonne** : la cellule variable
6. **OK**

### Exemple

| Prix | Marge |
|------|-------|
| 80 | =B2 |
| 90 | |
| 100 | |
| 110 | |

## Méthode 2 : Table de données à deux variables

### Principe

Varier deux paramètrès simultanement.

### Création

1. Valeurs du paramèêtre 1 : en colonne
2. Valeurs du paramèêtre 2 : en ligne
3. A l'intersection : formule de résultat
4. **Table de données**
5. **Cellule d'entrée en ligne** : paramèêtre 2
6. **Cellule d'entrée en colonne** : paramèêtre 1

### Résultat

Une matrice montrant toutes les combinaisons.

## Méthode 3 : Gestionnaire de scenarios

### Création

1. **Données** > **Analyse de simulation** > **Gestionnaire de scenarios**
2. **Ajouter**
3. Nommez le scenario
4. Sélectionnez les cellules variables
5. Entrez les valeurs
6. **OK**

### Comparer les scenarios

1. **Gestionnaire de scenarios** > **Synthèse**
2. Choisissez les cellules de résultat
3. **OK**

## Méthode 4 : Valeur cible

### Principe

Trouver la valeur d'entrée nécessaire pour atteindre un résultat cible.

### Utilisation

1. **Données** > **Analyse de simulation** > **Valeur cible**
2. **Cellule à définir** : résultat souhaité
3. **Valeur à atteindre** : votre objectif
4. **Cellule à modifier** : variable d'ajustement
5. **OK**

## Exemple pratique : Business plan

### Variables

- Prix de vente
- Cout unitaire
- Volume de ventes

### Formule de résultat

```excel
=Volume*(Prix-Cout)-ChargesFixes
```

### Analyse

Testez différentes combinaisons Prix/Volume pour la rentabilite.

## Visualisation

### Graphique tornado

Montre l'impact relatif de chaque variable.

### Graphique spider

Montre comment le résultat varie selon chaque paramèêtre.

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Identifier | Les variables les plus incertaines |
| Définir | Des plages realistes |
| Documenter | Les hypotheses |
| Interpreter | Les résultats avec prudence |

## Conseils

1. **Commencez** par une variable à la fois
2. **Utilisez** des plages realistes
3. **Visualisez** les résultats en graphique
4. **Documentez** vos hypotheses
