# Comment Connaitre sa Version d'Excel

Plusieurs methodes permettent d'identifier votre version d'Excel.

## Methode 1 : Via le menu Fichier

### Etapes

1. Ouvrez Excel
2. Cliquez sur **Fichier**
3. Cliquez sur **Compte** (ou **Aide** dans les versions anciennes)
4. Regardez "A propos d'Excel" ou "Version du produit"

### Informations affichees

- Nom du produit (Excel 2021, Microsoft 365, etc.)
- Numero de version
- Numero de build

## Methode 2 : Via A propos

### Excel moderne

1. **Fichier** > **Compte**
2. Cliquez sur **A propos d'Excel**

### Excel ancien

1. **?** (Aide) dans le menu
2. **A propos de Microsoft Excel**

## Methode 3 : Via la barre de titre

### Lecture directe

La barre de titre peut indiquer :
- "Microsoft Excel" (version classique)
- Nom du fichier seul (version moderne)

### Activer l'affichage

Dans les options, vous pouvez afficher le nom du produit.

## Identifier la version

### Par l'interface

| Element visuel | Version probable |
|----------------|------------------|
| Ruban colore | Excel 2007+ |
| Ruban plat | Excel 2013+ |
| Interface sombre possible | Excel 2019+ |
| Onglet Accueil simplifie | Microsoft 365 |

### Par le numero

| Numero | Version |
|--------|---------|
| 16.0 | Excel 2016, 2019, 2021, 365 |
| 15.0 | Excel 2013 |
| 14.0 | Excel 2010 |
| 12.0 | Excel 2007 |
| 11.0 | Excel 2003 |

## Version 32 bits ou 64 bits

### Verification

1. **Fichier** > **Compte**
2. **A propos d'Excel**
3. Regardez "(32 bits)" ou "(64 bits)"

### Difference

| 32 bits | 64 bits |
|---------|---------|
| Limite 2 Go RAM | Pas de limite |
| Compatible anciens add-ins | Meilleure performance |
| Par defaut | Recommande |

## Excel Desktop vs Excel Online

### Excel Desktop

- Installe sur l'ordinateur
- Toutes les fonctionnalites
- Fonctionne hors ligne

### Excel Online (Web)

- Dans le navigateur
- Fonctionnalites reduites
- Necessite internet

### Verification

- Desktop : application installee
- Online : acces via office.com

## Excel pour Mac

### Trouver la version

1. Ouvrez Excel
2. Menu **Excel** > **A propos d'Excel**

### Versions Mac

| Version | Annee |
|---------|-------|
| Excel pour Mac 16.xx | 2019+ |
| Excel pour Mac 15.xx | 2016 |
| Excel pour Mac 14.xx | 2011 |

## Verifier les mises a jour

### Windows

1. **Fichier** > **Compte**
2. **Options de mise a jour**
3. **Mettre a jour**

### Mac

1. Menu **Aide** > **Rechercher les mises a jour**
2. Ou via l'App Store

## Pourquoi connaitre sa version ?

### Raisons

1. **Compatibilite** : certaines fonctions sont recentes
2. **Support** : les anciennes versions ne sont plus supportees
3. **Formation** : adapter les tutoriels a votre version
4. **Depannage** : identifier les problemes specifiques

## Fonctionnalites par version

### Excel 365 / 2021

- RECHERCHEX, LET, LAMBDA
- Tableaux dynamiques
- Copilot (365 uniquement)

### Excel 2019

- CONCAT, JOINDRE.TEXTE
- Cartes 2D
- Entonnoirs

### Excel 2016

- Power Query integre
- Previsions
- Graphiques en cascade

### Excel 2013 et avant

- Fonctionnalites de base
- Pas de nouvelles fonctions

## Mettre a jour Excel

### Si ancienne version

1. Verifiez votre licence
2. Achetez une mise a jour ou abonnement
3. Telechargez depuis microsoft.com

### Si Microsoft 365

Les mises a jour sont automatiques.
