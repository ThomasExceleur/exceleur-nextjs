# Supprimer les Lignes Vides Excel

Nettoyez vos tableaux des lignes vides.

## Methode 1 : Atteindre

### Etapes

1. Selectionnez votre plage
2. **Ctrl+G** ou **F5** (Atteindre)
3. **Cellules** > **Cellules vides**
4. Clic droit > **Supprimer** > **Ligne entiere**

## Methode 2 : Filtre

### Etapes

1. Activez les filtres (**Ctrl+Maj+L**)
2. Filtrez pour afficher les vides
3. Selectionnez et supprimez
4. Retirez le filtre

## Methode 3 : Tri

### Etapes

1. Triez vos donnees
2. Les lignes vides se regroupent
3. Supprimez-les en bloc

## Methode Power Query

### Pour gros volumes

1. **Donnees** > **A partir d'un tableau**
2. **Accueil** > **Supprimer les lignes** > **Supprimer les lignes vides**

## Attention

| Point | Detail |
|-------|--------|
| Formules | Les cellules avec formules vides ne sont pas vides |
| Espaces | Un espace rend la cellule non vide |
| Sauvegarde | Faites une copie avant |

## Detecter les faux vides

### Formule

```excel
=NBCAR(A1)
```

Si resultat > 0, la cellule n'est pas vide.

### Nettoyer

```excel
=SUPPRESPACE(A1)
```

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Verification | Verifiez avant de supprimer |
| Tri prealable | Regroupez les vides |
| Source | Corrigez a la source si possible |
