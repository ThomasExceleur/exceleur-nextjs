# Supprimer les Lignes Vides Excel

Nettoyez vos tableaux des lignes vides.

## Méthode 1 : Atteindre

### Étapes

1. Sélectionnez votre plage
2. **Ctrl+G** où **F5** (Atteindre)
3. **Cellules** > **Cellules vides**
4. Clic droit > **Supprimer** > **Ligne entière**

## Méthode 2 : Filtre

### Étapes

1. Activez les filtres (**Ctrl+Maj+L**)
2. Filtrez pour afficher les vides
3. Sélectionnez et supprimez
4. Retirez le filtre

## Méthode 3 : Tri

### Étapes

1. Triez vos données
2. Les lignes vides se regroupent
3. Supprimez-les en bloc

## Méthode Power Query

### Pour gros volumes

1. **Données** > **A partir d'un tableau**
2. **Accueil** > **Supprimer les lignes** > **Supprimer les lignes vides**

## Attention

| Point | Detail |
|-------|--------|
| Formules | Les cellules avec formules vides ne sont pas vides |
| Espaces | Un espace rend la cellule non vide |
| Sauvegarde | Faites une copie avant |

## Detecter les faux vides

### Formule

```excel
=NBCAR(A1)
```

Si résultat > 0, la cellule n'est pas vide.

### Nettoyer

```excel
=SUPPRESPACE(A1)
```

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Vérification | Vérifiez avant de supprimer |
| Tri prealable | Regroupez les vides |
| Source | Corrigez à la source si possible |
