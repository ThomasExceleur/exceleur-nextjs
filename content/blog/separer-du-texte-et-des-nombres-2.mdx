# Séparer Texte et Nombres

Extrayez le texte et les nombres de vos cellules.

## Méthode Ctrl+E

### Étapes

1. Tapez le résultat souhaité (texte seul)
2. Descendez d'une cellule
3. **Ctrl+E**

## Formules pour le texte

### Supprimer les chiffres

```excel
=SUBSTITUE(SUBSTITUE(SUBSTITUE(A1;0;"");1;"");2;"")...
```

### TEXTE.APRES (Excel 365)

```excel
=TEXTE.APRES(A1;SEQUENCE(1;10;0;1))
```

## Formules pour les nombres

### SOMME matricielle

```excel
=SOMMEPROD(STXT(A1;SEQUENCE(NBCAR(A1));1)*ESTNUM(STXT(A1;SEQUENCE(NBCAR(A1));1)*1))
```

### Plus simple avec FILTRE (Excel 365)

Utilisez Ctrl+E pour les cas simples.

## Exemples

| Avant | Texte | Nombre |
|-------|-------|--------|
| ABC123 | ABC | 123 |
| 50kg | kg | 50 |
| Paris75001 | Paris | 75001 |

## Power Query

### Méthode

1. Sélectionnez la colonne
2. **Transformer** > **Extraire**
3. Choisissez le type d'extraction

### Options

- Premiers caracteres
- Derniers caracteres
- Plage de caracteres
- Texte avant/après delimiteur

## VBA

### Code simple

```vba
Function ExtraireNombre(texte As String) As Double
    Dim i As Integer, résultat As String
    For i = 1 To Len(texte)
        If IsNumeric(Mid(texte, i, 1)) Then
            résultat = résultat & Mid(texte, i, 1)
        End If
    Next i
    ExtraireNombre = Val(résultat)
End Function
```
