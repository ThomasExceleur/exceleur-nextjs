# Excel en Informatique

Le tableur incontournable.

## Définition technique

### Tableur

Logiciel permettant de manipuler des données dans une grille de cellules.

### Format

- .xlsx : Format actuel
- .xls : Ancien format
- .xlsm : Avec macros

## Place dans l'ecosysteme

### Suite Office

Excel fait partie de Microsoft Office avec Word, PowerPoint, etc.

### Alternatives

| Logiciel | Éditeur |
|----------|---------|
| Google Sheets | Google |
| LibreOffice Calc | Open Source |
| Numbers | Apple |

## Usages informatiques

### Analyse de données

- Traitement de fichiers
- Statistiques
- Reporting

### Prototypage

- Maquettes de calcul
- Tests d'algorithmes

### Intégration

- Export/Import CSV
- APIs
- Automatisation

## Limites

| Limite | Valeur |
|--------|--------|
| Lignes | 1 048 576 |
| Colonnes | 16 384 |
| Cellules | 17 milliards |

## Excel vs Base de données

| Critère | Excel | BDD |
|---------|-------|-----|
| Volume | Moyen | Grand |
| Requêtes | Basiques | Complexes |
| Multi-utilisateurs | Limite | Optimise |

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Usage approprie | Pas une BDD |
| Limites | Respectez les volumes |
| Alternatives | Power BI pour gros volumes |
