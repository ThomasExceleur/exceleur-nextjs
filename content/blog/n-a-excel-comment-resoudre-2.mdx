# Resoudre l'Erreur #N/A

Comprendre et corriger cette erreur.

## Signification

#N/A = "Non Applicable" - la valeur n'à pas ete trouvee.

## Causes principales

| Cause | Solution |
|-------|----------|
| Valeur absente | Vérifier les données |
| Espaces invisibles | SUPPRESPACE |
| Format différent | Convertir |
| Faute de frappe | Corriger |

## Diagnostic

### Vérifier les espaces

```excel
=NBCAR(A1)
```

### Vérifier le format

```excel
=ESTNUM(A1)
=ESTTEXTE(A1)
```

## Solutions

### SIERREUR

```excel
=SIERREUR(RECHERCHEV(...);"Non trouve")
```

### SI.NON.DISP

```excel
=SI.NON.DISP(RECHERCHEV(...);"Valeur par defaut")
```

### RECHERCHEX natif

```excel
=RECHERCHEX(valeur;plage;result;"Non trouve")
```

## Nettoyer les données

### SUPPRESPACE

```excel
=SUPPRESPACE(A1)
```

### Convertir texte en nombre

```excel
=CNUM(A1)
```

## Prevention

### Bonnes pratiques

| Action | Bénéfice |
|--------|----------|
| Validation | Evite les erreurs de saisie |
| Nettoyage | Données cohérentes |
