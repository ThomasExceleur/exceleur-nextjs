# Resoudre l'Erreur #N/A

Comprendre et corriger cette erreur.

## Signification

#N/A = "Non Applicable" - la valeur n'a pas ete trouvee.

## Causes principales

| Cause | Solution |
|-------|----------|
| Valeur absente | Verifier les donnees |
| Espaces invisibles | SUPPRESPACE |
| Format different | Convertir |
| Faute de frappe | Corriger |

## Diagnostic

### Verifier les espaces

```excel
=NBCAR(A1)
```

### Verifier le format

```excel
=ESTNUM(A1)
=ESTTEXTE(A1)
```

## Solutions

### SIERREUR

```excel
=SIERREUR(RECHERCHEV(...);"Non trouve")
```

### SI.NON.DISP

```excel
=SI.NON.DISP(RECHERCHEV(...);"Valeur par defaut")
```

### RECHERCHEX natif

```excel
=RECHERCHEX(valeur;plage;result;"Non trouve")
```

## Nettoyer les donnees

### SUPPRESPACE

```excel
=SUPPRESPACE(A1)
```

### Convertir texte en nombre

```excel
=CNUM(A1)
```
