### 2\. Configuration des Revenus

Dans votre **tableau Excel budget famille**, créez une section revenus avec ces colonnes :

```
| Date | Description | Catégorie | Montant | Récurrent |
```

**Formules essentielles pour l'onglet Revenus :**

-   Total mensuel : `=SUMIF(C:C,"Salaire",D:D)` pour additionner par catégorie
-   Total annuel : `=SUM(D:D)_12_` _pour les revenus récurrents_
_-   Moyenne mobile : `=AVERAGE(D2:D13)` pour analyser les tendances_

_


### 3\. Gestion des Dépenses par Catégories

Organisez vos dépenses en catégories principales et sous-catégories :

**Dépenses Fixes :**

-   Logement (loyer/prêt, charges, assurances)
-   Transport (carburant, assurance auto, transports publics)
-   Télécommunications (internet, téléphone, abonnements)

**Dépenses Variables :**

-   Alimentation (courses, restaurants)
-   Loisirs (sorties, vacances, hobbies)
-   Santé (pharmacie, consultations, mutuelles)

**Formule pour calculer le pourcentage par catégorie :**

_

```excel
=SUMIF(Catégorie:Catégorie,"Alimentation",Montant:Montant)/SUM(Montant:Montant)
```
