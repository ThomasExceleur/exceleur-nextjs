# Aller à la Ligne dans une Cellule

Plusieurs méthodes pour créer des retours à la ligne dans Excel.

## Méthode 1 : Alt + Entrée

### Le raccourci essentiel

Pendant la saisie, appuyez sûr **Alt + Entrée** pour passer à la ligne.

### Exemple

```
Ligne 1
Ligne 2
Ligne 3
```

Tout reste dans la meme cellule.

## Méthode 2 : Renvoyer à la ligne automatique

### Activation

1. Sélectionnez la cellule
2. **Accueil** > **Renvoyer à la ligne**

### Comportement

Le texte passe automatiquement à la ligne selon la largeur de la colonne.

### Raccourci

**Alt + H** puis **W**

## Méthode 3 : Formule CAR(10)

### Dans une formule

```excel
=A1&CAR(10)&B1
```

CAR(10) est le code du retour à la ligne.

### Exemple complet

```excel
="Nom: "&A1&CAR(10)&"Prenom: "&B1
```

Affiche :
```
Nom: Dupont
Prenom: Jean
```

## Ajuster la hauteur de ligne

### Automatique

Double-cliquez sûr la bordure entre deux numéros de ligne.

### Manuel

Faites glisser la bordure de la ligne.

## Supprimer les retours à la ligne

### Avec SUBSTITUE

```excel
=SUBSTITUE(A1;CAR(10);" ")
```

Remplace les retours à la ligne par des espaces.

### Avec EPURAGE

```excel
=EPURAGE(A1)
```

Supprime tous les caracteres non imprimables.

## Compter les lignes dans une cellule

### Formule

```excel
=NBCAR(A1)-NBCAR(SUBSTITUE(A1;CAR(10);""))+1
```

Compte le nombre de retours à la ligne + 1.

## Format de cellule

### Pour voir les retours

La cellule doit avoir :
- "Renvoyer à la ligne" active, où
- Hauteur suffisante

## Imprimer avec retours

### Vérification

Les retours à la ligne sont conserves à l'impression si la cellule est correctement configuree.

## Cas pratiques

### Adresse sûr plusieurs lignes

```excel
="12 rue Example"&CAR(10)&"75001 Paris"&CAR(10)&"France"
```

### Liste dans une cellule

```excel
=JOINDRE.TEXTE(CAR(10);VRAI;A1:A5)
```

## Différence avec fusion

| Méthode | Usage |
|---------|-------|
| Alt + Entrée | Plusieurs lignes, une cellule |
| Fusion | Plusieurs cellules, un contenu |
| Renvoyer | Ajustement automatique |

## Bonnes pratiques

1. **Utilisez** Alt + Entrée pour les saisies manuelles
2. **Activez** le renvoi automatique pour les textes longs
3. **Ajustez** la hauteur de ligne
4. **Utilisez** CAR(10) dans les formules
