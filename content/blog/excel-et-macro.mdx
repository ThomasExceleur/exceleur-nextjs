# Excel et Macro : Automatisez vos Taches

Les macros permettent d'automatiser les taches repetitives dans Excel. Gagnez des heures chaque semaine.

## Qu'est-ce qu'une macro ?

Une macro est un enregistrement d'actions que vous pouvez rejouer automatiquement. Elle permet de :

- Automatiser des taches repetitives
- Reduire les erreurs humaines
- Gagner un temps considerable
- Standardiser les processus

## Activer l'onglet Developpeur

### Etapes

1. **Fichier** > **Options**
2. **Personnaliser le ruban**
3. Cochez **Developpeur**
4. **OK**

L'onglet Developpeur apparait dans le ruban.

## Enregistrer une macro

### Methode simple

1. Onglet **Developpeur** > **Enregistrer une macro**
2. Donnez un nom a la macro (sans espaces)
3. Optionnel : Assignez un raccourci clavier
4. Effectuez vos actions
5. **Arreter l'enregistrement**

### Exemple pratique

**Objectif :** Mise en forme automatique d'un tableau

1. Demarrez l'enregistrement
2. Selectionnez la plage
3. Appliquez le gras aux en-tetes
4. Ajoutez des bordures
5. Ajustez les colonnes
6. Arretez l'enregistrement

## Executer une macro

### Methode 1 : Menu

**Developpeur** > **Macros** > Selectionnez > **Executer**

### Methode 2 : Raccourci clavier

Utilisez le raccourci defini lors de l'enregistrement (ex: Ctrl + Maj + M).

### Methode 3 : Bouton

1. Inserez un bouton (**Developpeur** > **Inserer** > **Bouton**)
2. Assignez la macro au bouton

## Securite des macros

### Niveaux de securite

1. **Fichier** > **Options** > **Centre de gestion de la confidentialite**
2. **Parametres** > **Parametres des macros**

Options :
- Desactiver toutes les macros
- Desactiver avec notification (recommande)
- Activer toutes les macros (non recommande)

### Format de fichier

Les macros ne sont conservees que dans les fichiers **.xlsm** (et non .xlsx).

## Introduction au VBA

### Qu'est-ce que VBA ?

VBA (Visual Basic for Applications) est le langage de programmation des macros.

### Acceder a l'editeur VBA

Appuyez sur **Alt + F11** ou **Developpeur** > **Visual Basic**.

### Structure d'une macro

```vba
Sub NomDeLaMacro()
    ' Commentaire
    Range("A1").Value = "Bonjour"
    Range("A1").Font.Bold = True
End Sub
```

### Exemple simple

```vba
Sub MiseEnFormeTableau()
    ' Selectionner la plage utilisee
    ActiveSheet.UsedRange.Select

    ' Bordures
    Selection.Borders.LineStyle = xlContinuous

    ' En-tetes en gras
    Rows(1).Font.Bold = True

    ' Ajuster les colonnes
    Cells.EntireColumn.AutoFit
End Sub
```

## Cas d'usage courants

### Mise en forme automatique

- Appliquer un style predetermine
- Formater les nombres
- Ajouter des bordures

### Import de donnees

- Ouvrir un fichier
- Copier les donnees
- Fermer le fichier

### Creation de rapports

- Consolider des donnees
- Creer des graphiques
- Enregistrer en PDF

### Nettoyage de donnees

- Supprimer les doublons
- Corriger les formats
- Remplacer des valeurs

## Bonnes pratiques

### Nommage

- Utilisez des noms descriptifs
- Pas d'espaces ni de caracteres speciaux
- Ex: `MiseEnFormeRapportMensuel`

### Commentaires

Documentez votre code :

```vba
' Cette macro formate le rapport mensuel
' Auteur: Thomas
' Date: 2024-01-15
```

### Sauvegarde

- Sauvegardez avant d'executer
- Testez sur une copie
- Utilisez Ctrl + Z si possible

### Modularite

Decoupez en petites macros reutilisables plutot qu'une grande macro.

## Limites des macros enregistrees

### Ce qu'elles ne peuvent pas faire

- Conditions (Si... Alors...)
- Boucles (Repeter...)
- Interactions utilisateur
- Gestion d'erreurs

### Solution

Pour des automatisations avancees, apprenez le VBA.

## Erreurs courantes

### La macro ne fonctionne pas

- Verifiez le format du fichier (.xlsm)
- Verifiez les parametres de securite
- Relancez Excel

### Resultat different

- Les macros enregistrent des references absolues
- Adaptez le code pour des references relatives
