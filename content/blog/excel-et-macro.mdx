# Excel et Macro : Automatisez vos Taches

Les macros permettent d'automatiser les taches repetitives dans Excel. Gagnez des heures chaque semaine.

## Qu'est-ce qu'une macro ?

Une macro est un enregistrement d'actions que vous pouvez rejouer automatiquement. Elle permet de :

- Automatiser des taches repetitives
- Réduire les erreurs humaines
- Gagner un temps considerable
- Standardiser les processus

## Activer l'onglet Developpeur

### Étapes

1. **Fichier** > **Options**
2. **Personnaliser le ruban**
3. Cochez **Developpeur**
4. **OK**

L'onglet Developpeur apparait dans le ruban.

## Enregistrer une macro

### Méthode simple

1. Onglet **Developpeur** > **Enregistrer une macro**
2. Donnez un nom à la macro (sans espaces)
3. Optionnel : Assignez un raccourci clavier
4. Effectuez vos actions
5. **Arrêter l'enregistrement**

### Exemple pratique

**Objectif :** Mise en forme automatique d'un tableau

1. Démarrez l'enregistrement
2. Sélectionnez la plage
3. Appliquez le gras aux en-têtes
4. Ajoutez des bordures
5. Ajustez les colonnes
6. Arrêtez l'enregistrement

## Executer une macro

### Méthode 1 : Menu

**Developpeur** > **Macros** > Sélectionnez > **Executer**

### Méthode 2 : Raccourci clavier

Utilisez le raccourci défini lors de l'enregistrement (ex: Ctrl + Maj + M).

### Méthode 3 : Bouton

1. Insérez un bouton (**Developpeur** > **Insérer** > **Bouton**)
2. Assignez la macro au bouton

## Sécurité des macros

### Niveaux de sécurité

1. **Fichier** > **Options** > **Centre de gestion de la confidentialite**
2. **Paramètrès** > **Paramètrès des macros**

Options :
- Desactiver toutes les macros
- Desactiver avec notification (recommandé)
- Activer toutes les macros (non recommandé)

### Format de fichier

Les macros ne sont conservees que dans les fichiers **.xlsm** (et non .xlsx).

## Introduction au VBA

### Qu'est-ce que VBA ?

VBA (Visual Basic for Applications) est le langage de programmation des macros.

### Acceder à l'éditeur VBA

Appuyez sûr **Alt + F11** où **Developpeur** > **Visual Basic**.

### Structuré d'une macro

```vba
Sub NomDeLaMacro()
    ' Commentaire
    Range("A1").Value = "Bonjour"
    Range("A1").Font.Bold = True
End Sub
```

### Exemple simple

```vba
Sub MiseEnFormeTableau()
    ' Sélectionner la plage utilisée
    ActiveSheet.UsedRange.Select

    ' Bordures
    Sélection.Borders.LineStyle = xlContinuous

    ' En-têtes en gras
    Rows(1).Font.Bold = True

    ' Ajuster les colonnes
    Cells.EntireColumn.AutoFit
End Sub
```

## Cas d'usage courants

### Mise en forme automatique

- Appliquer un style predetermine
- Formater les nombres
- Ajouter des bordures

### Import de données

- Ouvrir un fichier
- Copier les données
- Fermer le fichier

### Création de rapports

- Consolider des données
- Créer des graphiques
- Enregistrer en PDF

### Nettoyage de données

- Supprimer les doublons
- Corriger les formats
- Remplacer des valeurs

## Bonnes pratiques

### Nommage

- Utilisez des noms descriptifs
- Pas d'espaces ni de caracteres speciaux
- Ex: `MiseEnFormeRapportMensuel`

### Commentaires

Documentez votre code :

```vba
' Cette macro formate le rapport mensuel
' Auteur: Thomas
' Date: 2024-01-15
```

### Sauvegarde

- Sauvegardez avant d'executer
- Testez sûr une copie
- Utilisez Ctrl + Z si possible

### Modularite

Decoupez en petites macros reutilisables plutôt qu'une grande macro.

## Limites des macros enregistrees

### Ce qu'elles ne peuvent pas faire

- Conditions (Si... Alors...)
- Boucles (Répéter...)
- Interactions utilisateur
- Gestion d'erreurs

### Solution

Pour des automatisations avancées, apprenez le VBA.

## Erreurs courantes

### La macro ne fonctionne pas

- Vérifiez le format du fichier (.xlsm)
- Vérifiez les paramètrès de sécurité
- Relancez Excel

### Résultat différent

- Les macros enregistrent des références absolues
- Adaptez le code pour des références relatives

## Aller plus loin

Les macros sont le premier pas vers l'automatisation. Pour des taches plus complexes, le VBA offre des possibilités illimitees.
