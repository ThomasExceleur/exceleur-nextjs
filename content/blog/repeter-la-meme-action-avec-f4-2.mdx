# Répéter la Meme Action avec F4

Le raccourci polyvalent d'Excel.

## Deux usages de F4

### 1. Répéter la dernière action

Hors d'une formule, F4 répété l'action précédente.

### 2. Basculer les références

Dans une formule, F4 change le type de référence.

## Répéter une action

### Exemples

| Action précédente | F4 fait |
|-------------------|---------|
| Supprimer une ligne | Supprime la suivante |
| Appliquer une couleur | Applique la meme |
| Insérer une colonne | Inséré une autre |

### Workflow

1. Effectuez une action
2. Sélectionnez la cible suivante
3. **F4**

## Basculer les références

### Dans une formule

| Appui | Résultat |
|-------|----------|
| 1 | `$A$1` (absolue) |
| 2 | `A$1` (ligne fixe) |
| 3 | `$A1` (colonne fixe) |
| 4 | `A1` (relative) |

### Utilisation

1. Sélectionnez la référence dans la formule
2. Appuyez sûr **F4**
3. Répétez pour cycler

## Cas d'usage : mise en forme

### Scenario

Colorer 10 cellules en jaune.

### Méthode

1. Colorez la première cellule
2. Sélectionnez la suivante
3. **F4**
4. Répétez

## Limites

### Actions non repetables

- Actions multiples
- Certaines opérations complexes

### Alternative

**Ctrl+Y** pour refaire.

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Vérification | Vérifiez l'action avant de répéter |
| Sélection | Sélectionnez bien la cible |
| Combinaison | Utilisez avec d'autres raccourcis |
