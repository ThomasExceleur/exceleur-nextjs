# Repeter la Meme Action avec F4

Le raccourci polyvalent d'Excel.

## Deux usages de F4

### 1. Repeter la derniere action

Hors d'une formule, F4 repete l'action precedente.

### 2. Basculer les references

Dans une formule, F4 change le type de reference.

## Repeter une action

### Exemples

| Action precedente | F4 fait |
|-------------------|---------|
| Supprimer une ligne | Supprime la suivante |
| Appliquer une couleur | Applique la meme |
| Inserer une colonne | Insere une autre |

### Workflow

1. Effectuez une action
2. Selectionnez la cible suivante
3. **F4**

## Basculer les references

### Dans une formule

| Appui | Resultat |
|-------|----------|
| 1 | `$A$1` (absolue) |
| 2 | `A$1` (ligne fixe) |
| 3 | `$A1` (colonne fixe) |
| 4 | `A1` (relative) |

### Utilisation

1. Selectionnez la reference dans la formule
2. Appuyez sur **F4**
3. Repetez pour cycler

## Cas d'usage : mise en forme

### Scenario

Colorer 10 cellules en jaune.

### Methode

1. Colorez la premiere cellule
2. Selectionnez la suivante
3. **F4**
4. Repetez

## Limites

### Actions non repetables

- Actions multiples
- Certaines operations complexes

### Alternative

**Ctrl+Y** pour refaire.

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Verification | Verifiez l'action avant de repeter |
| Selection | Selectionnez bien la cible |
| Combinaison | Utilisez avec d'autres raccourcis |
