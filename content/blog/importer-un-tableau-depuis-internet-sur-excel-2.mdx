# Importer un Tableau depuis Internet

Recuperez des donnees web dans Excel.

## Methode Power Query

### Etapes

1. **Donnees** > **A partir du web**
2. Collez l'URL
3. Selectionnez le tableau
4. **Charger**

## Navigation dans la page

### Selecteur de tableau

Power Query detecte automatiquement les tableaux HTML.

### Choix multiple

Plusieurs tableaux ? Selectionnez celui souhaite.

## Avantages

| Avantage | Description |
|----------|-------------|
| Actualisation | Mise a jour automatique |
| Transformations | Nettoyage possible |
| Historique | Etapes enregistrees |

## Actualiser les donnees

### Methode manuelle

**Donnees** > **Actualiser tout** ou **Ctrl+Alt+F5**

### Automatique

1. Clic droit sur la requete
2. **Proprietes**
3. **Actualiser toutes les X minutes**

## Transformations courantes

### Dans Power Query

| Action | Utilite |
|--------|---------|
| Supprimer colonnes | Nettoyer |
| Filtrer | Garder l'essentiel |
| Changer types | Nombres, dates |
| Renommer | Clarifier |

## Limites

### Sites proteges

- Authentification requise
- CAPTCHA
- JavaScript dynamique

### Solutions

- Copier-coller manuel
- API si disponible
- Outils specialises
