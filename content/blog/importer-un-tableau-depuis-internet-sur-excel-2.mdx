# Importer un Tableau depuis Internet

Récupérez des données web dans Excel.

## Méthode Power Query

### Étapes

1. **Données** > **A partir du web**
2. Collez l'URL
3. Sélectionnez le tableau
4. **Charger**

## Navigation dans la page

### Selecteur de tableau

Power Query detecte automatiquement les tableaux HTML.

### Choix multiple

Plusieurs tableaux ? Sélectionnez celui souhaité.

## Avantages

| Avantage | Description |
|----------|-------------|
| Actualisation | Mise à jour automatique |
| Transformations | Nettoyage possible |
| Historique | Étapes enregistrees |

## Actualiser les données

### Méthode manuelle

**Données** > **Actualiser tout** où **Ctrl+Alt+F5**

### Automatique

1. Clic droit sûr la requête
2. **Propriétés**
3. **Actualiser toutes les X minutes**

## Limites

### Sites proteges

- Authentification requise
- CAPTCHA
- JavaScript dynamique

### Solutions

- Copier-coller manuel
- API si disponible
- Outils specialises
