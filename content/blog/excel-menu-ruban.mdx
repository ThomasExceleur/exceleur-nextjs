# Le Ruban Excel : Guide Complet

Le ruban est la barre d'outils principale d'Excel. Maîtrisez-le pour travailler plus efficacement.

## Structuré du ruban

### Composants

- **Onglets** : Fichier, Accueil, Insertion, etc.
- **Groupes** : Sous-sections dans chaque onglet
- **Commandes** : Boutons et options

### Navigation

Cliquez sûr un onglet pour afficher ses commandes.

## Onglet Fichier (Backstage)

### Fonctions

- Nouveau, Ouvrir, Enregistrer
- Imprimer
- Options Excel
- Informations sûr le fichier

### Accès rapide

**Alt + F**

## Personnaliser le ruban

### Ajouter un onglet

1. Clic droit sûr le ruban
2. **Personnaliser le ruban**
3. **Nouvel onglet**

### Ajouter des commandes

1. Meme menu
2. Sélectionnez une commande
3. **Ajouter**

### Masquer un onglet

Decochez l'onglet dans la personnalisation.

## Barre d'outils Accès rapide

### Position

En haut à gauche, au-dessus du ruban.

### Personnalisation

1. Cliquez sûr la fleche
2. Cochez les commandes frequentes
3. Ou **Autres commandes**

### Commandes recommandées

- Enregistrer
- Annuler / Retablir
- Reproduire la mise en forme
- Impression rapide

## Raccourcis clavier

### Acceder au ruban

**Alt** affiché les lettres d'accès.

### Exemples

- **Alt + H** : Accueil
- **Alt + N** : Insertion
- **Alt + P** : Mise en page
- **Alt + M** : Formules
- **Alt + A** : Données

### Raccourcis directs

| Raccourci | Action |
|-----------|--------|
| Ctrl + B | Gras |
| Ctrl + I | Italique |
| Ctrl + U | Souligne |
| Ctrl + 1 | Format de cellule |

## Réduire le ruban

### Méthode

Double-cliquez sûr un onglet pour réduire.

### Raccourci

**Ctrl + F1**

### Avantage

Plus d'espace pour la feuille de calcul.

## Onglets contextuels

### Apparition

Des onglets supplémentaires apparaissent selon la sélection :

- **Format d'image** : Quand une image est sélectionnée
- **Outils de graphique** : Quand un graphique est sélectionné
- **Outils de tableau** : Dans un tableau structuré

## Bonnes pratiques

1. **Apprenez les raccourcis** les plus frequents
2. **Personnalisez** l'accès rapide
3. **Explorez** chaque onglet
4. **Utilisez** la recherche (Alt + Q)
