# Excel pour Mac : Options Gratuites

Plusieurs options permettent d'utiliser Excel ou des alternatives sur Mac sans payer.

## Option 1 : Excel Online (Gratuit)

### Acces

1. Ouvrez Safari ou Chrome
2. Allez sur **office.com**
3. Connectez-vous avec un compte Microsoft gratuit
4. Cliquez sur Excel

### Avantages

- 100% gratuit
- Fonctionne sur Mac, PC, Linux
- Stockage OneDrive inclus
- Pas d'installation

### Limites

- Necessite internet
- Fonctionnalites reduites
- Pas de macros VBA

## Option 2 : Essai Microsoft 365

### Duree

1 mois gratuit avec toutes les fonctionnalites.

### Acces

1. Allez sur microsoft.com
2. Microsoft 365 > Essai gratuit
3. Creez ou connectez-vous a un compte
4. Telechargez Excel pour Mac

### Contenu

- Excel complet pour Mac
- Word, PowerPoint, etc.
- 1 To OneDrive

## Option 3 : Programmes etudiants

### Office 365 Education

Gratuit pour etudiants et enseignants.

1. Verifiez eligibilite sur microsoft.com/education
2. Utilisez votre email educatif
3. Telechargez la suite complete

## Option 4 : Alternatives gratuites

### Numbers (Apple)

- Gratuit sur Mac
- Interface moderne
- Import/export Excel
- Integration iCloud

### Google Sheets

- Gratuit dans le navigateur
- Excellente collaboration
- Compatible Excel
- Stockage Google Drive

### LibreOffice Calc

- Gratuit et open source
- Telechargement : libreoffice.org
- Tres compatible Excel
- Fonctionne hors-ligne

## Comparaison des options

| Option | Gratuit | Hors-ligne | Compatibilite |
|--------|---------|------------|---------------|
| Excel Online | Oui | Non | Excellente |
| Numbers | Oui | Oui | Bonne |
| Google Sheets | Oui | Limite | Tres bonne |
| LibreOffice | Oui | Oui | Bonne |

## Numbers vs Excel

### Avantages Numbers

- Design moderne
- Integration Apple
- Gratuit
- iCloud sync

### Avantages Excel

- Plus de fonctions
- Standard professionnel
- Macros VBA
- Plus d'options

## Convertir les fichiers

### Ouvrir un .xlsx dans Numbers

1. Double-cliquez sur le fichier
2. Ou : Numbers > Fichier > Ouvrir

### Exporter de Numbers vers Excel

1. Fichier > Exporter vers > Excel
2. Choisissez le format (.xlsx)

### Attention

Certaines fonctionnalites peuvent ne pas etre compatibles.

## Excel pour Mac (Payant)

### Achat unique

Office 2021 pour Mac : achat unique, pas d'abonnement.

### Abonnement

Microsoft 365 : paiement mensuel ou annuel.

### Differences

| | Office 2021 | Microsoft 365 |
|---|-------------|---------------|
| Prix | Unique | Mensuel |
| Mises a jour | Limitees | Continues |
| OneDrive | Non | 1 To |
| Multi-appareils | 1 Mac | 5 appareils |

## Configuration requise Mac

### Excel pour Mac

- macOS 10.14 ou plus recent
- 4 Go RAM minimum
- 10 Go espace disque

### Versions macOS supportees

- macOS Monterey
- macOS Ventura
- macOS Sonoma

## Raccourcis Mac

| Action | Windows | Mac |
|--------|---------|-----|
| Copier | Ctrl+C | Cmd+C |
| Coller | Ctrl+V | Cmd+V |
| Gras | Ctrl+B | Cmd+B |
| Enregistrer | Ctrl+S | Cmd+S |

3. **Evaluez** vos besoins avant d'acheter
4. **Verifiez** la compatibilite des fichiers
