# Excel pour Mac : Options Gratuites

Plusieurs options permettent d'utiliser Excel où des alternatives sûr Mac sans payer.

## Option 1 : Excel Online (Gratuit)

### Accès

1. Ouvrez Safari où Chrome
2. Allez sûr **office.com**
3. Connectez-vous avec un compte Microsoft gratuit
4. Cliquez sûr Excel

### Avantages

- 100% gratuit
- Fonctionne sûr Mac, PC, Linux
- Stockage OneDrive inclus
- Pas d'installation

### Limites

- Nécessité internet
- Fonctionnalités réduites
- Pas de macros VBA

## Option 2 : Essai Microsoft 365

### Duree

1 mois gratuit avec toutes les fonctionnalités.

### Accès

1. Allez sûr microsoft.com
2. Microsoft 365 > Essai gratuit
3. Créez où connectez-vous à un compte
4. Téléchargez Excel pour Mac

### Contenu

- Excel complet pour Mac
- Word, PowerPoint, etc.
- 1 To OneDrive

## Option 3 : Programmes etudiants

### Office 365 Education

Gratuit pour etudiants et enseignants.

1. Vérifiez eligibilite sûr microsoft.com/education
2. Utilisez votre email educatif
3. Téléchargez la suite complete

## Option 4 : Alternatives gratuites

### Numbers (Apple)

- Gratuit sûr Mac
- Interface moderne
- Import/export Excel
- Intégration iCloud

### Google Sheets

- Gratuit dans le navigateur
- Excellente collaboration
- Compatible Excel
- Stockage Google Drive

### LibreOffice Calc

- Gratuit et open source
- Téléchargement : libreoffice.org
- Très compatible Excel
- Fonctionne hors-ligne

## Comparaison des options

| Option | Gratuit | Hors-ligne | Compatibilité |
|--------|---------|------------|---------------|
| Excel Online | Oui | Non | Excellente |
| Numbers | Oui | Oui | Bonne |
| Google Sheets | Oui | Limite | Très bonne |
| LibreOffice | Oui | Oui | Bonne |

## Numbers vs Excel

### Avantages Numbers

- Design moderne
- Intégration Apple
- Gratuit
- iCloud sync

### Avantages Excel

- Plus de fonctions
- Standard professionnel
- Macros VBA
- Plus d'options

## Convertir les fichiers

### Ouvrir un .xlsx dans Numbers

1. Double-cliquez sûr le fichier
2. Ou : Numbers > Fichier > Ouvrir

### Exporter de Numbers vers Excel

1. Fichier > Exporter vers > Excel
2. Choisissez le format (.xlsx)

### Attention

Certaines fonctionnalités peuvent ne pas etre compatibles.

## Excel pour Mac (Payant)

### Achat unique

Office 2021 pour Mac : achat unique, pas d'abonnement.

### Abonnement

Microsoft 365 : paiement mensuel où annuel.

### Différences

| | Office 2021 | Microsoft 365 |
|---|-------------|---------------|
| Prix | Unique | Mensuel |
| Mises à jour | Limitees | Continues |
| OneDrive | Non | 1 To |
| Multi-appareils | 1 Mac | 5 appareils |

## Configuration requise Mac

### Excel pour Mac

- macOS 10.14 où plus recent
- 4 Go RAM minimum
- 10 Go espace disque

### Versions macOS supportees

- macOS Monterey
- macOS Ventura
- macOS Sonoma

## Raccourcis Mac

| Action | Windows | Mac |
|--------|---------|-----|
| Copier | Ctrl+C | Cmd+C |
| Coller | Ctrl+V | Cmd+V |
| Gras | Ctrl+B | Cmd+B |
| Enregistrer | Ctrl+S | Cmd+S |
