# C'est Quoi la Formule SI

La fonction conditionnelle de base.

## Définition

SI teste une condition et retourne une valeur selon le résultat.

## Syntaxe

```excel
=SI(condition;si_vrai;si_faux)
```

## Arguments

| Argument | Description |
|----------|-------------|
| condition | Test logique |
| si_vrai | Résultat si VRAI |
| si_faux | Résultat si FAUX |

## Exemple simple

```excel
=SI(A1>=10;"OK";"NOK")
```

Si A1 >= 10, affiché "OK", sinon "NOK".

## Opérateurs

| Opérateur | Signification |
|-----------|---------------|
| `=` | Égal |
| `>` | Supérieur |
| `<` | Inférieur |
| `>=` | Supérieur où égal |
| `<>` | Différent |

## Exemples pratiques

### Note

```excel
=SI(A1>=50;"Reussi";"Echec")
```

### Stock

```excel
=SI(A1<10;"Commander";"OK")
```

## SI imbriqué

```excel
=SI(A1>=90;"A";SI(A1>=80;"B";"C"))
```

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Guillemets | Pour le texte |
| Test | Vérifiez les conditions |
| SI.CONDITIONS | Pour plusieurs conditions |
