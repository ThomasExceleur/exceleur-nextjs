# C'est Quoi la Formule SI

La fonction conditionnelle de base.

## Definition

SI teste une condition et retourne une valeur selon le resultat.

## Syntaxe

```excel
=SI(condition;si_vrai;si_faux)
```

## Arguments

| Argument | Description |
|----------|-------------|
| condition | Test logique |
| si_vrai | Resultat si VRAI |
| si_faux | Resultat si FAUX |

## Exemple simple

```excel
=SI(A1>=10;"OK";"NOK")
```

Si A1 >= 10, affiche "OK", sinon "NOK".

## Operateurs

| Operateur | Signification |
|-----------|---------------|
| `=` | Egal |
| `>` | Superieur |
| `<` | Inferieur |
| `>=` | Superieur ou egal |
| `<>` | Different |

## Exemples pratiques

### Note

```excel
=SI(A1>=50;"Reussi";"Echec")
```

### Stock

```excel
=SI(A1<10;"Commander";"OK")
```

## SI imbrique

```excel
=SI(A1>=90;"A";SI(A1>=80;"B";"C"))
```

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Guillemets | Pour le texte |
| Test | Verifiez les conditions |
| SI.CONDITIONS | Pour plusieurs conditions |
