# Pourquoi Passer a RECHERCHEX

Les raisons de migrer vers RECHERCHEX.

## Recherche bidirectionnelle

### RECHERCHEV

Recherche uniquement a droite.

### RECHERCHEX

Recherche dans n'importe quelle direction.

## Gestion d'erreur integree

### RECHERCHEV

```excel
=SIERREUR(RECHERCHEV(...);"Non trouve")
```

### RECHERCHEX

```excel
=RECHERCHEX(val;plage;result;"Non trouve")
```

## Syntaxe plus claire

### RECHERCHEV

```excel
=RECHERCHEV(A1;B:E;3;FAUX)
```

3 = quelle colonne ?

### RECHERCHEX

```excel
=RECHERCHEX(A1;B:B;D:D)
```

La colonne resultat est explicite.

## Performance

RECHERCHEX est plus rapide sur gros volumes.

## Retour multiple

RECHERCHEX peut retourner plusieurs colonnes.

```excel
=RECHERCHEX(A1;B:B;C:E)
```

## Tableau comparatif

| Critere | RECHERCHEV | RECHERCHEX |
|---------|------------|------------|
| Direction | Droite | Les deux |
| Erreurs | SIERREUR | Integree |
| Lisibilite | Moyenne | Excellente |
| Performance | Bonne | Meilleure |
| Multi-colonnes | Non | Oui |

## Quand migrer ?

| Situation | Recommandation |
|-----------|----------------|
| Nouveau projet | RECHERCHEX |
| Fichier partage | Verifier compatibilite |
| Excel 365 | Migrer progressivement |
