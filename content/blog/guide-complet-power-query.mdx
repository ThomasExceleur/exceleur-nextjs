# Guide Complet Power Query

Power Query est l'outil de transformation de données intégré à Excel.

## Qu'est-ce que Power Query ?

### Définition

Power Query permet de :
- Importer des données de multiples sources
- Transformer et nettoyer les données
- Automatiser les processus

### Accès

**Données** > **Obtenir des données**

## Sources de données

### Fichiers

- Excel
- CSV/Texte
- XML/JSON
- PDF

### Bases de données

- SQL Server
- Access
- Oracle
- MySQL

### Web et Cloud

- Pages web
- SharePoint
- Azure
- API

## Première requête

### Import d'un fichier Excel

1. **Données** > **Obtenir des données** > **A partir d'un fichier**
2. Sélectionnez votre fichier
3. Choisissez la feuille/table
4. **Transformer les données**

## L'éditeur Power Query

### Interface

| Zone | Description |
|------|-------------|
| Volet gauche | Liste des requêtes |
| Barre formule | Code M |
| Apercu | Données transformees |
| Volet droit | Étapes appliquees |

## Transformations courantes

### Supprimer des colonnes

1. Sélectionnez les colonnes
2. Clic droit > **Supprimer les colonnes**

### Renommer une colonne

Double-cliquez sûr l'en-tête.

### Changer le type de données

1. Cliquez sûr l'icone de type
2. Sélectionnez le nouveau type

### Filtrer les lignes

1. Cliquez sûr la fleche de la colonne
2. Decochez les valeurs à exclure

### Remplacer des valeurs

1. Sélectionnez la colonne
2. **Transformer** > **Remplacer les valeurs**

## Étapes appliquees

### Historique des transformations

Chaque action créé une étape visible dans le volet droit.

### Modifier une étape

Double-cliquez sûr l'étape pour la modifier.

### Supprimer une étape

Cliquez sûr le X à côté de l'étape.

## Fusionner des requêtes

### Jointure de tables

1. **Accueil** > **Combiner** > **Fusionner des requêtes**
2. Sélectionnez les colonnes clés
3. Choisissez le type de jointure

### Types de jointure

| Type | Description |
|------|-------------|
| Externe gauche | Toutes les lignes de gauche |
| Externe droite | Toutes les lignes de droite |
| Externe complete | Toutes les lignes |
| Interne | Correspondances uniquement |

## Ajouter des requêtes

### Empiler des tables

1. **Accueil** > **Combiner** > **Ajouter des requêtes**
2. Sélectionnez les requêtes à empiler

## Colonnes personnalisees

### Créer une colonne

1. **Ajouter une colonne** > **Colonne personnalisee**
2. Entrez la formule M

### Exemple

```m
[Prix] * [Quantité]
```

## Charger les données

### Options de chargement

| Destination | Description |
|-------------|-------------|
| Tableau | Nouvelle feuille avec tableau |
| Rapport de TCD | Tableau croisé dynamique |
| Connexion uniquement | Pas de chargement |

### Actualiser

**Données** > **Actualiser tout**

## Bonnes pratiques

1. **Nommez** vos requêtes clairement
2. **Documentez** les étapes complexes
3. **Testez** sûr un echantillon d'abord
4. **Planifiez** l'actualisation

## Avantages de Power Query

- Transformations repetables
- Actualisation automatique
- Pas de VBA nécessaire
- Gestion des erreurs
