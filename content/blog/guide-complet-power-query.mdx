# Guide Complet Power Query

Power Query est l'outil de transformation de donnees integre a Excel.

## Qu'est-ce que Power Query ?

### Definition

Power Query permet de :
- Importer des donnees de multiples sources
- Transformer et nettoyer les donnees
- Automatiser les processus

### Acces

**Donnees** > **Obtenir des donnees**

## Sources de donnees

### Fichiers

- Excel
- CSV/Texte
- XML/JSON
- PDF

### Bases de donnees

- SQL Server
- Access
- Oracle
- MySQL

### Web et Cloud

- Pages web
- SharePoint
- Azure
- API

## Premiere requete

### Import d'un fichier Excel

1. **Donnees** > **Obtenir des donnees** > **A partir d'un fichier**
2. Selectionnez votre fichier
3. Choisissez la feuille/table
4. **Transformer les donnees**

## L'editeur Power Query

### Interface

| Zone | Description |
|------|-------------|
| Volet gauche | Liste des requetes |
| Barre formule | Code M |
| Apercu | Donnees transformees |
| Volet droit | Etapes appliquees |

## Transformations courantes

### Supprimer des colonnes

1. Selectionnez les colonnes
2. Clic droit > **Supprimer les colonnes**

### Renommer une colonne

Double-cliquez sur l'en-tete.

### Changer le type de donnees

1. Cliquez sur l'icone de type
2. Selectionnez le nouveau type

### Filtrer les lignes

1. Cliquez sur la fleche de la colonne
2. Decochez les valeurs a exclure

### Remplacer des valeurs

1. Selectionnez la colonne
2. **Transformer** > **Remplacer les valeurs**

## Etapes appliquees

### Historique des transformations

Chaque action cree une etape visible dans le volet droit.

### Modifier une etape

Double-cliquez sur l'etape pour la modifier.

### Supprimer une etape

Cliquez sur le X a cote de l'etape.

## Fusionner des requetes

### Jointure de tables

1. **Accueil** > **Combiner** > **Fusionner des requetes**
2. Selectionnez les colonnes cles
3. Choisissez le type de jointure

### Types de jointure

| Type | Description |
|------|-------------|
| Externe gauche | Toutes les lignes de gauche |
| Externe droite | Toutes les lignes de droite |
| Externe complete | Toutes les lignes |
| Interne | Correspondances uniquement |

## Ajouter des requetes

### Empiler des tables

1. **Accueil** > **Combiner** > **Ajouter des requetes**
2. Selectionnez les requetes a empiler

## Colonnes personnalisees

### Creer une colonne

1. **Ajouter une colonne** > **Colonne personnalisee**
2. Entrez la formule M

### Exemple

```m
[Prix] * [Quantite]
```

## Charger les donnees

### Options de chargement

| Destination | Description |
|-------------|-------------|
| Tableau | Nouvelle feuille avec tableau |
| Rapport de TCD | Tableau croise dynamique |
| Connexion uniquement | Pas de chargement |

### Actualiser

**Donnees** > **Actualiser tout**

## Bonnes pratiques

1. **Nommez** vos requetes clairement
2. **Documentez** les etapes complexes
3. **Testez** sur un echantillon d'abord
4. **Planifiez** l'actualisation

## Avantages de Power Query

- Transformations repetables
- Actualisation automatique
- Pas de VBA necessaire
- Gestion des erreurs
