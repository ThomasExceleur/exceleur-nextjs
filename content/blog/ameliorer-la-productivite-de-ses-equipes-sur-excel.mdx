# Ameliorer la Productivite sur Excel

Des strategies concretes pour optimiser le travail de votre equipe.

## Former aux raccourcis essentiels

### Les 10 raccourcis incontournables

| Raccourci | Action |
|-----------|--------|
| Ctrl + C | Copier |
| Ctrl + V | Coller |
| Ctrl + Z | Annuler |
| Ctrl + S | Sauvegarder |
| Ctrl + Fleche | Navigation rapide |
| Ctrl + Shift + Fleche | Selection rapide |
| F2 | Editer une cellule |
| F4 | Repeter / Reference absolue |
| Ctrl + ; | Date du jour |
| Alt + Entree | Retour a la ligne |

## Standardiser les fichiers

### Modeles communs

1. Creez des modeles pour les taches repetitives
2. Definissez des conventions de nommage
3. Etablissez une structure commune

### Exemple de convention

- `YYYY-MM_Nom-Projet_Type.xlsx`
- `2024-07_Ventes-Q3_Rapport.xlsx`

## Utiliser les tableaux structures

### Avantages

- Formules qui se propagent automatiquement
- Filtres et tris integres
- References lisibles (`Tableau[Colonne]`)

### Creation

1. Selectionnez vos donnees
2. **Ctrl + T** ou **Insertion** > **Tableau**

## Automatiser les taches repetitives

### Macros simples

Enregistrez les taches frequentes avec l'enregistreur de macros.

### Power Query

Automatisez l'import et le nettoyage des donnees.

## Partage et collaboration

### Bonnes pratiques

1. **Utilisez** OneDrive ou SharePoint
2. **Activez** la co-edition
3. **Commentez** les formules complexes
4. **Protegez** les cellules critiques
