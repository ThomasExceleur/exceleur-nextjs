# Excel en Word : Guide d'Integration

Inserer des tableaux et graphiques Excel dans Word est essentiel pour creer des rapports professionnels.

## Methode 1 : Copier-Coller Simple

### Etapes

1. Dans Excel, selectionnez les cellules
2. **Ctrl + C** pour copier
3. Dans Word, **Ctrl + V** pour coller

### Resultat

Le tableau est converti en tableau Word. Les donnees sont statiques (non liees a Excel).

### Avantages

- Simple et rapide
- Tableau modifiable dans Word
- Pas besoin du fichier Excel

### Inconvenients

- Pas de mise a jour automatique
- Perte de certains formats

## Methode 2 : Coller avec Liaison

### Etapes

1. Copiez dans Excel (**Ctrl + C**)
2. Dans Word : **Accueil** > **Coller** > **Collage special**
3. Selectionnez **Coller avec liaison**
4. Choisissez le format

### Options de format

- **Objet Feuille de calcul Microsoft Excel** : Conserve l'apparence Excel
- **Texte mis en forme (RTF)** : Tableau Word avec mise en forme
- **Image** : Image statique du tableau

### Mise a jour

- La liaison garde le lien avec le fichier Excel
- Modifications dans Excel = mise a jour dans Word
- Clic droit sur le tableau > **Mettre a jour la liaison**

## Methode 3 : Incorporation

### Etapes

1. Copiez dans Excel
2. Dans Word : **Collage special** > **Coller** (pas "avec liaison")
3. Selectionnez **Objet Feuille de calcul Microsoft Excel**

### Resultat

- Le tableau Excel est incorpore dans Word
- Double-clic pour modifier avec l'interface Excel
- Les donnees sont dans le document Word

### Avantages

- Pas besoin du fichier Excel original
- Modification complete possible
- Toutes les fonctionnalites Excel disponibles

## Methode 4 : Inserer un Objet

### Etapes

1. Dans Word : **Insertion** > **Objet**
2. Onglet **Creer a partir d'un fichier**
3. Parcourir et selectionner le fichier Excel
4. Cochez **Liaison** si vous voulez un lien

### Options

- **Liaison** : Mise a jour automatique
- **Afficher sous forme d'icone** : Affiche une icone cliquable

## Inserer un Graphique Excel

### Copier-Coller

1. Cliquez sur le graphique dans Excel
2. **Ctrl + C**
3. Dans Word, **Ctrl + V**

### Options de collage

Apres le collage, cliquez sur l'icone d'options :

- **Graphique (lie aux donnees Excel)** : Mise a jour auto
- **Graphique Excel (classeur entier)** : Incorpore
- **Coller en tant qu'image** : Image statique

## Mise en forme dans Word

### Redimensionner

1. Cliquez sur le tableau/graphique
2. Utilisez les poignees de coin
3. Maintenez **Maj** pour conserver les proportions

### Position

1. Clic droit > **Options de disposition**
2. Choisissez l'habillage du texte

### Bordures

Pour les tableaux colles en format Word :
- Selectionnez le tableau
- **Conception de tableau** > **Bordures**

## Problemes courants

### Tableau trop large

1. Selectionnez le tableau
2. **Disposition** > **Ajustement automatique** > **Contenu**

Ou reduisez le nombre de colonnes dans Excel avant de copier.

### Mise a jour ne fonctionne pas

1. Verifiez que le fichier Excel existe encore
2. Verifiez le chemin d'acces
3. Clic droit > **Mettre a jour la liaison**

### Perte de mise en forme

Utilisez l'option "Objet Feuille de calcul" au lieu du collage simple.

## Bonnes pratiques

### Pour les rapports reguliers

- Utilisez la liaison pour la mise a jour automatique
- Gardez Excel et Word dans le meme dossier
- Nommez clairement vos fichiers

### Pour les documents a partager

- Utilisez l'incorporation (pas besoin d'Excel)
- Ou convertissez en image pour un rendu fixe

### Pour les grandes donnees

- Selectionnez uniquement les donnees necessaires
- Evitez de copier des plages trop larges

## Comparatif des methodes

| Methode | Mise a jour | Modifiable | Fichier Excel requis |
|---------|-------------|------------|---------------------|
| Copier-coller simple | Non | Oui (Word) | Non |
| Avec liaison | Oui | Non | Oui |
| Incorporation | Non | Oui (Excel) | Non |
| Image | Non | Non | Non |
