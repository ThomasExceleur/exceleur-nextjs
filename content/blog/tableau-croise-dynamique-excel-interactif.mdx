# Tableau Croise Dynamique Interactif

Rendez vos TCD interactifs avec des segments et chronologies.

## Creer un TCD de base

### Etapes

1. Selectionnez vos donnees
2. **Insertion** > **Tableau croise dynamique**
3. Choisissez l'emplacement
4. Faites glisser les champs

## Ajouter des segments

### Definition

Les segments sont des boutons visuels pour filtrer le TCD.

### Creation

1. Cliquez dans le TCD
2. **Analyse** > **Inserer un segment**
3. Selectionnez les champs
4. **OK**

### Utilisation

Cliquez sur les valeurs pour filtrer. Multi-selection avec Ctrl.

## Ajouter une chronologie

### Pour les dates

1. Cliquez dans le TCD
2. **Analyse** > **Inserer une chronologie**
3. Selectionnez le champ date
4. **OK**

### Navigation

- Faites glisser pour selectionner une periode
- Changez le niveau (annee, trimestre, mois, jour)

## Connecter plusieurs TCD

### Un segment, plusieurs TCD

1. Clic droit sur le segment
2. **Connexions de rapport**
3. Cochez les TCD a connecter
4. **OK**

### Resultat

Filtrer un segment filtre tous les TCD connectes.

## Mise en forme

### Style du segment

1. Selectionnez le segment
2. **Outils Segment** > **Options**
3. Choisissez un style

### Colonnes du segment

1. Clic droit > **Parametres du segment**
2. Modifiez le nombre de colonnes

## Graphique croise dynamique

### Creation

1. Selectionnez le TCD
2. **Insertion** > **Graphique croise dynamique**
3. Choisissez le type

### Interaction

Le graphique se filtre avec les memes segments que le TCD.

## Tableau de bord complet

### Elements

- TCD de synthese
- Graphique croise dynamique
- Segments pour les filtres cles
- Chronologie pour les dates

### Disposition

| Zone | Contenu |
|------|---------|
| Haut | Segments et chronologie |
| Gauche | TCD resume |
| Droite | Graphique |
| Bas | TCD detail |

## Bonnes pratiques

### Segments

- Limitez a 3-5 segments
- Utilisez des noms clairs
- Positionnez-les logiquement

### TCD

- Groupez les dates si nombreuses
- Utilisez des pourcentages pour comparaison
- Ajoutez des totaux generaux

## Actualisation

### Manuelle

Clic droit > **Actualiser**

### Automatique

**Donnees** > **Proprietes** > **Actualiser lors de l'ouverture**

## Erreurs courantes

### Segment deconnecte

Reconnectez via **Connexions de rapport**.

### Chronologie grisee

Le champ doit etre au format date.
