# 6 Astuces sur les Onglets Excel

Gerez efficacement vos feuilles de calcul.

## 1. Renommer rapidement

Double-cliquez sur l'onglet pour le renommer.

## 2. Changer la couleur

1. Clic droit sur l'onglet
2. **Couleur d'onglet**
3. Choisissez une couleur

### Organisation par couleur

| Couleur | Usage suggere |
|---------|---------------|
| Vert | Donnees validees |
| Rouge | A verifier |
| Bleu | Parametres |
| Jaune | En cours |

## 3. Deplacer un onglet

### Dans le meme classeur

Glissez l'onglet vers sa nouvelle position.

### Vers un autre classeur

1. Clic droit > **Deplacer ou copier**
2. Selectionnez le classeur destination

## 4. Copier un onglet

1. **Ctrl + Glisser** l'onglet
2. Ou clic droit > **Deplacer ou copier** > **Creer une copie**

## 5. Masquer un onglet

1. Clic droit sur l'onglet
2. **Masquer**

### Afficher

1. Clic droit sur un onglet visible
2. **Afficher**
3. Selectionnez l'onglet

## 6. Naviguer rapidement

### Raccourcis

| Raccourci | Action |
|-----------|--------|
| Ctrl+Page Up | Onglet precedent |
| Ctrl+Page Down | Onglet suivant |
| Clic droit sur fleches | Liste des onglets |

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Noms explicites | Facilitez la comprehension |
| Ordre logique | Organisez par workflow |
| Couleurs coherentes | Code couleur standard |
