# Fonction FILTRE : Extraire des Donnees Dynamiquement

FILTRE est l'une des fonctions les plus puissantes d'Excel 365.

## Presentation de FILTRE

### Syntaxe

```excel
=FILTRE(tableau;inclure;[si_vide])
```

### Parametres

| Parametre | Description | Obligatoire |
|-----------|-------------|-------------|
| tableau | Donnees a filtrer | Oui |
| inclure | Condition booleenne | Oui |
| si_vide | Valeur si aucun resultat | Non |

### Resultat

Un tableau dynamique qui s'etend automatiquement.

## Exemples de base

### Filtrer par texte

```excel
=FILTRE(A2:C10;A2:A10="Paris")
```

### Filtrer par nombre

```excel
=FILTRE(A2:C10;B2:B10>1000)
```

### Filtrer par date

```excel
=FILTRE(A2:C10;C2:C10>=DATE(2024;1;1))
```

## Criteres multiples

### ET logique (toutes les conditions)

```excel
=FILTRE(A2:D10;(B2:B10="Paris")*(C2:C10>100))
```

La multiplication (*) agit comme ET.

### OU logique (au moins une condition)

```excel
=FILTRE(A2:D10;(B2:B10="Paris")+(B2:B10="Lyon"))
```

L'addition (+) agit comme OU.

### Combinaison ET/OU

```excel
=FILTRE(A2:D10;((B2:B10="Paris")+(B2:B10="Lyon"))*(C2:C10>100))
```

## Gestion du resultat vide

### Probleme

Sans le parametre si_vide, FILTRE retourne #CALC! si aucune ligne ne correspond.

### Solutions

```excel
' Message texte
=FILTRE(A2:C10;B2:B10="Inexistant";"Aucun resultat")

' Cellule vide
=FILTRE(A2:C10;B2:B10="Inexistant";"")

' Valeur specifique
=FILTRE(A2:C10;B2:B10="Inexistant";0)
```

## Filtrage dynamique

### Avec reference cellule

```excel
=FILTRE(Donnees;Donnees[Ville]=E1)
```

E1 contient la ville choisie par l'utilisateur.

### Avec liste deroulante

1. Creez une validation de donnees en E1
2. La formule FILTRE s'adapte automatiquement

## Combinaisons puissantes

### FILTRE + TRIER

```excel
=TRIER(FILTRE(A2:C10;B2:B10>100);2;-1)
```

Filtre puis trie par la 2e colonne decroissante.

### FILTRE + UNIQUE

```excel
=UNIQUE(FILTRE(A2:A10;B2:B10>100))
```

Valeurs uniques parmi les resultats filtres.

### FILTRE + INDEX

```excel
=INDEX(FILTRE(A2:C10;B2:B10>100);1;2)
```

Premiere valeur de la 2e colonne du resultat.

### FILTRE + LIGNES

```excel
=LIGNES(FILTRE(A2:A10;B2:B10>100))
```

Nombre de resultats.

## Reference debordante

### Syntaxe #

```excel
=SOMME(D1#)
```

Reference tout le resultat de la formule en D1.

### Utilisation

Utile pour calculer des statistiques sur le resultat filtre.

## Filtrage de colonnes

### Selectionner des colonnes

```excel
=FILTRE(CHOISIRCOLS(Donnees;1;3;5);Donnees[Statut]="Actif")
```

## Cas pratiques

### Commandes du jour

```excel
=FILTRE(Commandes;Commandes[Date]=AUJOURDHUI();"Aucune commande")
```

### Clients VIP

```excel
=FILTRE(Clients;Clients[CA]>=10000)
```

### Produits en rupture

```excel
=FILTRE(Produits;Produits[Stock]=0;"Tous les produits disponibles")
```

### Employes d'un departement

```excel
=FILTRE(Employes;Employes[Dept]=INDIRECT("E1"))
```

## Erreurs courantes

| Erreur | Cause | Solution |
|--------|-------|----------|
| #CALC! | Aucun resultat | Ajouter si_vide |
| #VALEUR! | Tailles differentes | Verifier les plages |
| #DEBORDEMENT | Cellules occupees | Liberer l'espace |

## Performance

### Optimisation

1. Limitez la taille du tableau source
2. Evitez les colonnes entieres (A:A)
3. Utilisez des tableaux structures

### Grandes donnees

Pour des millions de lignes, preferez Power Query.

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| si_vide | Toujours le definir |
| Noms | Utilisez des plages nommees |
| Espace | Prevoyez le debordement |
| Tests | Validez les criteres |
