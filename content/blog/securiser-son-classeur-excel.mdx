# Securiser son Classeur Excel

Protégez vos données et votre travail.

## Niveaux de protection

| Niveau | Description |
|--------|-------------|
| Chiffrement | Mot de passe ouverture |
| Structuré | Empeche modification onglets |
| Feuilles | Empeche modification contenu |
| Cellules | Contrôle fin |

## Chiffrer le fichier

1. **Fichier** > **Informations**
2. **Proteger** > **Chiffrer avec mot de passe**

## Proteger la structuré

1. **Révision** > **Proteger le classeur**
2. Cochez **Structuré**

## Proteger les feuilles

1. **Révision** > **Proteger la feuille**
2. Configurez les options

## Cellules editables

1. Sélectionnez les cellules
2. Format > Protection > Decochez Verrouillee
3. Protégez la feuille

## Masquer les formules

1. Sélectionnez les cellules
2. Format > Protection > Cochez Masquee
3. Protégez la feuille

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Mot de passe fort | 12+ caracteres varies |
| Documentation | Notez les protections |
| Test | Vérifiez les permissions |
| Sauvegarde | Copie non protegee |
