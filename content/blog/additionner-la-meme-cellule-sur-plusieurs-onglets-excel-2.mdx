# Additionner sur Plusieurs Onglets

Sommez la meme cellule a travers plusieurs feuilles.

## Reference 3D

### Syntaxe

```excel
=SOMME(Feuil1:Feuil12!A1)
```

### Explication

- **Feuil1:Feuil12** : Plage de feuilles
- **!A1** : Cellule a sommer

## Exemple pratique

### Scenario

12 feuilles mensuelles avec le CA en B10.

### Formule

```excel
=SOMME(Janvier:Decembre!B10)
```

## Conditions d'utilisation

| Condition | Importance |
|-----------|------------|
| Noms contigus | Feuilles consecutives |
| Meme structure | Cellules identiques |
| Ordre | Premiere et derniere feuille |

## Autres fonctions 3D

### MOYENNE

```excel
=MOYENNE(Feuil1:Feuil12!A1)
```

### MAX / MIN

```excel
=MAX(Feuil1:Feuil12!A1)
=MIN(Feuil1:Feuil12!A1)
```

### NB

```excel
=NB(Feuil1:Feuil12!A1)
```

## Attention

- Si vous ajoutez une feuille entre les bornes, elle est incluse
- Les noms de feuilles avec espaces necessitent des apostrophes

```excel
=SOMME('Jan 2024:Dec 2024'!B10)
```
