# Nettoyer son Tableau Excel

Préparez vos données pour l'analyse.

## Problèmes courants

| Problème | Solution |
|----------|----------|
| Espaces en trop | SUPPRESPACE |
| Lignes vides | Supprimer |
| Doublons | Supprimer les doublons |
| Formats inconsistants | Convertir |

## Supprimer les espaces

### SUPPRESPACE

```excel
=SUPPRESPACE(A1)
```

Supprime les espaces superflus.

### EPURAGE

```excel
=EPURAGE(A1)
```

Supprime les caracteres non imprimables.

## Supprimer les doublons

### Méthode

1. Sélectionnez vos données
2. **Données** > **Supprimer les doublons**
3. Choisissez les colonnes

## Convertir le texte

### Majuscules/Minuscules

| Fonction | Résultat |
|----------|----------|
| `MAJUSCULE(A1)` | TOUT EN MAJUSCULES |
| `MINUSCULE(A1)` | tout en minuscules |
| `NOMPROPRE(A1)` | Première Lettre Majuscule |

## Remplacer

### Rechercher et remplacer

**Ctrl+H**

- Remplacez les erreurs courantes
- Uniformisez les formats
