# Afficher la Date et l'Heure

Inserez des dates et heures automatiques ou statiques.

## Date du jour

### Dynamique

```excel
=AUJOURDHUI()
```

Se met a jour automatiquement.

### Statique

**Ctrl + ;** (point-virgule)

## Heure actuelle

### Dynamique

```excel
=MAINTENANT()
```

Date et heure actuelles.

### Statique

**Ctrl + Maj + ;**

## Formats personnalises

### Formats de date

| Format | Resultat |
|--------|----------|
| `jj/mm/aaaa` | 14/01/2024 |
| `jjjj j mmmm aaaa` | Lundi 14 janvier 2024 |
| `jj-mmm-aa` | 14-Jan-24 |

### Formats d'heure

| Format | Resultat |
|--------|----------|
| `hh:mm` | 14:30 |
| `hh:mm:ss` | 14:30:45 |
| `h "h" mm` | 14 h 30 |

## Appliquer un format

1. Selectionnez les cellules
2. **Ctrl + 1** (Format de cellule)
3. **Personnalise**
4. Entrez le format

## Calculs avec dates

### Difference de jours

```excel
=B1-A1
```

### Ajouter des jours

```excel
=A1+30
```

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Coherence | Meme format partout |
| Statique vs Dynamique | Choisissez selon le besoin |
| Validation | Verifiez les formats regionaux |
