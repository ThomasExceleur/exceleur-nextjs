# Excel qui ne Calcule Pas : Solutions

Quand Excel n'effectue pas les calculs, plusieurs causes sont possibles. Voici comment resoudre ce probleme.

## Cause 1 : Mode de calcul manuel

### Symptome

Les formules ne se mettent pas a jour quand vous modifiez les donnees.

### Verification

1. Onglet **Formules**
2. Regardez **Options de calcul**
3. Verifiez si "Manuel" est selectionne

### Solution

1. **Formules** > **Options de calcul**
2. Selectionnez **Automatique**

### Raccourci

- **F9** : Recalculer une fois
- **Ctrl + Alt + F9** : Forcer le recalcul complet

## Cause 2 : Format texte

### Symptome

La formule s'affiche au lieu du resultat.

### Verification

La cellule affiche "=A1+B1" au lieu du resultat.

### Solution

1. Selectionnez la cellule
2. **Accueil** > **Format** > **Standard**
3. Appuyez sur F2 puis Entree pour revalider

### Astuce

```excel
=CNUM(A1) + CNUM(B1)
```

Force la conversion en nombre.

## Cause 3 : Affichage des formules

### Symptome

Toutes les formules sont visibles au lieu des resultats.

### Verification

Vous voyez les formules dans toutes les cellules.

### Solution

Appuyez sur **Ctrl + `** (accent grave)

Ou : **Formules** > **Afficher les formules**

## Cause 4 : Cellule en mode edition

### Symptome

Une seule cellule ne calcule pas.

### Verification

Le curseur clignote dans la cellule.

### Solution

Appuyez sur **Entree** ou **Echap**.

## Cause 5 : Reference circulaire

### Symptome

Message d'avertissement sur la reference circulaire.

### Verification

**Formules** > **Verification des erreurs** > **References circulaires**

### Solution

Corrigez la formule qui se reference elle-meme.

## Cause 6 : Erreur dans la formule

### Symptome

La cellule affiche #VALEUR!, #REF!, etc.

### Verification

Regardez le message d'erreur affiche.

### Solution

Corrigez la formule selon le type d'erreur.

## Cause 7 : Nombres stockes comme texte

### Symptome

Les additions donnent des concatenations ou des erreurs.

### Indicateur

Triangle vert dans le coin de la cellule.

### Solutions

#### Option 1 : Corriger manuellement

1. Selectionnez les cellules
2. Cliquez sur l'icone d'avertissement
3. **Convertir en nombre**

#### Option 2 : Multiplication par 1

Dans une colonne vide :
```excel
=A1*1
```

Copie les valeurs en nombres.

#### Option 3 : Collage special

1. Tapez 1 dans une cellule vide
2. Copiez cette cellule
3. Selectionnez les nombres en texte
4. **Collage special** > **Multiplier**

## Cause 8 : Protection de feuille

### Symptome

Impossible de modifier les cellules.

### Verification

Un message indique que la feuille est protegee.

### Solution

**Revision** > **Oter la protection de la feuille**

Mot de passe requis si defini.

## Cause 9 : Macro en erreur

### Symptome

Les calculs s'arretent apres l'execution d'une macro.

### Solution

1. Fermez Excel
2. Rouvrez le fichier
3. Verifiez le code VBA

## Diagnostic rapide

| Symptome | Cause probable | Solution |
|----------|----------------|----------|
| Aucun calcul | Mode manuel | Automatique |
| Formule visible | Format texte | Standard + F2 |
| Toutes formules | Affichage formules | Ctrl + ` |
| Triangle vert | Texte au lieu nombre | Convertir |
| Message erreur | Formule incorrecte | Corriger |

## Parametres de calcul

### Acces

**Fichier** > **Options** > **Formules**

### Options

- **Automatique** : Calcul a chaque modification
- **Automatique sauf tables** : Ignore les tables de donnees
- **Manuel** : Calcul uniquement sur demande

### Recommandation

Laissez en **Automatique** sauf pour les tres gros fichiers.

## Prevention

### Bonnes pratiques

1. Verifiez le mode de calcul a l'ouverture
2. Formatez les cellules avant saisie
3. Evitez de coller du texte dans des zones numeriques
4. Testez vos formules sur des exemples

### Verification automatique

```excel
=TYPE(A1)
```

- 1 = Nombre
- 2 = Texte
