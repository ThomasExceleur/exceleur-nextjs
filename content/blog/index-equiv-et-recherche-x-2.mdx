# INDEX+EQUIV vs RECHERCHEX

Deux approches modernes pour les recherches.

## INDEX+EQUIV

### Syntaxe

```excel
=INDEX(plage_resultat;EQUIV(valeur;plage_recherche;0))
```

### Avantages

- Compatible toutes versions
- Tres flexible
- Deux fonctions = deux usages

## RECHERCHEX

### Syntaxe

```excel
=RECHERCHEX(valeur;plage_recherche;plage_resultat)
```

### Avantages

- Plus lisible
- Gestion erreur integree
- Mode de recherche avance

## Comparaison

| Critere | INDEX+EQUIV | RECHERCHEX |
|---------|-------------|------------|
| Disponibilite | Toutes versions | Excel 365+ |
| Lisibilite | Moyenne | Excellente |
| Performance | Tres bonne | Excellente |
| Gestion erreur | SIERREUR | Integree |
| Recherche inverse | Oui | Oui |

## Exemples equivalents

### INDEX+EQUIV

```excel
=INDEX(B:B;EQUIV("Paris";A:A;0))
```

### RECHERCHEX

```excel
=RECHERCHEX("Paris";A:A;B:B)
```

## Gestion des erreurs

### INDEX+EQUIV

```excel
=SIERREUR(INDEX(B:B;EQUIV("Paris";A:A;0));"Non trouve")
```

### RECHERCHEX

```excel
=RECHERCHEX("Paris";A:A;B:B;"Non trouve")
```

## Modes de recherche RECHERCHEX

| Mode | Description |
|------|-------------|
| 1 | Du debut (defaut) |
| -1 | De la fin |
| 2 | Binaire croissant |
| -2 | Binaire decroissant |

## Quand choisir quoi ?

| Situation | Recommandation |
|-----------|----------------|
| Excel 365 | RECHERCHEX |
| Compatibilite requise | INDEX+EQUIV |
| Recherche complexe | INDEX+EQUIV |
