# Quel Excel est Gratuit

Options pour utiliser Excel sans payer.

## Excel pour le Web

### Accès

1. Allez sûr office.com
2. Connectez-vous (compte Microsoft gratuit)
3. Utilisez Excel

### Fonctionnalités

| Inclus | Limite |
|--------|--------|
| Formules | Pas de macros |
| Graphiques | Fonctions réduites |
| Partage | Stockage limite |

## Excel Mobile

### Gratuit sûr

- Smartphones
- Tablettes < 10.1 pouces

### Apps

- iOS : App Store
- Android : Play Store

## Alternatives gratuites

### Google Sheets

- 100% gratuit
- Collaboratif
- Compatible Excel

### LibreOffice Calc

- Open source
- Hors ligne
- Fonctions completes

## Pour etudiants

### Office 365 Education

Gratuit avec email etudiant.

## Essai gratuit

### Microsoft 365

1 mois d'essai gratuit.
