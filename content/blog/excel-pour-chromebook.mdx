# Excel pour Chromebook

Les Chromebooks fonctionnent avec Chrome OS et ne supportent pas les applications Windows. Voici vos options pour Excel.

## Option 1 : Excel Online (Gratuit)

### Acces

1. Ouvrez Chrome
2. Allez sur **office.com**
3. Connectez-vous avec un compte Microsoft
4. Cliquez sur Excel

### Avantages

- Gratuit
- Pas d'installation
- Sauvegarde automatique sur OneDrive
- Fonctionne parfaitement

### Limites

- Necessite internet
- Fonctionnalites reduites par rapport au desktop
- Pas de macros VBA

## Option 2 : Application Android Excel

### Installation

1. Ouvrez le Google Play Store
2. Recherchez "Microsoft Excel"
3. Installez l'application

### Configuration requise

Votre Chromebook doit supporter les applications Android (la plupart des modeles recents).

### Fonctionnalites

- Interface tactile optimisee
- Mode hors-ligne
- Compatibilite fichiers .xlsx

### Limites

- Interface mobile
- Fonctions avancees limitees
- Ecran parfois petit

## Option 3 : Google Sheets (Gratuit)

### Avantages

- Natif sur Chrome OS
- Gratuit illimite
- Excellente collaboration
- Import/export Excel

### Differences avec Excel

| Aspect | Excel | Google Sheets |
|--------|-------|---------------|
| Formules | Plus nombreuses | Essentiel couvert |
| Macros | VBA | Apps Script |
| Hors-ligne | Oui | Limite |
| Collaboration | Bonne | Excellente |

### Importer un fichier Excel

1. Ouvrez Google Drive
2. Glissez-deposez le fichier .xlsx
3. Ouvrez avec Google Sheets

## Option 4 : Microsoft 365 Web

### Abonnement

Avec un abonnement Microsoft 365, vous accedez a Excel Online avec plus de fonctionnalites.

### Avantages

- 1 To de stockage OneDrive
- Fonctionnalites Premium
- Support technique

## Comparaison des options

| Option | Prix | Hors-ligne | Fonctionnalites |
|--------|------|------------|-----------------|
| Excel Online | Gratuit | Non | Basique |
| App Android | Gratuit | Oui | Moyen |
| Google Sheets | Gratuit | Limite | Bon |
| Microsoft 365 | Payant | Limite | Complet |

## Travailler hors-ligne

### Excel Android

L'application permet de travailler hors-ligne. Les fichiers se synchronisent a la reconnexion.

### Google Sheets

1. Ouvrez un fichier
2. **Fichier** > **Disponible hors connexion**
3. Le fichier est accessible sans internet

## Conseils pratiques

### Synchronisation

- Utilisez OneDrive pour Excel
- Utilisez Google Drive pour Sheets
- Les fichiers restent accessibles partout

### Raccourcis clavier

La plupart des raccourcis Excel fonctionnent dans Excel Online :

- Ctrl + C : Copier
- Ctrl + V : Coller
- Ctrl + Z : Annuler

### Performance

- Fermez les onglets inutiles
- Evitez les fichiers trop lourds
- Connexion internet stable recommandee

## Limites du Chromebook

### Ce qui ne fonctionne pas

- Excel Desktop (Windows/Mac)
- Macros VBA
- Certains complements
- Power Query/Power Pivot

### Alternatives

Pour les fonctions avancees, envisagez :
- Un ordinateur Windows/Mac
- Une machine virtuelle
- Un service de bureau a distance

## Choisir la bonne solution

### Pour un usage basique

Excel Online ou Google Sheets suffisent.

### Pour un usage regulier

Application Android Excel + OneDrive.

### Pour un usage professionnel intensif

Considerez un autre type d'ordinateur ou des solutions cloud avancees.

## FAQ

### Puis-je ouvrir mes fichiers Excel ?

Oui, avec toutes les options mentionnees.

### Les formules fonctionnent-elles ?

La plupart oui. Quelques formules avancees peuvent differer.

### Puis-je collaborer ?

Oui, Excel Online et Google Sheets offrent d'excellentes fonctionnalites de collaboration.
