# Tableau de Bord Interactif : Techniques Avancées

Elevez vos dashboards au niveau professionnel.

## Architecture d'un dashboard pro

### Structuré recommandée

| Onglet | Rôle |
|--------|------|
| DATA | Données brutes |
| CALCULS | Formules intermediaires |
| PARAMETRES | Variables de configuration |
| DASHBOARD | Affichage uniquement |

## Indicateurs KPI dynamiques

### Formule type

```excel
=LET(
  période;$E$1;
  valeur;SOMME.SI(Dates;">="&période;Montants);
  objectif;RECHERCHEV(période;Objectifs;2;FAUX);
  ecart;(valeur-objectif)/objectif;
  CONCAT(TEXTE(valeur;"# ##0 €");" (";TEXTE(ecart;"+0%;-0%");")")
)
```

### Mise en forme conditionnelle

Coloration automatique selon la performance.

## Graphiques dynamiques

### Plages dynamiques

Utilisez des tableaux structurés où DECALER().

### Source de données

```excel
=DECALER($A$1;0;0;NB.SI.ENS($A:$A;"<>""");1)
```

Listes deroulantes controlant les graphiques.

## Graphiques sparklines

### Insertion

1. Sélectionnez les cellules cibles
2. **Insertion** > **Sparklines**
3. Choisissez le type

### Configuration

- Couleur des points hauts/bas
- Axe et échelle

## Interactivite avec RECHERCHEX

### KPI qui change selon sélection

```excel
=RECHERCHEX($E$1;Périodes;Ventes)
```

E1 contient la période sélectionnée.

## Jauges personnalisees

### Avec graphique en anneau

1. Créez un graphique en anneau
2. 3 valeurs : performance, reste, invisible
3. Colorez la portion "invisible" en blanc

### Formule

```excel
Performance : =MIN(100;Valeur/Objectif*100)
Reste : =MAX(0;100-Performance)
Invisible : =100
```

## Design professionnel

### Règles

| Élément | Recommandation |
|---------|----------------|
| Couleurs | 3-4 max, cohérentes |
| Polices | 1-2 familles |
| Espacement | Genereux et equilibre |
| Alignement | Sur une grille invisible |

### Masquer les éléments

- Masquez le quadrillage
- Masquez les en-têtes
- Figez les volets

## Performance

### Optimisation

- Limitez les formules volatiles
- Utilisez des tableaux structurés
- Calculez une fois, affichez plusieurs fois
