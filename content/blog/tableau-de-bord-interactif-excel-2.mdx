# Tableau de Bord Interactif : Techniques Avancees

Elevez vos dashboards au niveau professionnel.

## Architecture d'un dashboard pro

### Structure recommandee

| Onglet | Role |
|--------|------|
| DATA | Donnees brutes |
| CALCULS | Formules intermediaires |
| PARAMETRES | Variables de configuration |
| DASHBOARD | Affichage uniquement |

## Graphiques dynamiques

### Plages dynamiques

Utilisez des tableaux structures ou DECALER().

### Source de donnees

```excel
=DECALER($A$1;0;0;NB.SI.ENS($A:$A;"<>""");1)
```

1. **Insertion** > **Formes**
2. Ajoutez un lien hypertexte vers l'onglet cible

Listes deroulantes controlant les graphiques.

## Graphiques sparklines

### Insertion

1. Selectionnez les cellules cibles
2. **Insertion** > **Sparklines**
3. Choisissez le type

### Configuration

- Couleur des points hauts/bas
- Axe et echelle

## Interactivite avec RECHERCHEX

### KPI qui change selon selection

```excel
=RECHERCHEX($E$1;Periodes;Ventes)
```

E1 contient la periode selectionnee.

## Jauges personnalisees

### Avec graphique en anneau

1. Creez un graphique en anneau
2. 3 valeurs : performance, reste, invisible
3. Colorez la portion "invisible" en blanc

### Formule

```excel
Performance : =MIN(100;Valeur/Objectif*100)
Reste : =MAX(0;100-Performance)
Invisible : =100
```

## Design professionnel

### Regles

| Element | Recommandation |
|---------|----------------|
| Couleurs | 3-4 max, coherentes |
| Polices | 1-2 familles |
| Espacement | Genereux et equilibre |
| Alignement | Sur une grille invisible |

### Masquer les elements

- Masquez le quadrillage
- Masquez les en-tetes
- Figez les volets

## Performance

### Optimisation

- Limitez les formules volatiles
- Utilisez des tableaux structures
- Calculez une fois, affichez plusieurs fois
