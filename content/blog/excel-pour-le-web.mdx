# Excel pour le Web : Guide Complet

Excel pour le Web (anciennement Excel Online) est la version gratuite d'Excel accessible depuis un navigateur.

## Acces a Excel pour le Web

### Via Office.com

1. Allez sur **office.com**
2. Connectez-vous avec un compte Microsoft
3. Cliquez sur l'icone Excel

### Via OneDrive

1. Connectez-vous a **onedrive.com**
2. Ouvrez un fichier Excel existant
3. Ou creez un nouveau fichier

### Compte requis

Un compte Microsoft gratuit suffit (Outlook, Hotmail, Live).

## Interface

### Similarites avec le desktop

- Ruban avec onglets
- Barre de formule
- Feuilles de calcul
- Grille de cellules

### Differences

- Interface simplifiee
- Moins d'options dans les menus
- Barre laterale pour certaines fonctions

## Fonctionnalites disponibles

### Formules

La majorite des formules fonctionnent :

- SOMME, MOYENNE, NB
- SI, ET, OU
- RECHERCHEV, RECHERCHEX
- INDEX, EQUIV

### Mise en forme

- Police, taille, couleur
- Bordures et remplissage
- Alignement
- Formats de nombres
- Mise en forme conditionnelle

### Graphiques

- Histogrammes
- Courbes
- Secteurs
- Nuages de points

### Tableaux

- Tableaux structures
- Tri et filtres
- Totaux automatiques

### Tableaux croises dynamiques

- Creation et modification
- Segments
- Actualisation

## Collaboration

### Partage

1. Cliquez sur **Partager**
2. Entrez les adresses email
3. Definissez les droits (lecture ou modification)
4. Envoyez

### Edition simultanee

Plusieurs personnes peuvent modifier en meme temps :

- Chaque utilisateur a une couleur
- Les modifications sont visibles en temps reel
- Historique des versions disponible

### Commentaires

- Ajoutez des commentaires sur les cellules
- Mentionnez des collegues avec @
- Discussions threaded

## Limites

### Fonctionnalites absentes

- Macros VBA
- Power Query (limite)
- Power Pivot
- Certaines fonctions avancees

### Performance

- Fichiers volumineux plus lents
- Limite de taille de fichier
- Necessite une connexion internet

### Compatibilite

Certains elements peuvent ne pas s'afficher correctement :
- Graphiques complexes
- Formes personnalisees
- Certains formats

## Hors-ligne

### Pas de mode hors-ligne natif

Excel pour le Web necessite internet. Pour travailler hors-ligne :

1. Telechargez le fichier
2. Travaillez avec Excel desktop
3. Reimportez sur OneDrive

### Alternative

Utilisez la synchronisation OneDrive sur Windows/Mac.

## Raccourcis clavier

La plupart des raccourcis Excel fonctionnent :

| Raccourci | Action |
|-----------|--------|
| Ctrl + C | Copier |
| Ctrl + V | Coller |
| Ctrl + Z | Annuler |
| Ctrl + S | Enregistrer |
| Ctrl + B | Gras |
| F2 | Modifier la cellule |

## Astuces

### Performance

- Limitez la taille des fichiers
- Evitez les formules sur colonnes entieres
- Fermez les onglets inutilises

### Organisation

- Utilisez des dossiers dans OneDrive
- Nommez clairement vos fichiers
- Utilisez des modeles

### Securite

- Verifiez les autorisations de partage
- Utilisez des liens avec expiration
- Revoquez les acces obsoletes

## Comparaison avec Excel Desktop

| Fonctionnalite | Desktop | Web |
|----------------|---------|-----|
| Prix | Payant | Gratuit |
| Macros | Oui | Non |
| Hors-ligne | Oui | Non |
| Collaboration | Bonne | Excellente |
| Fonctions | Toutes | Majorite |

## Cas d'utilisation ideaux

### Parfait pour

- Collaboration d'equipe
- Acces depuis n'importe quel appareil
- Fichiers simples a moyens
- Partage temporaire

### Moins adapte pour

- Macros et automatisation
- Fichiers tres volumineux
- Analyses complexes
- Travail hors-ligne frequent
