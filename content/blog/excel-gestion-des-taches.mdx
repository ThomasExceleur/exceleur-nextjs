# Gestion des Taches dans Excel

Excel permet de creer un systeme de gestion des taches simple et efficace.

## Structure de base

### Colonnes essentielles

| Colonne | Description |
|---------|-------------|
| ID | Identifiant unique |
| Tache | Description |
| Priorite | Haute/Moyenne/Basse |
| Statut | A faire/En cours/Termine |
| Date echeance | Deadline |
| Responsable | Personne assignee |
| Date creation | Quand ajoutee |

## Creer le tableau

### Etape 1 : En-tetes

```
A1: ID
B1: Tache
C1: Priorite
D1: Statut
E1: Date echeance
F1: Responsable
G1: Date creation
```

### Etape 2 : Tableau structure

1. Selectionnez vos donnees
2. **Ctrl + T** pour creer un tableau
3. Nommez le tableau "Taches"

## Listes deroulantes

### Priorites

1. Creez une liste : Haute, Moyenne, Basse
2. **Donnees** > **Validation** > **Liste**

### Statuts

- A faire
- En cours
- En attente
- Termine
- Annule

## Mise en forme conditionnelle

### Colorer par priorite

1. Selectionnez la colonne Priorite
2. **Mise en forme conditionnelle** > **Regles de mise en surbrillance**
3. Rouge = Haute, Orange = Moyenne, Vert = Basse

### Alerter sur les retards

```excel
=ET(E2<AUJOURDHUI();D2<>"Termine")
```

Format : fond rouge

### Barrer les taches terminees

```excel
=$D2="Termine"
```

Format : police barree, grise

## Cases a cocher

### Ajouter des cases

1. **Developpeur** > **Inserer** > **Case a cocher**
2. Liez a une cellule

### Formule conditionnelle

```excel
=SI(H2=VRAI;"Termine";"A faire")
```

## Calculs et statistiques

### Nombre de taches

```excel
=NB(Taches[ID])                            ' Total
=NB.SI(Taches[Statut];"A faire")           ' A faire
=NB.SI(Taches[Statut];"Termine")           ' Terminees
```

### Pourcentage d'avancement

```excel
=NB.SI(Taches[Statut];"Termine")/NB(Taches[ID])
```

### Taches en retard

```excel
=NB.SI.ENS(Taches[Date echeance];"<"&AUJOURDHUI();Taches[Statut];"<>Termine")
```

## Filtres et vues

### Filtrer les taches

1. Cliquez sur la fleche du filtre
2. Selectionnez les criteres

### Vues personnalisees

- Mes taches (filtre par responsable)
- Taches urgentes (priorite haute)
- En retard (echeance passee)
- Cette semaine (echeance dans 7 jours)

## Tri automatique

### Par priorite et echeance

1. **Donnees** > **Trier**
2. Premier niveau : Priorite
3. Deuxieme niveau : Date echeance

## Tableau de bord

### Resume des taches

| Indicateur | Valeur |
|------------|--------|
| Total taches | =NB(ID) |
| A faire | =NB.SI(Statut;"A faire") |
| En cours | =NB.SI(Statut;"En cours") |
| Terminees | =NB.SI(Statut;"Termine") |
| En retard | =FORMULE |

### Graphique d'avancement

1. Camembert des statuts
2. Ou barre de progression

## Taches recurrentes

### Gestion manuelle

Dupliquez la tache avec nouvelle echeance.

### Avec formule

```excel
=SI([@Statut]="Termine";[@[Date echeance]]+7;[@[Date echeance]])
```

## Notifications

### Mise en forme pour alertes

| Condition | Format |
|-----------|--------|
| Echeance aujourd'hui | Fond jaune |
| En retard | Fond rouge |
| Termine | Fond vert |

### Formule alerte

```excel
=SI(ET(E2<=AUJOURDHUI();D2<>"Termine");"URGENT";"")
```

## Assignation d'equipe

### Liste des responsables

Creez une liste deroulante avec les noms d'equipe.

### Charge par personne

```excel
=NB.SI(Taches[Responsable];G2)
```

## Export et partage

### Export PDF

1. **Fichier** > **Exporter** > **PDF**
2. Selectionnez la plage

### Partage OneDrive

1. Enregistrez sur OneDrive
2. Partagez le lien
3. Co-edition en temps reel

## Archivage

### Archiver les taches terminees

1. Creez une feuille "Archive"
2. Deplacez les taches terminees periodiquement

### Garder un historique

Ajoutez une colonne "Date cloture".

## Integration calendrier

### Export vers calendrier

1. Creez un fichier ICS
2. Ou copiez vers Outlook

### Rappels

Utilisez les rappels Outlook pour les echeances importantes.

## Bonnes pratiques

1. **Limitez** les taches en cours (max 3-5)
2. **Decomposez** les grandes taches
3. **Mettez a jour** quotidiennement
4. **Revoyez** les priorites regulierement
5. **Archivez** les taches terminees
