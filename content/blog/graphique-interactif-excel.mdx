# Graphiques Interactifs dans Excel

Rendez vos graphiques dynamiques et engageants.

## Methodes d'interactivite

### Segments

Filtres visuels pour les TCD et graphiques.

### Listes deroulantes

Selection qui modifie les donnees affichees.

### Boutons d'option

Choix entre plusieurs vues.

## Graphiques avec segments

### Etapes

1. Creez un TCD
2. Inserez un graphique croise dynamique
3. Ajoutez des segments

### Effet

Le graphique se met a jour selon les filtres.

## Graphiques avec listes

### Configuration

1. Liste deroulante en E1
2. Formule FILTRE pour les donnees
3. Graphique lie aux donnees filtrees

```excel
=FILTRE(Donnees;Region=E1)
```

## CHOISIR pour changer de vue

### Formule

```excel
=CHOISIR(E1;Donnees1;Donnees2;Donnees3)
```

E1 contient 1, 2 ou 3 selon le choix.

## Animation avec chronologie

### TCD avec dates

Utilisez une chronologie pour filtrer par periode.
