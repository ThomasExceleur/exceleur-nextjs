# Graphiques Interactifs dans Excel

Rendez vos graphiques dynamiques et engageants.

## Méthodes d'interactivite

### Segments

Filtres visuels pour les TCD et graphiques.

### Listes deroulantes

Sélection qui modifié les données affichées.

### Boutons d'option

Choix entre plusieurs vues.

## Graphiques avec segments

### Étapes

1. Créez un TCD
2. Insérez un graphique croisé dynamique
3. Ajoutez des segments

### Effet

Le graphique se met à jour selon les filtres.

## Graphiques avec listes

### Configuration

1. Liste deroulante en E1
2. Formule FILTRE pour les données
3. Graphique lie aux données filtrees

```excel
=FILTRE(Données;Region=E1)
```

## CHOISIR pour changer de vue

### Formule

```excel
=CHOISIR(E1;Donnees1;Donnees2;Donnees3)
```

E1 contient 1, 2 où 3 selon le choix.

## Animation avec chronologie

### TCD avec dates

Utilisez une chronologie pour filtrer par période.
