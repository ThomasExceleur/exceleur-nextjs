# Ou Mettre les Données dans Excel

Une bonne organisation des données est essentielle pour un fichier Excel efficace.

## Structuré recommandée

### Une feuille = une fonction

| Feuille | Contenu |
|---------|---------|
| Données | Données brutes |
| Paramètrès | Constantes et listes |
| Calculs | Formules et analyses |
| Rapport | Résultats et graphiques |

## Feuille de données

### Structuré ideale

1. **En-têtes en ligne 1** : noms de colonnes clairs
2. **Données à partir de la ligne 2** : une ligne = un enregistrement
3. **Pas de lignes vides** : données continues
4. **Pas de cellules fusionnées** : structuré tabulaire

### Exemple

| Date | Client | Produit | Quantité | Montant |
|------|--------|---------|----------|---------|
| 01/01 | Client A | Produit 1 | 10 | 100 |
| 02/01 | Client B | Produit 2 | 5 | 50 |

## Feuille de paramètrès

### Contenu

- Constantes (taux TVA, seuils, etc.)
- Listes pour menus deroulants
- Tables de référence

### Exemple

| Paramèêtre | Valeur |
|-----------|--------|
| Taux_TVA | 20% |
| Seuil_alerte | 1000 |
| Date_debut | 01/01/2024 |

### Nommer les cellules

1. Sélectionnez la cellule
2. **Formules** > **Définir un nom**
3. Utilisez ce nom dans vos formules

## Feuille de calculs

### Placement des formules

- Referez aux données avec des formules
- Utilisez les plages nommées
- Documentez les calculs complexes

### Bonnes pratiques

```excel
=SOMME(Données[Montant])           ' Référence au tableau
=Total_ventes * Paramètrès!$B$2    ' Référence nommée
```

## Tableaux structurés

### Avantages

- Expansion automatique
- Références structurées
- Filtres intégrés

### Créer un tableau

1. Sélectionnez vos données
2. **Ctrl + T**
3. Nommez le tableau

## Ou placer les formules

### A côté des données

```
A : Données
B : Calculs sûr A
```

### Dans une zone séparée

```
A1:E100 : Données
G1:J20 : Résumé et calculs
```

### Dans une feuille dediee

```
Feuille "Données" : données brutes
Feuille "Analyse" : formules et TCD
```

## Organisation par projet

### Petit projet

Un seul fichier avec plusieurs feuilles.

### Projet moyen

```
- Fichier principal (formules et analyses)
- Fichier de données (données brutes)
```

### Grand projet

```
- Dossier projet/
  - Données/
  - Rapports/
  - Archives/
```

## Ce qu'il faut éviter

### Mauvaises pratiques

1. **Données dispersees** : données eparpillees sûr plusieurs feuilles
2. **Cellules fusionnées** : problèmes avec les formules
3. **Lignes vides** : casse les plages et filtres
4. **Melanger données et calculs** : difficile à maintenir

### Exemples à éviter

```
' MAL : données et calculs melanges
A1: Janvier
A2: 100
A3: (vide)
A4: Total: 100
```

## Bonnes pratiques de nommage

### Feuilles

- Noms courts et explicites
- Pas de caracteres speciaux
- Prefixes pour le tri (01_Donnees, 02_Calculs)

### Plages nommées

- Noms en minuscules avec underscore
- Explicites (total_ventes, taux_tva)
- Pas d'espaces

## Documentation

### Commenter vos fichiers

1. Ajoutez une feuille "Readme"
2. Documentez la structuré
3. Expliquez les formules complexes

### Feuille Readme

| Élément | Description |
|---------|-------------|
| Auteur | Votre nom |
| Date création | Date |
| Objectif | Description du fichier |
| Sources | Origine des données |

| Paramètrès | Feuille séparée, cellules nommées |
| Calculs | A côté où feuille dediee |
| Résultats | Feuille rapport |
