# Excel en Ligne : Guide Complet

Excel Online est la version web gratuite d'Excel, accessible depuis n'importe quel navigateur.

## Acces a Excel Online

### Via Office.com

1. Allez sur **office.com**
2. Connectez-vous avec un compte Microsoft
3. Cliquez sur **Excel**

### Via OneDrive

1. Connectez-vous a **onedrive.com**
2. Cliquez sur **Nouveau** > **Classeur Excel**

### Compte gratuit

Un compte Microsoft gratuit suffit. Pas besoin d'abonnement Microsoft 365.

## Fonctionnalites disponibles

### Formules

La plupart des formules fonctionnent :

- SOMME, MOYENNE, NB
- RECHERCHEV, RECHERCHEX
- SI, ET, OU
- INDEX, EQUIV

### Mise en forme

- Couleurs et polices
- Bordures
- Mise en forme conditionnelle
- Formats de nombres

### Graphiques

- Histogrammes
- Courbes
- Secteurs
- Nuages de points

### Tableaux

- Tableaux structures
- Tri et filtres
- Tableaux croises dynamiques (limites)

## Collaboration en temps reel

### Partager un fichier

1. Cliquez sur **Partager** (en haut a droite)
2. Entrez les adresses email
3. Definissez les autorisations (lecture ou modification)
4. Envoyez

### Edition simultanee

Plusieurs personnes peuvent modifier le meme fichier en temps reel. Chaque utilisateur est identifie par une couleur.

### Commentaires

Ajoutez des commentaires pour communiquer avec les collaborateurs.

## Limites d'Excel Online

### Fonctionnalites absentes

- Macros VBA
- Power Query (limite)
- Power Pivot
- Certaines fonctions avancees

### Performance

- Fichiers volumineux : plus lent
- Limite de taille de fichier
- Necessite une connexion internet

### Mise en forme

Certaines options de mise en forme avancee ne sont pas disponibles.

## Comparaison avec Excel Desktop

| Fonctionnalite | Desktop | Online |
|----------------|---------|--------|
| Prix | Payant | Gratuit |
| Macros VBA | Oui | Non |
| Hors-ligne | Oui | Non |
| Collaboration | Oui | Excellente |
| Toutes les formules | Oui | Majorite |
| Power Query | Oui | Limite |

## Synchronisation avec OneDrive

### Sauvegarde automatique

Les fichiers sont sauvegardes automatiquement sur OneDrive.

### Acces multi-appareils

Retrouvez vos fichiers sur tous vos appareils.

### Historique des versions

Acces aux versions precedentes du fichier.

## Travailler hors-ligne

### Application de bureau

Telechargez le fichier pour travailler hors-ligne avec Excel Desktop.

### Synchronisation OneDrive

Installez OneDrive sur votre ordinateur pour synchroniser automatiquement.

## Cas d'utilisation ideaux

### Travail collaboratif

Parfait pour les projets d'equipe avec edition simultanee.

### Acces mobile

Consultez et modifiez vos fichiers depuis n'importe ou.

### Fichiers simples

Ideal pour les tableaux et calculs basiques.

### Partage temporaire

Partagez facilement un fichier sans envoyer de piece jointe.

## Raccourcis clavier

La plupart des raccourcis Excel fonctionnent :

| Raccourci | Action |
|-----------|--------|
| Ctrl + C | Copier |
| Ctrl + V | Coller |
| Ctrl + Z | Annuler |
| Ctrl + B | Gras |
| Ctrl + S | Enregistrer |

## Conseils d'utilisation

### Optimiser les performances

- Limitez la taille des fichiers
- Evitez les formules sur colonnes entieres
- Fermez les onglets inutiles

### Securite

- Verifiez les autorisations de partage
- Utilisez des liens avec expiration
- Protegez les feuilles sensibles

### Organisation

- Utilisez des dossiers dans OneDrive
- Nommez clairement vos fichiers
- Utilisez des modeles

## Comment commencer

1. Creez un compte Microsoft gratuit
2. Allez sur office.com
3. Cliquez sur Excel
4. Commencez a travailler
