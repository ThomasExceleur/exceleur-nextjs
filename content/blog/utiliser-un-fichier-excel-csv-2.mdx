# Utiliser un Fichier CSV

Importez et exportez des fichiers CSV avec Excel.

## Qu'est-ce qu'un CSV ?

### Definition

CSV = Comma Separated Values (valeurs separees par des virgules).

### Caracteristiques

- Format texte simple
- Universel
- Sans mise en forme

## Ouvrir un CSV

### Methode simple

1. Double-cliquez sur le fichier
2. Excel l'ouvre automatiquement

### Methode controlee

1. **Donnees** > **A partir d'un fichier texte/CSV**
2. Configurez les options

## Options d'import

| Option | Description |
|--------|-------------|
| Delimiteur | Virgule, point-virgule, tab |
| Encodage | UTF-8, ANSI, etc. |
| Detection | Automatique ou manuelle |

## Problemes courants

### Mauvais delimiteur

| Symptome | Cause |
|----------|-------|
| Tout dans une colonne | Mauvais separateur |
| Donnees decalees | Separateur dans les valeurs |

### Solution

Specifiez le bon delimiteur a l'import.

## Encodage

### Caracteres speciaux

| Probleme | Solution |
|----------|----------|
| Accents corrompus | Choisir UTF-8 |
| ??? a la place | Changer l'encodage |

## Exporter en CSV

### Etapes

1. **Fichier** > **Enregistrer sous**
2. **Type** : CSV (separateur: point-virgule)

### Types de CSV

| Type | Separateur |
|------|------------|
| CSV (virgule) | , |
| CSV UTF-8 | , avec encodage UTF-8 |
| CSV (point-virgule) | ; |
