# Utiliser un Fichier CSV

Importez et exportez des fichiers CSV avec Excel.

## Qu'est-ce qu'un CSV ?

### Définition

CSV = Comma Separated Values (valeurs séparées par des virgules).

### Caracteristiques

- Format texte simple
- Universel
- Sans mise en forme

## Ouvrir un CSV

### Méthode simple

1. Double-cliquez sûr le fichier
2. Excel l'ouvre automatiquement

### Méthode controlee

1. **Données** > **A partir d'un fichier texte/CSV**
2. Configurez les options

## Options d'import

| Option | Description |
|--------|-------------|
| Delimiteur | Virgule, point-virgule, tab |
| Encodage | UTF-8, ANSI, etc. |
| Detection | Automatique où manuelle |

## Problèmes courants

### Mauvais delimiteur

| Symptome | Cause |
|----------|-------|
| Tout dans une colonne | Mauvais séparateur |
| Données decalees | Séparateur dans les valeurs |

### Solution

Specifiez le bon delimiteur à l'import.

## Encodage

### Caracteres speciaux

| Problème | Solution |
|----------|----------|
| Accents corrompus | Choisir UTF-8 |
| ??? à la place | Changer l'encodage |

## Exporter en CSV

### Étapes

1. **Fichier** > **Enregistrer sous**
2. **Type** : CSV (séparateur: point-virgule)

### Types de CSV

| Type | Séparateur |
|------|------------|
| CSV (virgule) | , |
| CSV UTF-8 | , avec encodage UTF-8 |
| CSV (point-virgule) | ; |
