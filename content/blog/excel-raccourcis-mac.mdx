# Excel Raccourcis Mac : Les Essentiels

Sur Mac, les raccourcis Excel utilisent principalement **Cmd** au lieu de **Ctrl**. Voici tous les raccourcis indispensables.

## Raccourcis de base

| Raccourci | Action |
|-----------|--------|
| **Cmd + C** | Copier |
| **Cmd + X** | Couper |
| **Cmd + V** | Coller |
| **Cmd + Z** | Annuler |
| **Cmd + Y** | Retablir |
| **Cmd + S** | Enregistrer |
| **Cmd + O** | Ouvrir |
| **Cmd + N** | Nouveau classeur |
| **Cmd + P** | Imprimer |
| **Cmd + F** | Rechercher |
| **Cmd + H** | Remplacer |

## Navigation

| Raccourci | Action |

| **Ctrl + G** | Atteindre (equivalent F5) |
| **Fn + Fleche Bas** | Page suivante |
| **Fn + Fleche Haut** | Page precedente |
| **Ctrl + Tab** | Feuille suivante |
| **Ctrl + Maj + Tab** | Feuille precedente |

| **Cmd + Maj + L** | Activer/desactiver les filtres |

## Astuces Mac specifiques

### Utiliser Fn

De nombreuses touches de fonction (F1-F12) necessitent la touche **Fn** sur Mac :

- **Fn + F2** = Editer la cellule
- **Fn + F4** = Figer la reference
- **Fn + F7** = Orthographe
- **Fn + F9** = Calculer

### Preferences clavier

Pour utiliser F1-F12 sans Fn :

1. Preferences Systeme > Clavier
2. Cochez "Utiliser les touches F1, F2, etc. comme touches de fonction standard"

## Raccourcis personnalises

### Creer un raccourci

Excel Mac ne permet pas de personnaliser les raccourcis directement, mais vous pouvez :

1. Utiliser les macros avec des raccourcis assignes
2. Utiliser Automator ou Keyboard Maestro

## Comparaison Windows/Mac

| Windows | Mac | Action |
|---------|-----|--------|
| Ctrl + C | Cmd + C | Copier |
| F2 | Ctrl + U | Editer |
| F4 | Cmd + T | Figer |
| Alt + = | Cmd + Maj + T | Somme auto |
| Ctrl + Home | Fn + Cmd + Gauche | Aller en A1 |
