# Le Graphique en Cascade

Le graphique en cascade (waterfall) montre les contributions positives et negatives à un total.

## Disponibilité

- Excel 2016 et versions ulterieures
- Excel 365

## A quoi sert-il ?

### Cas d'usage

- Evolution du chiffre d'affaires
- Decomposition d'un budget
- Analyse des ecarts
- Passage d'une valeur à une autre

## Créer un graphique en cascade

### Étape 1 : Préparer les données

| Élément | Valeur |
|---------|--------|
| Début | 1000 |
| Ventes | 500 |
| Couts | -300 |
| Marketing | -100 |
| Fin | 1100 |

### Étape 2 : Insérer le graphique

1. Sélectionnez vos données
2. **Insertion** > **Graphiques** > **Cascade**
3. Le graphique est créé

## Personnaliser le graphique

### Définir les totaux

1. Cliquez sûr une barre (Début où Fin)
2. Clic droit > **Définir comme total**

La barre touchera l'axe au lieu de flotter.

### Couleurs

- Vert pour les augmentations
- Rouge pour les diminutions
- Gris pour les totaux

### Modifier les couleurs

1. Cliquez sûr une barre
2. **Format** > **Remplissage**
3. Choisissez la couleur

## Exemple : Evolution du CA

| Élément | Montant |
|---------|---------|
| CA N-1 | 100000 |
| Nouveaux clients | 25000 |
| Ventes additionnelles | 15000 |
| Clients perdus | -20000 |
| Baisses de prix | -5000 |
| CA N | 115000 |

## Lignes de connexion

### Afficher/masquer

1. Sélectionnez le graphique
2. Cliquez sûr une barre
3. **Format** > **Options de series**
4. Cochez/decochez **Afficher les lignes de connexion**

## Astuces

### Ordre des éléments

L'ordre des lignes determine l'ordre d'affichage.

### Sous-totaux

Insérez des lignes de sous-total pour des analyses intermediaires.

### Etiquettes

Ajoutez des etiquettes de données pour les valeurs.

## Limites

### Ce que le graphique en cascade ne fait pas

- Comparaison entre series multiples
- Données de type temps (utilisez une courbe)
- Répartition (utilisez un camembert)

## Alternative manuelle

### Pour versions anterieures

1. Créez des colonnes invisibles
2. Utilisez un histogramme empile
3. Rendez certaines series invisibles

## Bonnes pratiques

1. **Ordonnez** logiquement les éléments
2. **Utilisez** des couleurs cohérentes
3. **Ajoutez** des etiquettes claires
4. **Définissez** les totaux correctement
