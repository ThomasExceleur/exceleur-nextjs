# Le Graphique en Cascade

Le graphique en cascade (waterfall) montre les contributions positives et negatives a un total.

## Disponibilite

- Excel 2016 et versions ulterieures
- Excel 365

## A quoi sert-il ?

### Cas d'usage

- Evolution du chiffre d'affaires
- Decomposition d'un budget
- Analyse des ecarts
- Passage d'une valeur a une autre

## Creer un graphique en cascade

### Etape 1 : Preparer les donnees

| Element | Valeur |
|---------|--------|
| Debut | 1000 |
| Ventes | 500 |
| Couts | -300 |
| Marketing | -100 |
| Fin | 1100 |

### Etape 2 : Inserer le graphique

1. Selectionnez vos donnees
2. **Insertion** > **Graphiques** > **Cascade**
3. Le graphique est cree

## Personnaliser le graphique

### Definir les totaux

1. Cliquez sur une barre (Debut ou Fin)
2. Clic droit > **Definir comme total**

La barre touchera l'axe au lieu de flotter.

### Couleurs

- Vert pour les augmentations
- Rouge pour les diminutions
- Gris pour les totaux

### Modifier les couleurs

1. Cliquez sur une barre
2. **Format** > **Remplissage**
3. Choisissez la couleur

## Exemple : Evolution du CA

| Element | Montant |
|---------|---------|
| CA N-1 | 100000 |
| Nouveaux clients | 25000 |
| Ventes additionnelles | 15000 |
| Clients perdus | -20000 |
| Baisses de prix | -5000 |
| CA N | 115000 |

## Lignes de connexion

### Afficher/masquer

1. Selectionnez le graphique
2. Cliquez sur une barre
3. **Format** > **Options de series**
4. Cochez/decochez **Afficher les lignes de connexion**

## Astuces

### Ordre des elements

L'ordre des lignes determine l'ordre d'affichage.

### Sous-totaux

Inserez des lignes de sous-total pour des analyses intermediaires.

### Etiquettes

Ajoutez des etiquettes de donnees pour les valeurs.

## Limites

### Ce que le graphique en cascade ne fait pas

- Comparaison entre series multiples
- Donnees de type temps (utilisez une courbe)
- Repartition (utilisez un camembert)

## Alternative manuelle

### Pour versions anterieures

1. Creez des colonnes invisibles
2. Utilisez un histogramme empile
3. Rendez certaines series invisibles

## Bonnes pratiques

1. **Ordonnez** logiquement les elements
2. **Utilisez** des couleurs coherentes
3. **Ajoutez** des etiquettes claires
4. **Definissez** les totaux correctement
