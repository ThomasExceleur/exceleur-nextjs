# Nettoyer son Tableau Excel

Des données propres pour de meilleures analyses.

## Checklist de nettoyage

| Vérification | Action |
|--------------|--------|
| Lignes vides | Supprimer |
| Doublons | Supprimer |
| Espaces | SUPPRESPACE |
| Formats | Uniformiser |

## Supprimer les espaces

### SUPPRESPACE

```excel
=SUPPRESPACE(A1)
```

### EPURAGE

```excel
=EPURAGE(A1)
```

Supprime les caracteres non imprimables.

## Supprimer les doublons

### Via le ruban

**Données** > **Supprimer les doublons**

### Choisir les colonnes

Sélectionnez les colonnes de comparaison.

## Uniformiser le texte

### Majuscules/Minuscules

| Fonction | Résultat |
|----------|----------|
| MAJUSCULE | TEXTE |
| MINUSCULE | texte |
| NOMPROPRE | Texte |

## Remplacer

### Rechercher/Remplacer

**Ctrl+H**

- Corrigez les erreurs courantes
- Uniformisez les formats

## Convertir les formats

### Texte en nombre

Colonne > Données > Convertir

### Dates

Vérifiez le format régional.
