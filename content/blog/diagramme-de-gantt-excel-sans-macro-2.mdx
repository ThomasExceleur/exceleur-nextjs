# Diagramme de Gantt sans Macro - Techniques Avancees

Poussez les limites d'Excel sans utiliser de code VBA.

## Gantt avec sous-taches

### Structure hierarchique

| Niveau | Tache | Debut | Duree |
|--------|-------|-------|-------|
| 1 | Phase 1 | 01/01 | 15 |
| 2 | - Sous-tache 1.1 | 01/01 | 5 |
| 2 | - Sous-tache 1.2 | 06/01 | 10 |
| 1 | Phase 2 | 16/01 | 20 |

### Indentation visuelle

Utilisez des espaces ou prefixes pour montrer la hierarchie.

### Calculs automatiques

Duree phase parente :
```excel
=MAX(Fins_sous_taches)-MIN(Debuts_sous_taches)
```

## Dependances visuelles

### Sans fleches automatiques

Ajoutez une colonne "Predecesseur" et utilisez la mise en forme conditionnelle pour surligner.

### Indicateur visuel

```excel
=SI([@Predecesseur]<>"";
  SI(INDEX(Fins;EQUIV([@Predecesseur];IDs;0))>[@Debut];
    "⚠ Conflit";"✓");
  "")
```

## Calendrier avec jours ouvres

### Colonnes jour par jour

Generez les dates avec :
```excel
=SERIE.JOUR.OUVRE(Date_debut;Numero_colonne-1)
```

### Mise en forme weekends

Coloriez differemment samedi/dimanche :
```excel
=JOURSEM(C$1;2)>5
```

### Jours feries

Ajoutez une table de jours feries et excluez-les.

## Barres de ressources

### Structure

| Tache | Ressource | Charge |
|-------|-----------|--------|
| Dev 1 | Alice | 100% |
| Dev 2 | Bob | 50% |
| Test | Alice | 50% |

### Graphique de charge

Creez un graphique separe montrant la charge par ressource.

### Surcapacite

```excel
=SI(SOMME.SI(Ressource;[@Ressource];Charge)>100%;"Surcharge";"OK")
```

## Timeline dynamique

### Zoom sur la periode

Utilisez des listes deroulantes pour definir la plage visible :
- Date debut affichage
- Date fin affichage

### Formule de visibilite

```excel
=ET([@Debut]<=Fin_affichage;[@Fin]>=Debut_affichage)
```

## Indicateurs de performance

### Avancement global

```excel
=SOMMEPROD(Durees;Progressions)/SOMME(Durees)
```

### Valeur acquise

```excel
Valeur_acquise = Budget_total * Avancement_global
```

### Indice de performance

```excel
SPI = Valeur_acquise / Valeur_planifiee
```

## Formules avancees utiles

### Date la plus proche de la fin

```excel
=MAX(Fins)
```

### Taches en retard

```excel
=NB.SI.ENS(Fins;"<"&AUJOURDHUI();Statuts;"<>"&"Termine")
```

### Jours restants projet

```excel
=MAX(Fins)-AUJOURDHUI()
```

### Prochaine tache a commencer

```excel
=MIN(SI(Debuts>AUJOURDHUI();Debuts))
```
(Formule matricielle ou MINIFS)
