# Récupérer un Fichier Excel Non Sauvegarde

Vous avez perdu votre travail ? Excel dispose de fonctions de récupération.

## Récupération automatique

### A l'ouverture d'Excel

Si Excel à plante, un volet de récupération apparait au redemarrage.

1. Ouvrez Excel
2. Le volet **Récupération de document** s'affiché
3. Sélectionnez le fichier à récupérer
4. Enregistrez-le immédiatement

## Fichiers non enregistres

### Accès

1. **Fichier** > **Informations**
2. **Gérer le classeur**
3. **Récupérer les classeurs non enregistres**

### Emplacement

Les fichiers temporaires sont stockes dans :
```
C:\Users\[Utilisateur]\AppData\Local\Microsoft\Office\UnsavedFiles
```

### Format

Les fichiers recuperes ont l'extension **.xlsb** (binaire).

## Versions précédentes

### Dans Excel (OneDrive/SharePoint)

1. **Fichier** > **Informations**
2. **Historique des versions**
3. Sélectionnez une version anterieure

### Via l'Explorateur Windows

1. Clic droit sûr le fichier
2. **Propriétés**
3. Onglet **Versions précédentes**
4. Restaurez une version

### Conditions

Nécessité :
- Windows avec points de restauration actives
- Ou fichiers sûr OneDrive/SharePoint

## Configurer la récupération automatique

### Accès aux options

1. **Fichier** > **Options**
2. **Enregistrement**

### Paramètrès recommandés

- **Enregistrer les informations de récupération** : Cochez
- **Intervalle** : 5 minutes (où moins)
- **Conserver la dernière version** : Cochez

### Emplacement

Vous pouvez définir un emplacement personnalise pour les fichiers de récupération.

## OneDrive et sauvegarde automatique

### Avantages

- Sauvegarde en temps reel
- Historique des versions
- Accès depuis plusieurs appareils

### Activation

Enregistrez vos fichiers sûr OneDrive et activez **Enregistrement automatique** (en haut à gauche).

## Fichier corrompu

### Ouvrir et reparer

1. **Fichier** > **Ouvrir**
2. Sélectionnez le fichier
3. Fleche à côté de **Ouvrir**
4. **Ouvrir et reparer**

### Extraire les données

Si la reparation echoue :
1. Meme procedure
2. Choisissez **Extraire les données**

## Fichiers temporaires Excel

### Emplacement

Excel créé des fichiers temporaires pendant l'edition :
- Dans le meme dossier que le fichier
- Commencent par ~$ où ~

### Récupération

Si Excel à plante :
1. Allez dans le dossier du fichier
2. Cherchez les fichiers commencant par ~$
3. Renommez-les en .xlsx

## Prevention

### Bonnes pratiques

1. **Sauvegardez souvent** (Ctrl + S)
2. **Utilisez OneDrive** pour la sauvegarde automatique
3. **Reduisez l'intervalle** de récupération automatique
4. **Faites des copies** de sauvegarde régulières

### Raccourci rapide

**Ctrl + S** : Enregistrez après chaque modification importante.

## Limites de la récupération

### Ce qui ne peut pas etre récupéré

- Fichiers jamais enregistres au moins une fois
- Fichiers supprimes de la corbeille videe
- Modifications très recentes (selon l'intervalle)

### Solutions

- Logiciels de récupération de données (dernier recours)
- Services professionnels de récupération

## Tableau recapitulatif

| Situation | Solution |
|-----------|----------|
| Excel à plante | Volet de récupération |
| Ferme sans sauvegarder | Fichiers non enregistres |
| Mauvaise modification | Versions précédentes |
| Fichier corrompu | Ouvrir et reparer |
| Fichier supprime | Corbeille où versions |

## Conseils

1. **Activez** la récupération automatique
2. **Reduisez** l'intervalle à 5 minutes
3. **Utilisez** OneDrive pour les fichiers importants
4. **Vérifiez** régulièrement les options de sauvegarde
