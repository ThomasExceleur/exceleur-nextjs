# Recuperer un Fichier Excel Non Sauvegarde

Vous avez perdu votre travail ? Excel dispose de fonctions de recuperation.

## Recuperation automatique

### A l'ouverture d'Excel

Si Excel a plante, un volet de recuperation apparait au redemarrage.

1. Ouvrez Excel
2. Le volet **Recuperation de document** s'affiche
3. Selectionnez le fichier a recuperer
4. Enregistrez-le immediatement

## Fichiers non enregistres

### Acces

1. **Fichier** > **Informations**
2. **Gerer le classeur**
3. **Recuperer les classeurs non enregistres**

### Emplacement

Les fichiers temporaires sont stockes dans :
```
C:\Users\[Utilisateur]\AppData\Local\Microsoft\Office\UnsavedFiles
```

### Format

Les fichiers recuperes ont l'extension **.xlsb** (binaire).

## Versions precedentes

### Dans Excel (OneDrive/SharePoint)

1. **Fichier** > **Informations**
2. **Historique des versions**
3. Selectionnez une version anterieure

### Via l'Explorateur Windows

1. Clic droit sur le fichier
2. **Proprietes**
3. Onglet **Versions precedentes**
4. Restaurez une version

### Conditions

Necessite :
- Windows avec points de restauration actives
- Ou fichiers sur OneDrive/SharePoint

## Configurer la recuperation automatique

### Acces aux options

1. **Fichier** > **Options**
2. **Enregistrement**

### Parametres recommandes

- **Enregistrer les informations de recuperation** : Cochez
- **Intervalle** : 5 minutes (ou moins)
- **Conserver la derniere version** : Cochez

### Emplacement

Vous pouvez definir un emplacement personnalise pour les fichiers de recuperation.

## OneDrive et sauvegarde automatique

### Avantages

- Sauvegarde en temps reel
- Historique des versions
- Acces depuis plusieurs appareils

### Activation

Enregistrez vos fichiers sur OneDrive et activez **Enregistrement automatique** (en haut a gauche).

## Fichier corrompu

### Ouvrir et reparer

1. **Fichier** > **Ouvrir**
2. Selectionnez le fichier
3. Fleche a cote de **Ouvrir**
4. **Ouvrir et reparer**

### Extraire les donnees

Si la reparation echoue :
1. Meme procedure
2. Choisissez **Extraire les donnees**

## Fichiers temporaires Excel

### Emplacement

Excel cree des fichiers temporaires pendant l'edition :
- Dans le meme dossier que le fichier
- Commencent par ~$ ou ~

### Recuperation

Si Excel a plante :
1. Allez dans le dossier du fichier
2. Cherchez les fichiers commencant par ~$
3. Renommez-les en .xlsx

## Prevention

### Bonnes pratiques

1. **Sauvegardez souvent** (Ctrl + S)
2. **Utilisez OneDrive** pour la sauvegarde automatique
3. **Reduisez l'intervalle** de recuperation automatique
4. **Faites des copies** de sauvegarde regulieres

### Raccourci rapide

**Ctrl + S** : Enregistrez apres chaque modification importante.

## Limites de la recuperation

### Ce qui ne peut pas etre recupere

- Fichiers jamais enregistres au moins une fois
- Fichiers supprimes de la corbeille videe
- Modifications tres recentes (selon l'intervalle)

### Solutions

- Logiciels de recuperation de donnees (dernier recours)
- Services professionnels de recuperation

## Tableau recapitulatif

| Situation | Solution |
|-----------|----------|
| Excel a plante | Volet de recuperation |
| Ferme sans sauvegarder | Fichiers non enregistres |
| Mauvaise modification | Versions precedentes |
| Fichier corrompu | Ouvrir et reparer |
| Fichier supprime | Corbeille ou versions |

## Conseils

1. **Activez** la recuperation automatique
2. **Reduisez** l'intervalle a 5 minutes
3. **Utilisez** OneDrive pour les fichiers importants
4. **Verifiez** regulierement les options de sauvegarde
