# Quel Ecart-Type Utiliser dans Excel

Excel propose plusieurs fonctions d'ecart-type. Voici comment choisir.

## Les deux types d'ecart-type

### Ecart-type de l'echantillon (n-1)

Utilisez quand vos donnees representent un **echantillon** d'une population plus large.

### Ecart-type de la population (n)

Utilisez quand vos donnees representent la **population entiere**.

## Fonctions Excel

### Pour un echantillon

```excel
=ECARTYPE.STANDARD(A1:A100)
=ECARTYPE(A1:A100)            ' Equivalent (ancienne fonction)
```

### Pour une population

```excel
=ECARTYPEP(A1:A100)
=ECARTYPE.PEARSON(A1:A100)    ' Equivalent moderne
```

## Difference mathematique

### Echantillon (n-1)

Division par (n-1) pour corriger le biais.

### Population (n)

Division par n.

### Impact

L'ecart-type echantillon est legerement plus grand.

## Quand utiliser lequel ?

### Utilisez ECARTYPE.STANDARD si

- Vous analysez un echantillon
- Vos donnees sont un sous-ensemble
- Vous faites des statistiques inferentielles

### Utilisez ECARTYPEP si

- Vous avez toutes les donnees de la population
- Vous faites des statistiques descriptives
- Vous analysez un ensemble complet

## Exemples pratiques

### Notes d'une classe entiere

```excel
=ECARTYPEP(Notes)             ' Population complete
```

### Echantillon de clients

```excel
=ECARTYPE.STANDARD(Ventes)    ' Echantillon
```

## Fonctions associees

### Variance

| Type | Fonction |
|------|----------|
| Echantillon | VAR.S ou VAR |
| Population | VAR.P ou VARPA |

### Relation

```excel
Ecart-type = RACINE(Variance)
```

## Cas pratique : Controle qualite

### Mesures sur un lot

Si vous mesurez tous les produits d'un lot :

```excel
=ECARTYPEP(Mesures)           ' Population complete
```

### Mesures sur un echantillon de lot

```excel
=ECARTYPE.STANDARD(Mesures)   ' Echantillon
```

## Interpretation

### Ecart-type faible

Les valeurs sont proches de la moyenne.

### Ecart-type eleve

Les valeurs sont dispersees.

## Resume des fonctions

| Fonction | Type | Usage |
|----------|------|-------|
| ECARTYPE.STANDARD | Echantillon | Le plus courant |
| ECARTYPEP | Population | Population complete |
| ECARTYPE | Echantillon | Ancienne version |
| ECARTYPE.PEARSON | Population | Moderne |

## Conseil pratique

En cas de doute, utilisez **ECARTYPE.STANDARD** (echantillon).

C'est le choix le plus prudent statistiquement.

## Ignorer les valeurs texte

### ECARTYPEA et ECARTYPEPA

- Comptent les textes comme 0
- Comptent VRAI comme 1, FAUX comme 0

```excel
=ECARTYPEA(A1:A100)
```
