# Modifier Plusieurs Nombres d'un Coup

Appliquez des opérations sûr plusieurs cellules.

## Collage special

### Méthode

1. Tapez le facteur dans une cellule (ex: 1,2 pour +20%)
2. Copiez cette cellule (**Ctrl+C**)
3. Sélectionnez les nombres à modifier
4. **Collage special** (**Ctrl+Alt+V**)
5. Choisissez l'opération

### Opérations disponibles

| Opération | Effet |
|-----------|-------|
| Ajouter | Additionne |
| Soustraire | Soustrait |
| Multiplier | Multiplie |
| Diviser | Divise |

## Exemples pratiques

### Augmentation de 10%

1. Tapez 1,1 dans une cellule
2. Copiez
3. Sélectionnez les prix
4. Collage special > Multiplier

### Arrondir à la dizaine

1. Tapez 10
2. Collage special > Diviser
3. Puis Multiplier (après arrondi)

## Rechercher et remplacer

### Pour du texte dans les nombres

**Ctrl+H** pour remplacer des patterns.

## Formules

### Appliquer une formule

```excel
=A1*1,1
```

Puis copier les valeurs seulement.

## Convertir texte en nombre

### Problème

"123" en texte ne calcule pas.

### Solution collage special

1. Copiez une cellule contenant 1
2. Sélectionnez les textes
3. Collage special > Multiplier

## Power Query

### Pour gros volumes

1. Importez les données
2. **Transformer** > opération mathematique
3. Chargez
