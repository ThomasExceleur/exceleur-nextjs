# Extraire un Sous-ensemble avec FILTRE

Créez des extractions ciblees de vos données.

## Syntaxe

```excel
=FILTRE(tableau;inclure;[si_vide])
```

## Extraction simple

### Par critère

```excel
=FILTRE(A2:D100;B2:B100="Paris")
```

### Par valeur numérique

```excel
=FILTRE(A2:D100;C2:C100>1000)
```

## Extraction multicriteres

### ET logique

```excel
=FILTRE(Données;(Region="Nord")*(Montant>100))
```

### OU logique

```excel
=FILTRE(Données;(Statut="A")+(Statut="B"))
```

## Top N valeurs

### Les 10 plus grands

```excel
=FILTRE(Données;Montant>=GRANDE.VALEUR(Montant;10))
```

## Plage de dates

### Ce mois

```excel
=FILTRE(Données;(MOIS(Dates)=MOIS(AUJOURDHUI()))*(ANNEE(Dates)=ANNEE(AUJOURDHUI())))
```

## Exclure des valeurs

### Différent de

```excel
=FILTRE(Données;Statut<>"Exclu")
```

## Gestion des vides

```excel
=FILTRE(Données;Critère;"Aucun résultat")
```

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| si_vide | Toujours définir |
| Plages alignees | Meme taille |
| Test | Vérifiez les résultats |
