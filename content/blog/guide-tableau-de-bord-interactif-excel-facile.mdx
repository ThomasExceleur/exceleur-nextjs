# Creer un Tableau de Bord Interactif Facile

Un tableau de bord simple mais efficace en quelques etapes.

## Etape 1 : Preparer les donnees

### Structure recommandee

Utilisez un tableau structure avec :
- En-tetes clairs
- Une ligne par enregistrement
- Types de donnees coherents

### Creer un tableau

1. Selectionnez vos donnees
2. **Ctrl + T**
3. Validez la plage

## Etape 2 : Creer un TCD

### Creation

1. Cliquez dans le tableau
2. **Insertion** > **Tableau croise dynamique**
3. Nouvelle feuille
4. **OK**

### Configuration

Faites glisser les champs :
- Lignes : Categories
- Valeurs : Montants (Somme)

## Etape 3 : Ajouter des graphiques

### Graphique croise dynamique

1. Selectionnez le TCD
2. **Insertion** > **Graphique croise dynamique**
3. Choisissez le type

### Types recommandes

| Type | Usage |
|------|-------|
| Histogramme | Comparaison |
| Courbe | Evolution |
| Camembert | Repartition |
| Jauge | KPI |

## Etape 4 : Ajouter des segments

### Creation

1. Cliquez sur le TCD ou graphique
2. **Analyse** > **Inserer un segment**
3. Choisissez les champs
4. **OK**

### Placement

Disposez les segments en haut ou a gauche.

## Etape 5 : Chronologie

### Pour les dates

1. **Analyse** > **Inserer une chronologie**
2. Selectionnez le champ date
3. **OK**

## Etape 6 : Mise en page

### Organisation

| Zone | Contenu |
|------|---------|
| Haut | Titre + Filtres |
| Centre | Graphiques principaux |
| Bas | Details ou tableau |

### Masquer le quadrillage

**Affichage** > Decochez **Quadrillage**

## Etape 7 : KPI et indicateurs

### Formules simples

```excel
=SOMME(Tableau[Ventes])
=MOYENNE(Tableau[Marge])
=NB(Tableau[Client])
```

### Mise en forme

- Grande police pour les chiffres cles
- Couleurs pour les tendances
- Icones pour les statuts

## Etape 8 : Connecter les elements

### Segments multi-graphiques

1. Clic droit sur le segment
2. **Connexions de rapport**
3. Cochez tous les TCD
4. **OK**

## Astuces visuelles

### Couleurs coherentes

Utilisez 2-3 couleurs maximum.

### Titres explicites

Chaque graphique doit avoir un titre clair.

### Espacement

Laissez de l'espace entre les elements.

## Erreurs a eviter

| Erreur | Solution |
|--------|----------|
| Trop de graphiques | Limitez a 4-6 |
| Couleurs trop variees | 2-3 couleurs max |
| Pas de titre | Titrez chaque element |
| Trop de details | Synthetisez |

## Bonnes pratiques

1. **Commencez** par definir les KPI cles
2. **Testez** les segments avant de finaliser
3. **Demandez** un feedback utilisateur
4. **Actualisez** regulierement les donnees
