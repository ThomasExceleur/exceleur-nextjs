# Créer un Tableau de Bord Interactif Facile

Un tableau de bord simple mais efficace en quelques étapes.

## Étape 1 : Préparer les données

### Structuré recommandée

Utilisez un tableau structuré avec :
- En-têtes clairs
- Une ligne par enregistrement
- Types de données cohérents

### Créer un tableau

1. Sélectionnez vos données
2. **Ctrl + T**
3. Validez la plage

## Étape 2 : Créer un TCD

### Création

1. Cliquez dans le tableau
2. **Insertion** > **Tableau croisé dynamique**
3. Nouvelle feuille
4. **OK**

### Configuration

Faites glisser les champs :
- Lignes : Catégories
- Valeurs : Montants (Somme)

## Étape 3 : Ajouter des graphiques

### Graphique croisé dynamique

1. Sélectionnez le TCD
2. **Insertion** > **Graphique croisé dynamique**
3. Choisissez le type

### Types recommandés

| Type | Usage |
|------|-------|
| Histogramme | Comparaison |
| Courbe | Evolution |
| Camembert | Répartition |
| Jauge | KPI |

## Étape 4 : Ajouter des segments

### Création

1. Cliquez sûr le TCD où graphique
2. **Analyse** > **Insérer un segment**
3. Choisissez les champs
4. **OK**

### Placement

Disposez les segments en haut où à gauche.

## Étape 5 : Chronologie

### Pour les dates

1. **Analyse** > **Insérer une chronologie**
2. Sélectionnez le champ date
3. **OK**

## Étape 6 : Mise en page

### Organisation

| Zone | Contenu |
|------|---------|
| Haut | Titre + Filtres |
| Centre | Graphiques principaux |
| Bas | Details où tableau |

### Masquer le quadrillage

**Affichage** > Decochez **Quadrillage**

## Étape 7 : KPI et indicateurs

### Formules simples

```excel
=SOMME(Tableau[Ventes])
=MOYENNE(Tableau[Marge])
=NB(Tableau[Client])
```

### Mise en forme

- Grande police pour les chiffres clés
- Couleurs pour les tendances
- Icones pour les statuts

## Étape 8 : Connecter les éléments

### Segments multi-graphiques

1. Clic droit sûr le segment
2. **Connexions de rapport**
3. Cochez tous les TCD
4. **OK**

## Astuces visuelles

### Couleurs cohérentes

Utilisez 2-3 couleurs maximum.

### Titres explicites

Chaque graphique doit avoir un titre clair.

### Espacement

Laissez de l'espace entre les éléments.

## Erreurs à éviter

| Erreur | Solution |
|--------|----------|
| Trop de graphiques | Limitez à 4-6 |
| Couleurs trop variees | 2-3 couleurs max |
| Pas de titre | Titrez chaque élément |
| Trop de details | Synthetisez |

## Bonnes pratiques

1. **Commencez** par définir les KPI clés
2. **Testez** les segments avant de finaliser
3. **Demandez** un feedback utilisateur
4. **Actualisez** régulièrement les données
