# Simplifier vos Calculs avec LET et LAMBDA

La combinaison parfaite pour des formules elegantes.

## LET : Variables locales

### Syntaxe

```excel
=LET(nom1;valeur1;..;calcul)
```

### Exemple

```excel
=LET(
  moy;MOYENNE(A:A);
  ecart;ECARTYPE(A:A);
  (B1-moy)/ecart
)
```

## LAMBDA : Fonctions personnalisees

### Syntaxe

```excel
=LAMBDA(param1;...;expression)
```

### Nommer une fonction

Gestionnaire de noms > Fait reference a : =LAMBDA(...)

## Combinaison LET + LAMBDA

### Fonction avec logique complexe

```excel
Nom: CALCULER_SCORE
=LAMBDA(valeur;
  LET(
    min;0;
    max;100;
    normalise;(valeur-min)/(max-min);
    SI(normalise>0.8;\"A\";SI(normalise>0.6;\"B\";\"C\"))
  )
)
```

## Exemples pratiques

### Calcul de marge

```excel
Nom: MARGE
=LAMBDA(ca;couts;
  LET(
    marge;ca-couts;
    taux;marge/ca;
    SI(taux>0.2;\"Bonne\";\"A ameliorer\")
  )
)
```

### Validation email

```excel
Nom: EMAIL_VALIDE
=LAMBDA(email;
  LET(
    at;CHERCHE(\"@\";email);
    point;CHERCHE(\".\";email;at);
    ET(at>1;point>at+1)
  )
)
```
