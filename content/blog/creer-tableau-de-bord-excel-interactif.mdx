# Créer un Tableau de Bord Excel Interactif

Un tableau de bord efficace présenté les KPI de manière claire et actionnable.

## Planifier le tableau de bord

### Questions clés

- Qui sont les utilisateurs ?
- Quels KPI suivre ?
- Quelle frequence de mise à jour ?
- Quelles interactions nécessaires ?

### Structuré recommandée

| Zone | Contenu |
|------|---------|
| En-tête | Titre, date, filtres |
| KPI | Chiffres clés |
| Graphiques | Visualisations |
| Details | Tableaux de données |

## Préparer les données

### Organisation

1. **Feuille Données** : données brutes
2. **Feuille Calculs** : formules et agregations
3. **Feuille Dashboard** : visualisation

### Convertir en tableaux

**Ctrl+T** pour chaque plage de données.

## Créer les KPI

### Indicateurs avec mise en forme

```excel
=SOMME(Ventes[Montant])
```

### Format conditionnel

| Condition | Affichage |
|-----------|-----------|
| Objectif atteint | Vert |
| Proche | Orange |
| En dessous | Rouge |

### Icones

Utilisez des symboles : ✓ ✗ ● ▲ ▼

## Ajouter des graphiques

### Types recommandés

| Type | Usage |
|------|-------|
| Courbe | Evolution temporelle |
| Barres | Comparaison |
| Camembert | Répartition |
| Jauge | Performance vs objectif |

### Graphiques lies aux tableaux

Les graphiques se mettent à jour automatiquement avec les données.

## Interactivite avec segments

### Ajouter des segments

1. Cliquez dans un TCD lie aux données
2. **Insertion** > **Segment**
3. Sélectionnez les champs de filtre

### Connecter plusieurs graphiques

1. Clic droit sûr le segment
2. **Connexions de rapport**
3. Cochez les TCD/graphiques à connecter

## Chronologie

### Pour les dates

1. Cliquez dans un TCD avec des dates
2. **Insertion** > **Chronologie**
3. Sélectionnez le champ date

### Options de vue

- Années
- Trimestres
- Mois
- Jours

## Listes deroulantes

### Créer un filtre personnalise

1. Validation de données sûr une cellule
2. Source : votre liste de valeurs
3. Liez vos formules à cette cellule

### Exemple

```excel
=SOMME.SI(Ventes[Region];$B$1;Ventes[Montant])
```

Ou B1 contient la region sélectionnée.

## Mise en page

### Design professionnel

| Élément | Recommandation |
|---------|----------------|
| Couleurs | 3-4 maximum |
| Police | Lisible, cohérente |
| Espacement | Aere |
| Alignement | Grille |

### Masquer le quadrillage

**Affichage** > Decochez **Quadrillage**

### Proteger la mise en page

**Révision** > **Proteger la feuille**

## Navigation

### Liens entre feuilles

```excel
=LIEN_HYPERTEXTE("#Details!A1";"Voir les details")
```

### Boutons de navigation

1. **Insertion** > **Formes**
2. Ajoutez un lien hypertexte

## Actualisation

### Données externes

**Données** > **Actualiser tout**

### Automatique à l'ouverture

1. Propriétés de la connexion
2. Cochez **Actualiser à l'ouverture**

## Formules utiles

### Variation en pourcentage

```excel
=(Actuel-Précédent)/Précédent
```

### Objectif atteint

```excel
=SI(Actuel>=Objectif;"✓";"✗")
```

### Tendance

```excel
=SI(Actuel>Précédent;"▲";"▼")
```

## Export et partage

### PDF

**Fichier** > **Exporter** > **PDF**

### Image

Copiez le dashboard comme image.

### Excel Online

Partagez via OneDrive pour l'interactivite.
