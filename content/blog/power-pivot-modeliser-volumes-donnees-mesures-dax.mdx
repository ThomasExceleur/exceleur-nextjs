# Power Pivot : Modeliser des Volumes avec DAX

Gerez des millions de lignes efficacement.

## Avantages Power Pivot

### Capacite

- Millions de lignes
- Compression des donnees
- Moteur de calcul optimise

### Comparaison

| Excel standard | Power Pivot |
|---------------|-------------|
| 1M lignes max | Illimite |
| RAM intensive | Compresse |
| Formules cellules | Mesures DAX |

## Architecture optimale

### Schema en etoile

- Table de faits (transactions)
- Tables de dimensions (reference)
- Relations one-to-many

## Mesures performantes

### Bonnes pratiques

```dax
// Bon : mesure simple
Total := SUM(Ventes[Montant])

// A eviter : colonne calculee non necessaire
```

### CALCULATE efficace

```dax
VentesNord := CALCULATE([Total]; Region[Nom]=\"Nord\")
```

## Contexte et performance

### Comprendre le contexte

Les mesures DAX sont calculees dans un contexte de filtre.

### Eviter les calculs ligne par ligne

Preferez les agregations DAX aux colonnes calculees.
