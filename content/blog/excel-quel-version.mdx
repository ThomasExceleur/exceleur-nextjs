# Quelle Version d'Excel Utilisez-Vous ?

Connaitre sa version d'Excel est important pour utiliser les bonnes fonctionnalites.

## Methode rapide

### Via le menu Fichier

1. Ouvrez Excel
2. Cliquez sur **Fichier**
3. Cliquez sur **Compte**
4. Lisez "A propos d'Excel"

## Versions disponibles

### Versions actuelles

| Version | Annee | Type |
|---------|-------|------|
| Microsoft 365 | Continue | Abonnement |
| Excel 2021 | 2021 | Achat unique |
| Excel 2019 | 2018 | Achat unique |
| Excel 2016 | 2015 | Achat unique |

### Versions anciennes

| Version | Annee |
|---------|-------|
| Excel 2013 | 2013 |
| Excel 2010 | 2010 |
| Excel 2007 | 2007 |

## Differences entre versions

### Excel 365 vs versions perpetuelles

| Fonctionnalite | 365 | 2021 | 2019 |
|----------------|-----|------|------|
| Mises a jour | Continues | Limitees | Limitees |
| RECHERCHEX | Oui | Oui | Non |
| LET, LAMBDA | Oui | Oui | Non |
| Copilot | Oui | Non | Non |

### Nouvelles fonctions par version

**Excel 365 / 2021** :
- RECHERCHEX
- FILTRE
- TRIER
- UNIQUE
- LET
- LAMBDA

**Excel 2019** :
- CONCAT
- JOINDRE.TEXTE
- SI.CONDITIONS
- SI.MULTIPLE

## Comment mettre a jour

### Microsoft 365

Les mises a jour sont automatiques.

### Versions perpetuelles

Seules les mises a jour de securite sont disponibles.

## Quelle version choisir ?

### Pour les particuliers

- **Microsoft 365 Personnel** : mises a jour continues
- **Office 2021** : achat unique sans abonnement

### Pour les entreprises

- **Microsoft 365 Business** : collaboration cloud
- **Office LTSC** : environnements deconnectes

## Compatibilite des fichiers

### Formats

| Extension | Compatible avec |
|-----------|-----------------|
| .xlsx | Excel 2007+ |
| .xlsm | Excel 2007+ (macros) |
| .xls | Toutes versions |

### Attention

Les nouvelles fonctions ne fonctionnent pas dans les anciennes versions.
