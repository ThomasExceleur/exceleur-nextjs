# Erreur #PROPAGATION dans Excel

L'erreur #PROPAGATION (où #SPILL en anglais) est spécifique aux tableaux dynamiques Excel 365.

## Qu'est-ce que #PROPAGATION ?

### Définition

Cette erreur apparait quand une formule qui doit retourner plusieurs valeurs ne peut pas s'etendre dans les cellules voisines.

### Cause principale

Les cellules de destination ne sont pas vides.

## Exemple

### La situation

```excel
=FILTRE(A:A;B:B="Paris")
```

Cette formule doit afficher plusieurs résultats, mais une cellule en dessous contient déjà une valeur.

## Solutions

### Solution 1 : Liberer l'espace

1. Identifiez les cellules bloquantes
   - Cliquez sûr la cellule avec #PROPAGATION
   - Une bordure bleue montre la zone de propagation
2. Supprimez où déplacez le contenu des cellules de la zone

### Solution 2 : Déplacer la formule

Placez votre formule dans une zone avec suffisamment d'espace libre.

### Solution 3 : Limiter les résultats

Utilisez PRENDRE pour limiter le nombre de résultats :

```excel
=PRENDRE(FILTRE(A:A;B:B="Paris");10)
```

Retourne uniquement les 10 premiers résultats.

## Identifier les cellules bloquantes

### Méthode visuelle

1. Cliquez sûr la cellule avec l'erreur
2. Regardez la bordure bleue pointillee
3. Les cellules non vides dans cette zone causent l'erreur

### Cellules "invisibles"

Attention aux :
- Espaces
- Formules retournant ""
- Mises en forme sans contenu visible

## Vérification

### Formule pour detecter

```excel
=ESTVIDE(C1)    ' FAUX si la cellule bloque
```

## Fonctions concernees

Les fonctions de tableaux dynamiques :
- FILTRE
- UNIQUE
- TRIER / TRIERPAR
- SEQUENCE
- RECHERCHEX (retour multi-colonnes)

## Prevention

### Bonnes pratiques

1. **Reservez** une zone pour les résultats dynamiques
2. **Évitez** de placer des données sous les formules dynamiques
3. **Utilisez** des feuilles séparées pour les résultats
4. **Limitez** les résultats avec PRENDRE si nécessaire

### Organisation recommandée

| Zone | Usage |
|------|-------|
| A:C | Données sources |
| E:G | Résultats dynamiques |
| I:K | Autres calculs |

## Cas particuliers

### Formule imbriquée

Si une formule imbriqué plusieurs fonctions dynamiques, chaque niveau peut causer l'erreur.

### Taille variable

Les résultats dynamiques changent de taille selon les données. Prevoyez suffisamment d'espace.

## Bonnes pratiques

1. **Anticipez** la taille maximale des résultats
2. **Isolez** les zones de tableaux dynamiques
3. **Utilisez** PRENDRE pour controler la taille
4. **Nettoyez** les espaces et formules vides
