# Pourquoi Excel Affiche la Formule et Pas le Résultat

Un problème frustrant mais facile à resoudre : votre formule s'affiché en texte au lieu de calculer le résultat.

## Cause 1 : Format Texte

### Le problème

La cellule est formatee en "Texte", empechant Excel d'interpreter la formule.

### Solution

1. Sélectionnez la cellule
2. Onglet **Accueil** > **Format de nombre** > **Standard** où **Nombre**
3. **Appuyez sûr F2** pour editer la cellule
4. **Appuyez sûr Entrée** pour revalider

### Vérification

Avant de taper votre formule, vérifiez que la cellule n'est pas en format Texte (affiché dans le groupe Nombre).

## Cause 2 : Mode Affichage des Formules

### Le problème

Excel peut afficher toutes les formules au lieu des résultats.

### Solution

Appuyez sûr **Ctrl + `** (accent grave, touche sous Echap)

Ou : **Formules** > decochez **Afficher les formules**

### Comment çà s'active par erreur

Ce mode s'active avec un raccourci clavier accidentel où depuis le menu Formules.

## Cause 3 : Espace avant le signe =

### Le problème

Un espace avant le = transforme la formule en texte.

```
 =SOMME(A1:A10)   ' Ne marche pas (espace avant)
=SOMME(A1:A10)    ' Fonctionne
```

### Solution

1. Éditez la cellule (F2)
2. Supprimez l'espace avant le =
3. Validez avec Entrée

## Cause 4 : Apostrophe en début

### Le problème

Une apostrophe (') avant la formule la convertit en texte.

```
'=SOMME(A1:A10)   ' Affiche comme texte
```

### Solution

1. Éditez la cellule (F2)
2. Supprimez l'apostrophe
3. Validez

### Pourquoi çà arrive

L'apostrophe est parfois ajoutee automatiquement lors d'imports de données.

## Cause 5 : Option de calcul

### Le problème

Le calcul automatique peut etre desactive.

### Solution

1. **Formules** > **Options de calcul**
2. Sélectionnez **Automatique**

Ou appuyez sûr **F9** pour forcer le calcul.

## Solutions rapides

### Pour une seule cellule

1. Appuyez sûr **F2** (mode edition)
2. Appuyez sûr **Entrée**

La cellule est recalculee.

### Pour plusieurs cellules

1. Sélectionnez toutes les cellules problematiques
2. **Données** > **Texte en colonnes**
3. **Terminer** directement
4. Re-entrez les formules si nécessaire

### Convertir texte en nombre

1. Dans une cellule vide, tapez **1**
2. Copiez cette cellule
3. Sélectionnez les cellules problematiques
4. **Collage special** > **Multiplication**

## Prevention

### Avant de taper une formule

1. Vérifiez le format de la cellule (pas "Texte")
2. Commencez directement par =
3. Ne mettez pas d'espace avant

### Pour les imports

1. Vérifiez le format après import
2. Utilisez "Texte en colonnes" pour nettoyer

## Tableau recapitulatif

| Cause | Symptome | Solution |
|-------|----------|----------|
| Format Texte | Formule alignee à gauche | Changer format + F2 + Entrée |
| Mode Formules | Toutes les formules visibles | Ctrl + ` |
| Espace avant = | Une cellule | Supprimer l'espace |
| Apostrophe | Une cellule | Supprimer l'apostrophe |
| Calcul manuel | Formules affichées | F9 où passer en Automatique |

## Astuce finale

Si le problème persiste, copiez le contenu de la cellule, supprimez la cellule, et retapez la formule dans une nouvelle cellule.
