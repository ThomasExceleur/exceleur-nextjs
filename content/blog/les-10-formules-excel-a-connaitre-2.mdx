# Les 10 Formules Excel Essentielles

Maitrisez ces fonctions pour etre efficace.

## 1. SOMME

### Syntaxe

```excel
=SOMME(A1:A10)
```

Additionne une plage de cellules.

## 2. SI

### Syntaxe

```excel
=SI(condition;si_vrai;si_faux)
```

Test conditionnel.

## 3. RECHERCHEV / RECHERCHEX

### RECHERCHEV

```excel
=RECHERCHEV(valeur;tableau;colonne;FAUX)
```

### RECHERCHEX (moderne)

```excel
=RECHERCHEX(valeur;plage_recherche;plage_resultat)
```

## 4. NB.SI

### Syntaxe

```excel
=NB.SI(plage;critere)
```

Compte selon une condition.

## 5. SOMME.SI

### Syntaxe

```excel
=SOMME.SI(plage_critere;critere;plage_somme)
```

Somme conditionnelle.

## 6. MOYENNE

### Syntaxe

```excel
=MOYENNE(A1:A10)
```

Calcule la moyenne.

## 7. MAX / MIN

### Syntaxe

```excel
=MAX(A1:A10)
=MIN(A1:A10)
```

Valeurs extremes.

## 8. CONCATENER / CONCAT

### Syntaxe

```excel
=A1&" "&B1
=CONCAT(A1;" ";B1)
```

Assembler du texte.

## 9. GAUCHE / DROITE / STXT

### Syntaxe

```excel
=GAUCHE(A1;5)
=DROITE(A1;3)
=STXT(A1;2;4)
```

Extraire du texte.

## 10. AUJOURDHUI / MAINTENANT

### Syntaxe

```excel
=AUJOURDHUI()
=MAINTENANT()
```

Date et heure actuelles.
