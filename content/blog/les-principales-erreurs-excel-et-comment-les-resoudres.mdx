# Les Principales Erreurs Excel

Comprendre et resoudre les erreurs de formules dans Excel.

## #N/A - Valeur Non Disponible

### Cause
La valeur recherchee n'existe pas.

### Solution
```excel
=SIERREUR(RECHERCHEV(A1;B:C;2;FAUX);"Non trouve")
```

## #VALEUR! - Type Incorrect

### Cause
Operation entre types incompatibles (texte et nombre).

### Solution
```excel
=SOMME(SI(ESTNUM(A1:A10);A1:A10;0))
```

Ou nettoyez les donnees avec CNUM.

## #REF! - Reference Invalide

### Cause
Une cellule ou plage referencee a ete supprimee.

### Solution
1. **Annulez** avec Ctrl + Z si possible
2. **Corrigez** manuellement les references
3. **Evitez** de supprimer des cellules referencees

## #DIV/0! - Division par Zero

### Cause
Division par une cellule vide ou contenant zero.

### Solution
```excel
=SI(B1=0;0;A1/B1)
```

Ou avec SIERREUR :
```excel
=SIERREUR(A1/B1;0)
```

## #NOM? - Nom Non Reconnu

### Cause
- Nom de fonction mal orthographie
- Nom defini inexistant
- Texte sans guillemets

### Solution
```excel
' Incorrect
=RECHCHEV(A1;B:C;2)

' Correct
=RECHERCHEV(A1;B:C;2;FAUX)
```

## #NOMBRE! - Nombre Invalide

### Cause
- Argument numerique hors limites
- Calcul impossible

### Solution
Verifiez les valeurs d'entree de la formule.

## #NUL! - Intersection Nulle

### Cause
Deux plages qui ne se croisent pas.

### Solution
Utilisez le bon operateur :
- `;` pour union
- `:` pour plage
- ` ` (espace) pour intersection

## Tableau recapitulatif

| Erreur | Cause | Solution rapide |
|--------|-------|-----------------|
| #N/A | Valeur absente | SIERREUR |
| #VALEUR! | Type incorrect | CNUM, ESTNUM |
| #REF! | Reference supprimee | Corriger manuellement |
| #DIV/0! | Division par zero | SI(B=0;0;A/B) |
| #NOM? | Faute d'orthographe | Verifier syntaxe |
| #NOMBRE! | Nombre invalide | Verifier arguments |
| #NUL! | Pas d'intersection | Corriger operateurs |

## La fonction universelle : SIERREUR

### Syntaxe

```excel
=SIERREUR(formule;valeur_si_erreur)
```

### Exemple

```excel
=SIERREUR(A1/B1;"Erreur de calcul")
```

## Audit des formules

### Outils Excel

1. **Formules** > **Afficher les formules**
2. **Formules** > **Verification des erreurs**
3. **Formules** > **Evaluer la formule**

## Bonnes pratiques

1. **Encadrez** vos formules avec SIERREUR
2. **Testez** sur des valeurs connues
3. **Validez** les donnees d'entree
4. **Documentez** les formules complexes
