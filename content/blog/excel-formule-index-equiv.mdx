# INDEX EQUIV : Le Duo Puissant

La combinaison INDEX + EQUIV offre plus de flexibilite que RECHERCHEV pour les recherches dans Excel.

## Comprendre chaque fonction

### EQUIV (MATCH)

Trouve la position d'une valeur dans une plage.

```excel
=EQUIV(valeur_cherchee; plage_recherche; [type])
```

- **valeur_cherchee** : Ce que vous recherchez
- **plage_recherche** : Ou chercher (ligne ou colonne)
- **type** : 0 = correspondance exacte

### INDEX

Retourne la valeur a une position donnee.

```excel
=INDEX(plage; no_ligne; [no_colonne])
```

- **plage** : La plage de donnees
- **no_ligne** : Numero de ligne
- **no_colonne** : Numero de colonne (optionnel)

## La combinaison magique

### Formule de base

```excel
=INDEX(plage_resultat; EQUIV(valeur_cherchee; plage_recherche; 0))
```

### Exemple concret

Tableau de donnees en A1:C4 :

| Code | Produit | Prix |
|------|---------|------|
| A01 | Stylo | 2,50 |
| A02 | Cahier | 5,00 |
| A03 | Classeur | 8,00 |

Pour trouver le prix de A02 :

```excel
=INDEX(C1:C4; EQUIV("A02"; A1:A4; 0))
```

Resultat : 5,00

## Avantages sur RECHERCHEV

### Recherche vers la gauche

RECHERCHEV ne cherche que vers la droite. INDEX EQUIV n'a pas cette limite.

```excel
=INDEX(A:A; EQUIV("Cahier"; B:B; 0))
```

Trouve le code du produit "Cahier".

### Robustesse aux insertions

Le numero de colonne dans RECHERCHEV pose probleme si vous inserez des colonnes. INDEX EQUIV reste stable.

### Performances

INDEX EQUIV est souvent plus rapide sur de grands tableaux.

## Recherche bidimensionnelle

### Trouver une valeur dans un tableau

```excel
=INDEX(donnees; EQUIV(valeur_ligne; entetes_lignes; 0); EQUIV(valeur_colonne; entetes_colonnes; 0))
```

### Exemple

| | Jan | Fev | Mar |
|---|-----|-----|-----|
| Nord | 100 | 150 | 120 |
| Sud | 200 | 180 | 210 |

```excel
=INDEX(B2:D3; EQUIV("Sud"; A2:A3; 0); EQUIV("Fev"; B1:D1; 0))
```

Resultat : 180

## Correspondance approximative

### Type 1 (valeur inferieure)

```excel
=EQUIV(75; A:A; 1)
```

Trouve la plus grande valeur inferieure ou egale a 75.

**Attention** : Les donnees doivent etre triees par ordre croissant.

### Type -1 (valeur superieure)

```excel
=EQUIV(75; A:A; -1)
```

Trouve la plus petite valeur superieure ou egale a 75.

**Attention** : Les donnees doivent etre triees par ordre decroissant.

## Gestion des erreurs

### Avec SIERREUR

```excel
=SIERREUR(INDEX(C:C; EQUIV(E1; A:A; 0)); "Non trouve")
```

### Avec SI.NON.DISP

```excel
=SI.NON.DISP(INDEX(C:C; EQUIV(E1; A:A; 0)); "Non trouve")
```

## Cas pratiques

### Derniere valeur d'une colonne

```excel
=INDEX(A:A; EQUIV(9.99E+307; A:A; 1))
```

Pour du texte :
```excel
=INDEX(A:A; EQUIV("zzz"; A:A; 1))
```

### Recherche avec caracteres joker

```excel
=INDEX(B:B; EQUIV("*pomme*"; A:A; 0))
```

### Recherche multi-criteres

```excel
=INDEX(C:C; EQUIV(1; (A:A=E1)*(B:B=F1); 0))
```

Valider avec Ctrl + Maj + Entree (anciennes versions).

## Comparaison avec RECHERCHEV

| Aspect | RECHERCHEV | INDEX EQUIV |
|--------|------------|-------------|
| Direction | Droite uniquement | Toutes |
| Colonnes inserees | Probleme | Robuste |
| Performance | Bonne | Meilleure |
| Lisibilite | Simple | Plus complexe |
| Flexibilite | Limitee | Elevee |

## RECHERCHEX : Le meilleur des deux

Dans Excel 365, RECHERCHEX combine les avantages :

```excel
=RECHERCHEX(valeur; plage_recherche; plage_resultat; "Non trouve")
```

## Astuces avancees

### Retourner plusieurs colonnes

```excel
=INDEX(A:D; EQUIV(E1; A:A; 0); 0)
```

Retourne la ligne entiere.

### Avec des noms de plages

```excel
=INDEX(Prix; EQUIV(Code_cherche; Codes; 0))
```

Plus lisible avec des plages nommees.

## Bonnes pratiques

1. **Utilisez 0 pour EQUIV** : Correspondance exacte
2. **Evitez les colonnes entieres** : Limitez les plages
3. **Nommez vos plages** : Ameliore la lisibilite
4. **Testez les erreurs** : Utilisez SIERREUR
