# Comment Obtenir Excel Gratuitement

Plusieurs options legales existent pour utiliser Excel sans payer.

## Option 1 : Excel pour le web

### Gratuit et officiel

Microsoft propose Excel en ligne gratuitement.

### Acces

1. Allez sur [office.com](https://www.office.com)
2. Connectez-vous avec un compte Microsoft (gratuit)
3. Cliquez sur Excel

### Avantages

- Gratuit a vie
- Aucune installation
- Sauvegarde automatique sur OneDrive
- Collaboration en temps reel

### Limitations

- Necessite une connexion internet
- Pas toutes les fonctionnalites
- Pas de macros VBA

## Option 2 : Application mobile

### Excel pour smartphone et tablette

Gratuit sur Android et iOS pour les appareils de moins de 10.1".

### Installation

- [Google Play Store](https://play.google.com/store)
- [Apple App Store](https://www.apple.com/app-store/)

## Option 3 : Essai gratuit

### Microsoft 365 - 1 mois gratuit

1. Allez sur [microsoft.com/microsoft-365](https://www.microsoft.com)
2. Choisissez un plan
3. Demarrez l'essai gratuit
4. Annulez avant la fin si vous ne souhaitez pas payer

## Option 4 : Education

### Etudiants et enseignants

De nombreux etablissements offrent Microsoft 365 gratuitement.

1. Verifiez aupres de votre etablissement
2. Utilisez votre email educatif pour vous inscrire

## Alternatives gratuites

### Google Sheets

- Gratuit avec un compte Google
- Compatible avec les fichiers Excel
- Collaboration en temps reel

### LibreOffice Calc

- Logiciel libre et gratuit
- Installation locale
- Compatible Excel (.xlsx)

### WPS Office

- Version gratuite disponible
- Interface similaire a Excel
- Disponible sur toutes les plateformes

## Comparaison

| Option | Hors ligne | VBA | Gratuit |
|--------|------------|-----|---------|
| Excel Online | Non | Non | Oui |
| Excel Mobile | Oui | Non | Oui* |
| Google Sheets | Non | Non | Oui |
| LibreOffice | Oui | Partiel | Oui |

*Gratuit sur petits ecrans

## Fonctionnalites selon la version

### Excel Online inclut

- Formules de base
- Tableaux et graphiques
- Mise en forme
- Filtres et tris

### Excel Online n'inclut pas

- Macros VBA
- Power Query complet
- Power Pivot
- Certaines fonctions avancees

## Recommendation

| Besoin | Meilleure option |
|--------|------------------|
| Usage occasionnel | Excel Online |
| Mobile | Application Excel |
| Hors ligne gratuit | LibreOffice |
| Collaboration | Google Sheets |
