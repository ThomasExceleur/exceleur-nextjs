# Introduction a Power Pivot : Modeliser vos Donnees avec DAX

Power Pivot transforme Excel en outil d'analyse de donnees puissant.

## Qu'est-ce que Power Pivot ?

### Definition

Power Pivot est un complement Excel pour :
- Gerer des millions de lignes
- Creer des modeles de donnees
- Ecrire des formules DAX

### Activer Power Pivot

1. **Fichier** > **Options** > **Complements**
2. Gerer : Complements COM
3. Cochez Microsoft Power Pivot for Excel

## Le modele de donnees

### Concept

Un modele de donnees relie plusieurs tables entre elles.

### Composants

| Element | Description |
|---------|-------------|
| Tables | Ensembles de donnees |
| Relations | Liens entre tables |
| Mesures | Calculs DAX |
| Colonnes calculees | Valeurs derivees |

## Importer des donnees

### Sources

- Fichiers Excel
- Bases de donnees SQL
- Fichiers CSV
- Sources web

### Methode

1. **Power Pivot** > **Gerer**
2. **Accueil** > **Obtenir des donnees externes**
3. Selectionnez la source

## Creer des relations

### Types de relations

| Type | Description |
|------|-------------|
| Un a plusieurs | Le plus courant |
| Un a un | Cas particuliers |

### Etablir une relation

1. **Diagramme** dans Power Pivot
2. Glissez un champ sur l'autre
3. Ou **Conception** > **Creer une relation**

## Introduction a DAX

### Qu'est-ce que DAX ?

Data Analysis Expressions - langage de formules pour Power Pivot.

### Syntaxe de base

```dax
Mesure = FONCTION(arguments)
```

## Fonctions DAX essentielles

### Agregations

```dax
Total Ventes = SUM(Ventes[Montant])
Moyenne = AVERAGE(Ventes[Montant])
Compte = COUNT(Ventes[ID])
```

### Fonctions de filtre

```dax
Ventes France = CALCULATE(
    SUM(Ventes[Montant]);
    Ventes[Pays]="France"
)
```

## Mesures vs Colonnes calculees

### Colonnes calculees

- Calculees a l'import
- Stockees dans le modele
- Utilisent le contexte de ligne

```dax
Marge = [PrixVente] - [Cout]
```

### Mesures

- Calculees a la volee
- Non stockees
- Utilisent le contexte de filtre

```dax
Total Marge = SUM(Ventes[Marge])
```

## Contexte d'evaluation

### Contexte de ligne

S'applique ligne par ligne (colonnes calculees).

### Contexte de filtre

Determine par les filtres actifs (mesures dans TCD).

## Mesures courantes

### Pourcentage du total

```dax
% du Total =
DIVIDE(
    SUM(Ventes[Montant]);
    CALCULATE(SUM(Ventes[Montant]); ALL(Ventes))
)
```

### Variation annuelle

```dax
Ventes Annee Precedente =
CALCULATE(
    SUM(Ventes[Montant]);
    SAMEPERIODLASTYEAR(Calendrier[Date])
)
```

### Cumul annuel

```dax
Cumul YTD =
TOTALYTD(SUM(Ventes[Montant]); Calendrier[Date])
```

## Utiliser dans les TCD

### Ajouter une mesure

1. Creez la mesure dans Power Pivot
2. Elle apparait dans les champs du TCD
3. Glissez dans les valeurs

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Nommage | Noms clairs pour les mesures |
| Documentation | Commentez les calculs complexes |
| Performance | Evitez les colonnes calculees inutiles |
| Test | Verifiez avec des valeurs connues |

## Erreurs courantes

| Erreur | Cause | Solution |
|--------|-------|----------|
| Valeur vide | Division par zero | Utilisez DIVIDE() |
| Contexte incorrect | Mauvaise comprehension | Etudiez le contexte |
| Performance | Calculs lourds | Optimisez les mesures |
