# Introduction à Power Pivot : Modeliser vos Données avec DAX

Power Pivot transforme Excel en outil d'analyse de données puissant.

## Qu'est-ce que Power Pivot ?

### Définition

Power Pivot est un complement Excel pour :
- Gérer des millions de lignes
- Créer des modèles de données
- Écrire des formules DAX

### Activer Power Pivot

1. **Fichier** > **Options** > **Complements**
2. Gérer : Complements COM
3. Cochez Microsoft Power Pivot for Excel

## Le modèle de données

### Concept

Un modèle de données relie plusieurs tables entre elles.

### Composants

| Élément | Description |
|---------|-------------|
| Tables | Ensembles de données |
| Relations | Liens entre tables |
| Mesures | Calculs DAX |
| Colonnes calculees | Valeurs derivees |

## Importer des données

### Sources

- Fichiers Excel
- Bases de données SQL
- Fichiers CSV
- Sources web

### Méthode

1. **Power Pivot** > **Gérer**
2. **Accueil** > **Obtenir des données externes**
3. Sélectionnez la source

## Créer des relations

### Types de relations

| Type | Description |
|------|-------------|
| Un à plusieurs | Le plus courant |
| Un à un | Cas particuliers |

### Etablir une relation

1. **Diagramme** dans Power Pivot
2. Glissez un champ sûr l'autre
3. Ou **Conception** > **Créer une relation**

## Introduction à DAX

### Qu'est-ce que DAX ?

Data Analysis Expressions - langage de formules pour Power Pivot.

### Syntaxe de base

```dax
Mesure = FONCTION(arguments)
```

## Fonctions DAX essentielles

### Agregations

```dax
Total Ventes = SUM(Ventes[Montant])
Moyenne = AVERAGE(Ventes[Montant])
Compte = COUNT(Ventes[ID])
```

### Fonctions de filtre

```dax
Ventes France = CALCULATE(
    SUM(Ventes[Montant]);
    Ventes[Pays]="France"
)
```

## Mesures vs Colonnes calculees

### Colonnes calculees

- Calculees à l'import
- Stockees dans le modèle
- Utilisent le contexte de ligne

```dax
Marge = [PrixVente] - [Cout]
```

### Mesures

- Calculees à la volee
- Non stockees
- Utilisent le contexte de filtre

```dax
Total Marge = SUM(Ventes[Marge])
```

## Contexte d'evaluation

### Contexte de ligne

S'applique ligne par ligne (colonnes calculees).

### Contexte de filtre

Determine par les filtres actifs (mesures dans TCD).

## Mesures courantes

### Pourcentage du total

```dax
% du Total =
DIVIDE(
    SUM(Ventes[Montant]);
    CALCULATE(SUM(Ventes[Montant]); ALL(Ventes))
)
```

### Variation annuelle

```dax
Ventes Année Précédente =
CALCULATE(
    SUM(Ventes[Montant]);
    SAMEPERIODLASTYEAR(Calendrier[Date])
)
```

### Cumul annuel

```dax
Cumul YTD =
TOTALYTD(SUM(Ventes[Montant]); Calendrier[Date])
```

## Utiliser dans les TCD

### Ajouter une mesure

1. Créez la mesure dans Power Pivot
2. Elle apparait dans les champs du TCD
3. Glissez dans les valeurs

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Nommage | Noms clairs pour les mesures |
| Documentation | Commentez les calculs complexes |
| Performance | Évitez les colonnes calculees inutiles |
| Test | Vérifiez avec des valeurs connues |

## Erreurs courantes

| Erreur | Cause | Solution |
|--------|-------|----------|
| Valeur vide | Division par zero | Utilisez DIVIDE() |
| Contexte incorrect | Mauvaise compréhension | Etudiez le contexte |
| Performance | Calculs lourds | Optimisez les mesures |
