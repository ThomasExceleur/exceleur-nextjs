# Creer une Heatmap dans Excel

Visualisez vos donnees avec des couleurs.

## Methode : Echelles de couleurs

### Etapes

1. Selectionnez vos donnees
2. **Accueil** > **Mise en forme conditionnelle**
3. **Echelles de couleurs**
4. Choisissez un schema

## Schemas disponibles

| Schema | Usage |
|--------|-------|
| Vert-Rouge | Performance |
| Bleu-Rouge | Temperature |
| Blanc-Couleur | Intensite |

## Personnaliser

### Regles avancees

1. **Mise en forme conditionnelle** > **Gerer les regles**
2. **Nouvelle regle** > **Echelle a 3 couleurs**

### Options

| Option | Description |
|--------|-------------|
| Minimum | Couleur pour min |
| Milieu | Couleur intermediaire |
| Maximum | Couleur pour max |

## Types de valeurs

| Type | Usage |
|------|-------|
| Min/Max | Automatique |
| Nombre | Valeur fixe |
| Centile | Relatif |
| Formule | Calcule |

## Masquer les valeurs

### Afficher couleur seulement

1. Selectionnez les cellules
2. Format personnalise : `;;;`

## Cas d'usage

| Application | Exemple |
|-------------|---------|
| Ventes | Performance par region |
| RH | Absences par mois |
| Finance | Ecarts budget |

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Coherence | Meme echelle partout |
| Legende | Expliquez les couleurs |
| Contraste | Couleurs distinguables |
