# Créer une Heatmap dans Excel

Visualisez vos données avec des couleurs.

## Méthode : Échelles de couleurs

### Étapes

1. Sélectionnez vos données
2. **Accueil** > **Mise en forme conditionnelle**
3. **Échelles de couleurs**
4. Choisissez un schéma

## Schémas disponibles

| Schéma | Usage |
|--------|-------|
| Vert-Rouge | Performance |
| Bleu-Rouge | Temperature |
| Blanc-Couleur | Intensite |

## Personnaliser

### Règles avancées

1. **Mise en forme conditionnelle** > **Gérer les règles**
2. **Nouvelle règle** > **Échelle à 3 couleurs**

### Options

| Option | Description |
|--------|-------------|
| Minimum | Couleur pour min |
| Milieu | Couleur intermediaire |
| Maximum | Couleur pour max |

## Types de valeurs

| Type | Usage |
|------|-------|
| Min/Max | Automatique |
| Nombre | Valeur fixe |
| Centile | Relatif |
| Formule | Calcule |

## Masquer les valeurs

### Afficher couleur seulement

1. Sélectionnez les cellules
2. Format personnalise : `;;;`

## Cas d'usage

| Application | Exemple |
|-------------|---------|
| Ventes | Performance par region |
| RH | Absences par mois |
| Finance | Ecarts budget |

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Cohérence | Meme échelle partout |
| Legende | Expliquez les couleurs |
| Contraste | Couleurs distinguables |
