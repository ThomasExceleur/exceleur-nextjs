# Quelle Version d'Excel dans Office 365

Office 365 (maintenant Microsoft 365) inclut toujours la derniere version d'Excel.

## Version actuelle

### Excel dans Microsoft 365

Avec un abonnement Microsoft 365, vous avez toujours acces a la version la plus recente d'Excel, avec des mises a jour automatiques.

### Trouver votre version

1. Ouvrez Excel
2. **Fichier** > **Compte**
3. Regardez "A propos d'Excel"

Ou :
1. **Fichier** > **Aide** > **A propos de Microsoft Excel**

## Informations affichees

### Numero de version

```
Version XXXX (Build XXXXX.XXXXX)
```

### Canal de mise a jour

- Canal actuel (mises a jour mensuelles)
- Canal semi-annuel (mises a jour semestrielles)

## Difference Office 365 vs Office perpetuel

### Microsoft 365 (abonnement)

| Aspect | Detail |
|--------|--------|
| Mises a jour | Continues |
| Nouvelles fonctions | Oui |
| Support | Continu |
| Prix | Mensuel/Annuel |

### Office 2021 (achat unique)

| Aspect | Detail |
|--------|--------|
| Mises a jour | Securite uniquement |
| Nouvelles fonctions | Non |
| Support | Limite |
| Prix | Unique |

## Fonctionnalites exclusives Microsoft 365

### Fonctions recentes

- FILTRE
- TRIER
- UNIQUE
- RECHERCHEX
- JOINDRE.TEXTE
- LET
- LAMBDA

### Services cloud

- Co-edition en temps reel
- Enregistrement automatique OneDrive
- Historique des versions

### Intelligence artificielle

- Idees (analyses automatiques)
- Types de donnees lies
- Copilot (assistant IA)

## Plans Microsoft 365

### Pour les particuliers

| Plan | Contenu |
|------|---------|
| Microsoft 365 Personnel | 1 utilisateur |
| Microsoft 365 Famille | Jusqu'a 6 utilisateurs |

### Pour les entreprises

| Plan | Contenu |
|------|---------|
| Microsoft 365 Business Basic | Web uniquement |
| Microsoft 365 Business Standard | Desktop + Web |
| Microsoft 365 Business Premium | + Securite avancee |
| Microsoft 365 Apps for Enterprise | Desktop complet |

## Verifier les fonctionnalites disponibles

### Dans Excel

1. **Fichier** > **Compte**
2. Verifiez votre plan
3. Consultez "Nouveautes"

### En ligne

Consultez la page Microsoft pour les fonctionnalites par plan.

## Mettre a jour Excel

### Manuellement

1. **Fichier** > **Compte**
2. **Options de mise a jour**
3. **Mettre a jour**

### Verifier les mises a jour

1. **Options de mise a jour** > **Afficher les mises a jour**
2. Verifiez le statut

## Problemes de version

### Fonctionnalite manquante

1. Verifiez votre version
2. Mettez a jour Excel
3. Verifiez votre plan (certaines fonctions sont exclusives)

### Version anterieure

Si vous voyez "Office 2019" ou "Office 2021" :
- Vous n'avez pas d'abonnement Microsoft 365
- Vous avez une licence perpetuelle

## Passer a Microsoft 365

### Avantages

1. Toujours la derniere version
2. Nouvelles fonctionnalites
3. Support cloud
4. Mises a jour de securite

### Comment migrer

1. Souscrivez a Microsoft 365
2. Telechargez depuis office.com
3. Connectez-vous avec votre compte Microsoft
