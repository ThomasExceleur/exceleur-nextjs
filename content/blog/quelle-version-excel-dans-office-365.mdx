# Quelle Version d'Excel dans Office 365

Office 365 (maintenant Microsoft 365) inclut toujours la dernière version d'Excel.

## Version actuelle

### Excel dans Microsoft 365

Avec un abonnement Microsoft 365, vous avez toujours accès à la version la plus recente d'Excel, avec des mises à jour automatiques.

### Trouver votre version

1. Ouvrez Excel
2. **Fichier** > **Compte**
3. Regardez "A propos d'Excel"

Ou :
1. **Fichier** > **Aide** > **A propos de Microsoft Excel**

## Informations affichées

### Numéro de version

```
Version XXXX (Build XXXXX.XXXXX)
```

### Canal de mise à jour

- Canal actuel (mises à jour mensuelles)
- Canal semi-annuel (mises à jour semestrielles)

## Différence Office 365 vs Office perpetuel

### Microsoft 365 (abonnement)

| Aspect | Detail |
|--------|--------|
| Mises à jour | Continues |
| Nouvelles fonctions | Oui |
| Support | Continu |
| Prix | Mensuel/Annuel |

### Office 2021 (achat unique)

| Aspect | Detail |
|--------|--------|
| Mises à jour | Sécurité uniquement |
| Nouvelles fonctions | Non |
| Support | Limite |
| Prix | Unique |

## Fonctionnalités exclusives Microsoft 365

### Fonctions recentes

- FILTRE
- TRIER
- UNIQUE
- RECHERCHEX
- JOINDRE.TEXTE
- LET
- LAMBDA

### Services cloud

- Co-edition en temps reel
- Enregistrement automatique OneDrive
- Historique des versions

### Intelligence artificielle

- Idées (analyses automatiques)
- Types de données lies
- Copilot (assistant IA)

## Plans Microsoft 365

### Pour les particuliers

| Plan | Contenu |
|------|---------|
| Microsoft 365 Personnel | 1 utilisateur |
| Microsoft 365 Famille | Jusqu'à 6 utilisateurs |

### Pour les entreprises

| Plan | Contenu |
|------|---------|
| Microsoft 365 Business Basic | Web uniquement |
| Microsoft 365 Business Standard | Desktop + Web |
| Microsoft 365 Business Premium | + Sécurité avancée |
| Microsoft 365 Apps for Enterprise | Desktop complet |

## Vérifier les fonctionnalités disponibles

### Dans Excel

1. **Fichier** > **Compte**
2. Vérifiez votre plan
3. Consultez "Nouveautes"

### En ligne

Consultez la page Microsoft pour les fonctionnalités par plan.

## Mettre à jour Excel

### Manuellement

1. **Fichier** > **Compte**
2. **Options de mise à jour**
3. **Mettre à jour**

### Vérifier les mises à jour

1. **Options de mise à jour** > **Afficher les mises à jour**
2. Vérifiez le statut

## Problèmes de version

### Fonctionnalité manquante

1. Vérifiez votre version
2. Mettez à jour Excel
3. Vérifiez votre plan (certaines fonctions sont exclusives)

### Version anterieure

Si vous voyez "Office 2019" où "Office 2021" :
- Vous n'avez pas d'abonnement Microsoft 365
- Vous avez une licence perpetuelle

## Passer à Microsoft 365

### Avantages

1. Toujours la dernière version
2. Nouvelles fonctionnalités
3. Support cloud
4. Mises à jour de sécurité

### Comment migrer

1. Souscrivez à Microsoft 365
2. Téléchargez depuis office.com
3. Connectez-vous avec votre compte Microsoft
