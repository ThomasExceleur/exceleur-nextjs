# Deplacer et Changer la Couleur de ses Onglets

Organisez visuellement vos classeurs Excel en colorant et reorganisant vos onglets.

## Deplacer un onglet

### Methode 1 : Glisser-deposer

1. Cliquez sur l'onglet a deplacer
2. Maintenez le bouton enfonce
3. Faites glisser vers la nouvelle position
4. Relacchez

### Methode 2 : Menu contextuel

1. Clic droit sur l'onglet
2. **Deplacer ou copier**
3. Choisissez la position
4. **OK**

## Copier un onglet

### Dans le meme classeur

1. Maintenez **Ctrl** + glisser l'onglet
2. Une copie est creee

### Vers un autre classeur

1. Clic droit sur l'onglet
2. **Deplacer ou copier**
3. Selectionnez le classeur de destination
4. Cochez **Creer une copie**
5. **OK**

## Changer la couleur d'un onglet

### Etapes

1. Clic droit sur l'onglet
2. **Couleur d'onglet**
3. Choisissez une couleur

### Codes couleur suggeres

| Couleur | Usage |
|---------|-------|
| Vert | Donnees validees |
| Rouge | A corriger |
| Bleu | En cours |
| Jaune | Attention |

## Bonnes pratiques d'organisation

### Par type de contenu

- **Donnees brutes** : Bleu
- **Calculs** : Vert
- **Rapports** : Orange
- **Configuration** : Gris

### Par statut

- **Termine** : Vert
- **En cours** : Jaune
- **A faire** : Rouge

## Renommer un onglet

### Methode rapide

Double-cliquez sur l'onglet et tapez le nouveau nom.

### Regles de nommage

- Maximum 31 caracteres
- Pas de caracteres speciaux : / \ * ? : [ ]
- Noms uniques dans le classeur

## Raccourcis utiles

| Action | Raccourci |
|--------|-----------|
| Onglet suivant | Ctrl + Page Down |
| Onglet precedent | Ctrl + Page Up |
| Nouvel onglet | Shift + F11 |

## Conseils

1. **Utilisez** un code couleur coherent
2. **Groupez** les onglets par fonction
3. **Nommez** clairement chaque feuille
4. **Limitez** le nombre d'onglets (10-15 max)
