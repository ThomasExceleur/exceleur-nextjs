## Pourquoi utiliser les raccourcis Excel ?

Les raccourcis clavier peuvent vous faire gagner des heures chaque semaine. Au lieu de naviguer dans les menus, un simple appui sur quelques touches execute l'action souhaitee.

## Raccourcis de navigation

### Se deplacer rapidement

| Raccourci | Action |
|-----------|--------|
| Ctrl + Home | Aller a la cellule A1 |
| Ctrl + End | Aller a la derniere cellule utilisee |
| Ctrl + Fleche | Aller au bord de la zone de donnees |
| Ctrl + Page Up/Down | Changer d'onglet |

### Selectionner des plages

| Raccourci | Action |
|-----------|--------|
| Ctrl + Shift + End | Selectionner jusqu'a la fin |
| Ctrl + Shift + Home | Selectionner jusqu'au debut |
| Ctrl + A | Selectionner tout le tableau |
| Ctrl + Espace | Selectionner la colonne |
| Shift + Espace | Selectionner la ligne |

## Raccourcis d'edition

### Copier/Coller

| Raccourci | Action |
|-----------|--------|
| Ctrl + C | Copier |
| Ctrl + X | Couper |
| Ctrl + V | Coller |
| Ctrl + Alt + V | Collage special |
| Ctrl + D | Remplir vers le bas |
| Ctrl + R | Remplir vers la droite |

### Modifier les cellules

| Raccourci | Action |
|-----------|--------|
| F2 | Modifier la cellule |
| Echap | Annuler la modification |
| Entree | Valider |
| Tab | Valider et aller a droite |
| Ctrl + Z | Annuler |
| Ctrl + Y | Retablir |

<Callout type="success" title="Astuce pro">
Le raccourci F4 est magique : il repete la derniere action effectuee. Tres utile pour appliquer le meme formatage a plusieurs cellules !
</Callout>

## Raccourcis pour les formules

| Raccourci | Action |
|-----------|--------|
| F4 | Figer/defiger une reference |
| Ctrl + ` | Afficher les formules |
| Tab | Valider une suggestion |
| F9 | Calculer la selection |
| Alt + = | Somme automatique |

### Exemple avec F4

Quand vous tapez une reference comme `A1`, appuyez sur F4 pour la transformer :

1. Premier appui : `$A$1` (reference absolue)
2. Deuxieme appui : `A$1` (colonne relative, ligne absolue)
3. Troisieme appui : `$A1` (colonne absolue, ligne relative)
4. Quatrieme appui : `A1` (reference relative)
