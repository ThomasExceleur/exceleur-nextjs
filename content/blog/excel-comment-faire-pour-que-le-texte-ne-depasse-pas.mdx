# Empecher le Texte de Depasser dans Excel

Plusieurs solutions existent pour gerer le texte long dans les cellules.

## Solution 1 : Retour a la ligne automatique

### Activer le retour a la ligne

1. Selectionnez les cellules
2. **Accueil** > **Renvoyer a la ligne automatiquement**
3. Ou **Ctrl + 1** > **Alignement** > **Renvoyer a la ligne**

### Effet

Le texte reste dans la cellule et s'affiche sur plusieurs lignes.

## Solution 2 : Ajuster la largeur de colonne

### Manuellement

1. Placez le curseur entre deux en-tetes de colonnes
2. Double-cliquez pour ajuster automatiquement

### Automatiquement

1. Selectionnez la colonne
2. **Accueil** > **Format** > **Ajuster la largeur de colonne**

## Solution 3 : Reduire le texte

### Ajuster a la cellule

1. **Ctrl + 1** (Format de cellule)
2. Onglet **Alignement**
3. Cochez **Ajuster**

### Effet

Le texte est reduit pour tenir dans la cellule.

## Solution 4 : Fusionner des cellules

### Pour un titre long

1. Selectionnez plusieurs cellules adjacentes
2. **Accueil** > **Fusionner et centrer**

### Attention

Les cellules fusionnees peuvent poser des problemes avec les formules.

## Solution 5 : Tronquer le texte

### Avec une formule

```excel
=GAUCHE(A1;20)                ' Garde les 20 premiers caracteres
=GAUCHE(A1;20)&"..."          ' Avec points de suspension
```

### Formule intelligente

```excel
=SI(NBCAR(A1)>20;GAUCHE(A1;17)&"...";A1)
```

## Comparaison des solutions

| Solution | Avantage | Inconvenient |
|----------|----------|--------------|
| Retour a la ligne | Tout le texte visible | Ligne plus haute |
| Largeur colonne | Simple | Peut etre trop large |
| Reduire | Tout dans une cellule | Texte petit |
| Fusionner | Espace etendu | Problemes formules |
| Tronquer | Taille controlee | Texte coupe |

## Afficher le texte au survol

### Info-bulle

1. Selectionnez la cellule
2. **Revision** > **Nouveau commentaire**
3. Collez le texte complet

## Pour l'impression

### Verifier l'apercu

1. **Fichier** > **Imprimer**
2. Verifiez l'apercu
3. Ajustez si necessaire

### Repeter les en-tetes

1. **Mise en page** > **Imprimer les titres**
2. Definissez les lignes a repeter

## Bonnes pratiques

1. **Prevoyez** la largeur des colonnes
2. **Utilisez** le retour a la ligne pour les descriptions
3. **Evitez** les cellules fusionnees dans les donnees
4. **Testez** l'impression
