# Tableau de Bord Excel

Visualisez vos donnees cles en un coup d'oeil.

## Elements essentiels

### KPIs

Indicateurs cles de performance.

### Graphiques

Visualisation des tendances.

### Filtres

Segments et listes deroulantes.

## Structure

### Organisation

```
[KPI 1] [KPI 2] [KPI 3]

[Graphique principal]

[Details] [Repartition]
```

## Creer les KPIs

### Formule type

```excel
=SOMME(Ventes[Montant])
```

### Mise en forme

- Grande police
- Couleur selon performance

## Graphiques

### Types recommandes

| Type | Usage |
|------|-------|
| Colonnes | Comparaisons |
| Lignes | Evolution |
| Secteurs | Repartition |

## Interactivite

### Segments

Filtres visuels lies aux TCD.

### Listes deroulantes

Controles personnalises.

## Design

### Principes

- Coherence des couleurs
- Espacement equilibre
- Lisibilite

|----------|-------------|
