# Créer un Formulaire de Saisie

Facilitez la saisie de données avec un formulaire Excel.

## Méthode 1 : Formulaire intégré

### Activer le formulaire

1. Clic droit sûr le ruban
2. **Personnaliser le ruban**
3. Dans **Choisir les commandes**, sélectionnez **Toutes les commandes**
4. Trouvez **Formulaire** et ajoutez-le

### Utilisation

1. Cliquez dans votre tableau
2. Cliquez sûr le bouton **Formulaire**
3. Saisissez les données
4. **Nouveau** pour ajouter une ligne

## Méthode 2 : Validation des données

### Liste deroulante

1. Sélectionnez la cellule
2. **Données** > **Validation des données**
3. **Autoriser** : Liste
4. **Source** : vos valeurs

### Nombres limites

1. **Autoriser** : Nombre entier
2. Définissez min et max

### Message d'aide

Dans l'onglet **Message de saisie**, ajoutez des instructions.

## Méthode 3 : UserForm VBA

### Créer le formulaire

1. **Alt + F11** (Éditeur VBA)
2. **Insertion** > **UserForm**
3. Ajoutez des contrôles (zones de texte, boutons)

### Exemple simple

```vba
Private Sub CommandButton1_Click()
    Dim ws As Worksheet
    Set ws = ThisWorkbook.Sheets("Données")

    Dim derniereLigne As Long
    derniereLigne = ws.Cells(ws.Rows.Count, 1).End(xlUp).Row + 1

    ws.Cells(derniereLigne, 1) = TextBox1.Text
    ws.Cells(derniereLigne, 2) = TextBox2.Text
    ws.Cells(derniereLigne, 3) = TextBox3.Text

    TextBox1.Text = ""
    TextBox2.Text = ""
    TextBox3.Text = ""
End Sub
```

### Afficher le formulaire

```vba
Sub AfficherFormulaire()
    UserForm1.Show
End Sub
```

## Méthode 4 : Feuille de saisie

### Conception

1. Créez une feuille "Saisie"
2. Ajoutez des cellules pour chaque champ
3. Utilisez la validation des données
4. Ajoutez un bouton pour enregistrer

### Macro d'enregistrement

```vba
Sub EnregistrerSaisie()
    Dim ws As Worksheet
    Set ws = Sheets("Données")

    Dim derniereLigne As Long
    derniereLigne = ws.Cells(ws.Rows.Count, 1).End(xlUp).Row + 1

    ws.Cells(derniereLigne, 1) = Sheets("Saisie").Range("B2")
    ws.Cells(derniereLigne, 2) = Sheets("Saisie").Range("B3")
    ws.Cells(derniereLigne, 3) = Sheets("Saisie").Range("B4")

    ' Effacer les champs
    Sheets("Saisie").Range("B2:B4").ClearContents
End Sub
```

## Éléments de formulaire

| Élément | Usage |
|---------|-------|
| Zone de texte | Saisie libre |
| Liste deroulante | Choix limite |
| Case à cocher | Oui/Non |
| Bouton | Declencheur d'action |
