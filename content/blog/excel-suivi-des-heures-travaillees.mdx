# Suivi des Heures Travaillees dans Excel

Un bon suivi des heures permet de gerer efficacement le temps de travail.

## Structure de la feuille de temps

### Colonnes de base

| Colonne | Description |
|---------|-------------|
| Date | Jour de travail |
| Heure arrivee | Debut de journee |
| Pause debut | Debut pause dejeuner |
| Pause fin | Fin pause dejeuner |
| Heure depart | Fin de journee |
| Total heures | Calcul automatique |

## Tableau journalier

### Structure

| Date | Arrivee | Pause D | Pause F | Depart | Heures |
|------|---------|---------|---------|--------|--------|
| 01/01 | 9:00 | 12:00 | 13:00 | 18:00 | 8:00 |

### Formule de calcul des heures

```excel
=(E2-B2)-(D2-C2)
```

Cela calcule : (Depart - Arrivee) - (Pause)

## Format des heures

### Formatage correct

1. Selectionnez les cellules d'heures
2. **Ctrl + 1** > **Personnalise**
3. Format : `h:mm`

### Pour plus de 24h

```
[h]:mm
```

## Heures supplementaires

### Calcul des heures sup

```excel
=SI(F2>"7:00";F2-"7:00";0)    ' Base 7h/jour
=SI(F2>"8:00";F2-"8:00";0)    ' Base 8h/jour
```

### Differencier les majorations

```excel
=SI(Heures_sup<="2:00";Heures_sup*1.25;
  "2:00"*1.25+(Heures_sup-"2:00")*1.50)
```

## Tableau hebdomadaire

### Structure

| | Lun | Mar | Mer | Jeu | Ven | Total |
|-|-----|-----|-----|-----|-----|-------|
| Arrivee | 9:00 | 9:00 | 9:00 | 9:00 | 9:00 | |
| Depart | 18:00 | 18:00 | 18:00 | 18:00 | 17:00 | |
| Pause | 1:00 | 1:00 | 1:00 | 1:00 | 1:00 | |
| Heures | 8:00 | 8:00 | 8:00 | 8:00 | 7:00 | 39:00 |

### Formule total semaine

```excel
=SOMME(B4:F4)
```

## Tableau mensuel

### Resume par semaine

| Semaine | Heures travaillees | Heures sup | Absences |
|---------|-------------------|------------|----------|
| S1 | 39:00 | 4:00 | 0:00 |
| S2 | 35:00 | 0:00 | 4:00 |

### Total mensuel

```excel
=SOMME(Heures_travaillees)
=SOMME(Heures_sup)
```

## Calcul du solde d'heures

### Structure

| Element | Heures |
|---------|--------|
| Heures dues | 151:40 |
| Heures travaillees | 155:00 |
| Solde | +3:20 |

### Formule du solde

```excel
=B2-B1                        ' Peut etre negatif
```

### Afficher + ou -

```excel
=SI(Solde>=0;"+";"")&TEXTE(Solde;"[h]:mm")
```

## Validation des saisies

### Heures realistes

```excel
=ET(B2>="6:00";B2<="22:00")   ' Arrivee entre 6h et 22h
```

### Coherence des heures

```excel
=ET(E2>B2;D2>C2;C2>B2;E2>D2)  ' Ordre logique
```

## Absences et conges

### Types d'absence

| Type | Impact sur heures |
|------|-------------------|
| CP | Deduit des heures dues |
| RTT | Deduit des heures dues |
| Maladie | Justifie l'absence |
| Formation | Compte comme travail |

### Calcul jours travailles

```excel
=JOURS.OUVRES(Date_debut;Date_fin) - Jours_absence
```

## Rapport par projet

### Ventilation du temps

| Projet | Heures | % |
|--------|--------|---|
| Projet A | 20:00 | 50% |
| Projet B | 12:00 | 30% |
| Admin | 8:00 | 20% |

### Formule pourcentage

```excel
=B2/SOMME(B:B)
```

## Graphiques de suivi

### Evolution hebdomadaire

1. Courbe des heures par semaine
2. Ligne de reference (35h ou 39h)

### Repartition par projet

1. Camembert des heures par projet
2. Ajoutez les pourcentages

## Automatisation

### Remplissage automatique des dates

```excel
=SI(JOURSEM(A1;2)<=5;A1+1;A1+3)   ' Saute les weekends
```

### Calcul du jour

```excel
=TEXTE(A2;"jjjj")                  ' Affiche le jour de la semaine
```

## Export et reporting

### Rapport mensuel

1. Creez un tableau recapitulatif
2. Exportez en PDF
3. Archivez

### Pour la paie

```excel
=Heures_normales * Taux_horaire + Heures_sup * Taux_majore
```
