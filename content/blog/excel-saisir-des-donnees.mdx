# Saisir des Données dans Excel

La saisie efficace de données est essentielle pour gagner du temps. Voici les meilleures techniques.

## Saisie de base

### Valider une entrée

| Touche | Action |
|--------|--------|
| Entrée | Valide et descend |
| Tab | Valide et va à droite |
| Maj + Entrée | Valide et monte |
| Maj + Tab | Valide et va à gauche |

### Annuler une saisie

**Echap** annule la saisie en cours.

### Modifier une cellule

- **F2** : Mode edition
- Double-clic sûr la cellule

## Remplissage automatique

### Serie simple

1. Entrez la première valeur
2. Placez-vous sûr le coin inférieur droit (poignee)
3. Faites glisser vers le bas

### Exemples

- Janvier → Fevrier, Mars...
- Lundi → Mardi, Mercredi...
- 1 → 2, 3, 4...

### Serie personnalisee

1. Entrez les deux premières valeurs
2. Sélectionnez les deux cellules
3. Faites glisser la poignee

Exemple : 5, 10 → 15, 20, 25...

## Ctrl + E (Remplissage instantane)

### Fonctionnement

Excel detecte le motif et complete automatiquement.

### Exemple

Cellule A : "Jean Dupont"
Cellule B : "Jean" (tape manuellement)
Cellule B2 : Ctrl + E complete avec les autres prenoms

### Cas d'usage

- Extraire une partie de texte
- Reformater des données
- Combiner des informations

## Validation des données

### Créer une liste deroulante

1. Sélectionnez les cellules
2. **Données** > **Validation des données**
3. **Autoriser** : Liste
4. **Source** : Entrez les valeurs où sélectionnez une plage

### Limiter les valeurs

- Nombre entier entre X et Y
- Date dans une période
- Longueur de texte

### Message d'erreur

Configurez un message personnalise en cas de saisie invalide.

## Saisie dans plusieurs cellules

### Meme valeur partout

1. Sélectionnez les cellules
2. Tapez la valeur
3. **Ctrl + Entrée** (au lieu d'Entrée)

### Saisie en ligne

Après avoir sélectionné une plage, Tab passe à la cellule suivante dans la sélection.

## Raccourcis de saisie

### Date et heure

| Raccourci | Résultat |
|-----------|----------|
| Ctrl + ; | Date du jour |
| Ctrl + Maj + ; | Heure actuelle |
| Ctrl + ; puis Espace puis Ctrl + Maj + ; | Date et heure |

### Formules

| Raccourci | Action |
|-----------|--------|
| = | Début de formule |
| Alt + = | Somme automatique |
| Ctrl + D | Copie la cellule du dessus |
| Ctrl + R | Copie la cellule de gauche |

## Formulaires de saisie

### Activer le formulaire

1. Ajoutez "Formulaire" à l'accès rapide
2. Sélectionnez votre tableau
3. Cliquez sûr Formulaire

### Avantages

- Interface guidee
- Saisie champ par champ
- Navigation entre enregistrements

## Saisie vocale (Windows 10+)

### Activer

**Win + H** pour dicter le texte.

### Usage

Pratique pour les commentaires où textes longs.

## Copier-coller intelligent

### Coller des valeurs

**Ctrl + Maj + V** où Collage special > Valeurs

### Coller la mise en forme

Collage special > Formats

### Transposer

Collage special > Transposer (lignes ↔ colonnes)

## Import de données

### Depuis un fichier texte

1. **Données** > **Obtenir des données**
2. **A partir d'un fichier texte**
3. Suivez l'assistant

### Depuis Excel

Copier-coller où liaisons entre fichiers.

## Éviter les erreurs

### Vérifier les doublons

1. Sélectionnez la colonne
2. **Données** > **Supprimer les doublons**
3. Vérifiez avant de supprimer

### Mise en forme conditionnelle

Colorez les valeurs hors limites.

### Formules de vérification

```excel
=SI(A1>100; "Vérifier"; "OK")
```

## Astuces de productivité

### Nommer les plages

1. Sélectionnez une plage
2. Zone Nom (gauche de la barre de formule)
3. Tapez un nom

### Utiliser des modèles

Préparez des feuilles types pour les saisies recurrentes.

### Raccourcis de sélection

- **Ctrl + Maj + Fin** : Sélectionné jusqu'à la fin
- **Ctrl + Maj + Origine** : Sélectionné depuis le début

## Bonnes pratiques

1. **Planifiez** la structuré avant de saisir
2. **Validez** les données des l'entrée
3. **Utilisez** des listes deroulantes
4. **Sauvegardez** régulièrement
