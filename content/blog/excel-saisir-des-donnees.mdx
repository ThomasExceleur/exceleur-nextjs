# Saisir des Donnees dans Excel

La saisie efficace de donnees est essentielle pour gagner du temps. Voici les meilleures techniques.

## Saisie de base

### Valider une entree

| Touche | Action |
|--------|--------|
| Entree | Valide et descend |
| Tab | Valide et va a droite |
| Maj + Entree | Valide et monte |
| Maj + Tab | Valide et va a gauche |

### Annuler une saisie

**Echap** annule la saisie en cours.

### Modifier une cellule

- **F2** : Mode edition
- Double-clic sur la cellule

## Remplissage automatique

### Serie simple

1. Entrez la premiere valeur
2. Placez-vous sur le coin inferieur droit (poignee)
3. Faites glisser vers le bas

### Exemples

- Janvier → Fevrier, Mars...
- Lundi → Mardi, Mercredi...
- 1 → 2, 3, 4...

### Serie personnalisee

1. Entrez les deux premieres valeurs
2. Selectionnez les deux cellules
3. Faites glisser la poignee

Exemple : 5, 10 → 15, 20, 25...

## Ctrl + E (Remplissage instantane)

### Fonctionnement

Excel detecte le motif et complete automatiquement.

### Exemple

Cellule A : "Jean Dupont"
Cellule B : "Jean" (tape manuellement)
Cellule B2 : Ctrl + E complete avec les autres prenoms

### Cas d'usage

- Extraire une partie de texte
- Reformater des donnees
- Combiner des informations

## Validation des donnees

### Creer une liste deroulante

1. Selectionnez les cellules
2. **Donnees** > **Validation des donnees**
3. **Autoriser** : Liste
4. **Source** : Entrez les valeurs ou selectionnez une plage

### Limiter les valeurs

- Nombre entier entre X et Y
- Date dans une periode
- Longueur de texte

### Message d'erreur

Configurez un message personnalise en cas de saisie invalide.

## Saisie dans plusieurs cellules

### Meme valeur partout

1. Selectionnez les cellules
2. Tapez la valeur
3. **Ctrl + Entree** (au lieu d'Entree)

### Saisie en ligne

Apres avoir selectionne une plage, Tab passe a la cellule suivante dans la selection.

## Raccourcis de saisie

### Date et heure

| Raccourci | Resultat |
|-----------|----------|
| Ctrl + ; | Date du jour |
| Ctrl + Maj + ; | Heure actuelle |
| Ctrl + ; puis Espace puis Ctrl + Maj + ; | Date et heure |

### Formules

| Raccourci | Action |
|-----------|--------|
| = | Debut de formule |
| Alt + = | Somme automatique |
| Ctrl + D | Copie la cellule du dessus |
| Ctrl + R | Copie la cellule de gauche |

## Formulaires de saisie

### Activer le formulaire

1. Ajoutez "Formulaire" a l'acces rapide
2. Selectionnez votre tableau
3. Cliquez sur Formulaire

### Avantages

- Interface guidee
- Saisie champ par champ
- Navigation entre enregistrements

## Saisie vocale (Windows 10+)

### Activer

**Win + H** pour dicter le texte.

### Usage

Pratique pour les commentaires ou textes longs.

## Copier-coller intelligent

### Coller des valeurs

**Ctrl + Maj + V** ou Collage special > Valeurs

### Coller la mise en forme

Collage special > Formats

### Transposer

Collage special > Transposer (lignes ↔ colonnes)

## Import de donnees

### Depuis un fichier texte

1. **Donnees** > **Obtenir des donnees**
2. **A partir d'un fichier texte**
3. Suivez l'assistant

### Depuis Excel

Copier-coller ou liaisons entre fichiers.

## Eviter les erreurs

### Verifier les doublons

1. Selectionnez la colonne
2. **Donnees** > **Supprimer les doublons**
3. Verifiez avant de supprimer

### Mise en forme conditionnelle

Colorez les valeurs hors limites.

### Formules de verification

```excel
=SI(A1>100; "Verifier"; "OK")
```

## Astuces de productivite

### Nommer les plages

1. Selectionnez une plage
2. Zone Nom (gauche de la barre de formule)
3. Tapez un nom

### Utiliser des modeles

Preparez des feuilles types pour les saisies recurrentes.

### Raccourcis de selection

- **Ctrl + Maj + Fin** : Selectionne jusqu'a la fin
- **Ctrl + Maj + Origine** : Selectionne depuis le debut

## Bonnes pratiques

1. **Planifiez** la structure avant de saisir
2. **Validez** les donnees des l'entree
3. **Utilisez** des listes deroulantes
4. **Sauvegardez** regulierement
