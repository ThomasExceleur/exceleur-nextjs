# Astuces pour Nettoyer vos Donnees

Un nettoyage efficace est la base de toute bonne analyse.

## Supprimer les espaces

### SUPPRESPACE

Supprime les espaces en trop.

```excel
=SUPPRESPACE(A1)
```

### EPURAGE

Supprime les caracteres non imprimables.

```excel
=EPURAGE(A1)
```

### Combinaison

```excel
=SUPPRESPACE(EPURAGE(A1))
```

## Uniformiser la casse

### MAJUSCULE

```excel
=MAJUSCULE(A1)   ' PARIS
```

### MINUSCULE

```excel
=MINUSCULE(A1)   ' paris
```

### NOMPROPRE

```excel
=NOMPROPRE(A1)   ' Paris
```

## Corriger les dates

### Probleme courant

Dates stockees comme texte.

### Solution

```excel
=DATEVAL(A1)
```

### Format personnalise

```excel
=DATEVAL(DROITE(A1;4)&"-"&STXT(A1;4;2)&"-"&GAUCHE(A1;2))
```

## Corriger les nombres

### Nombres comme texte

```excel
=CNUM(A1)           ' Convertit en nombre
=A1*1               ' Methode rapide
=A1+0               ' Alternative
```

## Supprimer les doublons

### Methode rapide

1. Selectionnez les donnees
2. **Donnees** > **Supprimer les doublons**
3. Choisissez les colonnes a comparer

### Avec formule (marquer)

```excel
=SI(NB.SI($A$1:A1;A1)>1;"Doublon";"Unique")
```

## Fractionner le texte

### Remplissage instantane

**Ctrl+E** apres avoir saisi un exemple.

### Formules texte

```excel
=GAUCHE(A1;CHERCHE(" ";A1)-1)      ' Premier mot
=DROITE(A1;NBCAR(A1)-CHERCHE(" ";A1))  ' Reste
```

### TEXTE.APRES et TEXTE.AVANT

```excel
=TEXTE.AVANT(A1;" ")               ' Avant espace
=TEXTE.APRES(A1;" ")               ' Apres espace
```

## Valider les donnees

### Reperer les erreurs

```excel
=ESTERREUR(A1)     ' Detecte les erreurs
=ESTNUM(A1)        ' Verifie si nombre
=ESTTEXTE(A1)      ' Verifie si texte
```

### Mise en forme conditionnelle

1. Selectionnez la plage
2. **Mise en forme conditionnelle** > **Nouvelle regle**
3. Utilisez une formule pour detecter les anomalies

## Gerer les cellules vides

### Compter les vides

```excel
=NB.VIDE(A:A)
```

### Remplacer les vides

1. **Ctrl+H**
2. Laissez "Rechercher" vide
3. "Remplacer par" votre valeur
4. **Options** > Cochez "Cellules entieres"

### Avec formule

```excel
=SI(A1="";"N/A";A1)
```
