# Previsions et Tendances dans Excel

Anticipez l'avenir avec les outils de prevision.

## Courbe de tendance

### Ajouter

1. Cliquez sur le graphique
2. **+** > **Courbe de tendance**
3. Choisissez le type

### Types

| Type | Usage |
|------|-------|
| Lineaire | Evolution constante |
| Exponentiel | Croissance acceleree |
| Logarithmique | Croissance deceleree |
| Polynomiale | Fluctuations |

## Afficher l'equation

1. Clic droit sur la courbe
2. Format > Afficher l'equation

## Feuille de prevision

### Acces

**Donnees** > **Feuille de prevision**

### Fonctionnalites

- Detection automatique de saisonnalite
- Intervalle de confiance
- Graphique genere

## PREVISION.LINEAIRE

```excel
=PREVISION.LINEAIRE(x;valeurs_y;valeurs_x)
```

## PREVISION.ETS

```excel
=PREVISION.ETS(date_cible;valeurs;dates)
```

Meilleure pour les series temporelles avec saisonnalite.

## Intervalle de confiance

```excel
=PREVISION.ETS.CONFINT(date;valeurs;dates;;0.95)
```

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Donnees suffisantes | Min 2 cycles saisonniers |
| Validation | Comparez aux valeurs reelles |
| Prudence | Les previsions ont des limites |
