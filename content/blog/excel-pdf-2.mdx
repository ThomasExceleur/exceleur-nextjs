# Excel vers PDF

Convertissez vos classeurs en PDF.

## Méthode rapide

### Enregistrer sous

1. **Fichier** > **Enregistrer sous**
2. **Type** : PDF
3. **Enregistrer**

## Méthode Exporter

1. **Fichier** > **Exporter**
2. **Créer un document PDF**
3. **Publier**

## Options

### Quoi exporter

| Option | Description |
|--------|-------------|
| Classeur entier | Toutes les feuilles |
| Feuille active | Une seule |
| Sélection | Zone sélectionnée |

## Mise en page

### Avant export

1. **Mise en page** > **Zone d'impression**
2. Définissez les marges
3. Choisissez l'orientation

### Apercu

**Fichier** > **Imprimer** (apercu)

## Qualité

| Option | Usage |
|--------|-------|
| Standard | Publication |
| Minimum | Léger |

## Plusieurs feuilles

Ctrl+Clic sûr les onglets pour sélectionner.
