# Excel vers PDF

Convertissez vos classeurs en PDF.

## Methode rapide

### Enregistrer sous

1. **Fichier** > **Enregistrer sous**
2. **Type** : PDF
3. **Enregistrer**

## Methode Exporter

1. **Fichier** > **Exporter**
2. **Creer un document PDF**
3. **Publier**

## Options

### Quoi exporter

| Option | Description |
|--------|-------------|
| Classeur entier | Toutes les feuilles |
| Feuille active | Une seule |
| Selection | Zone selectionnee |

## Mise en page

### Avant export

1. **Mise en page** > **Zone d'impression**
2. Definissez les marges
3. Choisissez l'orientation

### Apercu

**Fichier** > **Imprimer** (apercu)

## Qualite

| Option | Usage |
|--------|-------|
| Standard | Publication |
| Minimum | Leger |

## Plusieurs feuilles

Ctrl+Clic sur les onglets pour selectionner.
