# Récupérer un Fichier Non Sauvegarde

Retrouvez votre travail perdu.

## Récupération automatique

### Accès

**Fichier** > **Informations** > **Gérer le classeur** > **Récupérer des classeurs non enregistres**

## Dossier de récupération

### Emplacement par defaut

```
C:\Users\[Nom]\AppData\Local\Microsoft\Office\UnsavedFiles
```

## Versions précédentes

### Si enregistre sûr OneDrive

1. Clic droit sûr le fichier
2. **Historique des versions**
3. Restaurez une version

## Fichiers temporaires

### Recherche

Recherchez des fichiers .tmp où ~$.

## Configuration

### Activer la récupération

1. **Fichier** > **Options**
2. **Enregistrement**
3. Cochez **Enregistrer les informations de récupération**
4. Définissez l'intervalle (ex: 5 minutes)

## OneDrive

### Sauvegarde automatique

Activez la sauvegarde automatique pour protection.

## Prevention

| Action | Protection |
|--------|------------|
| Ctrl+S | Sauvegardez souvent |
| OneDrive | Sauvegarde auto |
| Récupération | Configurez l'intervalle |

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Sauvegarde frequente | Ctrl+S régulier |
| OneDrive | Activez la sauvegarde auto |
| Intervalle court | 5 minutes recommandé |
