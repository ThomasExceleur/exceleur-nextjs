# Recuperer un Fichier Non Sauvegarde

Retrouvez votre travail perdu.

## Recuperation automatique

### Acces

**Fichier** > **Informations** > **Gerer le classeur** > **Recuperer des classeurs non enregistres**

## Dossier de recuperation

### Emplacement par defaut

```
C:\Users\[Nom]\AppData\Local\Microsoft\Office\UnsavedFiles
```

## Versions precedentes

### Si enregistre sur OneDrive

1. Clic droit sur le fichier
2. **Historique des versions**
3. Restaurez une version

## Fichiers temporaires

### Recherche

Recherchez des fichiers .tmp ou ~$.

## Configuration

### Activer la recuperation

1. **Fichier** > **Options**
2. **Enregistrement**
3. Cochez **Enregistrer les informations de recuperation**
4. Definissez l'intervalle (ex: 5 minutes)

## OneDrive

### Sauvegarde automatique

Activez la sauvegarde automatique pour protection.

## Prevention

| Action | Protection |
|--------|------------|
| Ctrl+S | Sauvegardez souvent |
| OneDrive | Sauvegarde auto |
| Recuperation | Configurez l'intervalle |

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Sauvegarde frequente | Ctrl+S regulier |
| OneDrive | Activez la sauvegarde auto |
| Intervalle court | 5 minutes recommande |
