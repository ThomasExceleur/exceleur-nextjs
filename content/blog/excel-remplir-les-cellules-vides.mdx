# Remplir les Cellules Vides

Plusieurs methodes pour combler les cellules vides dans vos donnees.

## Methode 1 : Atteindre + Formule

### Etapes

1. Selectionnez la plage contenant les vides
2. **Ctrl + G** (Atteindre)
3. **Special...**
4. Cochez **Cellules vides**
5. **OK** (toutes les cellules vides sont selectionnees)
6. Tapez `=` puis la cellule au-dessus (ex: `=A1`)
7. **Ctrl + Entree** (valide pour toutes les cellules)

### Resultat

Chaque cellule vide contient maintenant une formule copiant la valeur au-dessus.

### Convertir en valeurs

1. Copiez les cellules
2. Collez en valeurs (**Ctrl + Alt + V** > **Valeurs**)

## Methode 2 : Rechercher et remplacer

### Pour remplir avec une valeur fixe

1. Selectionnez la plage
2. **Ctrl + H**
3. **Rechercher** : laissez vide
4. **Remplacer par** : votre valeur (ex: "N/A")
5. **Options** > Cochez **Totalite du contenu de la cellule**
6. **Remplacer tout**

## Methode 3 : Formule SI

### Eviter les vides dans les calculs

```excel
=SI(A1="";"Valeur par defaut";A1)
```

### Copier la valeur precedente

```excel
=SI(A2="";A1;A2)
```

## Methode 4 : Power Query

### Etapes

1. Selectionnez vos donnees
2. **Donnees** > **A partir d'un tableau**
3. Dans Power Query, selectionnez la colonne
4. **Transformer** > **Remplir** > **Vers le bas**
5. **Fermer et charger**

### Avantage

Le remplissage se fait automatiquement lors des actualisations.

## Methode 5 : VBA

### Macro de remplissage

```vba
Sub RemplirVides()
    Dim rng As Range
    For Each rng In Selection
        If rng.Value = "" Then
            rng.Value = rng.Offset(-1, 0).Value
        End If
    Next rng
End Sub
```

## Cas pratiques

### Donnees importees

Les exports de bases de donnees ont souvent des cellules vides a remplir.

### Tableaux croises depivotes

Apres depivotement, les categories sont souvent vides.

### Donnees hierarchiques

Les sous-categories peuvent avoir des cellules de categorie vides.

## Comparaison des methodes

| Methode | Rapidite | Automatique | Sans formule |
|---------|----------|-------------|--------------|
| Atteindre | Rapide | Non | Non |
| Remplacer | Rapide | Non | Oui |
| Formule | Moyen | Oui | Non |
| Power Query | Moyen | Oui | Oui |
| VBA | Rapide | Non | Oui |

1. **Sauvegardez** avant de modifier
2. **Testez** sur une copie
3. **Preferez** Power Query pour les donnees recurrentes
4. **Documentez** les modifications effectuees
