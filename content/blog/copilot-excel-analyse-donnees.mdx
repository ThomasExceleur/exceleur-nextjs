# Copilot Excel : Analysez vos Données par l'IA

Copilot revolutionne l'analyse de données dans Excel.

## Qu'est-ce que Copilot dans Excel ?

### Fonctionnalités

- Analyser les données automatiquement
- Créer des formules en langage naturel
- Générer des graphiques
- Répondre aux questions sûr vos données

### Prerequis

- Microsoft 365 avec Copilot
- Données en tableau structuré
- Connexion internet

## Démarrer avec Copilot

### Acceder à Copilot

1. Ouvrez un classeur avec des données
2. Sélectionnez votre tableau
3. Cliquez sûr l'icone **Copilot** dans le ruban

### Préparer vos données

- Convertissez en tableau (Ctrl+T)
- Nommez clairement vos colonnes
- Supprimez les cellules vides

## Analyser les données

### Questions possibles

- "Quelle est la tendance des ventes ?"
- "Quels sont les 5 meilleurs clients ?"
- "Y à-t-il des anomalies dans les données ?"
- "Compare les ventes par region"

### Exemple de conversation

**Vous** : "Montre-moi les ventes par mois"

**Copilot** : Créé un tableau croisé dynamique et un graphique.

## Créer des formules

### Demander une formule

**Vous** : "Calcule la commission à 5% sûr le montant"

**Copilot** : Ajoute une colonne avec `=[@Montant]*5%`

### Formules complexes

**Vous** : "Classe les clients par niveau de vente : VIP au-dessus de 10000, Premium au-dessus de 5000, Standard sinon"

**Copilot** : Créé la formule SI imbriquée appropriee.

## Générer des visualisations

### Types de graphiques

- "Créé un histogramme des ventes par produit"
- "Montre l'evolution mensuelle en courbe"
- "Fais un camembert de la répartition par catégorie"

### Personnalisation

- "Change les couleurs du graphique"
- "Ajoute les etiquettes de données"
- "Mets un titre explicite"

## Insights automatiques

### Découvrir des patterns

Copilot peut identifier :

- Tendances
- Anomalies
- Correlations
- Valeurs extremes

### Exemple

**Copilot** : "J'ai remarque que les ventes chutent systematiquement en aout. Voici un graphique qui le montre."

## Mise en forme conditionnelle

### Demander une mise en forme

- "Colore en rouge les ventes inférieures à 1000"
- "Ajoute une barre de données sûr la colonne Montant"
- "Surligne les doublons"

## Limites actuelles

### Ce que Copilot ne fait pas (encore)

- Modifier le code VBA
- Gérer Power Query
- Créer des macros

### Precision

Les résultats doivent etre vérifiés - l'IA peut faire des erreurs.

## Bonnes pratiques

### Préparation des données

| Pratique | Impact |
|----------|--------|
| Tableaux structurés | Obligatoire |
| En-têtes clairs | Meilleure compréhension |
| Pas de cellules fusionnées | Evite les erreurs |
| Données propres | Résultats precis |

### Formulation des demandes

1. **Soyez spécifique** : "Somme des ventes de Paris" vs "Calcule"
2. **Utilisez** les noms de colonnes exacts
3. **Precisez** le format souhaité
4. **Decomposez** les demandes complexes

## Exemples de prompts efficaces

### Analyse

- "Quels produits ont vendu moins ce trimestre que le précédent ?"
- "Identifie les clients qui n'ont pas commande depuis 6 mois"
- "Calcule le panier moyen par catégorie de client"

### Formules

- "Ajoute une colonne qui calcule le nombre de jours entre commande et livraison"
- "Créé une formule qui extrait le code postal de l'adresse"

### Visualisation

- "Compare les performances des commerciaux avec un graphique"
- "Montre la répartition geographique des ventes"

## Flux de travail avec Copilot

1. Importer et nettoyer les données
2. Poser des questions pour explorer
3. Créer des calculs nécessaires
4. Générer des visualisations
5. Vérifier et affiner
