# Copilot Excel : Analysez vos Donnees par l'IA

Copilot revolutionne l'analyse de donnees dans Excel.

## Qu'est-ce que Copilot dans Excel ?

### Fonctionnalites

- Analyser les donnees automatiquement
- Creer des formules en langage naturel
- Generer des graphiques
- Repondre aux questions sur vos donnees

### Prerequis

- Microsoft 365 avec Copilot
- Donnees en tableau structure
- Connexion internet

## Demarrer avec Copilot

### Acceder a Copilot

1. Ouvrez un classeur avec des donnees
2. Selectionnez votre tableau
3. Cliquez sur l'icone **Copilot** dans le ruban

### Preparer vos donnees

- Convertissez en tableau (Ctrl+T)
- Nommez clairement vos colonnes
- Supprimez les cellules vides

## Analyser les donnees

### Questions possibles

- "Quelle est la tendance des ventes ?"
- "Quels sont les 5 meilleurs clients ?"
- "Y a-t-il des anomalies dans les donnees ?"
- "Compare les ventes par region"

### Exemple de conversation

**Vous** : "Montre-moi les ventes par mois"

**Copilot** : Cree un tableau croise dynamique et un graphique.

## Creer des formules

### Demander une formule

**Vous** : "Calcule la commission a 5% sur le montant"

**Copilot** : Ajoute une colonne avec `=[@Montant]*5%`

### Formules complexes

**Vous** : "Classe les clients par niveau de vente : VIP au-dessus de 10000, Premium au-dessus de 5000, Standard sinon"

**Copilot** : Cree la formule SI imbriquee appropriee.

## Generer des visualisations

### Types de graphiques

- "Cree un histogramme des ventes par produit"
- "Montre l'evolution mensuelle en courbe"
- "Fais un camembert de la repartition par categorie"

### Personnalisation

- "Change les couleurs du graphique"
- "Ajoute les etiquettes de donnees"
- "Mets un titre explicite"

## Insights automatiques

### Decouvrir des patterns

Copilot peut identifier :

- Tendances
- Anomalies
- Correlations
- Valeurs extremes

### Exemple

**Copilot** : "J'ai remarque que les ventes chutent systematiquement en aout. Voici un graphique qui le montre."

## Mise en forme conditionnelle

### Demander une mise en forme

- "Colore en rouge les ventes inferieures a 1000"
- "Ajoute une barre de donnees sur la colonne Montant"
- "Surligne les doublons"

## Limites actuelles

### Ce que Copilot ne fait pas (encore)

- Modifier le code VBA
- Gerer Power Query
- Creer des macros

### Precision

Les resultats doivent etre verifies - l'IA peut faire des erreurs.

## Bonnes pratiques

### Preparation des donnees

| Pratique | Impact |
|----------|--------|
| Tableaux structures | Obligatoire |
| En-tetes clairs | Meilleure comprehension |
| Pas de cellules fusionnees | Evite les erreurs |
| Donnees propres | Resultats precis |

### Formulation des demandes

1. **Soyez specifique** : "Somme des ventes de Paris" vs "Calcule"
2. **Utilisez** les noms de colonnes exacts
3. **Precisez** le format souhaite
4. **Decomposez** les demandes complexes

## Exemples de prompts efficaces

### Analyse

- "Quels produits ont vendu moins ce trimestre que le precedent ?"
- "Identifie les clients qui n'ont pas commande depuis 6 mois"
- "Calcule le panier moyen par categorie de client"

### Formules

- "Ajoute une colonne qui calcule le nombre de jours entre commande et livraison"
- "Cree une formule qui extrait le code postal de l'adresse"

### Visualisation

- "Compare les performances des commerciaux avec un graphique"
- "Montre la repartition geographique des ventes"

2. Poser des questions pour explorer
3. Creer des calculs necessaires
4. Generer des visualisations
5. Verifier et affiner
