# Premiers Pas avec Python dans Excel

Microsoft a integre Python directement dans Excel 365 pour l'analyse de donnees avancee.

## Disponibilite

### Actuellement

- Excel 365 (Insiders Program)
- Deploiement progressif

### Prerequis

- Abonnement Microsoft 365
- Version recente d'Excel

## Activer Python

### Dans une cellule

1. Tapez `=PY(` pour commencer
2. Ou utilisez **Formules** > **Inserer Python**

### Syntaxe

```excel
=PY("votre code Python")
```

## Exemple simple

### Dans Excel

```excel
=PY("import pandas as pd; df = xl('A1:C10'); df.describe()")
```

### Ce que ca fait

1. Importe pandas
2. Lit les donnees Excel (A1:C10)
3. Retourne les statistiques descriptives

## Fonction xl()

### Lire des donnees Excel

```python
xl("A1:D100")           # Plage
xl("Tableau1")          # Tableau nomme
xl("Feuil1!A:A")        # Colonne entiere
```

### Retourne un DataFrame pandas

## Bibliotheques disponibles

### Incluses

- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

### Installation

Pas d'installation necessaire - tout est dans le cloud Microsoft.

## Cas d'usage

### Analyse statistique

```python
import pandas as pd
df = xl("Ventes")
df.groupby("Region").sum()
```

### Visualisation

```python
import matplotlib.pyplot as plt
df = xl("A1:B100")
df.plot()
plt.show()
```

### Machine Learning

```python
from sklearn.linear_model import LinearRegression
# ... code de prediction
```

## Avantages

| Avantage | Description |
|----------|-------------|
| Integration | Python directement dans Excel |
| Cloud | Calculs dans le cloud Microsoft |
| Bibliotheques | Pandas, NumPy, etc. inclus |
| Collaboration | Partage facile des classeurs |

## Limites

### Actuelles

- Calcul dans le cloud (necessite internet)
- Temps de reponse variable
- Pas de pip install personnalise

| Critere | Python | VBA |

| Courbe d'apprentissage | Moyenne | Raide |
| Communaute | Tres large | Excel-specifique |
