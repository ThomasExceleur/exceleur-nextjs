# Bonnes Pratiques Excel Mobile

Tirez le maximum d'Excel sur votre smartphone ou tablette.

## Configuration initiale

### Telecharger l'application

- **iOS** : App Store
- **Android** : Google Play Store

Application gratuite avec compte Microsoft.

### Connexion OneDrive

1. Connectez votre compte Microsoft
2. Activez la synchronisation OneDrive
3. Vos fichiers sont accessibles partout

## Interface mobile

### Navigation

| Geste | Action |
|-------|--------|
| Pincer | Zoom avant/arriere |
| Balayer | Defiler |
| Double-tap | Editer cellule |
| Appui long | Menu contextuel |

### Barre de formule

- Apparait en haut lors de l'edition
- Bouton fx pour les fonctions
- Bouton v pour valider

## Creer et editer

### Nouveau classeur

1. Icone **+** en bas
2. **Classeur vide** ou modele
3. Nommez et enregistrez

### Saisie de donnees

- Clavier optimise pour les chiffres
- Suggestions de formules
- Autocompletion

### Formules

Les formules fonctionnent comme sur Desktop :

```excel
=SOMME(A1:A10)
=SI(B1>100;"Oui";"Non")
```

## Fonctionnalites mobiles

### Ce qui fonctionne

- Formules de base
- Mise en forme
- Graphiques simples
- Tableaux croises dynamiques (lecture)
- Filtres et tris

### Limitations

- VBA non supporte
- Power Query non disponible
- Power Pivot non disponible
- Fonctions avancees limitees

## Optimiser vos fichiers pour mobile

### Taille des fichiers

- Gardez les fichiers compacts
- Evitez les images lourdes
- Supprimez les donnees inutiles

### Mise en forme

- Utilisez des colonnes etroites
- Police lisible (taille 11+)
- Couleurs contrastees

### Structure

| Pratique | Avantage mobile |
|----------|-----------------|
| Tableaux structures | Navigation facile |
| Noms de plages | Acces rapide |
| Resume en debut | Vue rapide des KPI |

## Mode hors connexion

### Fichiers disponibles hors ligne

1. Ouvrez le fichier en ligne
2. Il reste en cache
3. Modifications synchronisees au retour

### Marquer pour hors ligne

Dans OneDrive, appuyez sur les 3 points > **Rendre disponible hors connexion**.

## Collaboration mobile

### Co-edition

- Travaillez a plusieurs en temps reel
- Voyez les modifications des autres
- Fonctionne comme sur Desktop

### Commentaires

1. Selectionnez une cellule
2. Menu > **Ajouter un commentaire**
3. @mentionnez des collegues

### Partage

1. Icone **Partager**
2. Choisissez les destinataires
3. Definissez les permissions

## Cas d'usage mobile

### Consultation rapide

- Verifier des chiffres
- Montrer des graphiques
- Acceder a des references

### Saisie terrain

- Inventaire
- Releves
- Notes de frais

### Validation

- Approuver des donnees
- Commenter des rapports
- Signaler des erreurs

## Accessoires recommandes

### Tablette

- Clavier Bluetooth
- Stylet pour precision
- Support ajustable

### Smartphone

- Clavier externe optionnel
- Bonne connexion reseau

## Raccourcis clavier (avec clavier externe)

| Raccourci | Action |
|-----------|--------|
| Ctrl+C | Copier |
| Ctrl+V | Coller |
| Ctrl+Z | Annuler |
| Ctrl+S | Enregistrer |

## Securite mobile

### Bonnes pratiques

1. **Activez** le verrouillage biometrique
2. **Utilisez** un code PIN fort
3. **Evitez** les Wi-Fi publics non securises
4. **Deconnectez** les appareils perdus

### Protection des fichiers

- Protection par mot de passe
- Gestion des permissions de partage
- Revocation d'acces a distance
