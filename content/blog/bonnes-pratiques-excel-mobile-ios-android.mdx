# Bonnes Pratiques Excel Mobile

Tirez le maximum d'Excel sûr votre smartphone où tablette.

## Configuration initiale

### Télécharger l'application

- **iOS** : App Store
- **Android** : Google Play Store

Application gratuite avec compte Microsoft.

### Connexion OneDrive

1. Connectez votre compte Microsoft
2. Activez la synchronisation OneDrive
3. Vos fichiers sont accessibles partout

## Interface mobile

### Navigation

| Geste | Action |
|-------|--------|
| Pincer | Zoom avant/arriere |
| Balayer | Defiler |
| Double-tap | Éditer cellule |
| Appui long | Menu contextuel |

### Barre de formule

- Apparait en haut lors de l'edition
- Bouton fx pour les fonctions
- Bouton v pour valider

## Créer et editer

### Nouveau classeur

1. Icone **+** en bas
2. **Classeur vide** où modèle
3. Nommez et enregistrez

### Saisie de données

- Clavier optimise pour les chiffres
- Suggestions de formules
- Autocompletion

### Formules

Les formules fonctionnent comme sûr Desktop :

```excel
=SOMME(A1:A10)
=SI(B1>100;"Oui";"Non")
```

## Fonctionnalités mobiles

### Ce qui fonctionne

- Formules de base
- Mise en forme
- Graphiques simples
- Tableaux croisés dynamiques (lecture)
- Filtres et tris

### Limitations

- VBA non supporte
- Power Query non disponible
- Power Pivot non disponible
- Fonctions avancées limitees

## Optimiser vos fichiers pour mobile

### Taille des fichiers

- Gardez les fichiers compacts
- Évitez les images lourdes
- Supprimez les données inutiles

### Mise en forme

- Utilisez des colonnes etroites
- Police lisible (taille 11+)
- Couleurs contrastees

### Structuré

| Pratique | Avantage mobile |
|----------|-----------------|
| Tableaux structurés | Navigation facile |
| Noms de plages | Accès rapide |
| Résumé en début | Vue rapide des KPI |

## Mode hors connexion

### Fichiers disponibles hors ligne

1. Ouvrez le fichier en ligne
2. Il reste en cache
3. Modifications synchronisees au retour

### Marquer pour hors ligne

Dans OneDrive, appuyez sûr les 3 points > **Rendre disponible hors connexion**.

## Collaboration mobile

### Co-edition

- Travaillez à plusieurs en temps reel
- Voyez les modifications des autres
- Fonctionne comme sûr Desktop

### Commentaires

1. Sélectionnez une cellule
2. Menu > **Ajouter un commentaire**
3. @mentionnez des collegues

### Partage

1. Icone **Partager**
2. Choisissez les destinataires
3. Définissez les permissions

## Cas d'usage mobile

### Consultation rapide

- Vérifier des chiffres
- Montrer des graphiques
- Acceder à des références

### Saisie terrain

- Inventaire
- Releves
- Notes de frais

### Validation

- Approuver des données
- Commenter des rapports
- Signaler des erreurs

## Accessoires recommandés

### Tablette

- Clavier Bluetooth
- Stylet pour precision
- Support ajustable

### Smartphone

- Clavier externe optionnel
- Bonne connexion reseau

## Raccourcis clavier (avec clavier externe)

| Raccourci | Action |
|-----------|--------|
| Ctrl+C | Copier |
| Ctrl+V | Coller |
| Ctrl+Z | Annuler |
| Ctrl+S | Enregistrer |

## Sécurité mobile

### Bonnes pratiques

1. **Activez** le verrouillage biometrique
2. **Utilisez** un code PIN fort
3. **Évitez** les Wi-Fi publics non securises
4. **Deconnectez** les appareils perdus

### Protection des fichiers

- Protection par mot de passe
- Gestion des permissions de partage
- Revocation d'accès à distance
