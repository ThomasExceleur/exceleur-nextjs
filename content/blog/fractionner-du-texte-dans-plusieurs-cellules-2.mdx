# Fractionner du Texte

Divisez vos cellules en plusieurs colonnes.

## Assistant Convertir

### Acces

**Donnees** > **Convertir**

### Types

| Type | Usage |
|------|-------|
| Delimiteur | Virgule, espace, etc. |
| Largeur fixe | Position exacte |

## Etapes Delimiteur

1. Selectionnez la colonne
2. **Donnees** > **Convertir**
3. **Delimiteur**
4. Choisissez le separateur
5. Definissez la destination

## TEXTE.SEPARER (Excel 365)

### Syntaxe

```excel
=TEXTE.SEPARER(texte;delimiteur)
```

### Exemple

```excel
=TEXTE.SEPARER(A1;" ")
```

Retourne un tableau dynamique.

## Formules classiques

### GAUCHE

```excel
=GAUCHE(A1;TROUVE(" ";A1)-1)
```

### DROITE

```excel
=DROITE(A1;NBCAR(A1)-TROUVE(" ";A1))
```

### STXT

```excel
=STXT(A1;debut;longueur)
```

## Ctrl+E (Flash Fill)

### Methode

1. Tapez le resultat souhaite
2. Descendez d'une cellule
3. **Ctrl+E**

## Power Query

### Avantages

- Reproductible
- Multi-delimiteurs
- Actualisation automatique

### Acces

**Donnees** > **A partir d'un tableau** > **Fractionner la colonne**

## Exemples

| Avant | Delimiteur | Resultat |
|-------|------------|----------|
| Jean Dupont | Espace | Jean, Dupont |
| Paris,France | Virgule | Paris, France |
| 2024-01-15 | Tiret | 2024, 01, 15 |
