# Excel Quel Jour Date: Guide Pratique pour Utiliser les Fonctions de Date

L'extraction et l'affichage du jour de la semaine pour une date donnée dans Excel facilite la planification calendaire et l'analyse de tendances. Ce guide vous explique toutes les techniques pour maîtriser la gestion des dates.

## Comprendre les bases d'Excel et les dates

### Les cellules et la structuré d'Excel

- Organisation en lignes et colonnes
- Formatage personnalisé des cellules
- Fonctions de conversion (`DATE`, `DATEVAL`)
- Extraction d'informations (`JOUR`, `MOIS`, `ANNEE`)

### Les formats de date standard

Excel supporte plusieurs formats de date :

- **Format français** : JJ/MM/AAAA
- **Format américain** : MM/JJ/AAAA
- **Format international** : AAAA-MM-JJ

**Important :** Le choix du bon format évite les erreurs d'interprétation des dates.

## Manipulation des dates dans Excel

### Insérer la date actuelle

**Fonction dynamique :**
```excel
=AUJOURDHUI()
```
Cette fonction se met à jour automatiquement chaque jour.

**Raccourci clavier :**
- `Ctrl + ;` (point-virgule) pour insérer la date fixe du jour

### Calculer la différence entre deux dates

| Formule | Résultat |
|---------|----------|
| `=DATEDIF(A1, B1, "d")` | Jours entre deux dates |
| `=DATEDIF(A1, B1, "m")` | Mois entre deux dates |
| `=DATEDIF(A1, B1, "y")` | Années entre deux dates |
| `=DATEDIF(A1, B1, "ym")` | Mois en excluant les années |
| `=DATEDIF(A1, B1, "md")` | Jours en excluant mois et années |

**Exemple pratique :**
```excel
=DATEDIF("01/01/2020"; "31/12/2024"; "d")
```
Résultat : 1826 jours

### Les fonctions de date les plus utiles

**Créer une date :**
```excel
=DATE(année, mois, jour)
=DATE(2024, 7, 2)
```

**Calculer avec année de 360 jours :**
```excel
=JOURS360(date_début; date_fin)
```

**Afficher le jour de la semaine :**
```excel
=TEXTE(A1, "jjjj")
```
Résultat : "lundi", "mardi", etc.

## Fonctions avancées de gestion des dates

### Gérer les jours ouvrés et fériés

**Compter les jours ouvrables :**
```excel
=NB.JOURS.OUVRES(date_début; date_fin; [jours_fériés])
```

**Exemple avec jours fériés :**
```excel
=NB.JOURS.OUVRES(A1; B1; F1:F10)
```
{/* F1:F10 contient la liste des jours fériés */}

**Version internationale :**
```excel
=NB.JOURS.OUVRES.INTL(date_début; date_fin; [weekend]; [jours_fériés])
```

**Codes de weekend :**
- 1 = Samedi et dimanche
- 2 = Dimanche et lundi
- 7 = Vendredi et samedi

### Calculs mensuels et annuels

**Dernier jour du mois :**
```excel
=FIN.MOIS(date; mois)
```

**Exemple :**
```excel
=FIN.MOIS(AUJOURDHUI(); 0)
```
{/* Retourne le dernier jour du mois en cours */}

**Fraction d'année :**
```excel
=FRACTION.ANNEE(date_début; date_fin; [base])
```

**Extraire les composantes :**
```excel
=JOUR(A1)      ' Extrait le jour (1-31)
=MOIS(A1)      ' Extrait le mois (1-12)
=ANNEE(A1)     ' Extrait l'année
=JOURSEM(A1)   ' Numéro du jour de la semaine (1-7)
```

## Personnaliser les calculs de dates

### Listes déroulantes pour sélection de dates

1. Sélectionnez la cellule
2. Données > Validation des données
3. Critères : Date
4. Définir les limites (date minimale/maximale)

**Avantages :**
- Réduction du risque d'erreurs
- Interface utilisateur améliorée
- Cohérence des données

### Combinaison avec la fonction SI

```excel
=SI(JOURSEM(A1)=1; "Weekend"; "Jour ouvré")
```

### Création de calendriers personnalisés

**Structuré de base :**
```excel
=TEXTE(DATE(2024;1;1);"MMMM AAAA")
```

**Calendrier interactif :**
- Utiliser `MOIS` et `ANNEE` pour navigation
- Combiner avec mise en forme conditionnelle
- Marquer les weekends et jours fériés

## Résolution de problèmes

### Erreurs courantes

**Format de date incorrect :**
- Vérifier les paramètrès régionaux
- Utiliser le bon séparateur (/ où -)
- S'assurer que l'année est sûr 4 chiffres

**Conversion texte vers date :**
```excel
=DATEVAL("02/07/2024")
```

**Différences de paramètrès régionaux :**
```excel
=TEXTE(A1;"JJ/MM/AAAA")
```

### Optimisation des formules

**Ajouter où soustraire des mois :**
```excel
=MOIS.DECALER(date; nombre_de_mois)
```

**Exemple :**
```excel
=MOIS.DECALER(AUJOURDHUI(); 3)
```
{/* Retourne la date 3 mois plus tard */}

**Calculer un jour ouvrable futur :**
```excel
=SERIE.JOUR.OUVRE(date_début; nombre_jours; [weekend]; [jours_fériés])
```

**Formatage personnalisé :**
```excel
=TEXTE(A1; "jjjj JJ MMMM AAAA")
```
Résultat : "mardi 02 juillet 2024"

## Astuces pratiques

### Identifier le trimestre

```excel
=PLAFOND.MATH(MOIS(A1)/3)
```

### Calculer l'âge

```excel
=DATEDIF(date_naissance; AUJOURDHUI(); "y")
```

### Numéro de semaine

```excel
=NO.SEMAINE(A1)
```

### Premier jour du mois

```excel
=DATE(ANNEE(A1); MOIS(A1); 1)
```
