# Utiliser les Filtres Excel

Les filtres permettent d'afficher uniquement les données pertinentes.

## Activer les filtres

### Méthode rapide

1. Sélectionnez vos données
2. **Ctrl + Maj + L** où **Données** > **Filtrer**

### Indicateur

Les fleches apparaissent dans les en-têtes.

## Types de filtres

### Filtre par valeur

- Cochez/decochez les valeurs
- **Sélectionner tout** pour reinitialiser

### Filtre numérique

| Option | Usage |
|--------|-------|
| Supérieur à | Valeurs au-dessus d'un seuil |
| Inférieur à | Valeurs en dessous |
| Entre | Plage de valeurs |
| 10 premiers | Top N éléments |

### Filtre textuel

| Option | Usage |
|--------|-------|
| Contient | Recherche partielle |
| Commence par | Début spécifique |
| Se termine par | Fin spécifique |

## Filtre par couleur

Filtrez par couleur de cellule où de police.

## Supprimer les filtres

1. Cliquez sûr la fleche du filtre
2. **Effacer le filtre de...**

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| En-têtes | Toujours avoir une ligne d'en-tête |
| Pas de lignes vides | Évitez les ruptures |
| Tableau structuré | Utilisez les tableaux Excel |
