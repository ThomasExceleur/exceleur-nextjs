# Utiliser les Filtres Excel

Les filtres permettent d'afficher uniquement les donnees pertinentes.

## Activer les filtres

### Methode rapide

1. Selectionnez vos donnees
2. **Ctrl + Maj + L** ou **Donnees** > **Filtrer**

### Indicateur

Les fleches apparaissent dans les en-tetes.

## Types de filtres

### Filtre par valeur

- Cochez/decochez les valeurs
- **Selectionner tout** pour reinitialiser

### Filtre numerique

| Option | Usage |
|--------|-------|
| Superieur a | Valeurs au-dessus d'un seuil |
| Inferieur a | Valeurs en dessous |
| Entre | Plage de valeurs |
| 10 premiers | Top N elements |

### Filtre textuel

| Option | Usage |
|--------|-------|
| Contient | Recherche partielle |
| Commence par | Debut specifique |
| Se termine par | Fin specifique |

## Filtre par couleur

Filtrez par couleur de cellule ou de police.

## Supprimer les filtres

1. Cliquez sur la fleche du filtre
2. **Effacer le filtre de...**

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| En-tetes | Toujours avoir une ligne d'en-tete |
| Pas de lignes vides | Evitez les ruptures |
| Tableau structure | Utilisez les tableaux Excel |
