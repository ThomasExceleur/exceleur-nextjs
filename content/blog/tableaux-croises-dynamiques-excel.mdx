# Tableaux Croisés Dynamiques Excel

Le TCD est l'outil d'analyse de données le plus puissant d'Excel.

## Qu'est-ce qu'un TCD ?

### Définition

Un tableau croisé dynamique résumé et analyse de grandes quantités de données.

### Capacités

- Regrouper par catégories
- Calculer des sommes, moyennes, etc.
- Filtrer et trier facilement
- Créer des rapports interactifs

## Créer un TCD

### Étapes

1. Sélectionnez vos données
2. **Insertion** > **Tableau croisé dynamique**
3. Choisissez l'emplacement
4. **OK**

### Raccourci

Alt+N+V

## Structuré du TCD

### Zones

| Zone | Usage |
|------|-------|
| Filtres | Filtrer le rapport entier |
| Lignes | Catégories en lignes |
| Colonnes | Catégories en colonnes |
| Valeurs | Données à calculer |

## Ajouter des champs

### Méthode

Glissez les champs dans les zones souhaitées.

### Exemple

- Lignes : Region
- Colonnes : Mois
- Valeurs : Somme de Ventes

## Types de calcul

### Options disponibles

| Calcul | Description |
|--------|-------------|
| Somme | Total des valeurs |
| Nombre | Compter les éléments |
| Moyenne | Moyenne des valeurs |
| Max | Valeur maximum |
| Min | Valeur minimum |
| Produit | Multiplication |
| Ecart type | Dispersion |

### Modifier le calcul

Clic droit sûr le champ > **Paramètrès des champs de valeurs**

## Filtrer les données

### Filtres de rapport

Ajoutez un champ dans la zone Filtres.

### Filtres de champ

Cliquez sûr la fleche du champ > Filtrez.

### Segments (Slicers)

1. Cliquez dans le TCD
2. **Insertion** > **Segment**
3. Sélectionnez les champs

## Grouper les données

### Dates

1. Clic droit sûr une date
2. **Grouper**
3. Choisissez : Mois, Trimestre, Année

### Nombres

Groupez par tranches (ex: 0-100, 100-200).

## Mise en forme

### Styles

**Conception** > **Styles de tableau croisé**

### Options d'affichage

- En-têtes de ligne/colonne
- Sous-totaux
- Totaux généraux
- Mise en forme conditionnelle

## Actualiser les données

### Manuel

Clic droit > **Actualiser**

### Automatique

**Options du TCD** > **Données** > **Actualiser à l'ouverture**

## Champs calcules

### Créer un champ

1. **Analyse** > **Champs, éléments et jeux**
2. **Champ calcule**
3. Définissez la formule

### Exemple

```
Marge = Ventes - Couts
```

## Afficher les valeurs comme

### Options

| Option | Description |
|--------|-------------|
| % du total général | Part du total |
| % du total de la colonne | Part par colonne |
| Différence par rapport à | Ecart vs référence |
| % de différence | Ecart en % |
| Cumul | Total cumulatif |

## Graphiques croisés dynamiques

### Créer

1. Cliquez dans le TCD
2. **Insertion** > **Graphique croisé dynamique**

### Avantage

Le graphique est lie au TCD et se met à jour.

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Données propres | Pas de lignes vides |
| Tableaux structurés | Source sous forme de tableau |
| Noms explicites | Renommez les champs |
| Documentation | Commentez les calculs |
