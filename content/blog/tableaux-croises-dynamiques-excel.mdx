# Tableaux Croises Dynamiques Excel

Le TCD est l'outil d'analyse de donnees le plus puissant d'Excel.

## Qu'est-ce qu'un TCD ?

### Definition

Un tableau croise dynamique resume et analyse de grandes quantites de donnees.

### Capacites

- Regrouper par categories
- Calculer des sommes, moyennes, etc.
- Filtrer et trier facilement
- Creer des rapports interactifs

## Creer un TCD

### Etapes

1. Selectionnez vos donnees
2. **Insertion** > **Tableau croise dynamique**
3. Choisissez l'emplacement
4. **OK**

### Raccourci

Alt+N+V

## Structure du TCD

### Zones

| Zone | Usage |
|------|-------|
| Filtres | Filtrer le rapport entier |
| Lignes | Categories en lignes |
| Colonnes | Categories en colonnes |
| Valeurs | Donnees a calculer |

## Ajouter des champs

### Methode

Glissez les champs dans les zones souhaitees.

### Exemple

- Lignes : Region
- Colonnes : Mois
- Valeurs : Somme de Ventes

## Types de calcul

### Options disponibles

| Calcul | Description |
|--------|-------------|
| Somme | Total des valeurs |
| Nombre | Compter les elements |
| Moyenne | Moyenne des valeurs |
| Max | Valeur maximum |
| Min | Valeur minimum |
| Produit | Multiplication |
| Ecart type | Dispersion |

### Modifier le calcul

Clic droit sur le champ > **Parametres des champs de valeurs**

## Filtrer les donnees

### Filtres de rapport

Ajoutez un champ dans la zone Filtres.

### Filtres de champ

Cliquez sur la fleche du champ > Filtrez.

### Segments (Slicers)

1. Cliquez dans le TCD
2. **Insertion** > **Segment**
3. Selectionnez les champs

## Grouper les donnees

### Dates

1. Clic droit sur une date
2. **Grouper**
3. Choisissez : Mois, Trimestre, Annee

### Nombres

Groupez par tranches (ex: 0-100, 100-200).

## Mise en forme

### Styles

**Conception** > **Styles de tableau croise**

### Options d'affichage

- En-tetes de ligne/colonne
- Sous-totaux
- Totaux generaux
- Mise en forme conditionnelle

## Actualiser les donnees

### Manuel

Clic droit > **Actualiser**

### Automatique

**Options du TCD** > **Donnees** > **Actualiser a l'ouverture**

## Champs calcules

### Creer un champ

1. **Analyse** > **Champs, elements et jeux**
2. **Champ calcule**
3. Definissez la formule

### Exemple

```
Marge = Ventes - Couts
```

## Afficher les valeurs comme

### Options

| Option | Description |
|--------|-------------|
| % du total general | Part du total |
| % du total de la colonne | Part par colonne |
| Difference par rapport a | Ecart vs reference |
| % de difference | Ecart en % |
| Cumul | Total cumulatif |

## Graphiques croises dynamiques

### Creer

1. Cliquez dans le TCD
2. **Insertion** > **Graphique croise dynamique**

### Avantage

Le graphique est lie au TCD et se met a jour.

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Donnees propres | Pas de lignes vides |
| Tableaux structures | Source sous forme de tableau |
| Noms explicites | Renommez les champs |
| Documentation | Commentez les calculs |
