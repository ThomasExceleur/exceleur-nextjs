# Excel 3D Maps

Visualisez vos donnees sur des cartes 3D.

## Qu'est-ce que 3D Maps ?

### Definition

Outil de visualisation geographique integre a Excel.

### Avantages

- Cartes interactives
- Visualisation temporelle
- Tours guides

## Acceder a 3D Maps

### Etapes

1. Selectionnez vos donnees geographiques
2. **Insertion** > **3D Map** > **Ouvrir 3D Maps**

## Types de donnees geographiques

| Type | Exemple |
|------|---------|
| Pays | France, Allemagne |
| Villes | Paris, Lyon |
| Codes postaux | 75001, 69000 |
| Coordonnees | Latitude, Longitude |

## Types de visualisations

### Cartes empilees

Colonnes verticales sur la carte.

### Carte thermique

Zones colorees selon l'intensite.

### Regions

Coloration par zone geographique.

### Points

Marqueurs individuels.

## Creer un tour guide

### Etapes

1. Ajoutez des scenes
2. Configurez les transitions
3. Ajoutez des annotations
4. Exportez en video

## Personnalisation

| Option | Description |
|--------|-------------|
| Hauteur | Dimension des colonnes |
| Couleur | Echelle de couleurs |
| Etiquettes | Texte sur les elements |
| Fond | Style de carte |

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Donnees propres | Nettoyez les localisations |
| Coherence | Un seul type par carte |
| Simplicite | Evitez la surcharge |
