# Excel pour Android : Guide Complet

Excel est disponible gratuitement sûr les appareils Android. Voici comment l'utiliser.

## Téléchargement

### Depuis Google Play

1. Ouvrez le **Play Store**
2. Recherchez "Microsoft Excel"
3. Appuyez sûr **Installer**
4. Connectez-vous avec un compte Microsoft

### Gratuit pour

- Smartphones (ecran < 10.1 pouces)
- Fonctionnalités de base
- Stockage OneDrive (5 Go)

## Fonctionnalités gratuites

### Édition de base

- Créer et modifier des classeurs
- Formules courantes
- Mise en forme basique
- Graphiques simples

### Collaboration

- Partage de fichiers
- Édition simultanee
- Commentaires

### Stockage

- OneDrive intégration
- Accès aux fichiers recents
- Mode hors-ligne

## Fonctionnalités Premium

### Avec Microsoft 365

- Toutes les fonctionnalités
- Pas de limite de taille d'ecran
- Tableaux croisés dynamiques complets
- Styles et themes avancés

### Prix

Abonnement Microsoft 365 requis pour les fonctions avancées.

## Interface mobile

### Navigation

- Barre d'outils en bas
- Onglets de feuilles en bas
- Zoom par pincement

### Saisie

- Clavier adapte aux formules
- Suggestions automatiques
- Saisie vocale possible

### Barre de formule

Accessible en appuyant sûr une cellule.

## Synchronisation

### OneDrive

Les fichiers se synchronisent automatiquement.

### Mode hors-ligne

1. Ouvrez le fichier
2. Menu **...** > **Disponible hors connexion**
3. Le fichier est accessible sans internet

### Conflit de versions

Excel gere automatiquement les conflits lors de la synchronisation.

## Formules supportees

### Toutes les formules courantes

- SOMME, MOYENNE, NB
- SI, ET, OU
- RECHERCHEV, INDEX, EQUIV
- Formules de date et texte

### Limites

Certaines formules avancées peuvent avoir des restrictions sûr mobile.

## Graphiques

### Types disponibles

- Histogrammes
- Courbes
- Secteurs
- Nuages de points

### Création

1. Sélectionnez les données
2. Menu **Insertion** > **Graphique**
3. Choisissez le type

## Raccourcis et gestes

### Gestes tactiles

| Geste | Action |
|-------|--------|
| Appui long | Menu contextuel |
| Pincement | Zoom |
| Glisser | Déplacer/sélectionner |
| Double-tap | Modifier cellule |

### Navigation rapide

Utilisez la zone Nom pour aller à une cellule spécifique.

## Partage de fichiers

### Méthodes

1. **Partager** > Envoyer le lien
2. **Partager** > Envoyer une copie
3. Exporter en PDF

### Autorisations

Choisissez entre lecture seule et modification.

## Comparaison avec Excel Desktop

| Fonctionnalité | Mobile | Desktop |
|----------------|--------|---------|
| Formules de base | Oui | Oui |
| Macros VBA | Non | Oui |
| Power Query | Non | Oui |
| Tableaux croisés | Limite | Complet |
| Graphiques | Basiques | Complets |

## Optimisation mobile

### Performance

- Évitez les fichiers trop lourds
- Limitez le nombre de formules
- Fermez les apps en arriere-plan

### Batterie

- Mode économie disponible
- Synchronisation à la demande

## Intégration Android

### Widgets

Ajoutez un widget Excel à votre ecran d'accueil.

### Raccourcis

Créez des raccourcis vers vos fichiers frequents.

### Partage système

Ouvrez les fichiers .xlsx depuis n'importe quelle app.

## Alternatives

### Google Sheets

- App native Android
- Gratuit complet
- Excellente collaboration

### WPS Office

- Compatible Excel
- Version gratuite disponible
- Interface similaire

## Conseils

1. **Connectez** OneDrive pour la synchronisation
2. **Utilisez** le mode hors-ligne pour les deplacements
3. **Simplifiez** vos fichiers pour le mobile
4. **Collaborez** avec les liens de partage
