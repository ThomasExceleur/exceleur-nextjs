# Excel pour Android : Guide Complet

Excel est disponible gratuitement sur les appareils Android. Voici comment l'utiliser.

## Telechargement

### Depuis Google Play

1. Ouvrez le **Play Store**
2. Recherchez "Microsoft Excel"
3. Appuyez sur **Installer**
4. Connectez-vous avec un compte Microsoft

### Gratuit pour

- Smartphones (ecran < 10.1 pouces)
- Fonctionnalites de base
- Stockage OneDrive (5 Go)

## Fonctionnalites gratuites

### Edition de base

- Creer et modifier des classeurs
- Formules courantes
- Mise en forme basique
- Graphiques simples

### Collaboration

- Partage de fichiers
- Edition simultanee
- Commentaires

### Stockage

- OneDrive integration
- Acces aux fichiers recents
- Mode hors-ligne

## Fonctionnalites Premium

### Avec Microsoft 365

- Toutes les fonctionnalites
- Pas de limite de taille d'ecran
- Tableaux croises dynamiques complets
- Styles et themes avances

### Prix

Abonnement Microsoft 365 requis pour les fonctions avancees.

## Interface mobile

### Navigation

- Barre d'outils en bas
- Onglets de feuilles en bas
- Zoom par pincement

### Saisie

- Clavier adapte aux formules
- Suggestions automatiques
- Saisie vocale possible

### Barre de formule

Accessible en appuyant sur une cellule.

## Synchronisation

### OneDrive

Les fichiers se synchronisent automatiquement.

### Mode hors-ligne

1. Ouvrez le fichier
2. Menu **...** > **Disponible hors connexion**
3. Le fichier est accessible sans internet

### Conflit de versions

Excel gere automatiquement les conflits lors de la synchronisation.

## Formules supportees

### Toutes les formules courantes

- SOMME, MOYENNE, NB
- SI, ET, OU
- RECHERCHEV, INDEX, EQUIV
- Formules de date et texte

### Limites

Certaines formules avancees peuvent avoir des restrictions sur mobile.

## Graphiques

### Types disponibles

- Histogrammes
- Courbes
- Secteurs
- Nuages de points

### Creation

1. Selectionnez les donnees
2. Menu **Insertion** > **Graphique**
3. Choisissez le type

## Raccourcis et gestes

### Gestes tactiles

| Geste | Action |
|-------|--------|
| Appui long | Menu contextuel |
| Pincement | Zoom |
| Glisser | Deplacer/selectionner |
| Double-tap | Modifier cellule |

### Navigation rapide

Utilisez la zone Nom pour aller a une cellule specifique.

## Partage de fichiers

### Methodes

1. **Partager** > Envoyer le lien
2. **Partager** > Envoyer une copie
3. Exporter en PDF

### Autorisations

Choisissez entre lecture seule et modification.

## Comparaison avec Excel Desktop

| Fonctionnalite | Mobile | Desktop |
|----------------|--------|---------|
| Formules de base | Oui | Oui |
| Macros VBA | Non | Oui |
| Power Query | Non | Oui |
| Tableaux croises | Limite | Complet |
| Graphiques | Basiques | Complets |

## Optimisation mobile

### Performance

- Evitez les fichiers trop lourds
- Limitez le nombre de formules
- Fermez les apps en arriere-plan

### Batterie

- Mode economie disponible
- Synchronisation a la demande

## Integration Android

### Widgets

Ajoutez un widget Excel a votre ecran d'accueil.

### Raccourcis

Creez des raccourcis vers vos fichiers frequents.

### Partage systeme

Ouvrez les fichiers .xlsx depuis n'importe quelle app.

## Alternatives

### Google Sheets

- App native Android
- Gratuit complet
- Excellente collaboration

### WPS Office

- Compatible Excel
- Version gratuite disponible
- Interface similaire

## Conseils

1. **Connectez** OneDrive pour la synchronisation
2. **Utilisez** le mode hors-ligne pour les deplacements
3. **Simplifiez** vos fichiers pour le mobile
4. **Collaborez** avec les liens de partage
