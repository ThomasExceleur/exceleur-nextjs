# Convertir Excel en PDF

Exportez vos classeurs au format PDF.

## Methode Enregistrer sous

### Etapes

1. **Fichier** > **Enregistrer sous**
2. Choisissez l'emplacement
3. **Type** : PDF
4. **Enregistrer**

## Methode Exporter

### Etapes

1. **Fichier** > **Exporter**
2. **Creer un document PDF/XPS**
3. **Publier**

## Options d'export

### Ce qui est exporte

| Option | Description |
|--------|-------------|
| Classeur entier | Toutes les feuilles |
| Selection | Zone selectionnee |
| Feuilles actives | Feuille courante |

### Qualite

| Option | Usage |
|--------|-------|
| Standard | Publication en ligne |
| Minimum | Fichier plus leger |

## Mise en page avant export

### Verifier l'apercu

**Fichier** > **Imprimer** (Apercu)

### Ajuster

- Marges
- Orientation
- Echelle
- Zone d'impression

## Plusieurs feuilles

### Selectionner plusieurs onglets

Ctrl+Clic sur les onglets souhaites.

## PDF protege

### Dans les options

Ajoutez un mot de passe si necessaire.

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Apercu | Verifiez avant export |
| Zone | Definissez la zone d'impression |
| Qualite | Adaptez au besoin |
