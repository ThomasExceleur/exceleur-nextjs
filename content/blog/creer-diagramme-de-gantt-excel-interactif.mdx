# Diagramme de Gantt Interactif

Ajoutez de l'interactivite à votre diagramme de Gantt Excel.

## Éléments interactifs

### Ce que nous allons créer

- Filtres par responsable
- Sélection de période
- Mise en evidence automatique
- Indicateurs de progression

## Préparer les données

### Structuré du tableau

| Tache | Responsable | Début | Fin | Progression | Statut |
|-------|-------------|-------|-----|-------------|--------|
| Tache 1 | Alice | 01/01 | 10/01 | 100% | Termine |
| Tache 2 | Bob | 05/01 | 20/01 | 60% | En cours |
| Tache 3 | Alice | 15/01 | 25/01 | 0% | A faire |

### Convertir en tableau

**Ctrl+T** pour beneficier des fonctionnalités de tableau.

## Ajouter des segments

### Étapes

1. Cliquez dans le tableau
2. **Insertion** > **Segment**
3. Sélectionnez : Responsable, Statut

### Positionnement

Placez les segments à côté du graphique pour un accès facile.

## Créer le Gantt dynamique

### Base du graphique

1. Sélectionnez Taches, Début, Duree
2. **Insertion** > **Barres empilees**
3. Masquez la serie de début

### Liaison avec le tableau

Le graphique se met à jour automatiquement quand les segments filtrent.

## Barre de progression

### Ajouter la progression visuelle

1. Créez une colonne "Progression_affichee"
2. Formule : `=[@Duree]*[@Progression]`
3. Ajoutez cette serie au graphique
4. Superposez sûr la barre principale

### Style

- Barre totale : gris clair
- Progression : couleur vive

## Indicateurs visuels

### Mise en forme conditionnelle

Pour le tableau source :

| Condition | Format |
|-----------|--------|
| Statut = "Termine" | Fond vert |
| Statut = "En retard" | Fond rouge |
| Progression > 75% | Icone check |

### Barre de données

Ajoutez une barre de données sûr la colonne Progression.

## Ligne de date actuelle

### Créer la ligne

1. Ajoutez une serie avec une seule valeur
2. Valeur = `AUJOURDHUI() - Date_debut_projet`
3. Formatez en ligne verticale

### Style recommandé

- Ligne rouge pointillee
- Epaisseur : 2 pt
- Etiquette : "Aujourd'hui"

## Sélection de période

### Liste deroulante pour la vue

1. Créez une liste : "Tout", "Ce mois", "Cette semaine"
2. Validation de données sûr une cellule
3. Filtrez les données selon la sélection

### Formule de filtre

```excel
=FILTRE(Données;
  SI(Sélection="Tout";1;
    SI(Sélection="Ce mois";MOIS(Données[Début])=MOIS(AUJOURDHUI());
      Données[Début]>=AUJOURDHUI()-7
    )
  )
)
```

## Tableau de bord associe

### KPI à afficher

| KPI | Formule |
|-----|---------|
| Taches totales | `=LIGNES(Tableau)` |
| Terminees | `=NB.SI(Statut;"Termine")` |
| En retard | `=NB.SI(Statut;"En retard")` |
| Avancement global | `=MOYENNE(Progression)` |

### Indicateurs visuels

Utilisez des jauge où des icones pour les KPI.

## Navigation clavier

### Activer les segments

Cliquez sûr un segment puis utilisez les fleches pour naviguer.

### Raccourcis utiles

| Raccourci | Action |
|-----------|--------|
| Alt+F5 | Actualiser |
| Ctrl+Shift+L | Filtres |

## Responsivite

### Ajuster pour l'ecran

1. Utilisez des cellules fusionnées pour le titre
2. Dimensionnez le graphique proportionnellement
3. Testez sûr différentes resolutions

## Export interactif

### PDF avec segments

Le PDF capture l'etat filtre actuel.

### Partage Excel

Les segments fonctionnent à l'ouverture du fichier.
