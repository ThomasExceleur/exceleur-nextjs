# Remplacer une Couleur par une Autre dans Excel

Plusieurs methodes permettent de changer la couleur des cellules en masse.

## Methode 1 : Rechercher et Remplacer

### Etapes

1. **Ctrl + H** (Rechercher et remplacer)
2. Cliquez sur **Options**
3. Cliquez sur **Format** (a cote de "Rechercher")
4. Onglet **Remplissage** > Selectionnez la couleur a trouver
5. Cliquez sur **Format** (a cote de "Remplacer par")
6. Selectionnez la nouvelle couleur
7. **Remplacer tout**

### Resultat

Toutes les cellules avec la couleur recherchee sont modifiees.

## Methode 2 : Selectionner par couleur

### Etapes

1. **Accueil** > **Rechercher et selectionner** > **Atteindre**
2. **Cellules...** > **Formats**
3. Selectionnez une cellule avec la couleur
4. Toutes les cellules de cette couleur sont selectionnees
5. Appliquez la nouvelle couleur

## Methode 3 : VBA

### Code simple

```vba
Sub RemplacerCouleur()
    Dim cell As Range
    Dim ancienneCouleur As Long
    Dim nouvelleCouleur As Long

    ancienneCouleur = RGB(255, 255, 0)  ' Jaune
    nouvelleCouleur = RGB(0, 176, 80)   ' Vert

    For Each cell In ActiveSheet.UsedRange
        If cell.Interior.Color = ancienneCouleur Then
            cell.Interior.Color = nouvelleCouleur
        End If
    Next cell
End Sub
```

### Utilisation

1. Modifiez les valeurs RGB
2. Executez la macro

### Trouver la valeur RGB

```vba
Sub TrouverCouleur()
    MsgBox "Couleur RGB: " & ActiveCell.Interior.Color
End Sub
```

## VBA avec selection de couleur

### Code interactif

```vba
Sub RemplacerCouleurSelection()
    Dim ancienne As Long
    Dim nouvelle As Long
    Dim cell As Range

    ' Demander a l'utilisateur de selectionner la cellule source
    ancienne = Application.InputBox("Cliquez sur une cellule avec la couleur a remplacer", Type:=8).Interior.Color

    ' Demander la cellule avec la nouvelle couleur
    nouvelle = Application.InputBox("Cliquez sur une cellule avec la nouvelle couleur", Type:=8).Interior.Color

    ' Remplacer
    For Each cell In ActiveSheet.UsedRange
        If cell.Interior.Color = ancienne Then
            cell.Interior.Color = nouvelle
        End If
    Next cell
End Sub
```

## Couleur du texte

### Remplacer la couleur de police

```vba
Sub RemplacerCouleurTexte()
    Dim cell As Range
    Dim ancienne As Long
    Dim nouvelle As Long

    ancienne = RGB(255, 0, 0)  ' Rouge
    nouvelle = RGB(0, 0, 255)  ' Bleu

    For Each cell In ActiveSheet.UsedRange
        If cell.Font.Color = ancienne Then
            cell.Font.Color = nouvelle
        End If
    Next cell
End Sub
```

## Plusieurs feuilles

### Code pour tout le classeur

```vba
Sub RemplacerToutClasseur()
    Dim ws As Worksheet
    Dim cell As Range
    Dim ancienne As Long
    Dim nouvelle As Long

    ancienne = RGB(255, 255, 0)
    nouvelle = RGB(0, 176, 80)

    For Each ws In ThisWorkbook.Worksheets
        For Each cell In ws.UsedRange
            If cell.Interior.Color = ancienne Then
                cell.Interior.Color = nouvelle
            End If
        Next cell
    Next ws
End Sub
```

## Mise en forme conditionnelle

### Note importante

La mise en forme conditionnelle n'est pas affectee par ces methodes.

### Modifier la MFC

1. **Accueil** > **Mise en forme conditionnelle**
2. **Gerer les regles**
3. Modifiez les regles individuellement

## Conseils

### Avant de remplacer

1. Sauvegardez votre fichier
2. Testez sur une copie
3. Verifiez le resultat

### Couleurs proches

Les couleurs peuvent paraitre identiques mais avoir des codes differents. Utilisez la formule pour verifier le code exact.

## Sans VBA : Methode manuelle

### Si peu de cellules

1. Selectionnez une cellule coloree
2. **Rechercher et selectionner** > **Selectionner les cellules de meme format**
3. Appliquez la nouvelle couleur
