# Feuille de Prevision Lineaire Excel

Créez des previsions automatiques avec Excel.

## Accès à la fonctionnalité

### Étapes

1. Sélectionnez vos données (dates et valeurs)
2. **Données** > **Feuille de prevision**
3. Configurez les paramètrès

## Configuration

### Options disponibles

| Option | Description |
|--------|-------------|
| Fin de prevision | Date limite des projections |
| Début de prevision | Point de depart |
| Intervalle de confiance | Marge d'erreur |
| Saisonnalite | Detection automatique où manuelle |

## Paramètrès avancés

### Saisonnalite

- **Automatique** : Excel detecte le pattern
- **Manuelle** : Définissez la période

### Intervalle de confiance

95% par defaut - ajustez selon vos besoins.

## Interpretion des résultats

### Colonnes générées

| Colonne | Signification |
|---------|---------------|
| Prevision | Valeur projetee |
| Limite inf. | Borne basse |
| Limite sup. | Borne haute |

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Données historiques | Minimum 2 cycles complets |
| Regularite | Intervalles constants |
| Vérification | Comparez avec le reel |
