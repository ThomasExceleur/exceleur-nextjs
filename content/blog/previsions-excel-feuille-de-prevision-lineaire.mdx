# Feuille de Prevision Lineaire Excel

Creez des previsions automatiques avec Excel.

## Acces a la fonctionnalite

### Etapes

1. Selectionnez vos donnees (dates et valeurs)
2. **Donnees** > **Feuille de prevision**
3. Configurez les parametres

## Configuration

### Options disponibles

| Option | Description |
|--------|-------------|
| Fin de prevision | Date limite des projections |
| Debut de prevision | Point de depart |
| Intervalle de confiance | Marge d'erreur |
| Saisonnalite | Detection automatique ou manuelle |

## Parametres avances

### Saisonnalite

- **Automatique** : Excel detecte le pattern
- **Manuelle** : Definissez la periode

### Intervalle de confiance

95% par defaut - ajustez selon vos besoins.

## Interpretion des resultats

### Colonnes generees

| Colonne | Signification |
|---------|---------------|
| Prevision | Valeur projetee |
| Limite inf. | Borne basse |
| Limite sup. | Borne haute |

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Donnees historiques | Minimum 2 cycles complets |
| Regularite | Intervalles constants |
| Verification | Comparez avec le reel |
