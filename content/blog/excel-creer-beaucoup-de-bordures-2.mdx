# Creer des Bordures Rapidement

Mettez en forme vos tableaux efficacement.

## Raccourci rapide

### Toutes les bordures

1. Selectionnez la plage
2. **Ctrl+Maj+7** (ou **Ctrl+Maj+&**)

## Via le ruban

### Acces

**Accueil** > **Bordures** (icone)

### Options

| Option | Description |
|--------|-------------|
| Toutes | Interieur et exterieur |
| Contour | Exterieur seulement |
| Interieur | Entre les cellules |

## Format de cellule

### Etapes

1. Selectionnez
2. **Ctrl+1**
3. Onglet **Bordure**

### Personnalisation

- Style de trait
- Couleur
- Position

## Dessiner des bordures

### Mode dessin

1. **Accueil** > **Bordures** > **Dessiner une bordure**
2. Cliquez et glissez

## Supprimer les bordures

### Methode

**Accueil** > **Bordures** > **Aucune bordure**

## Styles de bordures

| Style | Usage |
|-------|-------|
| Fin | Standard |
| Epais | Mise en evidence |
| Double | En-tetes |
| Pointille | Leger |

## Mise en forme de tableau

### Automatique

1. **Ctrl+T** pour creer un tableau
2. Les bordures s'appliquent automatiquement

| Pratique | Description |
|----------|-------------|
| Coherence | Meme style partout |
| Sobriete | Pas de surcharge |
| Tableaux | Utilisez les styles |
