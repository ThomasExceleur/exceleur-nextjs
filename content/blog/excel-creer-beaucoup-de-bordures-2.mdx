# Créer des Bordures Rapidement

Mettez en forme vos tableaux efficacement.

## Raccourci rapide

### Toutes les bordures

1. Sélectionnez la plage
2. **Ctrl+Maj+7** (où **Ctrl+Maj+&**)

## Via le ruban

### Accès

**Accueil** > **Bordures** (icone)

### Options

| Option | Description |
|--------|-------------|
| Toutes | Interieur et exterieur |
| Contour | Exterieur seulement |
| Interieur | Entre les cellules |

## Format de cellule

### Étapes

1. Sélectionnez
2. **Ctrl+1**
3. Onglet **Bordure**

### Personnalisation

- Style de trait
- Couleur
- Position

## Dessiner des bordures

### Mode dessin

1. **Accueil** > **Bordures** > **Dessiner une bordure**
2. Cliquez et glissez

## Supprimer les bordures

### Méthode

**Accueil** > **Bordures** > **Aucune bordure**

## Styles de bordures

| Style | Usage |
|-------|-------|
| Fin | Standard |
| Epais | Mise en evidence |
| Double | En-têtes |
| Pointille | Léger |

## Mise en forme de tableau

### Automatique

1. **Ctrl+T** pour créer un tableau
2. Les bordures s'appliquent automatiquement

## Bonnes pratiques

| Pratique | Description |
|----------|-------------|
| Cohérence | Meme style partout |
| Sobriete | Pas de surcharge |
| Tableaux | Utilisez les styles |
